1. Bagaimana cara membenarkan touchpad pada laptop?


Pada saat tertentu, kadang touchpad mengalami masalah seperti macet, tidak berfungsi sampai bergerak sendiri. Hal ini disebabkan oleh sebab sebagai berikut.
Laptop terkena serangan virusTouchpad kotor karena terkena debu / nodaTouchpad terkena minyak keringatDriver touchpad hilang atau belum terinstalKesalahan settingan pada program touchpadHardware touchpad telah rusak
Jika hardware anda belum dipastikan mengalami kerusakan pada hardware, tentunya anda tidak perlu khawatir untuk mengembalikannya menjadi normal seperti sediakala. Namun bila hardware touchpad sudah terkena imbasnya, maka satu - satunya cara yaitu membeli mouse, touchpad internal ataupun touchpad external. Untuk mengatasi masalah touchpad selain kerusakan pada hardware, yaitu :
1. Scanning Dengan Software Antivirus
Virus / malware memang menimbulkan dampak signifikan pada hardware, baik internal maupun eksternal. Biasanya virus mengacaukan program sistem, sehingga berbagai hardware tidak dapat berjalan dengan maksimal. Contohnya yang terjadi pada kasus touchpad yakni kursor tidak dapat bergerak atau kursor bergerak sendiri.
Sebagai solusinya ialah mengunduh antivirus untuk menghilangkan virus tersebut serta mencegah datangnya malware lain dikemudian hari. Penulis menyarankan antivirus yang perlu anda gunakan yakni  AVAST, Esset, Kapersky, Bit defender atau memakai Microsoft Security Essential sebagai antivirus utama. Dan yang kedua yaitu antivirus Smadav, meskipun tidak pantas disebut antivirus utama, namun efektif sebagai antivirus pendamping serta program untuk memperbaiki registry.
2. Bersihkan Registry Sampah Dengan Software Tune Up
Sistem pada komputer atau laptop perlulah dijaga agar lancar untuk digunakan ketika beraktifitas di laptop maupun komputer. Sistem yang tidak baik akan menimbulkan masalah lain, baik dari segi software maupun hardware, salah satunya touchpad.
Untuk menjaga sistem agar berjalan lancar dan ringan, maka anda memerlukan sebuah software agar anda tidak kesusahan dalam mengaturnya. Software yang saya maksud disini software tuneup, contohnya CCleaner, AVG tuneup Utilities maupun Norton Utilies, Software ini berfungsi untuk membuang file dan registry sampah serta memperbaiki registry dan sistem yang rusak.

3. Mouse dom touchpad pada laptoptidak berfungsi bersamaan, (laptop Lenovo)langkah yangharus dilakukan ?​


Jawaban:

Aktifkan touchpad yang disable

Pada beberapa laptop biasanya sudah dilengkapi tombol khusus di keyboard untuk mengaktifkan dan menonaktifkan touchpad. Jadi bisa saja tanpa sengaja kamu menekan tombol tersebut sehingga membuat touchpad tidak berfungsi. Untuk mengatasinya, carilah tombol di keyboard yang memiliki ikon bergambar touchpad. Biasanya tombol tersebut berada di antara tombol F1 – F12. Jika berhasil ketemu, kamu hanya perlu melakukan tekan kombinasi Fn + tombol tersebut untuk mengaktifkan kursor yang freeze, misalnya Fn + F6 atau Fn + F9.

Cara lainnya, kamu bisa mengaktifkan touchpad dari Mouse Setting menggunakan mouse eksternal. Untuk Windows 7 atau 8, dari menu Start, lalu ketik Mouse. Setelah jendela Mouse Setting terbuka, lalu pilih opsi Enable Touchpad, klik Apply, dan OK. Sementara pada Windows 10, klik tombol Windows, lalu masuk ke Setting dan klik Device. Setelah itu, pilih Mouse & Touchpad untuk masuk ke dalam jendela pengaturan touchpad. Lihat opsi di bagian bawah dan klik Additional Mouse Options untuk mengaktifkan touchpad.

9. gimana caranya balikin touchpad yang kursorny ga muncul di layar laptop​


Berikut ini adalah cara-cara yang dapat kita lakukan pada saat mendapatkan masalah seperti itu :

Memanfaatkan Tombol Fungsi

Cara mengatasi kursor hilang yang pertama ini sebenarnya sangat simple, yakni hanya dengan memanfaatkan tombol fungsi saja.

Adapun tombol fungsi yang bisa dimanfaatkan dalam proses ini sebenarnya berbeda-beda. Beda produsen, maka berbeda pula tombol fungsi yang digunakan. Intinya sih, anda bisa memadukan beberapa tombol FN + (F1 sampai F12) sebagai jalan untuk menemukan rumus terbaik.

Mengecek Pengaturan Kursor / Pointer

Untuk mengecek settingan pointer sebenarnya sangat mudah. Namun dibutuhkan perangkat tambahan untuk menggerakkan pointer anda yakni Mouse. Jadi untuk anda yang masih bingung bagaimana settingnya, bisa mengikuti tutorial ini :

Cari tombol Control Panel

Pada halaman tersebut, silakan pilih Hardware and Sound

Pada menu Device and Printer silakan pilih Mouse

Selanjutnya akan muncul pop up baru, dan silakan pilih tab Pointer Option

Lalu hapus centang pada kotak Hide Pointer While Typing

Install Ulang atau Update Driver Kursor / Pointer

Jika anda sudah merasa benar dalam mengikuti step by step yang saya berikan namun masih gagal. Maka langkah terakhir dalam mengatasi kursor yang tidak muncul adalah dengan Install Ulang atau Update Driver kursor anda.

17. Langka langka mengkopi file yang ada di laptop dari mnghidupkan laptop memasang flasdisk dan mematikan laptop secara benar dalam bahasa inggris


1. Turn the computer by pressing the power button.
2. Wait until the operating system and the computer is on.
3. Once the computer is on, plug the USB flash drive in the port.
4. Open the Windows Explorer. Wait until your name appears in the row stick Computer Drive.
5. Then open the flash and select the files to be copied. Right-click on the selected files, then select copy (or can use traditional Ctrl + C).
6. Then select the desired location to paste the file. Next, right click and select paste (or could with Ctrl + V) at that location.
7. After copying the file, right click on the flash icon and select eject.
8. Disconnect the USB stick from port.
9. Clik Start, and select the shut down button. Wait until the computer turns off.
