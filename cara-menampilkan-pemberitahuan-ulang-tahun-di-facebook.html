10. Perulangan java untuk menampilkan angka :


Code utk menampilkan gambar pertama:
int a=1;
for (int i=0; i<5; i++)
{
   for (int j=0; j<i; j++)
   {
     System.out.println(a);
     a+=1;
   }
   System.out.println("\n");
}

code untuk menampilkan gambar kedua :
int a;
for (int i=5; i>-1; i--)
{
   a=i;
   for (int j=0; j<i; j++)
   {
      System.out.println(a);
      a+=1;
   }
   System.out.println("\n");
}

21. sandi merayakan hari ulang tahunnya yang ke 25 pada hari jumat.Jika selama hidup sandi melewati 6 kali tahun kabisat , maka sandi lahir pada hari apa ????jelaskan yang tahu beritahu jawabannya


tahun kabisat iti muncul 4 th sekali
25 - 4 th = 21 ( kabisat 1)
21 - 4 th = 17 (kabisat 2)
17 - 4 = 13 ( kabisat ke 3)
13 - 4 = 9 (kabisat ke 4)
9 - 4 = 5 (kabisat ke 5)
5 - 5 = 1 (kabisat ke 6)

setiiap 7 hari aian berulang hari yg sama.
dr usia 25 th sampai 1 thn
= 24 x 365 : 7
= 1251 kali berulang hari jumat, sisa 3 hari
jadi sand merayakan ula g tahunnya yg pertama pad hari rabu,

setiap 7 hati aian berula g hari yg sama
365 : 7 = 52 kali hari rabu sisa 1 hari
maka, sandi lahir pada hari SELASA
