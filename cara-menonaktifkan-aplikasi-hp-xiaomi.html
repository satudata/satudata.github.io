12. Cara menonaktifkan auto correct code di aplikasi/program visual studio code gimana ya?​


Jawaban:

Buka Visual Studio Code.Buka menu "File" (berada di bagian atas layar).Pilih opsi "Preferences" (Preferensi), kemudian pilih "Settings" (Pengaturan). Anda juga dapat menggunakan pintasan keyboard "Ctrl + ," untuk membuka pengaturan.Halaman Pengaturan akan terbuka dalam tampilan JSON.Di bagian atas halaman Pengaturan, ada sebuah ikon pengaturan (ikon berbentuk ikat pinggang). Klik ikon tersebut untuk membuka "Settings UI" (Antarmuka Pengaturan).Di kotak pencarian di bagian atas halaman Pengaturan UI, ketik "Auto Correct".Dan akan melihat opsi "Editor: Auto Closing Brackets" dan "Editor: Auto Closing Quotes" yang berhubungan dengan koreksi otomatis.Untuk menonaktifkan koreksi otomatis, cukup nonaktifkan opsi tersebut dengan mengklik tanda centang di sebelahnya atau dengan mengubah nilainya menjadi "off" atau "false".
