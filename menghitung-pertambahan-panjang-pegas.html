4. Sebuah benda bermassa 50kg digantungkan pada ujung pegas sehingga pegas bertambah panjang 10m, hitunglah konstanta pegas​


Penjelasan:

Jika gaya kita berikan pada ujung pegas dan pegas mengalami perubahan panjang ( bertambah atau berkurang) maka berlaku persamaan/ Hukum Hooke :

F = k x

dimana :

F = Gaya yg diberikan pada ujung pegas (N)

k = konstanta pegas (N/m)

x = perubahan panjang pegas (m)

Data soal :

m = 50 kg

x = 10 m

F = W = m x g

= 50 x 10

= 500 N

F = k x

k = F/×

= 500/10

= 5 N/m

7. Sebuah pegas bertambah panjang 5 cm jika di beri gaya 10 N hitung pertambahan panjang pegas jika besar gayanya yang di berikan 16 N


Δx / Δx' = (F / k) / (F' / k)
Δx' = Δx F' / F
      = (5 cm)(16 N) / (10 N) = 8 cm

25. Suatu pegas akan bertambah panjang 10 cm jika diberi gaya 30 N. Hitunglah pertambahan panjang pegas jika diberi gaya 21 N


Jawaban :

7 cm

Penjelasan :

Diketahui :

[tex] x _{1} = 10cm[/tex][tex]f _{1} = 30N[/tex][tex]f _{2} = 21N[/tex]

Ditanya :

[tex]x _{2}...[/tex]

Pembahasan :

Dengan menggunakan persamaan Hukum Hooke :

[tex]f = k x [/tex]

Konstanta dari pegas adalah :

[tex]k = \frac{f}{x} [/tex]

Karena pada kondisi pertama dan kedua menggunakan pegas yang sama, maka pertambahan panjang pegas di kondisi kedua dapat ditentukan dengan cara berikut :

[tex]k = k[/tex]

[tex] \frac{f _{1} }{x _{1}} = \frac{f _{2} }{x _{2} } [/tex]

[tex]x _{2} = \frac{f _{2} \times x _{1}}{f_{1}} [/tex]

[tex]x_{2} = \frac{21 \times 0,1}{30} [/tex]

[tex]x_{2} = 0,07m[/tex]

[tex]x_{2} = 7cm[/tex]

Kesimpulan :

Jadi, pertambahan panjang pegas jika diberi gaya 21 N adalah 7 cm.

26. Sebuah pegas bertambah panjang sebesar 7 cm ketika di beri beban bermasa 350g hitunglah pertambahan panjang pegas tersebut bila di beri beban bermasa 500g


Yang diketahui :
Saat
Massa beban (m1) = 350 gram
Pertambahan panjang (x1) = 7 cm

Yang ditanya pertambahan panjang saat m2 = 500 gram.


Pembahasan.

Hukum hooke pada pegas.

F = k x

k = F /x

Dimana F = m g, g = 10 m/s^2


F1/x1 = F2 / x2

x2 = F2 x1 / F1
x2 = m2 g x1 / m1 g
x2 = m2 x1 / m1
x2 = 500 gram 7 cm / 350 gram
x2 = 10 cm


Kesimpulan.
Jadi pertambahan panjangnya 10 cm.
