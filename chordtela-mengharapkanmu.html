8. Onee-sama dikatakan ketika kita berhadapan dengan??HARAP BANTUANYA​


Onee-sama dikatakan ketika berhadapan dengan seorang kakak (pr) yang jauh lebih tua atau terhormat.

Pembahasan

お姉さま (onee-sama)

Untuk penggunaannya adalah kata tersebut sejenis panggilan kepada sang kakak dengan gender perempuan yang dimana kakak tersebut merupakan kakak yang paling tua (tertua), dan tentunya kehormatannya lebih tinggi.

Oleh karena itu, kata お姉さま (onee-sama) adalah bentuk panggilan yang sangat formal.

_________________________________________________

Selain itu, ada juga bentuk panggilan untuk kakak perempuan selain お姉さま (onee-sama) adalah sebagai berikut ini :

お姉さん (onee-san)

Bentuk kata ini juga termasuk penggunaan secara formal, namun tingkatan formal antara onee-sama dan onee-san itu jauh berbeda.

Dimana tingkat formalnya lebih tinggi お姉さま (onee-sama) dibandingkan dengan お姉さん (onee-san).

お姉ちゃん (onee-chan)

Berbeda dengan kata onee-sama dan onee-san, walau memiliki arti yang sama namun salah satu kata ini yakni お姉ちゃん (onee-chan) lebih spesifik atau informal.

Penggunaannya terkesan kasual atau informal, dikarenakan kata tersebut dapat ditujukan kepada kakak yang memiliki hubungan sangat akrab.

Jawaban :

お姉様 (Onee-sama) dikatakan ketika kita berhadapan dengan kakak perempuan yang sangat dihormati, dan juga ketika menyebut kakak perempuan orang lain yang berkedudukan tinggi sebagai orang ketiga dalam suatu percakapan formal.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :Arti dari お姉様 (Onee-sama)?

Kata ini terdiri dari Kanji 姉 yang berarti "Kakak perempuan", dan 様 yang merupakan gelar kehormatan untuk orang yang sangat terhormat atau berkedudukan tinggi. Huruf お di awal kata ini bertujuan untuk memberi semacam penghormatan kepada sesuatu. Jadi, dapat disimpulkan bahwa arti dari Onee-sama ini adalah kakak perempuan. Kemudian panggilan ini hanya digunakan kepada kakak perempuan yang sangat dihormati atau berkedudukan tinggi.

- - -

Panggilan-panggilan lain untuk "Kakak perempuan" dalam Bahasa Jepang?

Ada banyak cara untuk memanggil kakak perempuan dalam Bahasa Jepang. Salah satunya yaitu yang sudah saya jelaskan di atas. Untuk yang lainnya, kalian bisa menggunakan panggilan yang sama dengan yang di atas, tapi diganti 'Sama' nya dengan 'San' atau 'Chan'. Selain itu juga masih ada banyak lagi panggilan untuk kakak perempuan dalam Bahasa Jepang.

お姉さん (Onee-san) : Merupakan panggilan untuk kakak perempuan yang dihormati, tapi tidak setinggi dengan yang menggunakan akhiran 'Sama'. Selain itu juga merupakan panggilan untuk kakak perempuan orang lain sebagai orang ketiga dalam suatu percakapan formal.お姉ちゃん (Onee-chan) : Merupakan panggilan untuk kakak perempuan yang sudah sangat akrab.姉 (Ane) : Merupakan penyebutan untuk kakak perempuan sendiri sebagai orang ketiga dalam suatu percakapan formal.

- - -

Maksud dari Sama, San, Chan, dll?

Ini merupakan 敬称 (Keishou) atau yang dalam Bahasa Indonesia disebut dengan "Gelar kehormatan". Contohnya dalam Bahasa Indonesia itu seperti Bapak, Ibu, Kakak, Mas, Mba, Saudara, Tuan, Nyonya, Nona, dll. Dalam Bahasa Jepang juga ada yang seperti ini, tapi disebutkan setelah nama, baik nama keluarga maupun nama pertama nya.

Jumlah gelar kehormatan dalam Bahasa Jepang itu sangat banyak. Beberapanya yang cukup sering digunakan antara lain sebagai berikut.

さん (San) : Untuk panggilan yang sopan dan menunjukkan rasa hormat. Panggilan ini yang paling umum digunakan di Jepang.様 (Sama) : Untuk panggilan yang sangat sopan dan menunjukkan rasa hormat yang sangat tinggi. Panggilan ini bahkan juga digunakan untuk menyebut tuhan dalam Bahasa Jepang [disebut 神様 (Kami-sama)].ちゃん (Chan) : Untuk panggilan yang sangat akrab, dan terkesan imut. Biasa digunakan untuk memanggil teman perempuan yang akrab, perempuan yang lebih muda, dan juga untuk ke pacar perempuan. Panggilan ini sebenarnya juga bisa digunakan ke laki-laki, tapi jarang.君 (Kun) : Untuk panggilan yang akrab. Panggilan ini bisa digunakan oleh perempuan maupun laki-laki untuk memanggil temannya atau orang yang akrab. Biasa hanya digunakan ke laki-laki, tapi juga bisa ke perempuan.

Gelar kehormatan dalam Bahasa Jepang itu sangat penting, dan hampir selalu digunakan saat menyebut nama seseorang. Kalian bisa tidak menggunakan gelar kehormatan hanya saat memanggil orang yang sudah sangat akrab, seperti teman. Memanggil orang tanpa gelar kehormatan ini disebut dengan 呼び捨て (Yobisute).
