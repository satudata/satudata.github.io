5. sebuah benda tingginya 20cm diletakkan didepan cermin cekung seperti gambar.tinggi bayangan yang terbentuk 10cm berapa perbesaran bayangan jika digeser 10cm mendekati cermin??tolong dijawab yg benar gue isiin pulsa 25rb​


Diketahui :

h = 20 cm

h' = 10 cm

f = 16 cm

Ditanya :

M?

Jawab :

•Rumus yang dipakai :

✓ 1/f = 1/s + 1/s'

f = Fokus

s = Jarak Benda

s' = Jarak Bayangan

✓ M = |s'/s|

• Tahap I

-Mencari s awal(Jarak Benda Awal)-

1/f = 1/s + 1/s'

1/16 = 1/s + 1/0,5s

1/16 = 1/s + 2/s

1/16 = 3/s

s = 16 . 3

s = 48 cm

• Tahap II

-Mencari s akhir(Jarak Benda Akhir)-

s = 48 cm - 10 cm

s = 40 cm

• Tahap III

-Mencari s'(Jarak Bayangan)-

1/f = 1/s + 1/s'

1/16 = 1/40 + 1/s'

1/s' = 1/16 - 1/40

1/s' = 10 - 4/160

1/s' = 6/160

s'/1 = 160/6

s' = 26,6 cm

(Hasil coret-coretan saya 26,6 sedangkan hasil kalkulasi 26,666666667, saya akan pakai punya saya sendiri)

• Tahap IV

-Mencari M(Perbesaran)-

M = |s'/s|

M = 26,6 / 40

M = 0,665 kali

•Jadi perbesarannya ialah 0,665 kali.

Biaya seluruhnya adalah Rp1.100.000

2464m                  = [tex] \pi r^{2} [/tex]
2464m                  = 22/7 r^{2} [/tex]
2464m/ (22/7)       =  r^{2} [/tex]
784                       =  r^{2} [/tex]
28                         =  r

Keliling Lingkaran = [tex]2 \pi r[/tex]
                             = 2 x 22/7 x 28
                             = 2 x 22/7 x 28
                             = 176m

Banyak pohon cemara yang akan ditanam = 176m : 4m = 44 pohon cemara

Biaya seluruhnya = Banyak pohon x Harga/pohon
                             = 44 pohon x Rp25.000
                             = Rp1.100.000
