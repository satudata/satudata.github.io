Mengapa laptop asus yg saya gunakan tiba tiba sering mati sendiri

Permasalahan:
Kasus Laptop / komputer yang tiba - tiba mati atau hang (freeze), umumnya disebabkan karena hardware yang overheat. Hardware tersebut dapat menjadi overheat karena sistem terlalu banyak melakukan proses untuk melayani penggunanya.

Solusi:
Solusi yang tepat adalah mengganti (upgrade) dengan hardware yang baru. Atau, anda dapat mengecek dan membersihkan hardware laptop anda (cek kondisi RAM, Processor, dan Kipas). Jika masalah masih terjadi, anda dapat konsultasi melalui customer service vendor laptop anda atau dengan mengunjungi service center.

11. ada yang tahu cara perbaiki laptop, yang botloop, nanti mau masuk ke destop dia tiba tiba muncul kaya gini, setelah muncul bentar dia balik ke awal kita hidupin laptop


1. Cara Mengatasi Komputer / Laptop Restart Tidak Bisa Masuk Windows Dengan Safe Mode

Untuk memperbaiki komputer / laptop sering restart anda dapat mencoba melakukan booting melalui Safe Mode. Untuk menampilkan Safe Mode bagi pengguna Windows 7, anda harus restart kembali PC, dan secepatnya tekan tombol F8. Nantinya akan tampilan pilihan menu yang salah satunya akan menampilkan pilihan Safe Mode. Jika cara ini bisa membuat kamu masuk ke Windows, berarti permasalahan terletak pada driver atau aplikasi yang membuat crash atau bisa pula terjangkit virus.

2.Cara Mengatasi Komputer / Laptop Restart Tidak Bisa Masuk Windows Dengan Booting dari Linux

Cara lain untuk memperbaiki komputer / laptop sering restart anda dapat mencoba melakukan booting menggunakan bantuan CD Linux seperti Ubuntu atau Linux Mint ( dapat juga menggunakan distro Linux yang lain ). Lakukan booting dari sini dan lihat apakah sistem tersebut bisa bekerja dengan baik. Apabila dengan cara ini anda berhasil masuk ke dalam sistem operasi Linux dengan lancar, itu artinya permasalahan terdapat pada Windows anda dan solusi yang perlu dilakukan adalah instal ulang.

3. Cara Mengatasi Komputer / Laptop Restart Tidak Bisa Masuk Windows Dengan Menggunakan CD / DVD

Cara paling mudah untuk memulihkan komputer yang sering restart tidak dapat masuk ke windows adalah dengan cara repair windows. Caranya adalah masukkan CD instalasi windows kemudian ubah BIOS untuk mencoba start-up via CD atau DVD-ROM, kemudian setelah instalasi masuk ke bagian install windows maka pilih opsi repair windows. Apabila repair windows gagal anda dapat menginstal windows pada folder baru atau lakukan format hard disk. Pastikan bahwa anda tidak merubah setting BIOS sebelumnya. Jika memang anda telah merubah setting BIOS maka kembalikan setting BIOS seperti semula. Jika anda lupa, sebaiknya pilih Loading BIOS Default.

4. Cara Terakhir Mengatasi Komputer / Laptop Restart Tidak Bisa Masuk Windows

Langkah pertama hidupkan komputer atau laptop anda.
Ketika muncul tampilan BIOS, tekan F12 atau F2 untuk merubah setting Booting.
Lalu masukkan CD/DVD Master Windows anda.
Setelah itu pada panel BIOS, pilih BIOS Configuration untuk mengubah First Boot menjadi ke CD/DVD. Atau jika anda menggunakan flashdisk yang digunakan untuk instal windows, silahkan ubah First Boot menjadi USB.
Langkah selanjutnya ubah urutan First Bootnya dari Harddisk ( SATA ), menjadi CD /DVD.
Setelah itu SAVE.
Selanjutnya restart komputer anda.
Kemudian akan muncul tulisan ' Press Any Key From Boot to CD/DVD . . . ' Tekanlah tombol apapun pada keyboard anda.
Selanjutnya untuk repair computer pilih Repair Your Computer.
Tunggu beberapa saat.
Click Next.
Terakhir akan muncul jendela yang mempunyai 5 pilihan utnuk anda. Contohnya saja : Pilih Startup Repair jika Windows anda masih normal. Jika Anda pernah melakukan backup silahkan pilih System Restore. Jika anda pernah melakukan restore point image, pilih System Image Recovery. Jika tidak keduanya pilih opsi Command Prompt.

Jika anda memilih menggunakan Command Prompt pada bagian nomor 10 diatas akan muncul kotak berwarna hitam seperti cmd / dos. Dan ikuti langkah - langkah sebagai berikut ini :
Petama - tama ketik DIR C
Setelah itu ketiklah salah satu opsi yang akan muncul pada layar PC anda ( satu persatu tidak sekaligus ) mulailah dari yang paling atas. Kemudian tekan ENTER.
Lalu jika anda sudah menemukan tulisan ' The operation completed succesfully ' silahkan restart komputer. Tidak perlu menulis semua perintah diatas. Jika telah melakukan semua perintah diatas, namun belum ada yang berhasil, maka lakukan pemulihan pertama yaitu Startup Repair. Akan tetapi Anda harus mengubah First Boot kembali ke Hard disk jika sudah selesai perbaikan.

24. Materi: External Hard drive 1. Saat kamu tengah memformat external hard drive, tiba-tiba external hard drivemu mati total dan tidak bisa dikenali lagi oleh laptopmu. Bagaimana caramu memperbaikinya? Terima kasih.


Jawaban:

Format Harddisk

Jika harddisk masih tidak terbaca di File Explorer Windows coba buat partisi baru. Caranya buka menu Start dan ketik "disk management" kemudian tekan tombol Enter setelah opsi "Create and Format Hard Disk Partitions" muncul.

Scroll dan lihat jika harddisk eksternal kalian ada di daftar. Kalau iya, pastikan harddisk tersebut online dan sudah diformat. Jika offline klik kanan di nama disk dan pilih online.

Jika disk belum diformat atau ada tulisan Unallocated, klik kanan dan pilih "New Simple Volume" dan ikuti wizard yang ada untuk membuat partisi baru. Tapi hati-hati karena langkah ini bisa menghapus data yang ada di harddisk, jadi pastikan tidak ada data penting di dalamnya.
