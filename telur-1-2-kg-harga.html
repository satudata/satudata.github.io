Gula  = x

7x + 2y = 105.000

5x + 2y = 83.000

Step 1] - Eliminasi

7x + 2y = 105.0005x + 2y = 83.000______________-        2x = 22.000          x = 11.000

Step 2] - Substitusi nilai x

5x + 2y = 83.000

5(11.000) + 2y = 83.000

55.000 + 2y = 83.000

2y = 83.000 - 55.000

2y = 28.000

 y = 14.000

Step 3] - Mencari 3y + x

3y + x = 3(14.000) + 11.000

          = 42.000 + 11.000

          = 53.000

30. Harga 1 telur dan 3 kg daging Rp180.000,00. Jika harga 2 kg telur dan 5 kg daging Rp 309.000,00 , maka harga 2 kg telur dan 1 kg daging adalah …​


Penjelasan dengan langkah-langkah:

t + 3d = 180 (x2) 6d = 360

2t + 5d = 309 (x1) 5d = 309 –

d = 51

t = 180 - 3(51)

t = 27

2t + d = 54 + 51

2t + d = Rp. 105.000, –

Jawaban:

Rp. 105.000,-

Penjelasan lengkap :

Diketahui :

• Harga 1 kg telur dan 3 kg daging

= Rp. 180.000,00

• Harga 2 kg telur dan 5 kg daging

= Rp. 309.000,00

Ditanya :

Harga 2 kg telur dan 1 kg daging =…?

Jawaban :

Sistem persamaan linier dua variabel

• Telur = x

• Daging = y

x + 3y = 180.000 … persamaan 1

2x + 5y = 309.000 …persamaan 2

Dengan metode substitusi

• Langkah 1

Substitusi variabel x persamaan 1

x + 3y = 180.000

x = 180.000 - 3y

• Langkah 2

Substitusi variabel x dari persamaan 1

kita substitusikan ke persamaan 2

2x + 5y = 309.000

2(180.000 - 3y) + 5y = 309.000

360.000 - 6y + 5y = 309.000

- y = 309.000 - 360.000

- y = - 51.000

y = 51.000

• Langkah 3

Variabel x kita substitusi ke persamaan 1 atau persamaan 2,sehingga nilai x

adalah sbb:

x + 3y = 180.000

x + 3(51.000) = 180.000

x + 153.000 = 180.000

x = 180.000 - 153.000

x = 27.000

Kesimpulan:

Telur (x) per kg = Rp. 27.000,-

Daging (y) per kg = Rp. 51.000,-

• Harga 2 kg telur dan 1 kg daging :

2x + y = 2(27.000) + 51.000

= 54.000 + 51.000

= Rp. 105.000,-

Jadi harga 2 kg telur dan 1 kg daging

adalah 105.000,-
