21. ada 20 buku,8buku bersampul coklat dan sisanya bersampul biru.persentase buku bersampul adalah​


Jawab:

Buku sampul coklat = 40%

Buku sampul biru = 60%

Penjelasan dengan langkah-langkah:

Menghitung nilai persentase

Mapel : Matematika

Diketahui :

Banyaknya buku bersampul = 20 buku

Buku bersampul warna coklat = 8 buku

Buku bersampul warna biru = 12 buku

Ditanya :

Persentase buku bersampul warna coklat dan biru .... ?

Jawab :

Untuk menjawabnya, pertama hitung dulu persentase buku sampul warna coklat dengan menuliskan ....

C = [tex]\frac{8}{20}[/tex]

C = [tex]\frac{8 x 5}{20 x 5}[/tex] (Pembilang dan penyebutnya dikali 5)

C = [tex]\frac{40}{100}[/tex]

C = 40%

Kedua, hitung persentase buku sampul bewarna biru ....

B = [tex]\frac{12}{20}[/tex]

B = [tex]\frac{12 x 5}{20 x 5}[/tex]

B = [tex]\frac{60}{100}[/tex]

B = 60%

Jadi, persentase buku sampul bewarna coklat dan biru masing - masing 40% dan 60%

#LearnMathWithBrainly

#TetapSemangatBelajarnya

Penjelasan dengan langkah-langkah:

Dieket :

Total buku : 20

Sam Coklat : 8

Sam biru : sisa buku.

maka 20-8 = 12 (Sam biru)

*PERSENTASE SAMPUL COKELAT

rumus : Sampul cokelat / Total buku × 100

[tex] \frac{8}{20} x100 \\ = \frac{800}{20} \\ = 40\%[/tex]

*PERSENTASE SAMPUL BIRU

[tex] \frac{12}{20} \times 100 \\ = \frac{1200}{20} \\ = 60\%[/tex]

Moga membantu
