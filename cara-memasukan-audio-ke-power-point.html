Jawaban:

1. Editing Font (Huruf)

Untuk melakukan perubahan huruf pada PowerPoint

Kotak Latin text font : untuk memilih jenis huruf.

Kotak Font Style : untuk memilih tampilan huruf (biasa, tebal, miring, miring dan tebal).

Kotak Size : untuk menentukan ukuran besar huruf.

Kotak Font Color : untuk memilih warna huruf. Misalkan kita pilih jenis huruf Bernard MT Condensed, tampilan huruf Regular, besar huruf 54 point, dan warna huruf merah (red).

2. Mengatur Alignment (Perataan text)

Grup paragraph : untuk pengaturan tulisan dapat dilakukan dengan menggunakan fasilitas toolbars yang terdapat pada ribbons tab menu Home

Bullets and Numbering : untuk memberikan penomoran pada suatu teks atau naskah, baik itu penomoran dengan angka, huruf maupun dengan simbol.

3. Header and Footer dan page number

header and footer untuk memberikan informasi pada slide presentasi, pada semua halaman atau sebagian presentasi. Page numbering berfungsi untuk memberikan halaman pada slide secara otomatis.

4. Menambah Slide

Untuk melengkapi slide presentasi, kita dapat menambahkan beberapa halaman slide berikutnya dengan cara :

• Klik tab menu Home.

• Pada ribbons yang ada klik New Slide (), kemudian isi sesuai dengan bahan yang akan dipresentasikan

5. Memasukkan Gambar

Untuk membuat presentasi yang kita buat menjadi lebih menarik, salah satu caranya adalah dengan memasukkan objek berupa gambar ke dalam presentasi.

6. Memasukkan Textbox, Diagram, dan Movie

*Memasukkan Textbox

Jika ingin memasukkan teks pada suatu slide, namun tidak bisa menuliskan teks tersebut pada area yang diinginkan, maka kita harus memasukkan textbox dan menuliskan teks tersebut pada textbox yang kita masukkan. Kita dapat menggeser textbox ke area yang diinginkan.

*Memasukkan Diagram

Pada PowerPoint kita juga bisa memasukkan objek berupa diagram atau grafik (chart).

*Memasukkan Movie dan Sound

PowerPoint juga dapat memasukkan objek bergerak berupa movie dan dapat pula memasukkan objek berupa suara. Hal ini bertujuan untuk membuat presentasi lebih menarik, modern, dan tidak membosankan.

7. Membuat Tabel

Yang dimaksud dengan tabel pada PowerPoint adalah table yang sangat sederhana, artinya di dalam tabel tersebut kita tidak bisa membuat rumus dan fungsi seperti dalam Microsoft Excel. Pada slide presentasi memungkinkan kita memasukkan table.

8. Mengatur Background

Background adalah warna atau gambar yang digunakan untuk latar belakang suatu slide.

9. Hyperlink

Hyperlink digunakan untuk mengaitkan antara slide yang satu dengan slide yang lain. Hyperlink memudahkan kita apabila sedang presentasi dimana sudah menerangkan beberapa materi dan ada peserta menanyakan materi yang sudah dijelaskan sebelumnya. Dengan adanya hyperlink, kita langsung dapat menuju ke slide yang sudah dilewati.

20. Cara memisahkan teks/picture/audio/video dalam power point


Cara memisahkan teks / picture / audio/ video dalam power point mugnkin maksud kamu adalah cara memasukkan teks / picture / audio/ video ke dalam power point? Jawabannya adalah yaitu dengan menggunakan Tab insert yang berada di Toolbar Microsoft Office Power Point.  

Pembahasan

Di dalam tab tersebut terdapat beberapa ikon yaitu insert image, insert movie, dan insert sound.

Insert image digunakan untuk memasukkan file dengan jenis image / picture / gambar ke dalam laman lembar power point dengan format JPEG / JPG / PNG / GIF / dll.

Kemudian insert movie berguna untuk memasukkan file video dengan format video MP4 / WMV / dll.

Jika insert sound digunakan untuk memasukkan file dengan jenis audio / musik ke dalam laman lembar power point dengan format MP3 / WAV / dsb.

Namun bila pertanyaan kamu adalah memisahkan beberapa tipe file tersebut, maka jawabannya adalah file tersebut akan otomatis terpisah atau  terpilah sendiri apabila kamu memasukkan insert image, maka file video dan suara tidak akan terdeteksi di jendela explorer tersebut.

Cara memisahkan teks / picture / audio / video dalam powerpoint yaitu, pada saat ingin memasukkan teks, picture, audio, dan video dalam powepoint, kita dapat memisahkannya dengan menggunakan menu insert.

Saat ingin memasukkan teks, kita tinggal mengetikkan pada slide, jika ingin memasukkan picture (gambar) dapat menggunakan menu insert, image. Jika ingin memasukkan audio, insert sound, jika ingin memasukkan video, insert movie.

Pembahasan

Di dalam powerpoin, terdapat beberapa menu dengan fungsinya masing - masing. Jika ingin menambahkan gambar, audio, video dalam presentasi, kita dapat menggunakan menu insert. Didalam tab menu insert  terdapat beberapa sub menu yaitu insert image untuk memasukkan gambar, insert movie untuk memasukkan video, dan insert sound untuk memasukkan audio.

Insert image digunakan untuk memasukkan file dengan jenis image atau picture atau gambar ke dalam laman lembar power point dengan format JPEG / JPG / PNG / GIF / dan lain - lain.

Kemudian insert movie berguna untuk memasukkan file video dengan format video MP4 / WMV / dan lain - lain.

Jika insert sound digunakan untuk memasukkan file dengan jenis audio atau musik ke dalam laman lembar power point dengan format MP3 / WAV dan sebagainya.
