14. cara mengedit file word dari pc ke smartphone gimana


Memasang aplikasi Microsoft Office pertama kali
Buka file Office yang disimpan di Dropbox Anda dan ketuk ikon Edit. Atau, dari tab Files Anda bisa mengetuk tombol menu di sebelah file tertentu untuk dibuka dengan aplikasi yang lain.

Catatan: Saat ini pengeditan dapat dilakukan pada file yang disimpan di Office 2007 atau yang lebih baru (.docx; .xlsx; .pptx). File Office yang lebih tua bisa untuk hanya-baca.
Anda akan diminta memasang aplikasi Office Mobile, aplikasi yang sesuai untuk perangkat Anda. Selanjutnya, Anda akan dibawah langsung ke Google Play Store.
Pilih Pasang dan izinkan aplikasi ini diunduh.
Setelah dipasang, buka kembali aplikasi Dropbox Anda dan ketuk Berikutnya di layar untuk menunjukkan pemasangan telah selesai atau, jika Anda tidak melihat layar itu, silakan ketuk lagi tombol Edit. Kemudian Anda akan diminta untuk mengizinkan aplikasi Microsoft Office mengakses Dropbox Anda.

Catatan: Mengedit file di dalam akun Dropbox Business membutuhkan salah satu akun Office 365 berikut: Business, Business Premium, ProPlus, E3, atau E4. Untuk tablet Android dengan ukuran layar lebih besar dari 10,1, langganan Office 365 juga dibutuhkan untuk mengedit file yang disimpan pada akun Dropbox Basic dan Plus.
Catatan: Anda mungkin diminta masuk ke akun Microsoft untuk mengedit file Anda (akun ini bisa akun Microsoft mana saja yang sudah Anda miliki, mis. akun Hotmail atau Outlook.com).
File Anda akan dibuka di aplikasi seluler Office, dan Anda siap mengedit.
Ketika selesai mengedit, ketuk panah balik untuk menyimpan. Anda akan diantarkan kembali ke Dropbox Anda untuk melihat file yang telah diperbarui.

24. Cara mengunci file word agar tidak bisa diedit dan dicopy?


Berikut beberapa cara untuk mengunci file pekerjaanmu agar tidak mudah disalin dan diedit melalui word:

Buka file dokumen yang Anda kerjakan melalui Word.Pilih menu home 'Review' pada Word.Pilih icon 'Restrict Editing'.Klik Ceklis pada icon 'Editing Restrictions'.Klik 'Yes, Star Enforcing Protection'.Muncul icon password dan masukkan password yang Anda buat.Tulis kembali password yang Anda buat.KLik 'OK'.

Setelah melakukan langkah-langkah tersebut, maka file yang Anda kerjakan sudah terkunci. Selamat mencoba.

25. Word 2007 mampu menyimpan file ke dalam format lain yaitu​


Doc dan docx adalah dua format file atau ekstensi file yang digunakan di aplikasi Microsoft Word, salah satu aplikasi yang terdapat di paket aplikasi Microsoft Office. Format file doc digunakan di Microsoft Word 2003 dan versi sebelumnya, sedangkan docx digunakan di aplikasi Microsoft Word 2007 dan versi sesudahnya. Salah satu kendala yang ada sementara ini untuk format file terbaru (docx) adalah terkait kompatibilitas.
