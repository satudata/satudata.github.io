2. Kesimpulan dari premis berikut adalah! P1: jika saya tidur makan saya mengantuk P2: saya tidak mengantuk A. Saya mengantuk B. Saya tidak tidur C. Saya tidur D. Saya tidak mengantuk E. Saya tidak tidur dan mengantuk Maaf kak merepotkan, mohon untuk dijawab:')


Jawab:

Penjelasan dengan langkah-langkah:

premis 1 : p →q

premis 2 : ~q             ( modus tollens)

__________________

Kesimpulan: ~p

Modus Tollens berarti “jika diketahu p → q dan ~q, maka bisa ditarik kesimpulan ~p“.

premis 1 : Jika saya tidur maka saya mengantuk (p ⇒ q)

premis 2 : Saya tidak mengantuk (~q)

Kesimpulan : Saya tidak tidur (~p )    (B)
