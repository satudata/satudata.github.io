Rumus rata-rata excel dengan fungsi Average tersebut menghitung nilai rata-rata dari hasil A2:A6 dan B2:B6. Hasilnya adalah 20. Angka ini di dapat dari perhitungan berikut:

(10+15+30+20+15+20+30+40+0+20)/10 = 200/10 = 20

Pada microsoft excel, selain menggunakan rumus excel tersebut, cara mencari rata-rata dengan rumus average untuk hasil data tersebut, dapat juga dihitung dengan rumus excel berikut seperti contoh pada sel D4:

=AVERAGE(A2:B6)

Rumus Excel diatas akan menghasilkan nilai rata-rata yang sama dengan rumus sebelumnya. Yakni angka 20

maaf ya kak kalo jawabannya salah

Jawaban:

=SORTBY([range];RIGHT([range];2))

atau

=SORTBY([range];MOD([range];100))

dengan [range] adalah range sumber data yang ingin diurutkan.

Penjelasan:

Misalnya, data yang ingin diurutkan ada pada range: A1:A20 (sel A1 sampai sel A20).

Pada kolom B, ingin ditampilkan data dari range A1:A20 secara terurut dengan hanya melihat 2 angka terakhir saja, maka rumus yang dimasukkan pada sel B1 adalah:

=SORTBY(A1:A20;RIGHT(A1:A20;2))

yang artinya: urutkan data pada sel A1 hingga A20, dengan urutan berdasarkan 2 karakter terakhir pada sel A1 hingga A20 tersebut.

ALTERNATIF LAIN

Jika ingin benar-benar "matematis", dan data yang diurutkan adalah angka/number, bisa memanfaatkan fungsi MOD (sisa pembagian bilangan bulat) dengan pembagi/divisor 100. Rumus/formulanya menjadi:

=SORTBY(A1:A20;MOD(A1:A20;100))

Sebagai ilustrasi, silahkan memperhatikan gambar terakhir. Range sumber pengurutan adalah A1:A20. Nilai-nilai pada range ini saya isi secara random dengan fungsi =RANDBEETWEEN(100000;999999) ==> menghasilkan nilai acak dalam rentang 100000 hingga 999999.

Kolom B (B1 hingga B20) adalah hasil pengurutan dengan fungsi SORTBY di atas. Kolom C (C1 hingga C20) hanya untuk menampilkan 2 karakter terakhir dari sel B1 hingga B20, agar lebih bisa terlihat hasil pengurutannya.


27. Sebutkan cara mengubah urutan nomor halaman di excel


Jawaban:

How to Provide Page Numbers in Microsoft Excel

As with other documents, documents created using Microsoft Excel should also be given a page number so that no errors occur in the order in which they are read. Numbering in Microsoft Excel is relatively more complicated than numbering in Microsoft Word but that does not mean that in Microsoft Excel numbering cannot be done. To create page numbers in Microsoft Excel you can use the following method.

Click the Page Layout ribbon menu, then click the page setup button, or

Press the keyboard keys ALT + P, S, P

Figure: How to display the Page Setup dialog in Microsoft Excel

Then in the Page Setup dialog select the Header / Footer tab

Decide where to place the page number whether in the header (the top of the document), or in the Footer (the bottom of the document). For example, select and click on the footer then click the Custom Footer button .

Next in the Footer dialog , select where to place the page number, whether on the left ( left section ), in the middle ( Center Section ), or on the right ( Right Section ). Suppose we want to put it in the middle of the page, then click an empty area in the Center Section .

Then click the Insert Page Number button.

In the center section area there will be a text & [Page] . (you can also type it directly)

Click the OK button to close the Footer dialog

Click the OK button again to close the page setup dialog .

The page number will now be used when printing a document.

besides giving the number, you can also add the excel page number directly to the footer / excel header .
