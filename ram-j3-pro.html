8. berapa a dan b jika J3 = 180 dan S4 = 0


Nilai a dan b yang diketahui J₃ = 180 dan S₄ = 0 adalah 90 dan -30.

Untuk soal tersebut merupakan matematika ekonomi tentang deret hitung (barisan aritmatika).

Deret hitung (barisan aritmatika) adalah barisan bilangan dengan hasil pengurangan setiap suku dengan suku sebelumnya selalu sama atau tetap.

Rumus deret hitung

Suku ke-n → [tex]\boxed {S_n = a + (n-1) b}[/tex]Jumlah n suku pertama → [tex]\boxed {J_n = \frac{n}{2}[2a + (n-1) b] }[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

J₃ = 180 dan S₄ = 0

Ditanya:

Nilai a dan b.

Penyelesaian:

Membuat persamaan dari nilai yang diketahui

[tex]\boxed {J_n = \frac{n}{2}[2a + (n-1) b] }[/tex]

J₃ = 180

[tex]\displaystyle \frac{3}{2}[/tex] [2a + (3 - 1) b] = 180

[tex]\displaystyle \frac{3}{2}[/tex] × 2 (a + b) = 180

3a + 3b = 180   ... persamaan I

[tex]\boxed {S_n = a + (n-1) b}[/tex]

S₄ = 0

a + (4 - 1) b = 0

a + 3b = 0       ... persamaan II

Menentukan nilai a dan b

Eliminasikan b pada persamaan I dan II.

3a + 3b = 180

 a + 3b = 0     -

2a         = 180

         a = [tex]\displaystyle \frac{180}{2}[/tex]

         a = 90

Subtitusikan a = 90 ke dalam persamaan II

a + 3b = 0

90 + 3b = 0

       3b = -90

         b = [tex]\displaystyle \frac{-90}{3}[/tex]

         b = -30

Jadi nilai a dan b pada deret hitung adalah 90 dan -30.

16. Apakah kepanjangan dari RAM..?? Dan Apakah itu RAM?


RAM (Random Access Memory)

Merupakan jenis memori yang isinya dapat diganti-ganti selama komputer dihidupkan sebagai suatu penyimpanan data yang dapat dibaca atau ditulis dan dapat dilakukan secara berulang-ulang dengan data yang berbeda-beda. Random artinya data yang disimpan pada RAM dapat diakses secara acak. Modul memori RAM yang umum diperdagangkan berkapasitas 128 MB, 256 MB, 512 MB, 1 GB, 2 GB, dan 4 GB

RAM dibagi lagi menjadi dua jenis, yaitu jenis statik dan dinamik.

RAM statik menyimpan satu bit informasi dalam sebuah flip-flop. RAM statik biasanya digunakan untuk aplikasi-aplikasi yang tidak memerlukan kapasitas memori RAM yang besar.RAM dinamik menyimpan satu bit informasi data sebagai muatan. RAM dinamik menggunakan kapasitas gerbang substrat sebuah transistor MOS sebagai sel memori elementer.
