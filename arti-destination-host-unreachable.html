1. Arti pesan : 1. Destinations Host Unreachable 2. Destinations Net Unreachable 3. Request Time out 4. General Failure 5. Transmit Failed General Failure


1. Destinations Host Unreachable
Jaringan yang dituju ada, namun anda tidak bisa mengakses target karena tidak ada jalur kesana atau memang di blok di firewall atau router.

2. Destinations Net Unreachable
jaringan yang dituju tidak bisa diakses.

3. Request Time out
Request Time out (RTO) adalah ketika komputer server tidak merespon permintaan koneksi dari client setelah beberapa lama (jangka waktu timeout bervariasi). Biasanya terjadi karena adanya firewall, koneksi rendah dan koneksi terputus atau tidak terkoneksi.

4. General Failure
Biasanya terjadi ketika system tidak bisa mengirimkan paket ping karena suatu masalah.

5. Transmit Failed General Failure
Jaringan telah terputus.

5. ARTI SHIPMENT FORWARDER TO DESTINATION


Arti Shipment Forwarded to Destination adalah STATUS PENGIRIMAN PESANAN OLEH KURIR PENGIRIMAN YANG MENYATAKAN BAHWA PESANAN TELAH DISAMPAIKAN KEPADA PETUGAS PENGANTAR UNTUK MENGIRIMKAN PESANAN KE ALAMAT PENERIMA.

Pembahasan

Shipment forwarded to destination adalah salah satu status pengiriman pesanan yang diterapkan oleh salah satu kurir pengiriman di Indonesia, yaitu JNE. Status ini menerangkan bahwa pesanan telah diterima di gudang di kota tujuan pengiriman dan telah diserahkan kepada petugas pengantar untuk kemudian disampaikan ke alamat penerima.

Biasanya, setelah status ini muncul, barang akan diterima oleh penerima pada hari yang sama.

28. Perintah untuk membuat static routing adalah A. ip route destination mask nexthop B. ip host nama host ip address C. ip route source mask nexthop D. ip route mask nexthop source E. ip route destination nexthop


Perintah untuk membuat static routing adalah ip route destination mask nexthop. Ip route adalah perintah untuk membuat static routing. Destination adalah jaringan tujuan yang ingin ditambahkan ke routing table. Mask adalah subnet mask yang digunakan dalam jaringan. Nexthop adalah address yang akan menerima paket dan memforwardnya ke jaringan remote.

Penjelasan

Static routing adalah proses meneruskan paket jaringan oleh admin atau pengelola jaringan dari satu jaringan kejaringan lainnya yang lebih spesifik. Router melakukan routing berdasarkan informasi yang diterima dari tabel routing.
