18. apa saja icon yang ada di aplikasi AutoCAD?


Ada berbagai icon yang terdapat dalam aplikasi AutoCAD. Berikut ini adalah beberapa icon yang sering digunakan dan fungsinya.

Pull-down menus: menu-menu AutoCAD yang disusun secara menurun.Properties toolbar: mengakses toolbar untuk properti ruang kerja.Standard toolbar: mengakses toolbar dasar.Layer toolbar: mengakses toolbar untuk later.Edit toolbar: mengakses fitur edit gambar.Draw toolbar: mengakses fitur menggambar.Save: menyimpan file yang sudah pernah disimpan sebelumnya.Save as: menyimpan file baru.Open: membuka dokumen AutoCAD.Close: menutup lembar kerja AutoCAD.Export: mengubah jenis ekstensi file hasil kerja.Page setup: mengatur ukuran kertas, orientasi, dan hal-hal lain yang berhubungan dengan kertas lembar kerja.Zoom: memperbesar maupun memperkecil ukuran tampilan lembar kerja.Text style: mengubah jenis huruf yang digunakan.Line: membuat garis lurus sambung.Circle: membuat lingkaran.Erase: menghapus objek yang tidak diinginkan dari lembar kerja..

19. jelaskan perbedaan antara ikon pemintas shortcut icon yang terdapat pada desktop window dan ikon pemintas shortcut icon yang terdapat pada taskbar​


Jawaban:

Icon pemintas shortcut

Yaitu icon bisa diakses dg 2 cara, yaitu diklik memakai mouse maupun diakses memakai keyboard.

Icon yang terdapat pada desktop window

Yaitu icon yg bisa diakses dg cara dobel klik memakai mouse (2 kali menekan tombol kiri mouse).

Ikon pemintas shortcut icon yang terdapat pada taskbar​

Yaitu icon yg bisa diakses dg cara klik memakai mouse (1 kali menekan tombol kiri mouse).

Penjelasan:

- Contoh icon pemintas shortcut yaitu Copy. Bisa diakses dg menekan simbol icon itu (memakai mouse) dan bisa juga memakai keyboard (menekan tombol Ctrl+C).

- Contoh icon yg terdapat pada desktop Windows: icon di gambar berikut (ditunjukkan tanda panah kuning). Bisa diakses dg mouse (dobel klik pada icon itu).

- Contoh ikon pemintas shortcut icon yang terdapat pada taskbar​: icon di gambar berikut (ditunjukkan tanda panah merah). Bisa diakses dg mouse (klik pada icon itu).

6. Systray digunakan untuk menginformasikan jam dan tanggal.
