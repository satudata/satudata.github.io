Jawab:

Penjelasan dengan langkah-langkah:

Jika Alif membeli 2 karung gula pasir dengan berat masing-masing 50 kg dan tara 2% setiap karung, maka berat gula pasir yang sebenarnya adalah:

50 kg - (50 kg x 2%) = 49 kg per karung

Jadi, Alif harus membayar 49 kg x Rp.10.000/kg = Rp. 490.000 untuk setiap karung gula pasir.

Sehingga, Alif harus membayar Rp.490.000 x 2 = Rp.980.000 untuk 2 karung gula pasir.


18. Pak Salim hendak berjualan beras dan gula pasir. Ia berbelanja beras dan gula pasir di pasar induk. Harga satu karung beras Rp 120.000,00 dan harga satu karung gula pasir Rp 100.000,00. modal yang ia miliki adalah Rp 10.000.000,00. Kios Pak Salim hanya dapat menampung tidak lebih dari 85 karung (beras dan gula pasir). Tiap satu karung beras dijual dengan laba Rp. 7.000,00 dan tiap satu karung gula pasir dijual dengan laba Rp 6.000,00. keuntungan maksimum yang diperoleh Pak Salim adalah


Jawab:

Penjelasan dengan langkah-langkah:

misakan:

beras = x

gula pasir = y

Objective function:

7.000x + 6.000y = z

dimana z = keuntungan maksimal

subject to

(1)

120.000x + 100.000 y ≤ 10.000.000

6x + 5y ≤ 500

(2)

x + y ≤ 85

(3)

x ≥ 0

(4)

y ≥ 0

Penyelesaian (graphic lihat gambar 1)

untuk fungsi (1)

titik potong sumbu y ( x = 0)

5y = 500

y = 100

di titik (0,100)

titik potong sumbu x ( y = 0)

6x = 500

x = 83.33

di titik (83,33; 0)

untuk fungsi (2)

titik potong sumbu y (x = 0)

y = 85

di titik (0, 85)

titik potong sumbu x (y = 0 )

x = 85

di titik (85,0)

dari graphic dapat ditentukan titik titik pojok yaitu

(1) (0,0)

(2) (0,85)

(3) (83.33 , 0 )

(4) titik potong fungsi x + y ≤ 85 dan 6x + 5 y ≤ 500: (75,10)

cara perhitungan:

x = 85 - y

6( 85 - y) + 5y = 500

510 - 6y + 5y = 500

- y = -10

y = 10

x = 85 - y

x = 85 - 10 = 75

Perhitungan keuntungan:

fungsi objective: 7.000x + 6.000y = z

1. untuk (0,0)

z = 0

2. untuk (0,85)

7.000 (0) + 6.000 (85) = 510.000

3. untuk (83, 0)

7.000 (83) + 6.000 (0) = 581.000

4. untuk (75, 10)

= 7.000 (75) + 6.000 (10)

= 525.000 + 60.000

= 585.000

dari perhitungan di atas dapat disimpulkan

keuntungan maksimum yang diperoleh Pak Salim = Rp. 585.000 dengan penjualan 75 karung beras dan 10 karung gula pasir.

neto/karung = 50×(100%-4%) = 50×96/100 = 48 kg

harga pembelian/karung = 900.000:5 = 180.000/karung

harga pembelian/kg = 180.000:48 = 3.750/kg

Netto = 200 Kg - 5 Kg
