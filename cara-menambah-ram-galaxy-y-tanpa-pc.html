1. Kill Aplikasi yang tidak perlu / tidak sedang digunakan
Langkah ini diambil untuk mematikan aplikasi-aplikasi yang sedang tidak digunakan. Caranya, silahkan masuk ke Settings kemudian buka Apps, lalu pilih Running. Disana Anda akan menemukan aplikasi-aplikasi yang sedang running. Untuk cara mematikannya, cukup buka kemudian klik Stop untuk mematikan aplikasi.
2. Hapus Aplikasi yang Memakan RAM Besar
Jika memang tidak terlalu diperlukan, Anda bisa menghapus aplikasi yang tergolong memakan ram besar. Pengecekan seberapa besar penggunaan RAM bisa Anda lihat pada Settings, kemudian buka Apps, kemudian pilih Running.
3. Gunakan Aplikasi XBooster
Pada awalnya, nama aplikasi ini adalah Boost My Experia. Dibuat oleh seorang member forum xda ber-username gagan.u20, dan aplikasi ini banyak digunakan pada ROM-ROM xperia yang muncul kemudian.

4. Gunakan Aplikasi RAM Manager
RAM Manager tersedia dalam versi gratis dan pro. Berfungsi untuk mengaktifkan profil tertentu untuk manajemen RAM android sesuai kebutuhan. Beberapa profil yang tersedia antara lain more free memory untuk memori bebas lebih banyak, balance for everyday using untuk kestabilan penggunaan setiap hari, multitasking mode untuk multitasking berat, sampai mode hard gaming untuk penggunaan game berat.

29. bagaimana cara menambahkan ram pada windows 7​


Jawaban:

Cara yang pertama adalah dengan menambah RAM pada komputer maupun laptop. Khusus untuk menambah RAM, ada beberapa pertimbangan yang harus kamu perhatikan. Pertimbangan tersebut antara lain sebagai berikut.

Slot Kosong

Pada beberapa laptop biasanya sudah dipasang dua slot RAM. Tujuannya adalah kita dapat menambah memori RAM sendiri jika dibutuhkan. Akan tetapi, apabila slot memori RAM hanya satu saja atau dua-duanya sudah terpasang, maka kamu tinggal mengganti saja kartu memori RAM-nya dengan RAM yang diinginkan.

Cek Jenisnya

Berhubungan dengan poin sebelumnya, kamu harus mengetahui ukuran memori RAM itu sendiri. Sebab, jenis RAM pada komputer dan laptop itu berbeda-beda, ada yang DDR2, DDR3, DDR4, dan lain sebagainya. Di bagian ini harusnya kamu sudah paham cara membedakan RAM DDR1, DDR2, DDR3. Sebab, tempat slot memori RAM pada komputer maupun laptop sudah didesain sesuai dengan jenisnya masing-masing. Dengan begitu jenis memori yang berbeda dengan slotnya akan susah dimasukan. untuk kasus ini, kamu juga perlu mempelajari cara mengetahui max RAM support Motherboard.

Cek Ukurannya

Jika sudah mengetahui jenis memori RAM yang digunakan, selanjutnya adalah mengecek ukuran memori RAM yang dibutuhkan. Apabila memori RAM sebelumnya sudah dipasang sebesar 2GB, sebaiknya dipasang juga dengan ukuran yang sama yakni 2GB. Kalau misalkan kamu membutuhkan memori RAM sebesar 4 GB dengan cara 1GB ditambah 3GB, hal ini bisa juga diterapkan. Akan tetapi, performanya lebih menurun dibandingkan menggunakan memori RAM yang dipasangkan kapasitas sama besar.

Cek Kecepatan Transfernya

Berikutnya adalah mengecek kecepatan transfer pada memori RAM. Setiap memori RAM mempunyai kecepatan transfer rate yang berbeda-beda. Bisa kamu lihat pada bagian kodenya seperti PC 1900, PC 5000, PC 18050, dan lain-lainnya. Nah, perhatikanlah pada angkanya! Jika angkanya semakin besar, maka kecepatan rate-nya juga semakin kencang.

Cobalah lihat kecepatan rate pada memori RAM di laptop atau komputer. Usahakan kecepatan rate kedua memori RAM harus sama. Jika kecepatannya berat sebelah, komputer akan membaca kecepatan rate memori RAM yang paling kecil. Dengan begitu kamu akan sia-sia menambah memori RAM yang baru.

Menggunakan Virtual Memory

Jika kamu tidak ingin menambah atau pun mengganti memori RAM secara fisik, maka kamu bisa menggunakan cara virtual memori. Virtual memori sendiri adalah memori yang yang ditambahkan pada RAM dengan menggunakan sisa memori yang ada pada hardisk. Dengan kata lain, memori RAM akan “bertambah” dengan ditambahkan memori pada hardisk komputer maupun laptop. Caranya sendiri ikutilah langkah-langkah berikut.

Pertama-tama bukalah Windows Explorer.

Setelah itu, sorot kursor pada bagian My Computer lalu klik kanan dan pilih Properties.

Setelah itu, sorot kursor pada menu Advanced System Settings lalu klik.

Nanti akan terbuka sebuah pop up Advanced, lalu klik properties bada bagian kolom performance.

Setelah terbuka, pilih tab Advanced kemudian klik Change.

Akan terbuka tab virtual memory. Di sinilah kamu hilangkan ceklist pada bagian “Automatically manage paging file size for all drives” yang posisinya paling awal.

Jika sudah diklik, pilih memori harddisk mana yang akan dibuat partisinya.

Setelah itu, klik “Costum Size” lalu masukan pada kolom “Initial Size” dengan jumlah memori yang ingin digunakan. Tentunya dalam satuan MB. Contohnya, kamu membutuhkan 5GB, maka kamu tulis 5000 MB.

Isi juga pada bagian “Maximum Size” yakni batas memori yang digunakan. Contohnya batasnya sampai 9GB, maka kamu tulis 9000MB.

Jika semuanya sudah selesai, klik OK dan restart komputer atau laptop.

Memori RAM otomatis bertambah.

Namun, sebelum kamu menggunakan virtual memori ini, kamu harus mengetahui dulu bahwa memori Harddisk yang digunakan pada memori RAM otomatis akan berkurang. Tentunya kamu sudah mempersiapkan risikonya ya.

Itulah cara menambah RAM Windows 7 32bit. Jika kamu ingin membeli RAM yang baru. cobalah baca referensi

ram android tidak bisa di tambah karna sudah paten..

semoga membantuBisa menggunakan metode "Swap RAM". Yaitu memanipulasi size external sd card menjadi RAM. Tapi resikonya jika gagal cukup besar, bisa bootloop, bahkan brick (mati total). Untuk cara, singkat aja

1. Pastikan HP sudah di root

2. Install Swapper on Root
2a. Buka Swapper
2b. Centang active swap on boot
2c. Pilih SD Card FAT Partition pada SwapFile Partition
2d. Isi value dengan kapasitas swap RAM yang Anda kehendaki
2e. Besar kapasitas swap RAM tergantung dengan besar free space memori eksternal
2f. Save
2g. Tunggu hingga swap file selesai dibuat
2h. Reboot perangkat Android Anda

3. Install Sd-Booster
3a. Buka SD-Booster
3b. Isi cache size antara 1 hingga 8192,disarankan 1024 atau 2048
3c. Tekan gambar centang hijau

4. Install Juwe RAM Script
4a. Pastikan CWM Recovery sudah terinstall di perangkat Android Anda
4b. Referensi install CWM Recovery bisa di cari di search engine
4c. Download Juwe RAM Script, simpan di SD Card, taruh dibagian luar
4d. Buka CWM Recovery Anda
4e. Pilih install zip from sd card
4f. Arahkan ke Juwe RAM Script
4g. Install

***
Tapi ada kekurangannya loh kak. Kalau kita menambah RAM, otomatis akan terjadi Overclock. HP bisa lebih cepat panas dan lama lama bisa rusak. Dan Swap RAM dengan external sd card ini, juga bakal mengurangi umur sd card kakak
