Soal belum lengkap

Nilai a pada parabola untuk menentukan arah parabola itu terbuka ke atas atau terbuka ke bawah.

1. a > 0, titik balik minimum sehingga kurva terbuka (cekung) ke atas,

2. a < 0, titik puncak maksimum sehingga kurva terbuka (cekung) ke bawah.

Nilai b dan a pada parabola untuk menentukan letak titik puncak.

1. Titik puncak ada di kiri sumbu y, ab > 0,

2. Titik puncak pada sumbu y, ab = 0,

3. Titik puncak ada di kanan sumbu y, ab < 0.

Nilai c menunjukkan perpotongan parabola dengan sumbu y.

1. Memotong sumbu y positif, c > 0,

2. Memotong sumbu y di pusat, c = 0,

3. Memotong sumbu y negatif, c <0.

Nilai diskriminan menunjukkan kedudukan grafik dengan sumbu x.

1. Memotong sumbu x di dua titik, D > 0,

2. Menyinggung sumbu x, D = 0,

3. Tidak memotong sumbu x, D < 0.

Diketahui:

Parabola: f(x) = ax² + bx + c,

dengan a, b, dan c bilangan real dan a ≠ 0, x dinamakan variabel, a dinamakan koefisien dari x², b dinamakan koefisien dari x, dan c dinamakan konstanta.

Parabola terbuka ke bawah, titik puncak ada di kiri sumbu y, memotong sumbu y negatif, dan memotong sumbu x di dua titik.

Ditanyakan:

Penjelasan gambar pada lampiran.

(1) a < 0, karena parabola terbuka ke bawah

(2) ab > 0, karena titik puncak berada di sebelah kiri sumbu y dan

a < 0 maka b < 0.

(3) c < 0, karena c memotong sumbu y.

(4) D > 0, karena ada 2 titik yang memotong di sumbu x.

D > 0

⇔ b² - 4ac > 0

⇔ b² > 4ac

Jadi, penjelasan gambar pada lampiran adalah a < 0, b < 0, c < 0, dan b² > 4ac.

Tentukan titik potong grafik dengan sumbu x c. Tentukan titik potong grafik dengan sumbu y d.Tentukan titik puncak/ optimum/balik e. Gambarlah dengan menghubungkan titik-titik tersebut (bentuk kurva) pada diagram Kartesius


Penjelasan dengan langkah-langkah:

Perhatikan sketsa grafik fungsi kuadrat di bawah ini! Tentukan

a. sumbu simetri ⇒ xp = 1

b. rumus fungsi kuadrat ⇒ y = a(x² – 2x – 3)

c. nilai optimum ⇒ yp = –4a

d. titik puncak ⇒ (1, –4a)

e. titik potong grafik dengan sumbu y ⇒ (0,–3a)

Sebenarnya soal ini kurang lengkap. Dalam gambar yang diketahui hanya titik potong terhadap sumbu x, seharusnya ada satu titik lagi yang diketahui agar diperoleh koefisien dari x².

Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c dengan a ≠ 0.

Jika a > 0 maka kurva terbuka ke atas dan jika a < 0 maka kurva terbuka ke bawah

Ada beberapa cara untuk menentukan persamaan fungsi kuadrat, yaitu

Jika diketahui titik puncak (xp, yp) dan melalui titik (x, y)

y = a(x – xp)² + yp

Jika diketahui titik potong terhadap sumbu x di titik (x₁, 0) dan (x₂, 0) serta melalui titik (x, y)

y = a(x – x₁)(x – x₂)

Pembahasan

Dalam gambar kurva terbuka ke bawah maka

a < 0 (a = bilangan negatif)

Kurva memotong sumbu x di titik (–1, 0) dan (3, 0)

a. sumbu simetri

xp = ½ (x₁ + x₂)

xp = ½ (–1 + 3)

xp = ½ (2)

xp = 1

b. rumus fungsi kuadrat

y = a(x – x₁)(x – x₂)

y = a(x – (–1))(x – 3)

y = a(x + 1)(x – 3)

y = a(x² – 3x + x – 3)

y = a(x² – 2x – 3)

c. nilai optimum

diperoleh dengan mensubstitusikan x = xp = 1

y = a(x² – 2x – 3)

yp = a(xp² – 2xp – 3)

yp = a(1² – 2(1) – 3)

yp = a(1 – 2 – 3)

yp = a(–4)

yp = –4a

d. titik puncak

(xp, yp)

= (1, –4a)

e. titik potong grafik dengan sumbu y

Diperoleh jika x = 0

y = a(x² – 2x – 3)

y = a(0² – 2(0) – 3)

y = a(0 – 0 – 3)

y = a(–3)

y = –3a

(0, –3a)

Jadi agar diperoleh nilai a, dalam soal harus diketahui titik lainnya.

Kita misalkan nilainya adalah a = –1

maka

b. rumus fungsi kuadrat

y = a(x² – 2x – 3) = –1(x² – 2x – 3) = –x² + 2x + 3

c. nilai optimum

yp = –4a = –4(–1) = 4

d. titik puncak

(xp, yp) = (1, 4)

e. titik potong grafik dengan sumbu y

(0,–3a) = (0, –3(–1)) = (0, 3)
