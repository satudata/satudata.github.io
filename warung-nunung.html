27. Tiga tahun lalu, jumlah usia Nunung, Jeriko, dan Helmi adalah 33 tahun. Sekarang, usia Nunung 2 tahun kurang dari usia Jeriko. Sedangkan jumlah usia Jeriko dan Helmi adalah 30 tahun. Jika sekarang tahun 2020, Nunung lahir pada tahun


Jawaban:

//(N-3) + (J-3) + (H-3) = 33

N + J + H = 33+9

= 42

// N = J - 2

// J + H = 30

Nunung lahir tahun = ?

Penjelasan dengan langkah-langkah:

//N + J + H = 42

N + (N+2) + H = 42

2N + 2 + H = 42

2N + H = 40

H = 40 - 2N

//J + H = 30

(N+2) + (40-2N) = 30

42 - N = 30

N = 12

Tahun ini = 2020 (umur Nunung 12)

maka Nunung lahir tahun = 2020 - 12

= 2008
