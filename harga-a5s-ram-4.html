12. sebutkan dan jelaskan 4 struktur RAM. ....


4 Struktur RAM yaitu :

Input Storage

Bagian ini digunakan dalam memuat masukan - masukan yang diterima dari input devices. Masukan / input sendiri dapat berupa perintah suatu  program maupun hasil pengolahan data.

Program StorageWorking Storage

Program dan working storage sama - sama menyimpan masukan data akan tetapi perbedaanya terletak pada jenis inputan.

Program storage akan menyimpan masukan yang berupa perintah suatu program ,sedangkan working storage akan menyimpan yang berupa data dan hasil olahannya

Output Storage

Bagian ini berfungsi untuk memuat hasil olahan data hingga final untuk dimunculkan pada perangkat output.

Pembahasan

RAM / Random Access Memory adalah media penyimpanan primer yang ada dan karenanya, RAM sendiri terhubung langsung media processor dan tentunya terlibat dalam pemrosesan  data sehingga sangat berpengaruh terhadap kecepatan akses suatu device. Salah satu kekurangan RAM sebagai salah satu media penyimpanan primer adalah sangat mudah kehilangan data jika mendadak terjadi situasi power cut.

RAM sendiri memiliki beberapa jenis diantaranya :

Dynamic Random Access MemorySychronous Dynamic Random Access MemoryDouble Rate Random Access MemoryRambus Dynamic Access MemoryStatic Random Access Memoryxtended Data Out Random Access Memory

13. Apakah kepanjangan dari RAM..?? Dan Apakah itu RAM?


RAM (Random Access Memory)

Merupakan jenis memori yang isinya dapat diganti-ganti selama komputer dihidupkan sebagai suatu penyimpanan data yang dapat dibaca atau ditulis dan dapat dilakukan secara berulang-ulang dengan data yang berbeda-beda. Random artinya data yang disimpan pada RAM dapat diakses secara acak. Modul memori RAM yang umum diperdagangkan berkapasitas 128 MB, 256 MB, 512 MB, 1 GB, 2 GB, dan 4 GB

RAM dibagi lagi menjadi dua jenis, yaitu jenis statik dan dinamik.

RAM statik menyimpan satu bit informasi dalam sebuah flip-flop. RAM statik biasanya digunakan untuk aplikasi-aplikasi yang tidak memerlukan kapasitas memori RAM yang besar.RAM dinamik menyimpan satu bit informasi data sebagai muatan. RAM dinamik menggunakan kapasitas gerbang substrat sebuah transistor MOS sebagai sel memori elementer.

21. Seorang pedagang smart phone ingin membeli 25 smart phone untuk persediaan. Ia ingin membeli smartphone RAM 1 Gb dengan Harga Rp. 1.500.000,- per buah dan smartphone RAM 2 Gb dengan harga Rp. 2.000.000,- per buah. Ia merencanakan tidak akan mengeluarkan yang lebih dari Rp. 42.000.000,- keuntungan sebuah smartphone RAM 1 Gb Rp. 500.000,- dan sebuah smartphone RAM 2 Gb Rp. 600.000,- . hitunglah keuntungan maksimum yang diterima pendatang


misalkan : x = ram 1gb, y = ram 2gb
maka F(x,y) = 1.500.000x + 2.000.000y
persamaan : x + y ≤ 25 ...(1)
1.500.000x + 2.000.000y ≤ 42.000.000 ...(2)
fungsi tujuan :
Misal Z-› Z(x,y) = 500.000x + 600.000y ...(3)
_____________
Sketsa (mencari titik potong)

x + y = 25 ...(1)
maka jika x = 0, y = 25 -› (0,25)
jika y = 0, x = 25 -› (25,0)

1.500.000x + 2.000.000y ≤ 42.000.000
disederhanakan menjadi:
3x + 4y = 84
jika x = 0, y = 21 -› (0,21)
jika y = 0, x = 28 -› (28,0)

eliminasi (1) & (2)
x + y = 25 | x3
ex + 4y = 84 | x1

3x + 3y = 75
3x + 4y = 84
____________ -
-y = -9
y = 9
x + y = 25
x + 9 = 25
x = 25 - 9
x = 16
titik potong (16,9)

karena model soal menggunakan kurang dari maka titik potong yang dipakai menggunakan yang mendekati (0,0). x terdekat adalah (25,0),y terdekat adalah (0,21) dan terakhir menggunakan titik potong eliminasi (16,9)
__________
uji titik:
1. (25,0) -› 500.000x25 + 600.000x0 = 12.500.000
2. (0,21) -› 500.000x0 + 600.000x21 = 12.600.000
3. (16,9) -› 500.000x16 + 600.000x9 = 13.400.000

keuntungan maksimal yang dapat diperoleh adalah 13.400.000

28. seorang pedagang smartphone membeli 25 smartphone untuk persediaan. ia ingin membeli smartphone RAM 1 Gb dengan harga Rp. 1.500.000,- per buah dan smartphone RAM 2 Gb dengan harga Rp. 2.000.000,- per buah. ia merencanakan tidak akan mengeluarkan uang lebih dari Rp. 42.000.000,- . keuntungan sebuah smartphone RAM 1 Gb Rp. 500.000,- dan sebuah amartphone RAM 2 Gb Rp. 600.000,-. hitunglah keuntungan maksimum yan diterima pedagang


Misal , jumlah hp 1 gb = a, dan jumlah hp 2 gb = b

Dari data di atas dapat didapatkan 2 persamaan yaitu ... 
(1) 15a + 20 b = 420
(b) a +b = 25 
kemudian mengeliminasi a dengan cara 
- persamaan 1 di semua ruas dikalikan 1
- persamaan 2 semua ruas dikalikan 15
sehingga akan terbentuk persamaan 
(1) 15a + 20b = 420
(2) 15a + 15b + 375 kemudian pers 1 dikurangankan pers 2
Hasilnya yaitu 5b = 45, maka b= 9. 
a = 25-b
a = 25-9 = 16 

Maka keuntungan yang akan didapatkan yaitu 
= untung dr 16 hp 1 gb +untung dr 9 hp 2 gb 
= (16 x 500000) + (9 x 600000)
= 8.000.000 + 5.400.000
= 13 400 000 
TOTAL KEUTUNGANNYA ADALAH Rp.13.400.000,00
