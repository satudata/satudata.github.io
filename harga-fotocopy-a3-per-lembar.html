Jika lembar fotocopy lebih dari 100 lembar baik member atau tidak maka dapat diskon lagi 1.5% 3. Jika lembar fotocopy antara 200-300 lembar maka baik member atau tidak akan mendapatkan diskon 2.5% 4. Harga perlembar ditentukan jika hitam putih :150 rupiah perlembar jika warna : 500 rupiah perlembar


Jawaban:

public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       

       String nama, member, jenis;

       int lembar;

       double diskon = 0, total = 0;

       

       System.out.print("Masukan Nama  : ");

       nama = input.nextLine();

       

       System.out.print("Member (Y/N)  : ");

       member = input.next();

       

       System.out.print("Jumlah Lembar : ");

       lembar = input.nextInt();

       

       System.out.println("Jenis Fotokopi");

       System.out.println("a. Hitam Putih");

       System.out.println("b. Berwarna");

       System.out.print("Pilihan anda : ");

       jenis = input.next();

       

       if (jenis.toUpperCase().equals("A")) {

           total = lembar * 150;

       } else {

           total = lembar * 500;

       }

       

       if (member.toUpperCase().equals("Y")) {

           diskon = diskon + total * 0.5 / 100;

       }

       

       if (lembar > 100) {

           diskon = diskon + total * 1.5 / 100;

       }

       

       if (lembar >= 200 && lembar <= 300) {

           diskon = diskon + total * 2.5 / 100;

       }

       

       total = total - diskon;

       

       System.out.println("");

       System.out.println("Hasil Perhitungan");

       System.out.println("=================");

       System.out.println("Diskon : Rp" + diskon);

       System.out.println("Total  : Rp" + total);

   }

210 ÷ 3 = 70 lembar

Karena dalam 1 menit dapat mencetak 70 lembar, dalam waktu 5 menit dapat mencetak :

70 × 5 = 350 lembar

Jadi, banyak lembar yang dapat dicetak oleh mesin fotocopy itu dalam 5 menit adalah 350 lembar
