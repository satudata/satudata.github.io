1. cara buat windows dengan aplikasi RUFUS


Hai.. Selamat pagi,

Maaf maksud dari pertanyaan anda itu Cara install Windows dengan Rufus?

Langkah - langkah untuk menginstall Windows dengan Rufus :

1. Pertama, silakan download softwarenya

2. Masukkan flashdisk anda kemudian buka aplikasi Rufus. Device anda secara otomatis akan terdeteksi

3. Klik gambar CD drive untuk memilih lokasi file ISO windowsnya.  Kemudian klik Open.

4. Secara otomatis Rufus akan merubah file system dari  FAT32 menjadi NTFS. Anda juga bisa merubah nama drive tersebut pada New volume label (opsional). Kemudian klik Start.

5. Akan muncul notifikasi bahwa semua data yang terdapat pada drive tersebut akan diformat, jadi pastikan bahwa anda sudah membackup semua data yang penting. Selanjutnya klik OK.

6. Proses ini memakan waktu kira-kira 20 menit. Aplikasi ini sangatlah cepat, tetapi semua itu tergantung berapa besar ukuran sistem operasi tersebut dan kecepatan flashdisk yang digunakan.

7. Selesai! Flashdisk anda sekarang sudah bisa digunakan untuk media instalasi windows. Kemudian klik Close.

2. Cara membuat multi installer windows 10 di flashdisk rufus


Jawaban:

Cara Memasukkan Beberapa Installer ke Flashdisk (Multiboot USB) Beranda Windows Cara Memasukkan Beberapa Installer ke Flashdisk (Multiboot USB)

Cara membuat multiboot usb installer pada Flashdisk atau Hardisk

Pembuatan sebuah bootable Flashdisk installer merupakan salah satu langkah penting jika hendak melakukan install ulang sebuah sistem operasi. Ketimbang menggunakan CD / DVD untuk install OS, maka Flashdisk akan membuat proses install menjadi semakin cepat. Selain itu, sangat mudah memasukkan sebuah file ISO sistem operasi ke dalam Flashdisk agar menjadi USB Bootable. Anda bisa memasukkan OS apa saja, seperti Windows, Linux, dll.

Namun, yang menjadi masalah adalah ketika anda hanya bisa memasukkan satu file ISO ke dalam satu Flashdisk. Artinya, satu Flashdisk hanya bisa anda jadikan sebagai satu installer OS saja. Dengan demikian ketika anda hendak menginstall OS lain, maka harus memasukkan ulang file ISO OS yang lain dan menghapus yang lama. Hal ini tentu saja akan sangat merepotkan, terlebih lagi jika kapasitas Flashdisk / Hardisk anda sebenarnya bisa menampung beberapa file ISO sistem operasi sekaligus.

Ada banyak jenis sistem operasi yang bisa anda install pada setiap perangkat Komputer atau Laptop, dan semuanya memiliki beberapa versi dan fitur unggulan tersendiri. Sedangkan, tidak mungkin anda harus menyedikan Flashdisk sebanyak jumlah sistem operasi yang akan anda buatkan installer. Akan lebih efektif jika anda memasukkan semua file ISO sistem operasi tersebut ke dalam satu Flashdisk atau membuat sebuah Multiboot USB. Jadi, anda bisa hemat dana karena tidak perlu beli banyak Flashdisk dan saat ingin install OS mudah menemukannya karena tersimpan dalam satu Flashdisk.

4. sebutkan kelemahan dan kelebijan sistem operasi pada : windows 95 windows 98 windows xp windows vista windows 7 windows 8


kelemahan:
Sistem Operasi Windows 3.1.1
Kelebihan dari sistem operasi ini adalah :
a. Memudahkan komunikasi antar anggota workgroup lainnya
b. Menggunakan sharing printer
c. Memudahkan mengakses file dan program aplikasi yang berada di komputer lain.
d. Administrasi sistem jaringan (network) relative lebih mudah

Sistem Operasi Windows 95
Keunggulan dari sistem operasi ini adalah :
a. Sangat mudah dan cepat :
· Setiap saat kita dapat menggunakan seluruh kemampuan Windows 95, yaitu dengan mengklik tombol start untuk menjalankan program aplikasi, membuka dokumen, mencari data dan menggunakan fasilitas lainnya.
· Pada taskbar Windows 95 ditampilkan seluruh program aplikasi yang sedang anda gunakan, hal ini akan memudahkan untuk berpindah program aplikasi.
b. Berbagai fasilitas baru
1. Dengan sistem 32-bit, Kemampuan pengkodeaan 32-bit dari prosesor seri 80386, 80486, dan 80586 dapat dimanfaatkan secara optimal.
2. Dilengkapi fasilitas multithreading dan preemptive multitasking : multitasking lebih responsive dan dengan fasilitas ini dapat menjalankan beberapa program aplikasi secara bersama-sama.
3. Kemampuan memberi nama yang panjang pada file dan folder (maksimal 250 karakter).
4. Lalu lintas input/output yang lebih cepat yang membuat operasi baca tulis, dan mencetak ke printer menjadi lebih ditingkatkan kecepatannya.
5. Visual device driver 32-bit yang mengelola sumber daya sistem sehingga memungkinkan lebih dari satu aplikasi dapat mengunakan daya pada saat yang bersamaan.
Securiti secara terpusat : Sistem pengamanan dapat dilakukan secara terpusat pada server.
c. Fasilitas Bantuan (Help) yang terintegrasi Fasilitas Help apabila pemakai sedang menghadapi kesulitan dalam pengoprasian, program ini lebih luas.

Sistem Operasi Windows 98
Keunggulan dari sistem operasi ini dalah :
a. Lebih mudah digunakan membuka file, serta banyak tambahan tombol jelajah pada tiap jendela. meningkatkan ukuran bidang kerja. Dan instalasi perangkat keras baru akan lebih mudah karena Windows 98 mendukung Standar Universal serial Bus (USB). Standar ini memungkinkan kita menancapkan perangkat keras baru dan langsung menggunakannya tanpa perlu melakukan restart computer.
b. Lebih terjamin : memiliki akses ke Windows Update, ekstensi berbasis web untuk Windows 98. Fasilitas pemecahan masalah yang baru dilengkapi perangkat
c. Lebih cepat : windows dan program yang ada didalamnya dapat dijalankan lebih cepat daripada sebelumnya. Memanfaatkan Windows Tune-Up Wizard, kita dapat secara mudah meningkatkan kecepatan dan efisiensi komputer. Begitu juga dengan fasilitas Power Management dan kita pun dapat menggunakan FAT 32 untuk meningkatkan efisiensi penyimpanan file dan menghemat pemakaian ruang hard disk.
d. Lebih menghibur : windows 98 dapat mendukung DVD dan digital audio, jadi kita dapat memutar film dan audio digital berkualitas sangat tinggi pada komputer kita dan didukung oleh paket wallpaper, screen saver, icon 3-D, pointer mouse yang unik

12. Apakah fungsi Windows XP,Windows Me,dan Windows Vista????


Windows XP, Windows Me, dan Windows Vista adalah suatu OS (Operating System) yg terdapat di dalam sebuah komputer. OS berguna sebagai suatu kesatuan sistem yg terdiri dari berbagai proses dimana terdapat input-proses-output dalam sebuah komputer. Sebuah komputer tidak akan dapat difungsikan jika tidak terdapat OS terinstal didalamnya.

13. Jaringan workgrroup standard windows adalah


Jaringan Workgroup

Jaringan ini terdiri dari beberapa unit komputer yang dihubungkan dengan menggunakan Network Interface Card atau yang biasa disebut dengan Local Area Network Card, serta dengan menggunakan kabel BNC maupun UTP. Semua unit komputer yang terhubung dapat mengakses data dari unit komputer lainnya dan juga dapat melakukan print document pada printer yang terhubung dengan unit komputer lainnya.

Keuntungan Jaringan Workgroup.

Pertukaran file dapat dilakukan dengan mudah (File Sharing).Pemakaian printer dapat dilakukan oleh semua unit komputer (Printer Sharing).Akses data dari/ke unit komputer lain dapat di batasi dengan tingkat sekuritas pada password yang diberikan.Komunikasi antar karyawan dapat dilakukan dengan menggunakan E-Mail & Chat.Bila salah satu unit komputer terhubung dengan modem, maka semua atau sebagian unit komputer pada jaringan ini dapat mengakses ke jaringanInternet atau mengirimkan fax melalui 1 modem.

KEKURANGAN WINDOWS 95

Tampilannya kurang menarik dan masih terlalu kaku.

Kestabilan sistem kurang  

Windows 98 lebih boros memori dan ruang harddisk daripada Windows 95

Sering terjadinya blue screen

windows 98

Kekurangan :

* Dukungan support sudah dicabut oleh produsennya (microsoft)

* Susah cari driver, karena produsen hardware menarik dukungan untuk win 98

* Tidak mendukung dan tidak bisa diinstall di perangkat terbaru (bila dipaksakan akan blue screen)

* Tidak ada versi 64bit

* Ringkih, mudah crash

* Kemampuan multi tasking yang sangat buruk

* Kemampuan multi thread yang sangat buruk

* Penanganan device yang buruk, DMA sering konflik

* Manajemen resource cpu yang kacau balau

* Tidak bisa membuat file dg ukuran lebih dari 2GB

* Tidak bisa mendeteksi memory besar (1GB masih stabil, dipaksa 2GB-hasil BlueScreen)

* Utility sangat terbatas, apalagi tool network

* Gampang terserang virus, gampang di hack, di crack, gampang di acak-acak

* Terlalu beresiko melakukan transaksi perbankan dari browser win 98, karena fitur networknya minim

* Tidak mendukung applikasi keluaran terbaru

* Tampilan terlalu sederhana (meskipun sempat dibanggakan di era nya)

Kelebihan :

* Cocok untuk hardware lama (misalnya pentium I,II,III)

* Ringan

* Ukuran installer kecil, bisa diinstal dari disket

* Bisa diinstall dari dos

* Bisa booting ke dos

* Cocok untuk development applikasi under dos terutama applikasi recovery

* Cocok untuk main game tempoe doeloe

* System (boot) mudah ditransfer ke media lain seperti cd dan disket

* Dos protected mode (DPMI) yang sempurna, baik untuk applikasi DOS32 dan DOS16

* Sangat cocok untuk development bagi programmer BIOS, atau programmer bahasa mesin

1. Biaya Operasi relatif mahal.

3. Jika tidak ada server Windows 2000 Server client tidak bisa mangoneksikan ke server begitupun sebaliknnya.
