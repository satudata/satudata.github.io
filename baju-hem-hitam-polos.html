1. harga dua baju hem dan satu oblong 170.00 sedangkan harga satu baju hem dan tiga baju oblong RP 185.00, harga tiga baju hem dan dua baju oblong adalah ?​


Jawab:

Harga 3 baju hem dan 2 oblong baju adlah Rp 275.000

Penjelasan dengan langkah-langkah:

misalkan :
1 baju hem = x
1 baju oblong = y

maka,
2x + y = Rp 170.000.....Pers 1
x + 3y = Rp 185.000.....Pers 2

gunakan rumus eliminasi dan substitusi
2x + y = Rp 170.000 ║×3║
x + 3y = Rp 185.000 ║×1║

6x + 3y = Rp 510.000
 x + 3y = Rp 185.000
--------------------------------- (-)
5x = Rp 325.000
x = [tex]\frac{Rp 325.000}{5}[/tex]
x = Rp 65.000

substitusi x ke persamaan 2
x + 3y = Rp 185.000
Rp 65.000 + 3y = Rp 185.000
3y = Rp 185.000 - Rp 65.000
3y = Rp 120.000
y =  [tex]\frac{Rp 120.000}{3}[/tex]
y = Rp 40.000

jadi,
3x + 2y

3(Rp 65.000) + 2(Rp 40.000)
Rp 195.000 + Rp 80.000
= Rp 275.000

17. Sebuah butik memiliki 8m kain polos dan 10m kain bercorak. Dari bahan tersebut akan dibuat dua buah model baju . Baju model I memerlukan 2m kain polos dan 1m kain bercorak, sedangkan baju model II memerlukan 1m kain polos dan 2m kain bercorak. Jika harga jual baju model I sebesar Rp350.000,00 dan baju model II sebesar Rp200.000,00, hasil penjualan maksimum butik tersebut adalah …


kelas : X SMA
mapel : matematika
kategori : pertidaksamaan linier dua variabel
kata kunci : soal cerita

kode : 10.2.4 [matematika SMA kelas X bab 4 pertidaksamaan linier dua variabel]

Pembahasan:

untuk menyelesaikan soal seperti ini kita buat terlebih dahulu kalimat matematikanya, dan kita tentukan nilai untuk variabel x dan nilai untuk variabel y.
pertidaksamaan linier dua variabel adalah kalimat terbuka yang menggunakan 2 variabel, dan masing-masing variabel berderajat 1, serta menggunakan tanda penghubung <, >, ≤, ≥.
bentuk pertidaksamaan linier dapat ditulis:
ax + by ≤ c
ax + by ≥ c
ax + by < c
ax + by > c
dan hasilnya berbeda dibandingkan dengan persamaan, jika dalam persamaan hasilnya berupa himpunan pasangan titik-titik, namun jika pertidaksamaan linier dua variabel hasilnya berupa daerah arsiran

dalam soal, kita buat pertidaksamaannya terlebih dahulu:
baju model 1 = x
baju model 2 = y

                                polos    corak    harga jual
baju model 1 = x        2           1           350.000
baju model 2 = y       1            2          200.000
    total                       8           10
dari bagan ini dapat kita buat pertidak

2x + y ≤ 8
x + 2y ≤ 10
z = 350.000x + 200.000y
ditanya hasil penjualan maksimum dari penjualan kedua model baju = ...?

jawab:
kita buat nol fungsi terlebih dahulu
2x + y ≤ 8
memotong sumbu x di titik y = 0
2x + y = 8
2x + 0 = 8
2x = 8
x = 8/2
x = 4
memotong sumbu y di titik x = 0
2x + y = 8
0 + y = 8
y = 8

x + 2y ≤ 10
memotong sumbu x apabila y = 0
x + 2y = 10
x + 0 = 10
x = 10
memotong sumbu y apabila x = 0
x + 2y = 10
0 + 2y = 10
y = 10/2
y = 5

langkah selanjutnya kita eliminasi
2x + y = 8   I*1I   2x + y = 8
x + 2y = 10 I*2I  2x + 4y = 20
                          --------------------- -
                                -3y         = -12
                                    y         = -12/-3
                                    y         = 4
x + 2y = 10
x + 2(4)  = 10
x + 8 = 10
x = 10 - 8
x = 2

kita tentukan titik uji yang memenuhi
kita dapat 3 titik uji, perhatikan gambar pada lampiran
1) (0, 5)
2) (2, 4)
3) (4, 0)
subsitusikan titik uji pada nilai z = 350.000x + 200.000y
1) titik uji pertama (0, 5)
    z = 350.000x + 200.000y
      = 350.000(0) + 200.000(5)
      = 0 + 10.000.000
      = 1.000.000
2) titik uji kedua (2, 4)
    z = 350.000x + 200.000y
       = 350.000(2) + 200.000(4)
       = 700.000 + 800.000
      = 1.500.000
3) titik uji ketiga (4, 0)
    z = 350.000x + 200.000y
      = 350.000(4) + 200.000(0)
      = 1.400.000 + 0
      = 1.400.000

hasil penjualan maksimum dari penjualan 2 model baju yaitu 1.500.000 dengan menjual 2 baju model 1  dan 4 baju model 2

untuk lebih jelas cara menentukan titik uji, perhatikan lampiran

27. Sebuah butik memiliki 8m kain polos dan 10m kain bercorak. Dari bahan tersebut akan dibuat dua buah model baju . Baju model I memerlukan 2m kain polos dan 1m kain bercorak, sedangkan baju model II memerlukan 1m kain polos dan 2m kain bercorak. Jika harga jual baju model I sebesar Rp350.000,00 dan baju model II sebesar Rp200.000,00, hasil penjualan maksimum butik tersebut adalah …


kelas : X SMA
mapel : matematika
kategori : pertidaksamaan linier dua variabel
kata kunci : soal cerita

kode : 10.2.4 [matematika SMA kelas X bab 4 pertidaksamaan linier dua variabel]

Pembahasan:

untuk menyelesaikan soal seperti ini kita buat terlebih dahulu kalimat matematikanya, dan kita tentukan nilai untuk variabel x dan nilai untuk variabel y.
pertidaksamaan linier dua variabel adalah kalimat terbuka yang menggunakan 2 variabel, dan masing-masing variabel berderajat 1, serta menggunakan tanda penghubung <, >, ≤, ≥.
bentuk pertidaksamaan linier dapat ditulis:
ax + by ≤ c
ax + by ≥ c
ax + by < c
ax + by > c
dan hasilnya berbeda dibandingkan dengan persamaan, jika dalam persamaan hasilnya berupa himpunan pasangan titik-titik, namun jika pertidaksamaan linier dua variabel hasilnya berupa daerah arsiran

dalam soal, kita buat pertidaksamaannya terlebih dahulu:
baju model 1 = x
baju model 2 = y

                                polos    corak    harga jual
baju model 1 = x        2           1           350.000
baju model 2 = y       1            2          200.000
    total                       8           10
dari bagan ini dapat kita buat pertidak

2x + y ≤ 8
x + 2y ≤ 10
z = 350.000x + 200.000y
ditanya hasil penjualan maksimum dari penjualan kedua model baju = ...?

jawab:
kita buat nol fungsi terlebih dahulu
2x + y ≤ 8
memotong sumbu x di titik y = 0
2x + y = 8
2x + 0 = 8
2x = 8
x = 8/2
x = 4
memotong sumbu y di titik x = 0
2x + y = 8
0 + y = 8
y = 8

x + 2y ≤ 10
memotong sumbu x apabila y = 0
x + 2y = 10
x + 0 = 10
x = 10
memotong sumbu y apabila x = 0
x + 2y = 10
0 + 2y = 10
y = 10/2
y = 5

langkah selanjutnya kita eliminasi
2x + y = 8   I*1I   2x + y = 8
x + 2y = 10 I*2I  2x + 4y = 20
                          --------------------- -
                                -3y         = -12
                                    y         = -12/-3
                                    y         = 4
x + 2y = 10
x + 2(4)  = 10
x + 8 = 10
x = 10 - 8
x = 2

kita tentukan titik uji yang memenuhi
kita dapat 3 titik uji, perhatikan gambar pada lampiran
1) (0, 5)
2) (2, 4)
3) (4, 0)
subsitusikan titik uji pada nilai z = 350.000x + 200.000y
1) titik uji pertama (0, 5)
    z = 350.000x + 200.000y
      = 350.000(0) + 200.000(5)
      = 0 + 10.000.000
      = 1.000.000
2) titik uji kedua (2, 4)
    z = 350.000x + 200.000y
       = 350.000(2) + 200.000(4)
       = 700.000 + 800.000
      = 1.500.000
3) titik uji ketiga (4, 0)
    z = 350.000x + 200.000y
      = 350.000(4) + 200.000(0)
      = 1.400.000 + 0
      = 1.400.000

hasil penjualan maksimum dari penjualan 2 model baju yaitu 1.500.000 dengan menjual 2 baju model 1  dan 4 baju model 2

untuk lebih jelas cara menentukan titik uji, perhatikan lampiran
