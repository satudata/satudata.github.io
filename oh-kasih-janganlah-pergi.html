Dokter memegang perban Via dan setelah perbannya dibuka kepala Via terasa ringan.

Dokter    : Bukalah matamu dengan perlahan-lahan.

Kemudian kedua mata Via di buka perlahan sehingga membuat jantung Via berdebar kencang, aku melihat seberkas cahaya dan semakin lama semakin jelas pandanganku.

Ada seorang wanita cantik berwajah keibuan Oh Tuhan apakah dia mamaku batin Via?

Via       : Mama, apakah ini mama?

Mama  : Oh Tuhan, kau bisa melihat. Iya ini mama nak.

Mama memeluk Via sambil menangis.

Via       : Terima kasih dokter, ucap Via.

Dokter : Via, berterima kasihlah kepada Tuhan YME karena Tuhanlah yang telah mengembalikan penglihatanmu ini.

Setelah memberi nasehat dokter bergegas pergi.
