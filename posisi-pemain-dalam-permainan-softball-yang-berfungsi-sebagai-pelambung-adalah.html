11. apa saja posisi pemain bertahan dalam permainan softball


Pada pertandingan softbal, pemain bertahan dibagi menjadi dua, yaitu
1. Posisi bertahan di dalam lapangan (infield)
Pemain yang bertahan di dalam lapangan ini terdiri dari 
- Pitcher 
- Catcher
- First base man (penjaga base satu)
- Second base man (penjaga base dua)
- Third base man (penjaga base tiga)
- Short stop
2. Posisi bertahan di luar lapangan (outfield), terdiri dari
- Left fielder : penjaga daerah garis batas base ketiga hingga mendekati base kedua.
- Right fielder : penjaga garis batas base pertama sampai base kedua.
- Centre fielder : penjaga bagian tengah dari garis batas tengah luar lapangan.

20. pemain nomor satu dalam permainan softball berposisi sebagai...


PERMAINAN SOFTBALL

a. Posisi penjagaan lapangan dalam ( infield)
1). Posisi first baseman ( Penjaga Base Satu )
Posisi ini berada di tengah antara home plate dan base pertama, serta di antara base pertama dan base kedua.tugas - tugasnya adalah sebagai berikut:
Menjaga dan menangkap arah bola yang di pukul atau di lempar ke arah base pertama
Membuat mati terpaksa dengan menyentuh pertama atau melempar bola kepada pemain lain yang menjaga base pertama
melempar bola pada base kedua untuk membuat mati terpaksa
memotong pada home plate
meneruskan lemparan dari lapangan sebelah kanan
membantu menjaga belakang pada base kedua
2). Posisi second baseman ( Penjaga base kedua )
Posisi base kedua berada di luar base dan harus menjaga base kedua. Oleh sebab itu second base tidak melakukan lemparan jauh.tugas - tugasnya adalah sebagai berikut :
Menjaga pukulan bola pada daerah yang harus di kuasai
Melempar bola ke arah base kedua atau base pertama untuk membuat mati terpakasa
Mengetik pelari yang bergerak dari base pertama menuju base kedua
Membuat mati terpaksa dan mengetik pelari pada base kedua
Menjaga base pertama jika pukulan bunt pada saat first baseman menjaga atau menuggu bola
membantu menjaga daerahnya belakang base dua, jika short stop menjaga base kedua
3.) Posisi Short Stop
Posisi terbaik short stop berada mulai kira - kira di tengah antara base kedua dan base ketiga. Tugas - tugasnya adalah sebagi berikut.
Menjaga semua bola hasil pukulan di daerah yang di kuasainya
Melempar bola ke arah base satu dan base dua untuk membuat pelari mati terpaksa
Memotong lemparan dari outfield
Mnejaga base kedua pada saat pukulan bunt ke arah base kedua
4.) Posisi third baseman
Third baseman menjaga daerah infield, kira- kira di tengah garis batas tiga sampai di belakang posisi short stop dan daerah bagian dalam left fielder. Tugas - tugasnya adalah sebagai berikut :
Melempar bola ke arah base satu dan base dua untuk membuat pelari mati terpaksa
Menjaga base ketiga
Memotong lemparan dari outfield
Menguasai pukulan bunt ke arah base ketiga di bagian samping infield
Menjaga belakang short stop
5.) Posisi Pitcher
Selain bertugas melempar bola ke arah pemukul, tugas lainnya adalah sebagai berikut:
Membantu pemain di belakang home plate
Menjaga base pertama ketika penjaga base pertama mengambil bola
membantu menjaga belakang base kedua dan ketiga, Jika ada lemparan dari outfield
Membantu menjaga belakang lemparan ke arah home plate
6.) Posisi catcher
Tugasnya lebih banyak menangkap bola lemparan dari pitcher. Daerah penjagaannya berada di antara home plat dan back stop. Tugas Lainnya adalah sebagai berikut :
Menjaga Home Plate untuk mematikan pelari dan membuat nilai
Menjaga pukulan bunt di sekitar home plate
melempar bola ke base satu,dua,tiga untuk mematikan lawan
membantu menjaga belakang base pertama
menjaga pukulan bunt

24. Dalam permainan softball, pemain yang bertugas melambungkan bola adalah ....


Dalam permainan softball, pemain yang bertugas melambungkan bola adalah pitcher atau pengepul. Pitcher adalah pemain yang bertugas melempar bola ke arah pemukul, dengan tujuan untuk menghambat pemukul dari memukul bola dengan baik atau mengecoh pemukul agar salah mengantisipasi arah dan kecepatan bola.

Pitcher harus memiliki kemampuan melempar bola dengan tepat sasaran dan menggunakan berbagai jenis pukulan, seperti pukulan lurus, pukulan curve, pukulan screw, dan pukulan slider, untuk mengimbangi strategi dan gaya bermain pemukul. Selain itu, pitcher juga harus memiliki kemampuan fisik yang kuat dan tangkas, karena posisi pitcher adalah posisi yang paling sering terpapar bola yang dipukul oleh pemukul.
