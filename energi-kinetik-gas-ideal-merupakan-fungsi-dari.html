12. Tentukan energi kinetik dan energi dalam 0,5 mol gas ideal pada suhu 1227ºc


KELAS            = 12
PELAJARAN   = FISIKA
KATEGORI      = TEORI KINETIK GAS
KATA KUNCI   = gas monoatomik, gas diatomik, energi kinetik rata-rata

Diketahui :
n = 0,5 mol
T = 1227 C = 1227 + 273 = 1.500 K = 1,5x10³ K
Ditanyakan : Energi kinetik rata-rata?

Dijawab :
k = 1,38 x 10⁻²³ 

Ada 2 jenis keadaan :
a. Gas monoatomik
Ek = 3/2 kT

     = 3/2 x 1,38 x 10⁻²³  x 1,5x10³

     = 1,63x10⁻²⁰ J

b. Gas diatomik
Ek = 3/2 kT + kT

     = 5/2 kT

     = 5/2 x 1,38 x 10⁻²³  x 1,5x10³

     = 5,175x10⁻²⁰ J

20. gas ideal bersuhu 37°c menunyai energi kinetik Ek.untuk mengubah energi kinetik menjadi 2Ek,maka suhu gas harus dijadikan


Diket:
T1 = 37'c = 310 K
Ek2 = 2Ek

Ditanya: T2....?

jawab:
Ek1 / Ek2 = 3/2kT1 / 3/2kT2
Ek / Ek2 = T1 / T2
Ek / 2Ek = 310 / T2
1/2 = 310/T2
T2 = 310 x 2
T2 = 620 K

atau T2 = 620 - 273 = 347'c
