4. Bagaimana cara membuat grafik menggunakan excel dalam sebuah data tabel?


Cara untuk membuat grafik pada microsoft excel adalah dengan melakukan seleksi pada sebuah tabel yang berisi data yang akan kita buat menjadi grafik. Setelah data terblok/ terseleksi dengan benar langkah selanjutnya adalah memilih menu Insert, kemudian pilih charts. Pilih chart atau grafik sesuai dengan keinginan kita.

Pembahasan

Pembuatan grafik merupakan salah satu fitur dari microsoft excel. Dengan memakai grafik, penyajian data akan semakin mudah dan lebih nyaman untuk dibaca. Pada microsoft excel terdapat beberapa macam jenis grafik yang bisa kita gunakan. Jenis-jenis grafik pada microsoft excel adalah sebagai berikut

Column ChartPie ChartLine ChartBar ChartArea Chart Scatter Chart Stock ChartsDoughnut charts Surface Chart Radar Chart (Spider Chart)Buble Chart

Setelah kita mengetahui jenis-jenis grafik yang ada pada microsoft excel, maka kita bisa memilihnya untuk kebutuhan penyajian data kita. Penggunaan grafik yang tepat akan mempermudah kita dalam menyajikan data.

13. Cara menjumlahkan data yang sama di excel


Untuk melakukan penjumlahan dalam Microsoft Excel dapat menggunakan rumus SUM. Rumus SUM ini memiliki fungsi untuk penjumlahan. Cara menggunakan rumus SUM ini sangat mudah, yaitu dengan =SUM(data 1: data 2). Jika masing-masing angka berada di cell A1, B1 dan C1 jadi rumus yang digunakan adalah =SUM(A1:C1)

26. Cara memilter data di microsoft excel


Cara Filter data di excel

Menu filter dapat anda akses pada Tab Data dan dikelompokan kedalam group Sort and filter seperti pada gambar siatas. Semua tipe data (text, angka, tanggal, jam) pada excel dapat dilakukan pemfilteran data, hanya saja kondisi untuk masing-masing tipe data akan berbeda. pada menu filter terdapat 3 icon yaitu :

Clear : Digunakan untuk mengembalikan data seperti semula setelah data terfilter.
Reapply : Digunakan untuk memfilter ulang ketika terjadi perubahan data.
Advance filter : Digunakan untuk memfilter data dengan kriteria yang kompleks.
Langkah-langkah untuk memfilter data sebagai berikut.

Pastikan data anda mempunyai Judul kolom atau header.
Letakan kursor (pointer) pada area data.
Masuk ke tab data pilih menu Filter atau dapat anda gunakan shortcut CTRL+SHIFT+L
Pastikan Setiap judul kolom atau header terdapat dropdown filter.

cara filter data di excel

Sampai disini langkah awal untuk pemfilteran data sudah selesai. selanjutnya kita dapat memfilter data untuk satu kolom atau multi kolom berdasarkan kondisi tertentu.
