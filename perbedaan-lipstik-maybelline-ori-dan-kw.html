10. Cahce Memori vs DDR Memori bedanya apa?​


1.Memory cache sering disebut sebagai cache RAM. Ini adalah bagian dari memori yang dibuat dengan RAM statis kecepatan tinggi. Jenis cache ini lebih efektif karena hampir semua program dapat berulang kali mengakses data atau perintah yang sama. Ini berarti bahwa semakin banyak informasi yang disimpan dalam, semakin sering komputer menggunakan cache ini daripada menggunakan RAM yang relatif lebih lambat.Cache memori memiliki tiga level. Anda pasti akan menemukan label L1 cache, L2 cache atau L3 cache dalam kemasan perangkat komputer Anda atau dalam informasi teknis. L1 adalah nama untuk cache internal, itu berada di area antara CPU dan RAM. Cache ini memiliki kecepatan akses maksimum. Ukuran memori berkisar dari 8 KB, 64 KB dan 128 KB.

Sedangkan L2 adalah cache eksternal dengan kapasitas lebih besar dari 256 KB hingga 2 MB. Dalam hal kecepatan L2 sebenarnya lebih lambat dari L1. Terakhir, cache L3 biasanya digunakan dalam model komputer baru yang memiliki lebih dari satu prosesor, misalnya Dual Core atau Quad Core. L3 bertindak sebagai pengontrol data yang diakses oleh cache L2 dan oleh masing-masing inti prosesor.

21. Apa perbedaan flashdisk dan memori


Media penyimpanan data digital yang sering digunakan adalah memori dan flashdisk. Flashdisk media penyimpanan data digital berupa USB kecil memiliki kapasitas penyimpanan rendah. Sedangkan memori adalah media penyimpanan data digital berupa chip dengan kapasitas penyimpanan data yang besar dengan maksimum 1 TB.

Pembahasan:

Data adalah sekumpulan informasi atau keterangan-keterangan dari suatu hal yang diperoleh dari hasil pengukuran atau pengamatan suatu variabel dari sumber tertentu. Kegunaan data adalah mempermudah orang untuk mengolah sesuatu agar menjadi sumber informasi yang terpercaya. Bentuk data dapat berupa kata-kata ataupun angka. Untuk menjaga data agar tidak hilang ketika dibuka dan diproses kembali di waktu yang akan datang kita dapat melakukan penyimpanan. Di era globalisasi ini penyimpanan data dilakukan secara berbentu digital. Media penyimpanan terdiri dari tiga kategori yaitu:

Media penyimpanan magnetikMedia penyimpanan opticalMedia penyimpanan awan

Media penyimpanan data terlebih untuk data digital yang kita olah di komputer, dapat disimpan dalam dua media, yaitu memori dan flashdisk.

Memori

Media penyimpanan data memori dapat menyimpan data dengan kapasitas maksimum 1 TB atau 1.000 GB. Wujud fisik memori berupa chip kecil.

Flashdisk

Media penyimpanan data flashdisk dapat menyimpan data dengan kapasitas yang lebih kecil dari memori , dengan kata lain flashdisk hanya mampu menyimpan file-file yang kecil. Flashdisk berupa benda kecil seperti USB yang dapat dibawa kemana-mana.

25. apa perbedaan memori secara fisik dan memori komputer secara virtual


Memori fisik merupakan istilah generik yang merujuk pada media penyimpanan data sementara pada komputer. Setiap program dan data yang sedang diproses oleh prosesor akan disimpan di dalam memori fisik. Data yang disimpan dalam memori fisik bersifat sementara, karena data yang disimpan di dalamnya akan tersimpan selama komputer tersebut masih dialiri daya (dengan kata lain, komputer itu masih hidup). Ketika komputer itu direset atau dimatikan, data yang disimpan dalam memori fisik akan hilang. Oleh karena itulah, sebelum mematikan komputer, semua data yang belum disimpan ke dalam media penyimpanan permanen (umumnya bersifat media penyimpanan permanen berbasis disk, semacam hard disk atau floppy disk), sehingga data tersebut dapat dibuka kembali pada lain waktu.

Memori fisik umumnya diimplementasikan dalam bentuk Random Access Memory (RAM), yang bersifat dinamis (DRAM). Mengapa disebut Random Access, adalah karena akses terhadap lokasi-lokasi di dalamnya dapat dilakukan secara acak (random), bukan secara berurutan (sekuensial). Meskipun demikian, kata random access dalam RAM ini sering menjadi salah kaprah. Sebagai contoh, memori yang hanya dapat dibaca (ROM), juga dapat diakses secara random, tetapi ia dibedakan dengan RAM karena ROM dapat menyimpan data tanpa kebutuhan daya dan tidak dapat ditulisi sewaktu-waktu. Selain itu, hard disk yang juga merupakan salah satu media penyimpanan juga dapat diakses secara random, tapi ia tidak digolongkan ke dalam Random Access Memory.
Memori virtual merupakan suatu teknik yang memisahkan antara memori logis dan memori fisiknya. Menyediakan memori virtual yang sangat besar diperuntukkan untuk programmer bila tersedia memori fisik yang lebih kecil.

Konsep memori virtual yang dikemukakan Fotheringham pada tahun 1961 pada sistem komputer Atlas di Universitas Manchester, Inggris:
“Kecepatan maksimum eksekusi proses di memori virtual dapat sama, tetapi tidak pernah melampaui kecepatan eksekusi proses yang sama di sistem tanpa menggunakan memori virtual.”

Keuntungan Memori Virtual:

Lalu lintas I/O menjadi rendah.

Berkurangnya memori yang dibutuhkan.

Meningkatnya respon.

Bertambahnya jumlah user yang dapat dilayani.

Memori virtual melebihi daya tampung dari memori utama yang tersedia.

Untuk mengimplementasikan memori maya tersebut dapat dilakukan dengan tiga cara:

Sistem Paging.

Sistem Segmentasi.

Sistem kombinasi Paging dan Segmentasi.

28. . Apa perbedaan RAM,ROM,PROM, EPROM,Register dan Cache, sebutkan perbedaan masing2 fungsi dari memori-memori tersebut


Jawaban:

RAM adalah Random Access Memory yang berfungsi untuk menyimpan sementara data yang sedang diproses. RAM berfungsi untuk menyimpan data yang kemungkinan akan sering digunakan oleh processor sehingga data tersebut disimpan semantara di RAM.

ROM adalah Read Only Memory yang berfungsi untuk menyimpan data secara permanen namun datanya tidak bisa diubah, dihilangkan, atau ditambahkan. Contohnya : BIOS pada motherboard.

PROM adalah Programmable Read Only Memory yang berfungsi untuk menyimpan data secara permanen namun datanya tidak bisa diubah, dihilangkan, atau ditambahkan oleh user namun bisa ditambahkan oleh perusahaan yang membuat setelah proses manufaktur.

EEPROM adalah Electrically Eraseable Programmable Read Only Memory yang berfungsi untuk menyimpan data secara permanen dan datanya dapat diubah, dihilangkan, atau ditambahkan oleh pengguna namun ukuran data yang ditambahkan harus kecil. Contoh smart card dan kunci keyless pada kendaraan bermotor.

Register adalah jenis memory yang ditemukan pada processor dimana memori ini berfungsi untuk menyimpan sementara data berupa instruksi perhitungan yang dilakukan oleh processor. Misal register 1 menyimpan perhitungan A+B dan register 2 menyimpan perhitungan C+D dimana register 3 akan menjumlahkan keduanya.

Cache adalah jenis memori yang biasa ditemukan pada processor juga. Cache berfungsi untuk menyimpan sementara data yang akan sering digunakan oleh processor. Semakin besar cache maka data yang dapat disimpan semakin banyak dan proses yang dapat dijalankan akan semakin banyak. Jadi processor akan mengakses cache terlebih dahulu baru mengakses RAM dan baru mengakses main memory (seperti SSD atau HDD) jika ingin mengambil data.
