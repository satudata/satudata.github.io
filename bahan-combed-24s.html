8. Tito berjalan cepat ke timur dan menempuh 50m dalam 10s kemudian berbelok ke utara dan menempuh 120m dalam 24s tentukan : A. perpindahan B. jarak C. kecepatan D. kelajuan


Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Jarak,Perpindahan,Kecepatan,dan Kelajuan.

Pembahasan Soal :
Tito berjalan cepat ke timur dan menempuh 50 m dalam 10 s kemudian berbelok ke utara dan menempuh 120 m dalam 24 s tentukan :
A. perpindahan
B. jarak
C. kecepatan
D. kelajuan

Pembahasan Jawaban :
a.) Rumus mencari Perpindahan 
Yaitu :
Perpindahan = √50² + √120²
Perpindahan = √2.500 + √14.400
Perpindahan = √16.900
Perpindahan = 130 meter

b.) Rumus mencari Jarak 
Yaitu :
= Jarak 1 + Jarak 2 
= 50 m + 120 m 
= 170 m

c.) Rumus mencari Kecepatan 
Yaitu :
Kecepatan Arah Timur = 50 : 10 = 5 m/s 
Kecepatan Arah Utara = 120 : 24 = 5 m/s
Kecepatan total = √5² + √5² = 5√2 m/s

d.) Rumus mencari Kelajuan 
Yaitu :
Kelajuan = Kecepatan Arah Timur + Kecepatan Arah Utara 
Kelajuan = 5 + 5 
Kelajuan = 10 m/s

= 4xy²+12x²y+4xy

= 4xy (y + 3x + 1)

Bentuk-bentuk aljabar berikut  dapat disederhanakan dalam bentuk perkalian faktor-faktornya menjadi:

A. 8s ( 3s + 1 )

B. 3pq ( 5p - 2 )

C. 4xy ( y + 3x + 1 )

D. ( 5b - 2 )( 9a + 1 )

PEMBAHASANSoal A:

[tex]24s^2 + 8s = 3s(8s) +1(8s) = 8s( 3s + 1 )[/tex]

Soal B:

[tex]15p^2q - 6 pq = 3pq( 5p ) - 3pq ( 2) = 3pq ( 5p - 2 )[/tex]

Soal C:

[tex]4xy^2 - 5xy + 12x^2y + 9xy = xy(4y) - xy(5) + xy(12x) + xy(9)[/tex]

[tex]4xy^2 - 5xy + 12x^2y + 9xy = xy ( 4y - 5 + 12x + 9 )[/tex]

[tex]4xy^2 - 5xy + 12x^2y + 9xy = xy ( 4y + 12x + 4 )[/tex]

[tex]4xy^2 - 5xy + 12x^2y + 9xy = 4xy ( y + 3x + 1 )[/tex]

Soal D:

[tex]6a( 5b-2 ) + (3a+1)( 5b-2 ) = (5b - 2 )(6a + 3a + 1) = (5b - 2)(9a + 1)[/tex]
