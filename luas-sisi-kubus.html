20. jika panjang rusuk kubus 15 cmA. tentukan rumus luas permukaan kubus tersebutLuas sisi alas = Luas sisi atas ....Luas sisi depan = Luas sisi Belakang.... Luas sisi kanan = Luas sisi Kiri...​


Jawab:

Rumus luas permukaan kubus = luas sisi alas + luas sisi atas + luas sisi depan + luas sisi belakang + luas sisi kanan + luas sisi kiri

Kubus terbentuk dari gabungan bangun persegi (luas persegi = s x s)

Ingat:

Luas sisi alas = Luas sisi atas                 => misal sisinya = s1

Luas sisi depan = Luas sisi Belakang    => misal sisinya = s2

Luas sisi kanan = Luas sisi Kiri               => misal sisinya = s3

Jadi luas permukaan kubus = (s1 x s1) + (s1 x s1) + (s2 x s2) + (s2 x s2) + (s3 x s3) + (s3 x s3) = 2(s1 x s1) + 2 (s2 x s2) + 2 (s3 x s3)

karena kubus memiliki panjang rusuk yang sama, maka sisi (s) = s1 = s2 = s3  

Luas permukaan kubus = 2(s1 x s1) + 2 (s2 x s2) + 2 (s3 x s3)

                                       = 2(s x s) + 2(s x s) + 2(s x s)

                                       = 6 (s x s)

                                       = 6 s²
