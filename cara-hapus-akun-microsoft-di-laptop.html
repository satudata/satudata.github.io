10. Bagaimana cara menghapus page di Microsoft Word?


Jawaban:

Misalkan ada 3 halaman (Page) di Ms. Word dan ingin menghapus halaman ketiga, caranya:

- posisikan kursor di halaman 2

- tekan tombol Delete di keyboard secara terus menerus

- bila halaman 3 sudah terhapus, hentikan menekan tombol Delete

Penjelasan tambahan:

Amatilah contohnya lampiran:

- awalnya ada 3 halaman, bila posisi kursor sudah ada di halaman 2, maka keterangan di status bar adalah PAGE 2 OF 3 (ditunjukkan tanda panah merah)

- tekan tombol Delete di keyboard terus menerus hingga tampilan selanjutnya ditunjukkan tanda panah biru

- bila halaman 3 sudah terhapus, maka keterangan di status bar menjadi PAGE 2 OF 2 (ditunjukkan tanda panah hijau)

Cukup blok di halaman yang berlebih, lalu klik tombol "del" atau "delete", maka halaman tersebut secara otomoatis akan terhapus.

Untuk set password, klik ikon akun yang baru terbentuk.

27. bagaimana install microsoft di laptop?


1. Masukkan CD/DVD Master Microsoft Office pada CD/DVD ROM Anda.
2. Kemudian masuklah ke explorer (ctrl-e) dan kliklah Icon CD/DVD ROM Anda, setelah itu klik setup.
3. Setelah meng-klik setup. Proses instalasipun dimulai.
4. Langkah selanjutnya installer akan membawa anda pada software license terms, pada tahap ini pilih I accept the terms of this agreement, setelah itu klik continue.
5. Tahap selanjutnya Anda akan memilih paket instalasi pada Ms Office yang akan di install pada pc / laptop Anda. Pilih Upgrade untuk menginstal semua paket aplikasi Ms Office atau Custumize untuk memilih salah satu dari paket aplikasi pada Ms Office.
6. Tunggu sampai selesai..

28. cara menginstal microsoft window di laptop


Cara menginstal Windows di laptop :
1. Siapkan instalasi Windowsnta, atau download Windows 10 dari situs resminya dan pilih file ISO, yang sudah punya CD langsung ke No 5
2. Download aplikasi rufus sebagai tools burningnya
3. Siapkan flashdiks minimal 5GB
4. Colokkan flashdisk yang ingin dibuat instalasinya, lalu buka riufus, dan pada pengaturan pilih Mode GPT partition lalu pilih file ISO Windows nya, lalu klik start tunggu sampai selesai lalu cabut
5. Masukkan Flashdisk atau CD ke laptop yang akan diinstal
6. Masuk ke BIOS biasa tombol Esc, F2 tergantung laptop
7. Pilih Boot nemlalui flashdisk atau CD yang kita colokkan tadi
8. Klik simpan, lalu komputer restart sendiri
9. Maka tampilan instalasi Windows muncul, pilih bahasa dan negara Indonesia, lalu klik instal
10. Pilih versi Windows yang akan di instal lalu klik next
11. Jika muncul permintaan kode serial, klik tombol i dont have activate license key
12. Pilih partisi dimana Windows akan dj instal, lalu klik instal
13. Tunggu proses instal selesai
14. Lakukan pengaturan nama dan user jika perlu tambahkan password.
15. Selesai
