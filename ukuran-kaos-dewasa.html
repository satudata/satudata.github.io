y = 40

2x + 2y = 140

2x + 2(40) = 140

2x + 80 = 140

20. Sebuah toko pakaian mempunyai kaos dewasa.Harga kaos anak lebih murah dari pada harga kaos dewasa.Selisih harga kedua jenis kaos Rp40.000,00.Jumlah harga 1 kaos anak dan 2 kaos dewasa Rp290.000,00.Jika Pak Andi membeli 1 kaos dewasa,ia harus membayar sebanyak....


Misalkan :
- Kaos Dewasa = a
- Kaos Anak = b

Dari Soal di atas, kita dapat membuat persamaan : 

a - b = 40.000 ......................... (1)
2a + b = 290.000 .................... (2)

(i) Dari Persamaan (1) diperoleh :
     a - b = 40.000 
          a = 40.000 + b ...............(3) 

(ii) Subtitusi Persamaan (3) kedalam Persamaan (2) :
     2a + b = 290.000
     2.(40.000 + b) + b = 290.000
     80.000 + 2b + b = 290.000
     80.000 + 3b = 290.000
     3b = 290.000 - 80.000 
     3b = 210.000
       b = 70.000. (Harga Kaos Anak)

(iii) Untuk b = 70.000 Subtitusi ke Persamaan (1)  :
      a - b = 40.000 
      a - 70.000 = 40.000
      a = 40.000 + 70.000
      a = 110.000 (Harga Kaos Dewasa)

Jadi, Jika pak Andi membeli 1 Kaos Dewasa, ia harus membayar sebanyak 110.000,-
