1. Jelaskan tentang rumus perkalian excel


Untuk menggunakan rumus Perkalian di program Microsoft Excel ada 2 cara, pertama menggunakan operator asterisk (*) dan yang kedua menggunakan formula PRODUCT. Contoh: pada cell A2 masukkan angka 100 dan pada cell B2 masukkan angka 5, kemudian pada cell C2 masukkan rumus = A2*B2 kemudian tekan ENTER hasilnya 500. Sekarang coba rubah rumus pada C2 menjadi =PRODUCT(A2:B2) maka hasilnya akan sama 500

8. tuliskan rumus penjumlahan, pengurangan, perkalian, dan pembagian pada Microsoft Excel​


Rumus penjumlahan dalam Microsoft Excel dapat menggunakan tombol "+" pada keyboard atau menggunakan rumus SUM dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =10+12 atau dengan cell seperti =A1+A2. Di sisi lain, penjumlahan menggunakan rumus SUM dapat dituliskan dengan =SUM(A1:A2).Rumus pengurangan dalam Microsoft Excel dapat menggunakan tombol "-" pada keyboard dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =12-10 atau dengan cell seperti =A1-A2.Rumus perkalian dalam Microsoft Excel dapat menggunakan tombol "*" pada keyboard dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =10*2 atau dengan cell seperti =A1*A2.Rumus pembagian dalam Microsoft Excel dapat menggunakan tombol "/" pada keyboard dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =10/2 atau dengan cell seperti =A1/A2.

Pembahasan

Microsoft Excel merupakan salah satu software penunjang kerja dengan berbagai fungsi yang mempermudah kerja user atau pengguna. Beberapa fungsi dari Microsoft Excel adalah sebagai berikut.

Mampu membuat, menganalisa, meringkas, mengedit, dan mengurutkan berbagai data yang dapat diolah dalam pekerjaan.Mampu mempermudah user untuk menghitung aritmatika dan statistika.Mampu menyelesaikan berbagai persoalan logika atau matematika.Mampu membantu user dalam membuat berbagai grafik dan diagram yang diinginkan.

23. pengertian excel dan rumus rumusnya


excel versi pribadi : tempat pengolahan data baik berupa text dan angka. excel sendiri bisa dibilang sebagai database termudah untuk digunakan.

Rumusan Umum :

Tentang Sum
Sum() = Penjumlahan seluruh data dari sel 1 dan sel ke 2 misal : Sum(A1:A5) artinya menjumlahkan data dari sel A1 sampai A5.

Tentang IF
IF() = Logika percabangan, misal kita mau buat rumus jika cel A1 = 1 maka sel yg kita beri rumus muncul teks "Anda memasukan Angka Satu" tapi selain A1 maka "Bukan Angka Satu"

kode rumus : =IF(A1=1,"Anda memasukan Angka Satu","Bukan Angka Satu")

Tentang Count
Kalau kita mau menghitung banyaknya data yg terisi di cel cukup buat rumus dgn ini (hanya dalam penggunaan angka)

=count(A1:A9)

jika km mengisi angka di antra cel 1 sampai 9 maka akan muncul bnyaknya data yg terisi.

Tentang CountIf
Sama halnya dengan count tetapi bedanya ini menjumlahkan data berdasarkan text.

contoh di cel A1 ditulis "Sapta"
cel A2 ditulis "Dewi"
cel A3 ditulis "Sapta"

jika km menulis rumus di cel lain :
=countif(A1:A3,"Sapta") maka akan muncul angka 2.

Tentang Min dan Max
Jika kamu mau mencari data angka terkecil pada cel kamu bisa memanfaatkan =Min() dan sebaliknya =Max tapi jgn lupa isi parameter cel-nya sperti dicontoh sebelumnya.
