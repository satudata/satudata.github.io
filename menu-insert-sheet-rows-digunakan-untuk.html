Insert Sheet Rows termasauk perintah yg dapat dilakukan melalui tab ribbon Home di menu Insert.

Row atau dalam istilah Indonesianya adalah baris

Row : artinya baris

Column: artinya kolom

Insert Sheet Rows : ditunjukkan tanda panah merah (gambar atas)

Delete Sheet Columns : ditunjukkan tanda panah biru (gambar bawah)
