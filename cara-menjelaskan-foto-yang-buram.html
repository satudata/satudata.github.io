9. Maaf ya fotonya agak buram cuma ini yng bisa aku dapet


Jawaban:

1. Misal = mobil = x

motor = y

3x + 5y = 17000 (×4)

4x + 2y = 18000 (×-3)

------------------------

12x + 20y = 68000

-12x - 6y = -54000

------------------------ +

14y = 14000

y = 1000

Masukkan y ke persamaan 3x + 5y = 17000

3x + 5(1000) = 17000

3x + 5000 = 17000

3x = 17000 - 5000

3x = 12000

x = 4000

Jadi, tarif parkir 1 mobil adalah Rp4000 dan motor Rp1000

20 mobil dan 20 motor = 20x + 20y

= 20(4000) + 20(1000)

= 80.000 + 20.000

= 100.000

Jadi, tarif parkir 20 mobil dan 20 motor adalah Rp100.000 (D)

2. Misal = buku = x

pensil = y

3x + 2y = 18000 (×1)

5x + y = 23000 (×-2)

-----------------------

3x + 2y = 18000

-10x - 2y = -46000

----------------------- +

-7x = -28000

x = 4000

Masukkan x ke persamaan 3x + 2y = 18000

3(4000) + 2y = 18000

12000 + 2y = 18000

2y = 18000 - 12000

2y = 6000

y = 3000

Jadi, harga 1 buku adalah Rp4000 dan pensil Rp3000

4 buku dan 5 pensil = 4x + 5y

= 4(4000) + 5(3000)

= 16000 + 15000

= 31000

Jadi, harga 4 buku dan pensil adalah Rp31.000 (C)

3. 7x + 2y = 19 (×3)

4x - 3y = 15 (×2)

-----------------

21x + 6y = 57

8x - 6y = 30

----------------- +

29x = 87

x = 3

Masukkan x ke persamaan 4x - 3y = 15

4(3) - 3y = 15

12 - 3y = 15

-3y = 15 - 12

-3y = 3

y = -1

(x, y) = (3, -1)

Nilai dari 3x - 2y = 3(3) - 2(-1)

= 9 - (-2)

= 9 + 2

= 11

Jadi, nilai dari 3x - 2y adalah 11 (D)

4. 3p + 4q = -16 (×1)

2p - q = -18 (×4)

-------------------

3p + 4q = -16

8p - 4q = -72

------------------- +

11p = -88

p = -8

Masukkan p ke persamaan 2p - q = -18

2(8) - q = -18

16 - q = -18

-q = -18 + 16

-q = -2

q = 2

(p, q) = (-8, 2)

Nilai p + q = -8 + 2

= -6

Jadi, nilai p + q adalah -6 (A)

5. y = 2x + 1

3x - 5y = 16

-----------------

-2x + y = 1 (×5)

3x - 5y = 16 (×1)

-----------------

-10x + 5y = 5

3x - 5y = 16

----------------- +

-7x = 21

x = -3

Masukkan x ke persamaan -2x + y = 1

-2(-3) + y = 1

6 + y = 1

y = 1 - 6

y = -5

(x, y) = (-3, -5)

Jadi, penyelesaian dari persamaan y = 2x + 1

dan 3x - 5y = 16 adalah {-3, -5} (C)
