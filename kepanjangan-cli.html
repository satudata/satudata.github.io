8. perbedaan GUI sama CLI?


GUI adalah tipe antarmuka yang digunakan oleh pengguna untuk berinteraksi dengan sistem operasi melalui gambar-gambar grafik, ikon, menu, dan menggunakan perangkat penunjuk (pointing device) seperti mouse atau track ball. Elemen-elemen utama dari GUI bisa diringkas dalam konsep WIMP (window, icon, menu, pointing device).

CLI adalah tipe antarmuka dimana pengguna berinteraksi dengan sistem operasi melalui text terminal. Pengguna menjalankan perintah dan program di sistem operasi tersebut dengan cara mengetikkan barisbaris tertentu.

9. jelaskan yang dimasksud dengan CLI


Antarmuka baris perintah (bahasa Inggris: command-line interface, CLI) adalah mekanisme interaksi dengan sistem operasi atau perangkat lunak komputer dengan mengetikkan perintah untuk menjalankan tugas tertentu. Antarmuka hanya-teks ini merupakan kontras dari penggunaan peranti penunjuk untuk mengeklik pilihan pada antarmuka pengguna grafis (GUI), atau penggunaan menu untuk memilih pilihan pada antarmuka pengguna teks (TUI)

10. Jelaskan proses instalasi menggunakan CLI​


Penjelasan:

Proses instalasi menggunakan Command Line Interface (CLI) biasanya melibatkan beberapa langkah seperti:

Pastikan bahwa perangkat Anda memiliki akses ke internet, serta memiliki aplikasi terminal atau command prompt yang dapat digunakan untuk menjalankan perintah-perintah CLI.

Buka aplikasi terminal atau command prompt, lalu akses direktori yang diinginkan untuk menginstal aplikasi yang akan diinstal.

Gunakan perintah cd untuk mengakses direktori yang tepat, lalu gunakan perintah ls atau dir untuk melihat daftar file dan direktori yang ada pada direktori tersebut.

Unduh atau salin file instalasi yang dibutuhkan dari internet ke direktori yang telah dipilih. Jika menggunakan perintah wget atau curl, gunakan perintah wget [alamat file] atau curl -O [alamat file] untuk mengunduh file instalasi.

Setelah file instalasi telah berhasil diunduh, jalankan file tersebut dengan menggunakan perintah ./[nama file instalasi] atau [nama file instalasi] tergantung dari jenis file instalasi yang digunakan.

Ikuti instruksi-instruksi yang diberikan oleh program instalasi untuk menyelesaikan proses instalasi. Jika diperlukan, Anda mungkin harus mengizinkan aplikasi untuk mengakses atau memodifikasi file-file sistem.

Setelah proses instalasi selesai, Anda dapat mulai menggunakan aplikasi yang telah diinstal. Jika diperlukan, Anda juga dapat menjalankan perintah [nama aplikasi] -h atau [nama aplikasi] --help untuk menampilkan bantuan atau dokumentasi tentang cara penggunaan aplikasi tersebut.

14. Buatlah kode CLI pada router​


Penjelasan:

Salah satu cara konfigurasi pada router cisco adalah menggunakan CLI (Command Line Interface). Kita harus belajar menghapal beberapa perintah yang cukup penting pada konfigurasi dasar router ini. Perintah-peritah harus diketikan pada terminal. Software yang menangani input/output dan pemerosesan perintah-perintah adalah Cisco IOS (Internetwork Operating System).

Cisco IOS adalah sistem operasi berbasis teks yang digunakan pada perangkat switch dan router buatan Cisco. Perintah-perintah Cisco IOS dikelompokan menjadi beberapa mode yang disebut Exec mode. Secara umum ada dua mode utama, User Exec mode dan Privileged Exec mode. Pada Privileged Exec mode ini kita bisa masuk pada Global Configuration yang biasa digunakan untuk mengkonfigurasi IP pada tiap interface, Routing, Vlan dsb.

17. Jelaskan tentang aplikasi berbasis CLI​


Jawaban:

Aplikasi berbasis CLI (Command Line Interface) adalah aplikasi yang menggunakan perintah-perintah yang diketikkan dalam command prompt atau terminal, daripada menggunakan tampilan grafis seperti yang digunakan dalam aplikasi berbasis GUI (Graphic User Interface). Aplikasi CLI cenderung digunakan oleh administrator sistem, pengguna avansa, atau pengembang, karena mereka dapat mengakses fitur-fitur yang lebih kompleks dan memberikan kontrol yang lebih besar daripada aplikasi berbasis GUI.

Aplikasi CLI juga dapat digunakan untuk automatisasi tarek, menjalankan perintah yang ditentukan dalam skrip, atau membuat perintah yang disesuaikan. Beberapa contoh aplikasi CLI yang populer adalah terminal pada sistem operasi Linux dan macOS, PowerShell pada Windows, dan Git untuk manajemen versi kode.

Keuntungan dari aplikasi CLI adalah konsumsi sumber daya yang lebih rendah dibandingkan dengan aplikasi berbasis GUI, dan juga mudah digunakan dalam skrip otomatisasi. Namun, aplikasi CLI memerlukan pengetahuan yang lebih tinggi dalam perintah dan struktur sistem operasi, sehingga mungkin kurang user friendly bagi pengguna yang tidak terbiasa dengan perintah-perintah command line.
