4. langkah langkah mengirim email secara singkat


Langkah-langkah mengirim email secara singkat:

Membuka website penyedia email yang akan digunakan, Contoh penyedia email, Google Mail dan Yahoo MailBila belum masuk ke dalam akun pribadi, gunakan fitur sign-in/login menggunakan alamat email terdaftar dan passwordSetelah masuk ke halaman utama, cari menu compose/new/tulis yang pada umumnya berada di sisi kiri atau kananAkan muncul pop-up window yang digunakan untuk menuliskan pesan ataupun terdirect ke halaman penulisan emailMasukkan alamat email yang dituju dengan tepat, lalu masukkan subject email (subject dapat dikosongkan, namun disarankan untuk tetap diisi agar memudahkan pihak penerima), tulis pesan yang akan dikirimkan beserta attachment atau lampiran bila diperlukan. (penyedia layanan email biasanya akan membatasi ukuran attachment)Jika sudah lengkap, klik menu send. Setelah itu, cek kembali pada menu Terkirim atau Sent, jika sudah muncul di section tersebut, maka email sudah terkirim. Namun, jika email tersebut kembali ke draft atau masih dalam antrean, email tersebut gagal terkirim atau memang masih dalam proses.

Pembahasan

Mengirim email adalah bentuk tindakan menggunakan layanan internet dalam hal komunikasi. Pengguna dapat saling mengirimkan email dari manapun dan kapan pun selama pengguna yang dituju memiliki alamat email yang aktif. Pengiriman email dimasa sekarang memungkinkan seseorang mengirimkan email dari surel yang berbeda seperti Google Mail ke Yahoo mail atau sebaliknya tanpa proses tambahan.

21. jelaskan langkah mengirim file lewat email(bantu dong)


1. Klik teks link new. Halama web untuk menulis email akan ditampilkan.

2. Isikan alamat email penerima dan subjek email

3. Klik Isi email.

4. Selanjutnya cari tombol Attach / Lampirkan dari komputer dan Klik tombol tsb. dalam kotak dialog select file akan ditampilkan.

5. Bukafolder tempat file yang akan dikirim, anda dapat mengirim beberapa file sekaligus. selanjutnya, Klik tombol select file.

6. Selanjutnya klik tombol Send untuk mengirim email bersama file yang sudah dipilih.tunggu sampai halaman web untuk mengkonfirmasikan pengiriman email berhasil telah ditampilkan.

Jelaskan langkah mengirim file lewat email(bantu dong)

Untuk mengirim file menggunakan email, langkah-langkahnya adalah:

1. Buka situs penyedia email

2. Masuk dengan akun yang akan digunakan untuk mengirim email

3. Klik tombol Compose

4. Masukkan Subject, To, dan isi email

5. Klik tombol Attach File untuk melampirkan file

6. Ketika dialog Attach File muncul, pilih file yang akan dikirim, dan klik tombol OK

7. Klik tombol Send untuk mengirim email

8. Anda telah mengirim file dengan menggunakan email
