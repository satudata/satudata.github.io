16. Bagaimana cara memindahkan folder atau alamat website

3) Pilih folder atau alamat website yang ingin dipindahkan,kemudian pilih dan klik tombol Move to folder maka kotak dialog Browse for Folder akan ditampilkan.

22. Cara memindahkan folder dari satu sebuah drive ke darive lainnya. dimana drive yang di maksud bisa saja removable atau floppy disk! jelas kan beserta tampilam gambar


Jawaban:

Memindahkan file atau folder dari folder bersama ke Drive Saya adalah memindahkan, bukan menyalin, sehingga konten yang dipindahkan akan dihapus dari folder bersama. Sebagai hasilnya:

Pengguna tidak akan lagi melihat file atau folder yang telah dipindahkan dalam folder bersama.

Semua izin yang diwarisi dari folder bersama pada konten yang dipindahkan akan dihapus, dan izin baru akan diwarisi dari folder tujuan (beserta izin yang ditetapkan secara eksplisit pada file atau folder).

Untuk memindahkan file atau folder dari folder bersama ke Drive Saya:

Pastikan untuk membagikan item dengan akses Edit.

Aktifkan setelan Editor dapat mengubah izin dan berbagi untuk file atau folder tersebut.

Catatan: Jika Anda memindahkan folder dengan banyak file atau subfolder, mungkin dibutuhkan waktu cukup lama untuk memindahkan semua item ke lokasi baru.

25. Jelaskan bagaimana cara mengunci folder di drive windows 7 ?


Jadi, di Windows 7,8 dan 10 sudah terdapat sistem enkripsi folder yang bisa kamu gunakan untuk mengamankan folder dan isinya.

Berikut cara menggunakannya:

1. Arahkan ke file atau folder yang ingin kamu enkripsi, pada contoh ini saya akan memilih sebuah folder.

2. Klik kanan di file atau folder tersebut lalu klik Properties.

3. Pada menu General, klik tombol Advanced.

4. Centang bagian Encrypt contents to secure data, kemudian klik OK lalu Apply.

5. Akan muncul pop up Windows yang memintamu untuk memilih apakah ingin melakukan enkripsi di folder only atau folder, subfolder dan file didalamnya. Lalu klik OK.

6. Kamu akan melihat tanda gembok kecil berwarna kuning di folder atau file yang terenkripsi sebagai tanda. Metode enkripsi ini akan melindungi file penting milikmu dari tangan yang tidak bertanggungjawab. Enkripsi ini tersambung dengan akun Windows komputermu. Jadi saat kamu log in menggunakan akun Windows-mu, file tersebut akan terdekripsi otomatis. Simpelnya, cara ini hanya akan melindungin file terenkripsi jika kamu log out dari akun Windows-mu dan orang lain log in dengan akun Windows-nya lalu mencoba membuka file yang kamu enkripsi.
