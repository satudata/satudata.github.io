mungkin hardisknya tidak kompitable dgn Pcnya atau Ukurannya Kegedean.Driver hard disk eksternal tidak dikenali.

Bisa jadi karena kesalahan dalam file system pada hardisk karena sering langsung cabut tanpa di eject terlebih dahulu.

Port USB pada PC/Laptop rusak

Bisa jadi karena kerusakan fisik seperti port USB kendor/longgar. Coba colokkan pada port USB pada PC/Laptop lain

HDD External tidak mendapat pasokan daya yang cukup

Coba ganti kabel USB-nya dengan kabel USB yang ujungnya 2 (kabel Y).
Kabel Y

Driver Windows yang bermasalah.

Begitu sebuah perangkat kita colokkan ke port USB, Windows akan berusaha mengenali perangkat tersebut. Untuk mengatasi driver Windows yang bermasalah, silahkan ikuti cara berikut:
1.Coba colokkan hardisk eksternal-mu ke port USB dan lihat apakah Windows membacanya

2.Lalu, jika hardisk eksternal sudah dikenali tapi tidak muncul di Windows Explorer gimana tuh? Ikuti cara berikut:

1.Buka Run (tombol windows + R), ketik diskmgmt.msc di dalam kolom dialog Run, dan tekan Enter.
Lalu akan masuk ke Disk Management. Jika di Windows Explorer partisi hardisk tidak terbaca, maka di Disk Management akan terbaca.
Di dalam Disk Management ini kamu bisa memformat, menata ulang partisi, mengganti nama drive, dan sebagainya

18. Hardisk ssd bagus Atau apakelebihan dibanding hardisk biasa


Kelebihan:

Hardisk Bgus : Tidak gampang rusak, membaca file lebih cepat, tidak ada suara, garansi resmi, sparepart cari gampang

Hardisk Biasa: Murah,gampang mencarinya

Kekurangan:

Hardisk Bagus: Mahal

Hardisk Biasa: Gampang rusak, membaca file lambat, berbunyi suara yang keras, garansi tidak resmi, spaprepart susah mencarinya
