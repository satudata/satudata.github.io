19. Persilangan antara sapi jantan gemuk dan tidak tahan penyakit (GGtt) dengan sapi betina kurus tahan penyakit (ggTT) apabila sifat gemuk dan tahan penyakit dominan terhadap kurus dan tidak tahan penyakit.a. Berapakah prosentase F2 sapi jantan gemuk dan tahan penyakit? b. Berapakah perbandingan fenotif hasil keturunan kedua? ​


Persilangan Monohibrid

P1 : GGtt × ggTT

Gamet : Gt × gT

F1 : GgTt (gemuk dan tahan penyakit)

F1 disilangkan dengan sesamanya :

P2 : GgTt × GgTt

Gamet : GT, Gt, gT, gt >< GT, Gt, gT, gt

F2 : GGTT, GGTt, GgTT, GgTt, GGTt, GGtt, GgTt, Ggtt, GgTT, GgTt, ggTT, ggTt, GgTt, Ggtt, ggTt, ggtt

a. persentase F2 sapi jantan gemuk dan tahan penyakit :

= 9/16 × 100%

= 56,25%

b. perbandingan fenotif F2 :

= gemuk tahan penyakit : gemuk tidak tahan penyakit : kurus tahan penyakit : kurus tidak tahan penyakit

= 9 : 3 : 3 : 1
