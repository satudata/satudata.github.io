. Hubungkan smartphone Android kamu ke PC menggunakan kabel USB.Kemudian buka aplikasi Vysor di smartphone Android kamu. Kemudian luncurkan aplikasi Vysor di Google Chrome pada laptop atau PC kamu.Lalu klik tombol “Find Device”, kemudian pilih “Android from Android” lalu klik “Sent”. Nama perangkat mungkin akan berbeda di komputer kamu, tergantung dari perangkat Android yang kamu gunakan. 13. jelaskan cara mengirim file di laptop ke hp​


Jawaban:

Memindahkan file dengan kabel USB

1. Membuka kunci ponsel Anda.

2. Dengan kabel USB, hubungkan ponsel ke komputer.

3. Di ponsel, ketuk notifikasi "Mengisi daya perangkat ini via USB".

4. Di bagian "Gunakan USB untuk," pilih Transfer File.

5. Jendela transfer file akan terbuka di komputer.
