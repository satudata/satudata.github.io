4. Apakah Shortcut untuk pindah halaman sesuai dengan Yang kita inginkan



Banyak orang yang mengalami kesulitan ketika mendapat pekerjaan yang mengharuskan untuk membuka banyak file. Membaca file yang satu dan membuat rangkuman di file lainnya atau menyalin data di file yang satu ke file yang lain kerap sulit dilakukan karena tidak bisa di- copy. Seperti kita ketahui, setiap aplikasi memiliki shortcut untuk melakukan perintah tertentu seperti “ctrl+c” untuk copy atau “ctrl+v” untuk paste . Terdapat juga shortcut untuk memindahkan halaman atau memindahkan aplikasi. Saat membuka satu buah file word dan satu buah file ppt dan bermaksud untuk menyalin isi di ppt ke word, Anda dapat dapat menggunakan “alt+tab” untuk mempermudah berpindah aplikasi. Alt+tab adalah shortcut untuk pindah aplikasi atau pindah halaman di browser (ctrl+shift+tab).

Jawaban singkat:

Penjelasan tambahan:

Misalkan pada program Ms. Excel 2013, terdapat beberapa tombol kombinasi keyboard yg bisa dipakai. Antara lain:

Ctrl + B = untuk menebalkan karakter dalam sel yg disorot/ diseleksi

shift + F11 = untuk menambahkan sheet

Alt + F4 = untuk menutup aplikasi Ms. Excel

Ctrl + F2 = untuk mengaktifkan tampilan Print Preview (saat mau mencetak)
