Jika set top box Apple Matrix Anda tidak dapat menemukan siaran meskipun telah di-scan dan Anda menggunakan antena tambahan pada TV, ada beberapa langkah yang dapat Anda lakukan untuk mencoba memperbaikinya:

Periksa koneksi antena: Pastikan bahwa koneksi antena ke set top box dan TV terhubung dengan benar dan tidak terputus.

Periksa setelan antena: Pastikan bahwa setelan antena pada set top box dan TV sudah diatur dengan benar sesuai dengan lokasi Anda.

11. pada suatu pameran kuliner,salsa menjual rice box dalam tiga jenis yaitu premium set,simple set dan kids set. premium dijual dengan rp. 33.000,00, simple 24.000,00, dan kids set 9.000,00. pada pameran tersebut, simple set dan kids set yang terjual 30 lebih banyak dari setengah premium set yang terjual. jumlah simple yang terjual 5 lebih banyak dari 4 kali jumlah kids set yang terjual.jika jumlah hasil penjualan rice box seluruhnya rp.89.820.000,00, maka simple set yg terjual sebanyak ?​


Jawab:

Mari kita sebut jumlah premium set yang terjual sebagai x.

Dari soal, kita tahu bahwa jumlah simple set dan kids set yang terjual 30 lebih banyak dari setengah premium set yang terjual. Dengan kata lain,

jumlah simple set dan kids set yang terjual = 1.5x + 30

Kita juga tahu bahwa jumlah simple set yang terjual 5 lebih banyak dari 4 kali jumlah kids set yang terjual. Dengan kata lain,

jumlah simple set yang terjual = 4 * jumlah kids set yang terjual + 5

Kita bisa menggunakan kedua persamaan di atas untuk menyelesaikan masalah ini. Kita tahu bahwa total penjualan dari semua jenis rice box adalah Rp. 89.820.000,00, sehingga kita dapat menuliskan persamaan:

33.000x + 24.000(1.5x + 30) + 9.000(4x + 1) = 89.820.000

Setelah menyelesaikan persamaan tersebut, kita akan mendapatkan x = 1.800.

Kita dapat menggunakan nilai x tersebut untuk mencari jumlah simple set yang terjual:

jumlah simple set yang terjual = 4 * jumlah kids set yang terjual + 5 = 4 * (1.800 - 30)/5 + 5 = 573

Jadi, jumlah simple set yang terjual adalah 573.

Nada membeli dessert box yang dijual oleh temannya di Instagram. Harga 1 box tiramisu sama dengan harga 2 box red velvet. Harga 3 box tiramisu dan 2 box red velvet Rp480.000. Apabila Nada ingin membeli 2 box tiramisu dan 3 box red velvet, harganya adalah [tex]\boxed{\sf{Rp420.000}}.[/tex]

PEMBAHASAN

SPLDV adalah singkatan dari Sistem Persamaan Linier Dua Variabel. SPLDV adalah persamaan matematika yang terdiri dari dua variabel berbeda dan lebih dari satu persamaan. Adapun bentuk umum dari SPLDV adalah:

[tex]\boxed{\boxed{\left\{\begin{array}{c} \sf{ax+by=c}\\\sf{px+qy=r}\end{array}\right.}}[/tex]

ㅤ

Cara penyelesaian SPLDV diantaranya:

Metode grafikMetode eliminasiMetode substitusiMetode campuranMetode matriksDll

ㅤ

ㅤ

Diketahui:

Harga 1 box tiramisu sama dengan harga 2 box red velvet

Harga 3 box tiramisu dan 2 box red velvet Rp480.000

ㅤ

Ditanyakan:

Harga 2 box tiramisu dan 3 box red velvet adalah …

ㅤ

Jawab:

Pertama-tama ubahlah kalimat diatas menjadi kalimat matematika agar lebih mudah untuk diselesaiakan.

ㅤ

Misalkan:

x = Harga box tiramisu

y = Harga box red velvet

ㅤ

Maka:

Harga 1 box tiramisu sama dengan harga 2 box red velvet dapat ditulis menjadi [tex]\boxed{\sf{x=2y}}.[/tex]Harga 3 box tiramisu dan 2 box red velvet Rp480.000 dapat ditulis menjadi [tex]\boxed{\sf{3x+2y=480.000}}.[/tex]Harga 2 box tiramisu dan 3 box red velvet adalah … dapat ditulis menjadi [tex]\boxed{\sf{2x+3y=...}}.[/tex]

ㅤ

Diperoleh bentuk umum SPLDV:

[tex]\left\{\begin{array}{c}\sf{\:\:\:x =2y}\\ \sf{3x+2y=480.00}\end{array}\right.[/tex]

ㅤ

Untuk penyelesaiannya gunakan metode substitusi agar lebih mudah. Ada dua cara yang bisa dipilih untuk menyelesaikan permasalahan diatas.

CARA I: Cari nilai x dan nilai y.

[tex]\begin{array}{cl}\sf{\:\:\:x =2y}&...\:\sf{(i)}\\\sf{3x+2y=480.00}&...\:\sf{(ii)}\end{array}[/tex]

ㅤ

Substitusikan persamaan (i) ke persamaan (ii).

[tex]\begin{array}{rll}\sf{3x+2y}&=&\sf{480.000}\\\sf{3x+\:\:x}&=&\sf{480.000}\\\sf{4x}&=&\sf{480.000}\\\sf{x}&=&\sf{120.000}\end{array}[/tex]

ㅤ

Substitusikan nilai x = 120.000 ke persamaan (i)

[tex]\begin{array}{lll}\sf{\:\:x}&=&\sf{2y}\\\sf{2y}&=&\sf{x}\\\sf{2y}&=&\sf{120.000}\\\sf{\:\:y}&=&\sf{60.000}\end{array}[/tex]

ㅤ

Sehingga harga 2 box tiramisu dan 3 box red velvetnya adalah

[tex]\begin{array}{lll}\sf{2x+3y}&=&\sf{2(120.000)+3(60.000)}\\&=&\sf{240.000+180.000}\\&=&\sf{420.000}\end{array}[/tex]

ㅤ

Jadi harga 2 box tiramisu dan 3 box red velvetnya adalah [tex]\boxed{\sf{Rp420.000}}.[/tex]

ㅤ

CARA II: Cukup cari nilai y.

[tex]\begin{array}{cl}\sf{\:\:\:x =2y}&...\:\sf{(i)}\\\sf{3x+2y=480.00}&...\:\sf{(ii)}\end{array}[/tex]

ㅤ

Substitusikan persamaan (i) ke persamaan (ii).

[tex]\begin{array}{rll}\sf{3x+2y}&=&\sf{480.000}\\\sf{3(2y)+2y}&=&\sf{480.000}\\\sf{6y+2y}&=&\sf{480.000}\\\sf{8y}&=&\sf{480.000}\\\sf{y}&=&\sf{60.000}\end{array}[/tex]

ㅤ

Sehingga harga 2 box tiramisu dan 3 box red velvetnya adalah

[tex]\begin{array}{lll}\sf{2x+3y}&=&\sf{2(2y)+3y}\\&=&\sf{4y+3y}\\&=&\sf{7y}\\&=&\sf{7(60.000)}\\&=&\sf{42.000}\end{array}[/tex]

ㅤ

Jadi harga 2 box tiramisu dan 3 box red velvetnya adalah [tex]\boxed{\sf{Rp420.000}}.[/tex]

14. harga Lap Top Rp. 10.000.000;- maka penawaran sebanyak 15 unit. Jika harga Lap Top Rp. 15.000.000;- maka penawaran sebanyak 20 unit. Jika harga Lap Top Rp. 20.000.000;- maka penawaran sebanyak


Pola bilangan adalah aturan dalam barisan bilangan sehingga dapat menentukan bilangan berikutnya dari barisan tersebut. Polanya bisa antar dua huruf berdekatan atau bisa juga loncat 1 bilangan, 2 bilangan atau 3 bilangan.

Berdasarkan perhitungan jika harga Lap Top Rp. 20.000.000;- maka penawaran sebanyak 25 Unit.

Penjelasan dengan langkah-langkah:

Diketahui :

harga Lap Top Rp. 10.000.000;- maka penawaran sebanyak 15 unit.

Jika harga Lap Top Rp. 15.000.000;- maka penawaran sebanyak 20 unit.

Ditanya:

Jika harga Lap Top Rp. 20.000.000;- maka penawaran sebanyak?

Penyelesaian:

Pola bilangan adalah aturan dalam barisan bilangan sehingga dapat menentukan bilangan berikutnya dari barisan tersebut. Polanya bisa antar dua huruf berdekatan atau bisa juga loncat 1 bilangan, 2 bilangan atau 3 bilangan.

Jadi untuk menyelesaikan soal diatas adalah

harga laptop kita misalkan A dan Penawaran kita misalkan B

A = Rp. 10.000.000 maka B = 15 unit

A = Rp. 15.000.000 maka B = 20 unit

Jika

A = Rp. 20.000.000 maka B = ....?

terlihat bahwa untuk A pola mengalami kenaikan Rp. 5.000.000 maka B akan ikut mengalami kenaikan 5 unit. Jadi bisa disimpulkan bahwa A = Rp. 20.000.000 maka B = 25 Unit  atau harga LapTop Rp. 20.000.000 maka penawaran sebanyak 25 Unit.

29. Harga sepidol satu set 2/3 harga satu set pastel. Harga satu set pastel 3/4 harga buku. Nandita membeli 3 set sepidol, dan satu set pastel membayar dengan uang lima puluh ribuan dan mendapat kembalian Rp.14.000, 00.- selang beberapa hari Nandita membeli lagi 5 set sepidol, 3 set pastel, dan beberapa buku. Banyak buku yang dibeli


Jawaban:

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Harga buku gambar adalah ¹/₃ harga jangka" yaitu :

x = ¹/₃ y ...(Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Harga jangka adalah ²/₃ harga pastel warna" yaitu :

y = ²/₃ z ...(Persamaan 2)

LANGKAH KETIGA (III)

Hitung nilai z dengan menggunakan persamaan dari pernyataan "Harga 2 pastel warna adalah Rp. 22.500" yaitu :

2z = 22.500

z = \frac{22.500}{2}

z = 11.250

LANGKAH KEEMPAT (IV)

Subitusikan nilai z pada persamaan 2 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

y = ²/₃ z

y = ²/₃ (11.250)

y = 7.500

LANGKAH KELIMA (V)

Subitusikan nilai y pada persamaan 1 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

x = ¹/₃ y

x = ¹/₃ (7.500)

x = 2.500

LANGKAH KEENAM (VI)

Hitung jumlah jangka dengan menggunakan cara sebagai berikut :

8x + ay + 2z = 65.000

8 (2.500) + a (7.500) + 2 (11.250) = 65.000

20.000 + 7.500a + 22.500 = 65.000

42.500 + 7.500a = 65.000

7.500a = 65.000 - 42.500

7.500a = 22.500

a = \frac{22.500}{7.500}

a = 3

∴ Kesimpulan jumlah jangka yang dibeli doni adalah 3 buah.
