21. Pernyataan yang benar tentang tipe data berdasarkan gambar mind map tersebut adalah .... (Jawaban lebih dari satu) A. Array dan structure merupakan jenis data yang dikelompokkan atau dimodifikasi B. Float digunakan untuk mendeklarasikan angka negatif, netral, dan positif C. Void merupakan tipe data yang dibuat sendiri D. Double digunakan untuk mendeklarasikan angka desimal dengan ketetapan hingga 15 desimal digit E. Tipe data yang terdiri dari kumpulan tipe data dasar merupakan pengertian integer 3. Void merupakan salah satu tipe data dalam bahasa pemrograman C. Bagaimana bentuk tipe data void? Jawab : ...


A. Benar

B. Salah. Float digunakan untuk mendeklarasikan angka desimal dengan ketetapan hingga 6-7 desimal digit.

C. Benar. Void digunakan untuk mendefinisikan fungsi yang tidak mengembalikan nilai apapun.

D. Benar. Double digunakan untuk mendeklarasikan angka desimal dengan ketetapan hingga 15-16 desimal digit.

E. Salah. Tipe data yang terdiri dari kumpulan tipe data dasar bukanlah pengertian dari integer. Integer adalah tipe data yang hanya memuat bilangan bulat tanpa pecahan.

Tipe data void dalam bahasa pemrograman C merupakan sebuah tipe data yang tidak memiliki nilai atau memuat nilai kosong. Void digunakan sebagai jenis data yang tidak mengembalikan nilai apa pun dari sebuah fungsi. Contoh deklarasi variabel dengan tipe data void adalah sebagai berikut:

void hello_world() {

  printf("Hello, World!\n");

}

Dalam contoh di atas, fungsi hello_world() digunakan untuk mencetak pesan "Hello, World!" pada layar tetapi tidak mengembalikan nilai apa pun.

28. Apa itu tipe data di C++ dan apa saja jenis-jenis tipe data yang tersedia? Bagaimana cara deklarasi dan penggunaan tipe data dalam program C++? Bagaimana menentukan jenis tipe data yang tepat untuk menyimpan nilai yang dibutuhkan dalam program?


Jawaban:

Tipe data di C++ merupakan jenis data yang digunakan untuk menyimpan nilai dalam program. Tipe data digunakan untuk memberikan informasi kepada compiler tentang jenis data yang akan digunakan dalam program. Dalam C++, terdapat beberapa jenis tipe data, yaitu:

Tipe data primitif, seperti int, float, double, char, dan bool.Tipe data turunan, seperti array, pointer, dan struct.Tipe data user-defined, seperti class dan enum.

Untuk mendeklarasikan sebuah variabel dengan tipe data tertentu, kita perlu menggunakan sintaks seperti berikut:

<tipe data> <nama variabel>;

Sebagai contoh, untuk mendeklarasikan sebuah variabel bertipe integer dengan nama "umur", kita dapat menggunakan sintaks berikut:

int umur;

Setelah kita mendeklarasikan variabel, kita dapat menggunakan variabel tersebut dalam program dengan cara memberikan nilai ke variabel tersebut, dan mengakses nilai dari variabel tersebut. Sebagai contoh:

int umur;

umur = 20; // memberikan nilai 20 ke variabel umur

cout << "Umur saya adalah " << umur << endl; // mencetak nilai dari variabel umur

Dalam menentukan jenis tipe data yang tepat untuk menyimpan nilai dalam program, kita perlu mempertimbangkan ukuran data yang akan disimpan, serta jenis operasi yang akan dilakukan terhadap data tersebut. Sebagai contoh, jika kita hanya perlu menyimpan nilai bilangan bulat kecil, kita dapat menggunakan tipe data "short" atau "char" untuk menghemat penggunaan memori. Namun, jika kita perlu melakukan operasi matematika kompleks, kita mungkin memerlukan tipe data "double" atau "float" untuk menghindari kehilangan presisi.

Tipe data dalam C++ adalah jenis nilai yang dapat disimpan dalam variabel atau konstan. Berikut adalah beberapa jenis tipe data yang tersedia dalam C++:

Tipe Data Bilangan Bulat (Integer)

long int

signed int
