3. cara mengirim file dan video dari hp ke laptop dgn kabel chargerpliss penyimpanan hp ku sekarat kakkkumohonnnmkasih​


Jawaban:

Sambungkan HP memakai kabel data (yang support transfer file).

Cek di HP. Pilih Mode Transfer File.

Buka laptop, kemudian buka drive HP yang telah terdeteksi.

Cari dan pilih file-file yang diinginkan dengan klik Ctrl + mouse kiri.

Klik kanan mouse, lalu pilih Copy.

Pergi ke drive laptop, lalu buat folder baru.

Setelah itu klik kanan, lalu pilih Paste.

Penjelasan:

Langkah 1: Ambil kabel data yang sudah support keperluan transfer file (bukan cuma buat cas saja), kemudian sambungkan ke HP dan ke laptop.

Langkah 2: Setelah itu, tunggu beberapa saat. Maka di HP Anda nanti akan muncul sebuah jendela. Silakan atur ke mode Transfer File.

Langkah 3: Selanjutnya, buka laptop Anda. Anda akan melihat drive baru yang biasanya sesuai dengan nama HP. Silakan klik drive tersebut untuk melihat file-file di dalamnya

Langkah 4: Langkah berikutnya, cari dan pilih file-file apa saja yang ingin disalin. Bisa klik mouse sambil menekan tombol Ctrl. Kalau sudah, klik kanan lalu klik copy.

Langkah 5: Buka drive lain di laptop Anda, lalu klik kanan mouse > New > New Folder. Beri nama folder tersebut, kemudian buka foldernya.

Langkah 6: Klik kanan lagi, lalu pilih opsi Paste. Setelah itu tinggal tunggu saja proses salin datanya sampai benar-benar selesai

Langkah 7: Dengan begitu, data-data dari HP akan berpindah ke laptop.

Note:

Kelebihan cara memakai kabel data ini adalah kecepatan transfernya yang cepat.

Biasanya file-file ukuran belasan hingga puluhan GB, bisa selesai dalam waktu 10-20 menit saja.

Kalau jendelanya nggak keluar, coba ganti kabelnya

11. Jelaskan cara mengirim file dari HP ke HP lainnya​


Jawaban:

Terdapat berbagai cara untuk mengirim file dari HP ke HP lainnya, dapat dilakukan melalui koneksi data, Bluetooth, atau aplikasi komunikasi seperti email ataupun WhatsApp.

1. Mengirim file melalui koneksi data

- Pastikan kedua ponsel memiliki koneksi data yang stabil

- Buka aplikasi File Manager yang terdapat di HP Anda lalu pilih file yang akan dikirim.

- Tekan dan tahan file tersebut hingga muncul opsi “Bagikan”

- Setelah itu pilih metode pengiriman melalui koneksi data seperti email, pesan atau pesan singkat.

2. Mengirim file melalui Bluetooth

- Pastikan koneksi Bluetooth pada kedua HP dalam keadaan aktif

- Buka aplikasi File Manager yang terdapat di HP Anda dan pilih file yang akan dikirim.

- Tekan dan tahan file tersebut hingga muncul opsi “Bagikan”

- Pilih file yang ingin dikirim melalui Bluetooth dan pilih penerima dari daftar perangkat yang terhubung.

3. Mengirim file melalui aplikasi komunikasi

- Unduh aplikasi komunikasi seperti WhatsApp atau Email pada kedua perangkat

- Buka aplikasi tersebut dan pilih kontak yang akan menerima file

- Pilih opsi “Lampirkan File ” dan pilih file yang akan dikirim dari file manager pada HP Anda.

- Setelah memilih file, tekan tombol “Kirim” atau “Send”.

Perlu diingat bahwa cara ini bisa bervariasi tergantung jenis dan merek HP yang digunakan. Koneksi internet dan jaringan perlu dipastikan untuk mengirimkan file secara berkualitas dan lancar.

15. Cara mencari file file yang hilang pada laptop cara nya gimana?


Kemungkinan pertama. Kalau "hilang". Tinggal ketik nama file yang hilang di Search Bar di Windows Explorer (Yang ada tulisan "Search" di pojok kanan agak atas). Sebelum mengetik, pilih folder dimana kemungkinan file hilang. Kalau belum ketemu, ganti folder lain atau ganti ketikan nama yang dikira lebih sesuai.

Kemungkinan kedua. Ternyata bukan hilang, melainkan terhapus (belum permanen). Biasanya file masih terdapat di Recycle Bin di Windows Explorer. Tinggal cari gambar tempat sampah (Recycle Bin) >> Klik 2x >> Cari file yang terhapus >> Klik 2x (Perhatikan Origin, disitulah tempat file nanti kembali) >> Pilih Restore. File akan kembali ke lokasi di Origin tadi.

Kemungkinan ketiga. File terhapus permanen, sudah dicari di Recycle Bin tidak ada berarti file terhapus permanen dari komputer.

Kemungkinan tambahan. File terkena virus dan disembunyikan di suatu tempat. Ini agak susah, file yang disembunyikan virus besar kemungkinan kalau virusnya dihapus file ikut terhapus. Cari di google kalau yang ini.
