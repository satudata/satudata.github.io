6. 1. langkah membuka gmail2. cara masuk akun gmail3. cara mendaftarkan aku baru4. cara mengirim file lewat Emali


Misalkan membuka gmail memakai komputer/ laptop:
1. yang pertamakali dilakukan adalah membuka program web browser (misalnya Mozilla Firefox) lalu ketikkan www.gmail.com pada address bar.

2. saat tampil jendela situs www.gmail.com klik tombol Masuk, yang mana nantinya kita diminta mengisikan user name dan password email.

3. untuk mendaftarkan akun baru, klik tombol buat akun (berwarna merah), yang mana nantinya kita diminta mengisikan biodata diri secara lengkap.

4. untuk mengirim file lewat email, dapat memakai tombol Lampirkan file saat menulis email.

12. Cara menghapus gmail tapi kata sandinys lupa , hapusnya lewat hp gimanas??


Akses akun anda melalui alamat http://accounts.google.com/Login



Setelah tampil halaman login, masuk-kan e-mail dan password anda (pada contoh kasus ini saya menggunakan email kosur2013@gmail.com untuk dihapus), klik Masuk



Setelah proses loading selesai, jika tampil dialog untuk masuk-kan nomor ponsel dan alamat pemulihan, klik tombol Simpan dan lanjutkan untuk mengabaikan.



Setelah masuk ke halaman pengaturan akun, pilih menu Product> edit (untuk mengakses halaman ini, dapat juga melalui https://www.google.com/settings/products)





Muncul halaman untuk meng-edit layanan. Untuk menghapus akun secara pemanen, pilih menu yang paling bawah yakni pilihan Tutup akun dan hapus semua ayanan dan info....





Pada halaman penghapusan akun yang muncul, centangi semua pilihan serta masuk-kan password anda. Lalu klik tombol HAPUS GOOGLE ACCOUNT





Jika muncul halaman dengan tulisan seperti di bawah ini, berati anda telah berhasil menghapus akun google/gmail anda
