2. pak ardi membeli 3 kaleng cat untuk dinding berukuran 5kg dan 2 kaleng cat untuk kayu seharga Rp 1.140.000 disuatu toko bangunan , pada saat yang bersamaan seorang pelanggan lain ditoko tersebut harus membayar Rp 1.540.000 untuk membeli 4 kaleng cat untuk dinding berukuran 5kg dan 3 kaleng cat untuk kayu. biaya yang harus dikeluarkan untuk membeli masing masing 2 kaleng pada kedua jenis cat tersebut adalah


Pak Ardi membeli 3 kaleng cat untuk dinding berukuran 5 kg dan 2 kaleng cat untuk kayu seharga Rp 1.140.000 disuatu toko bangunan, pada saat yang bersamaan seorang pelanggan lain ditoko tersebut harus membayar Rp 1.540.000 untuk membeli 4 kaleng cat untuk dinding berukuran 5 kg dan 3 kaleng cat untuk kayu. Biaya yang harus dikeluarkan untuk membeli masing masing 2 kaleng pada kedua jenis cat tersebut adalah Rp. 800.000


Pembahasan

Untuk menyelesaikan soal diatas dibuat persamaan linier dua variabelnya dengan cat untuk dinding sebagai x dan cat untuk kayu sebagai y.

Pak Ardi membeli 3 kaleng cat untuk dinding berukuran 5 kg dan 2 kaleng cat untuk kayu seharga Rp 1.140.000

3x + 2y = 1.140.000

Seorang pelanggan lain ditoko tersebut harus membayar Rp 1.540.000 untuk membeli 4 kaleng cat untuk dinding berukuran 5 kg dan 3 kaleng cat untuk kayu

4x + 3y = 1.540.000

Eliminasi variabel x

3x + 2y = 1.140.000    | x4

4x + 3y = 1.540.000   | x3

12x + 8y = 4.560.000

12x + 9y = 4.620.000   -

-y = -60.000

y = 60.000

Substitusi variabel y

3x + 2y = 1.140.000

3x + 2 ( 60.000 ) = 1.140.000

3x + 120.000 = 1.140.000

3x = 1.140.000 - 120.000

3x = 1.020.000

x = 340.000

Jadi harga 1 kaleng cat untuk dinding adalah Rp. 340.000 dan harga satu kaleng cat untuk kayu adalah Rp. 60.000

Jika ingin membeli masing-masing 2 kaleng cat maka uang yang harus dikeluarkan:

Harga = 2 x 340.000 + 2 x 60.000

Harga = 680.000 + 120.000

Harga = 800.000

Jadi untuk membeli masing-masing 2 kaleng cat biaya yang harus dikeluarkan adalah Rp. 800.000
