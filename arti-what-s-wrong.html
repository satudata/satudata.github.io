23. was it wrong artinya...


Soal:

Was it wrong

artinya...?

Jawab:

Apakah itu salah?

Pembahasan:

To Be yg tertera adalah WAS yg berarti Past Tense bentuk Nominal

(+) S + was/were + Adj/Adv/Noun

= it + was + wrong

(Itu salah)

(-) S + was/were + not + Adj/Adv/Noun

= it + was + not + wrong

(Itu tidak salah)

(?) was/were + S + Adj/Adv/Noun?

= was + it + wrong?

(Apakah itu salah?)

Catatan:

Pertanyaan “was it wrong” berarti menanyakan suatu hal atau tindakan itu salah atau tidak yg terjadi di masa lampau.

==== Semoga Membantu ====

Mapel: Bahasa Inggris

Kelas: SMP

Materi: Past Tense

Kode Mapel: 5
