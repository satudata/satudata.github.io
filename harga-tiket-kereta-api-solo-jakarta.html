6. Sebuah kereta api melaju dari jakarta ke solo dengan kecepatan 125 km/jam. Jarak tempuh yaitu 725 km. Jika kereta api berangkat pukul 07.15, maka berapa kereta api sampai tempat tujuan ?


Kereta api sampai tempat tujuan pada pukul 13.03

Pembahasan

Jarak = s

Kecepatan = v

Waktu = t

s = v × t

v = s/t

t = s/v

Diketahui:

v = 125 km/jam

s = 725 km

berangkat pukul 07.15

Ditanyakan:

Tiba pada pukul = .... ?

Jawab:

waktu perjalanan

t = s/t

t = 725/125

t = 29/5

t = 5 4/5 jam

t = 5 jam 48 menit

Tiba pukul

= 07.15 + 5 jam 48 menit

= 12.60 + 3 menit

= 13.03
