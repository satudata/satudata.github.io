18. langkah langkah untuk mengaktifkan program aplikasi Microsoft office word 2010 dari desktop windows XP adalah​


Misalkan memakai sistem operasi Windows XP:

1. Langkah pertama, klik Start

2. Langkah kedua, klik All Programs

3. Langkah ketiga, klik Microsoft Office

4. Langkah keempat, klik Microsoft Office Word 2010

Setelah langkah keempat dilakukan, maka akan tampil program Ms. Word 2010 yang menandakan bahwa program itu telah aktif.
