27. Cari median, kuartil atas, dan kuartil bawah dari data kelompok berikut!


Nilai median, kuartil atas dan kuartil bawah berturut-turut adalah 23.3, 38.42 dan 28.9375. Soal ini berkaitan dengan materi statistika data berkelompok.

Penjelasan dengan langkah-langkah

Diketahui :

Terlampir pada gambar soal

Ditanyakan :

Terlampir pada gambar soal

Jawab :

Mencari median

Rumus median data berkelompok

Me = Tb + [(n/2 - fkk)/fi].C

Dengan

Tb = batas bawah kelas median

n = frekuensi total

fkk = frekuensi kumulatif sebelum kelas median

fi = frekuensi kelas median

C = Panjang kelas

Kita ketahui bahwa n = 30, maka median terletak di data ke 15 dan 16

Perhatikan bahwa

xii = 20.5

fkii = 10

fi = 9

C = 5

Maka mediannya adalah

Me = 20.5 + [(30/2 - 10)/9].5

     = 20.5 + [5/9].5

     = 20.5 + 2.8

     = 23.3

Mencari kuartil atas

Rumus kuartil atas data berkelompok

Q3 = Tb + [((3/4)n – fkk)/fQ3].C

Dengan

Tb = batas bawah kelas

fkk = frekuensi kumulatif sebelum kelas kuartil

C = panjang kelas

Letak kuartil atas yaitu di antara (3/4).30 = 22.5 dan (3/4).30 + 1 = 23.5 yaitu di kelas interval 36-40,

Dan kita ketahui bahwa

Tb = 35.5

fkk = 19

fQ3 = 6

C = 5

maka nilai Q3

Q3 = 35.5 + [((3/4)n – fkk)/fQ3].C

     = 35.5 + [((3/4)30 – 19)/6].5

     = 35.5 + [(22.5 – 19)/6].5

     = 35.5 + [3.5/6].5

     = 35.5 + 2.92

     = 38.42

Mencari kuartil bawah

Rumus kuartil bawah data berkelompok

Q1 = Tb + [((1/4)n – fkk)/fQ1].C

Dengan

Tb = batas bawah kelas

fkk = frekuensi kumulatif sebelum kelas kuartil

C = panjang kelas

Letak kuartil bawah yaitu di antara (1/4).30 = 7.5 dan (1/4).30 + 1 = 8.5 yaitu di kelas interval 26-30,

Dan kita ketahui bahwa

Tb = 25.5

fkk = 2

fQ1 = 8

C = 5

maka nilai Q1

Q1 = 25.5 + [((1/4)n – fkk)/fQ1].C

     = 25.5 + [((1/4)30 – 2)/8].5

     = 25.5 + [(7.5 – 2)/8].5

     = 25.5 + [5.5/8].5

     = 25.5 + 3.4375

     = 28.9375

30. suatu data tunggal mempunyai hamparan 20 sedangkan kuartil atasnya 85. nilai kuartil bawah data tersebut adalah...


Suatu data tunggal mempunyai hamparan 20 sedangkan kuartil atasnya 85. Nilai kuartil bawah data tersebut adalah 65

Beberapa ukuran penyebaran data (khususnya data tunggal), terdiri dari

Jangkauan (Rentang)

J = data terbesar – data terkecil

        J = [tex]x_{max} - x_{min}[/tex]

Hamparan (jangkauan antar kuartil)

H = kuartil atas – kuartil bawah

        H = Q₃ – Q₁

Simpangan kuartil  

Sk = ½ (Q₃ – Q₁)

Rataan kuartil

½ (Q₃ + Q₁)

Rataan Tiga Kuartil

¼ (Q₃ + 2Q₂ + Q₁)

Pembahasan    

Diketahui

Hamparan (H) = 20 Kuarti atas (Q₃) = 85

Ditanyakan

Nilai kuartil bawah (Q₁) = … ?

Jawab

H = Q₃ – Q₁

20 = 85 – Q₁

Q₁ = 85 – 20

Q₁ = 65

Jadi nilai kuartil bawah data tersebut adalah 65
