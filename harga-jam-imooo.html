8. sebuah parkiran terdapat 1000 kendaraan. ternyata bannya ada 3120 buah. jika harga parkir motor 3000/jam, dan parkir mobil 5000/jam. tentukan uang yang didapat tukang parkir jika 20% motor parkir selama 3jam dan motor sisanya selama 5jam. 50%mobil yang ada parkir selama 2jam, 20% selama 1jam dan sisanya 6 jam


Ban motor = 2
Ban mobil = 4
Jumlah motor = m
Jumlah mobil = c

m + c = 1000
c = 1000 - m

2m + 4c = 3120
2m + 4 x (1000 - m) = 3120
2m + (4000 - 4m) = 3120
2m + (4000 - 4m) - 3120 = 0
4000 - 3120 = 4m - 2m
880 = 2m
880/2 = m
440 = m (motor)
c = 1000 - m = 560 (mobil)

20% x 440 = 88 motor parkir selama 3 jam
440 - 88 = 352 motor parkir selama 5 jam

50% x 560 = 280 mobil parkir selama 2 jam
20% x 560 = 112 mobil parkir selama 1 jam
560 - 280 - 112 = 168 mobil parkir selama 6 jam

Uang dari motor:
= 88 x (3000 x 3) + 352 x (3000 x 5)
= 88 x 9000 + 352 x 15000
= 6.072.000

Uang dari mobil:
= 280 x (5000 x 2) + 112 x 5000 + 168 x (5000 x 6)
= 280 x 10000 + 112 x 5000 + 168 x 30000
= 8.400.000

Total uang:
= 6.072.000 + 8.400.000
= 14.472.000
