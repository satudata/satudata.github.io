12. tentukan akar persamaan kuadrat b. 4x2 + 4x + 1 = 0


Kategori Soal : Matematika - Persamaan Kuadrat
Kelas : X (1 SMA)
Pembahasan :
Diketahui persamaan kuadrat : 4x² + 4x + 1 = 0
Kita cek dahulu diskriminannya.
Jika a = 4, b = 4, dan c = 1, maka
D = b² - 4ac
⇔ D = 4² - 4.4.1
⇔ D = 16 - 16
⇔ D = 0
Karena D = 0, artinya persamaan kuadrat tersebut memiliki akar-akar real yang sama atau kembar.
Sehingga 4x² + 4x + 1 = 0
⇔ (2x + 1)² = 0
⇔ 2x + 1 = 0
⇔ 2x = -1
⇔ x = -1/2

Jadi, akar atau penyelesaian dari persamaan kuadrat 4x² + 4x + 1 = 0 adalah -1/2.

Semangat!
