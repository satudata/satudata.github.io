12. Jelaskan cara mengirim file dari HP ke HP lainnya​


Jawaban:

Terdapat berbagai cara untuk mengirim file dari HP ke HP lainnya, dapat dilakukan melalui koneksi data, Bluetooth, atau aplikasi komunikasi seperti email ataupun WhatsApp.

1. Mengirim file melalui koneksi data

- Pastikan kedua ponsel memiliki koneksi data yang stabil

- Buka aplikasi File Manager yang terdapat di HP Anda lalu pilih file yang akan dikirim.

- Tekan dan tahan file tersebut hingga muncul opsi “Bagikan”

- Setelah itu pilih metode pengiriman melalui koneksi data seperti email, pesan atau pesan singkat.

2. Mengirim file melalui Bluetooth

- Pastikan koneksi Bluetooth pada kedua HP dalam keadaan aktif

- Buka aplikasi File Manager yang terdapat di HP Anda dan pilih file yang akan dikirim.

- Tekan dan tahan file tersebut hingga muncul opsi “Bagikan”

- Pilih file yang ingin dikirim melalui Bluetooth dan pilih penerima dari daftar perangkat yang terhubung.

3. Mengirim file melalui aplikasi komunikasi

- Unduh aplikasi komunikasi seperti WhatsApp atau Email pada kedua perangkat

- Buka aplikasi tersebut dan pilih kontak yang akan menerima file

- Pilih opsi “Lampirkan File ” dan pilih file yang akan dikirim dari file manager pada HP Anda.

- Setelah memilih file, tekan tombol “Kirim” atau “Send”.

Perlu diingat bahwa cara ini bisa bervariasi tergantung jenis dan merek HP yang digunakan. Koneksi internet dan jaringan perlu dipastikan untuk mengirimkan file secara berkualitas dan lancar.

30. Jelaskan cara mengirim dan membaca file attachment.Terimakasih.


1. Pertama, buka akun gmail atau yahoo mail kamu dulu. Bila belum punya account, silahkan belajar dulu cara membuat email baru di tautan ini.

2. Selanjutnya, anggap saja kita pakai layanan email Gmail, klik tombol merah besar di sebelah kiri atas yang ada kata “Tulis” pada tombol merahnya.

3. Berikutnya akan terbukalah halaman kecil tempat penulisan email anda. Nah, misalnya mau mengirim berkas, entah itu file dari ms. Word dan excel, gambar, lagu, video, dan lainnya, maka yang pertama dilakukan adalah mengetik alamat email penerimanya (ada tulisan ‘Kepada’ di sampingnya), kemudian judul dari pesan tersebut (atau istilahnya ‘Subjek’)

4. Jika sudah mengisi dua kolom di atasnya, selanjutnya pada kolom penulisan isi pesan silahkan berikan sedikit pengantar bahwa kamu mengirim file dalam bentuk lampiran biar yang menerima pesannya paham.

5. Terakhir, ini inti pembahasan kita, klik icon seperti jepitan kertas yang ada di paling bawah. Perhatikan gambar di bawah ini:

6. Setelah diklik, akan terbuka halaman kecil melayang, nah tinggal cari dan pilih di komputermu file atau berkas yang mau kamu kirim, lalu setelah ketemu tinggal klik, kemudian akhiri dengan mengklik tombol “Open” pada halaman tersebut.

7. Tentunya, biar bisa terkirim emailnya, silahkan tekan tombol “Kirim” berwarna biru seperti biasaya kalau mengirim pesan tanpa lampiran berkas atau attachment.
