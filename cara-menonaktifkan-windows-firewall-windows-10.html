3. carilah cara menonaktifkan sistem windows 7,8,9,10​


Jawaban:

Untuk menonaktifkan sistem Windows, Anda dapat melakukan hal-hal berikut:

Melalui Control Panel: Buka Control Panel dan pilih "System and Security." Kemudian, pilih "Power Options" dan pilih "Choose what the power buttons do." Di sini, Anda dapat mengatur tombol daya untuk menonaktifkan sistem saat ditekan.

Melalui Command Prompt: Buka Command Prompt dengan hak akses administrator dan ketik "shutdown /s /t 0" tanpa tanda petik. Ini akan menonaktifkan sistem secara instan.

Melalui Shortcut Keyboard: Anda dapat membuat shortcut keyboard untuk menonaktifkan sistem Windows. Caranya adalah dengan membuat shortcut baru di desktop dan menambahkan "shutdown /s /t 0" tanpa tanda petik sebagai target. Kemudian Anda tinggal menekan tombol shortcut yang telah anda buat.

Perlu diingat bahwa menonaktifkan sistem secara tiba-tiba tanpa menyimpan data Anda dapat menyebabkan kerusakan pada data Anda.

6. Jelaskan cara menonaktifkan komputer berbasis Windows


1.Menekan tombol power. Ini adalah cara tercepat untuk mematikan komputer. Tombol power di laptop atau PC jika ditekan akan segera terjadi penutupan beragam aplikasi secara instan, lalu mati. Hanya saja cara ini sebaiknya digunakan ketika benar-benar dalam posisi terpaksa, seperti komputer hang. Pasalnya, langkah ini memungkinkan terjadinya kerusakan sistem file, sistem operasi, hingga program yang saat itu masih terbuka.

2.Menerapkan shortcut combo. Ini dilakukan dengan memencet beberapa tombol yang ada di keyboard. Bahkan, Anda tidak perlu menggunakan campur tangan mouse untuk urusan ini. Pada sistem operasi Windows 7, penerapannya sangat sederhana. Tinggal tekan tombol Windows, tekan panah kanan, lalu tekan tombol Enter. Sementara pada Windows 10 hanya dengan menekan tombol Windows, lalu tekan X,u,u.

3.Membuat shortcut taskbar di laman desktop. Bisa pula mematikan komputer dengan mengeklik shortcut buatan sendiri yang diletakkan di desktop. Cara seperti ini:

Klik kanan pada desktop, tekan New lalu Shortcut.Untuk isian di kota Location, tuliskan perintah ini: “%windir%system32shutdown.exe -s -t 0” (tanpa tanda “)Klik Next, lalau namai shortcut sesuai keinginan.Shortcut yang sudah dibuat dapat diklik saat Anda ingin mematikan komputer.

22. Apakah yang harus diloakukan apabila sudah memverifikasi windows firewall tetapi tetap tidak berjalan?


Jawaban:

Panggil fungsi "Setup API InstallHinfSection" untuk menginstal Windows Firewall

Untuk menginstal Windows Firewall, ikuti langkah-langkah:

Klik Start, klik Run, ketik cmd, and klik OK.

Pada command prompt, ketik baris perintah berikut, kemudian tekan ENTER:

Rundll32 setupapi,InstallHinfSection Ndi-Steelhead 132 %windir%\inf\netrass.inf

Restart Windows,

Klik Start, klik Run, ketik cmd, dan klik OK.

Pada command prompt, ketik baris perintah berikut, kemudian tekan ENTER:

Netsh firewall reset

Klik Start, klik Run, ketik firewall.cpl, dan tekan ENTER. Pada kotak dialog Windows Firewall, klik On (recommended), kemudian klik OK.

26. Cara mematikan windows update windows 10 permanen adalah .....


Jawaban:

Cara-Cara

1. Buka Control Panel dengan cara mengetikkan “Control Panel” di kolom pencarian di samping Start Menu.

2. Kemudian, pilih System and Security dan tekan menu Administrative Tools.

3. Akan muncul halaman baru yang berisikan berbagam macam fitur dengan fungsi yang berbeda-beda. Di sini, Anda pilih Services saja.

4. Scroll ke bawah sampai Anda menemukan Windows Update, dan klik dua kali. Muncul halaman Windows Update Properties.

5. Sekarang, ganti opsi pada Startup Type menjadi Disabled, dan klik tombol Stop pada Service Status. Kemudian tekan Ok.

windows 7

Fitur di Windows 7 kurang complete.

Browser bawaan Microsoft yaitu Internet Explorer,hanya sampai versi 8.

windows  95

KELEBIHAN WINDOWS 95

Kinerja lebih baik dari Windows 95

Kecepatan waktu start up dan shutdown yang lebih cepat dari Windows 95

Sistem FAT32 (File Allocation Table 32-bit) yang ukuran clusternya lebih kecil dibandingkan FAT16, sehingga menghemat ruang harddisk, fungsi untuk menyimpan file dlm bentuk table.

Windows 98 juga memiliki kemampuan mendownload sendiri file-file update Windows 98.

Kemampuan multi display card yang dimiliki Windows 98

Internet Explorer 4.0 yang terintegrasi dengan desktop sehingga memudahkan browsing internetUtility baru untuk tune up dan repair Windows,peningkatan dalam menjalankan  

kemampuan multimedia baik dalam kualitas grafis maupun kualitas suara

Dukungan yang lebih baik bagi device baru seperti USB,DVD, dll

Windows 95Lebih cepat jika digunakan untuk mengakses internet Cocok untuk maen game-game ringan atau game playstation 1 dan jalannya pun stabil

KEKURANGAN WINDOWS 95

Tampilannya kurang menarik dan masih terlalu kaku.

Kestabilan sistem kurang  

Windows 98 lebih boros memori dan ruang harddisk daripada Windows 95

Sering terjadinya blue screen

windows 98

Kekurangan :

* Dukungan support sudah dicabut oleh produsennya (microsoft)

* Susah cari driver, karena produsen hardware menarik dukungan untuk win 98

* Tidak mendukung dan tidak bisa diinstall di perangkat terbaru (bila dipaksakan akan blue screen)

* Tidak ada versi 64bit

* Ringkih, mudah crash

* Kemampuan multi tasking yang sangat buruk

* Kemampuan multi thread yang sangat buruk

* Penanganan device yang buruk, DMA sering konflik

* Manajemen resource cpu yang kacau balau

* Tidak bisa membuat file dg ukuran lebih dari 2GB

* Tidak bisa mendeteksi memory besar (1GB masih stabil, dipaksa 2GB-hasil BlueScreen)

* Utility sangat terbatas, apalagi tool network

* Gampang terserang virus, gampang di hack, di crack, gampang di acak-acak

* Terlalu beresiko melakukan transaksi perbankan dari browser win 98, karena fitur networknya minim

* Tidak mendukung applikasi keluaran terbaru

* Tampilan terlalu sederhana (meskipun sempat dibanggakan di era nya)

Kelebihan :

* Cocok untuk hardware lama (misalnya pentium I,II,III)

* Ringan

* Ukuran installer kecil, bisa diinstal dari disket

* Bisa diinstall dari dos

* Bisa booting ke dos

* Cocok untuk development applikasi under dos terutama applikasi recovery

* Cocok untuk main game tempoe doeloe

* System (boot) mudah ditransfer ke media lain seperti cd dan disket

* Dos protected mode (DPMI) yang sempurna, baik untuk applikasi DOS32 dan DOS16

* Sangat cocok untuk development bagi programmer BIOS, atau programmer bahasa mesin

3. Jika tidak ada server Windows 2000 Server client tidak bisa mangoneksikan ke server begitupun sebaliknnya.

5. Terlalu banyak service yang tidak berguna.
