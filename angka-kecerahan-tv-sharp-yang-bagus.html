25. jawablah pertanyaan logika dan algoritma berikut dengan menggunakan simbol" logika. Hitunglah total bayar tv, yang mendapat diskon berdasarkan tanggal pembelian, inputlah nama pelanggan, jumlah tv, merek tv, dan tanggal pembelian. jika merek tv = sharp maka harga 1.840.000, jika merek tv = sonic maka harga 2.000.000, jika merek tv = toshiba maka harga 1.620.000, jika tanggal pembelian = 0 maka kembali mengimput dati awal, jika tanggal pembelian <=10 maka diakon = harga - 200.000, bila tidak diskon = harga - 0. rumus total harga = (diskon × jumlah)?.


﻿ Berikut adalah algoritma dari soal :

initialize nama_pelanggan, jumlah_tv, merek_tv, tanggal_pembelian, harga, diskon, total_harga

procedure daftar_harga( )

begin

read nama_pelanggan

read jumlah_tv

read merek_tv

read tanggal_pembelian

if merek_tv = "sharp" then

harga = 1840000

else if merek_tv = "sonic" then

harga = 2000000

else if merek_tv = "toshiba" then

harga = 1620000

end if

end procedure

begin

daftar_harga()

if tanggal_pembelian = 0 then

daftar_harga()

else if tanggal_pembelian <=10 then

diskon = harga - 200000

else

diskon = harga - 0

end if

total_harga = diskon*jumlah_tv

print total_harga

end main

PEMBAHASAN

Algoritma terdiri dari beberapa instruksi yang biasa digunakan oleh para programmer untuk mempermudah penulisan kode. Didalam algoritma terdapat beberapa perintah logika , seperti if-then (sebab-akibat) dan lain-lain.

Prosedur adalah sebuah fungsi yang dapat dipanggil berkali-kali tanpa mengembalikan nilai dari fungsi tersebut. Lain hal nya dengan fungsi , fungsi juga dapat dipanggil berkali-kali dengan mengembalikan nilai dari fungsi tersebut.

Kembali ke soal, di soal kita diminta untuk membuat algoritma dari pertanyaan logika tersebut. Pertama-tama, kita perlu membaca input dari nama pelanggan, jumlah tv, merek tv, dan tanggal pembelian. Karena jika tanggal pembelian sama dengan 0, maka kita harus membaca lagi input dari keempat variabel tersebut. Jika tanggal pembelian masih sama dengan 0, maka kita membaca lagi input dari keempat variabel tersebut. Hal ini jelas tidak efisien, sehingga diperlukan suatu prosedur untuk membaca input dan menyiapkan harga.

Jika tanggal pembelian kurang dari sama dengan 10, maka diskonnya adalah harga dikurangi 200000, selain itu diskonnya adalah harga dikurangi 0. Setelah itu, total harga didapat dari diskon dikali jumlah tv yang akan dibeli. Lalu cetak total harga yang harus dibayar oleh pelanggan.
