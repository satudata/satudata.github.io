4. Jika sambungan rel kereta api tidak dibuat renggang maka saat pemuaian rel akan


Jika sambungan rel kereta api tidak dibuat renggang maka saat pemuaian rel akan memanjang dan mengakibatkan membengkok.

PEMBAHASAN

A) Pengertian Pemuaian

Jika kita perhatikan rel kereta api dibuat celah dengan tujuan agar tidak terjadi pembengkokan rel kereta. Selain itu, pada bingkai jendela juga terdapat celah. Fungsi dari celah bingkai jendela tersebut adalah sebagai tempat pemuaian kaca sehingga kaca tidak pecah. Pemuaian kaca terjadi saat siang hari maupun malam hari. Faktor penyebabnya adalah perubahan suhu.

Dari uraian diatas, dapat disimpulkan bahwa pemuaian adalah bertambahnya besaran ukuran suatu benda karena kenaikan suhu yang terjadi pada benda tersebut. Kenaikan suhu yang terjadi menyebabkan benda tersebut mendapatkan tambahan energi berupa kalor sehingga partikel - partikel kecil benda begerak cepat.
