21. harga 2 kilo jeruk dan 3 kilo apel rp194.000 harga 5 kilo jeruk dan 2 kilo apel rp221.000 maka harga 3 kilo jeruk dan 2 kilo apel =​


Harga 3 kilo jeruk dan 2 kilo apel adalah Rp171.000,00.

Pembahasan:

Sistem Persamaan Linear Dua Variabel (SPLDV) adalah persamaan matematika yang terdiri dari dua variabel dan semua penyusunnya berpangkat satu.

Cara-cara menyelesaikan persamaan linear dua variabel:

Grafik. Prinsipnya, kedua persamaan dijadikan persamaan garis lurus, dan titik potong kedua garis merupakan penyelesaiannya.Substitusi. Prinsipnya mengubah nilai variabel lain ke dalam satu bentuk variabel saja.Eliminasi. Prinsipnya "menghilangkan" variabel lain hingga tersisa satu variabel dan konstanta saja.Gabungan. Pertama-tama gunakan eliminasi untuk mendapat salah satu nilai variabel, setelah itu substitusikan ke salah satu persamaan untuk mendapat nilai variabel lainnya.

-

Diketahui:

Harga 2 kilo jeruk dan 3 kilo apel adalah Rp194.000,00.

Harga 5 kilo jeruk dan 2 kilo apel adalah Rp221.000,00.

-

Ditanya:

Harga 3 kilo jeruk dan 2 kilo apel adalah...

Penyelesaian:

28. harga 1 kilo jeruk dan 3 kilo Apel adalah Rp90.000 sedangkan harga 2 kilo jeruk dan 1 kilo Apel adalah Rp55.000 Berapakah harga 3 kilo jeruk dan 1 kilo apel​

Diketahui :

1 kilo jeruk + 3 kilo Apel = Rp. 90,000

2 kilo jeruk + 1 kilo Apel = Rp. 55,000

Ditanyakan :

Harga 3 kilo jeruk + 1 kilo Apel ?

Jawab :

Soal ini membahas tentang sistem persamaan linear dua variabel atau SPLDV Didalam SPLDV terdapat beberapa metode yang digunakan yaitu metode eliminasi subsitusi grafik determinan dll..

Kita akan gunakan metode Eliminasi untuk mengerjakan ini , maka ;

Model matematika soal diatas adalah :

misal :

x = jeruk

y = Apel

maka,

x + 3y = 90,000

2x + y = 55,000

Metode Eliminasi :

eliminasi x untuk dapat kan nilai y maka :

x + 3y = 90,000 |×2

2x + y = 55,000 |×1

2x + 6y = 180,000

2x + y = 55,000

-------------------------- (-)

5y = 125,000

y = 125,000 : 5

y = 25,000

substitusi kan y = 25,000 ke :

2x + y = 55,000

2x + (25,000) = 55,000

2x = 55,000 - 25,000

2x = 30,000

x = 30,000 : 2

x = 15,000

jadi, harga

x = jeruk 1 kilonya Rp. 15,000

y = Apel 1 kilonya Rp. 25,000

maka, total harga dari :

= 3 kilo jeruk + 1 kilo Apel

= 3x + y

= 3(15,000) + (25,000)

= 45,000 + 25,000

= Rp. 70,000
