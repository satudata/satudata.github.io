2. TULISKAN LANGKAH - LANGKAH MEMBUKA MICROSOFT WORD DAN MICROSOFT EXCEL MELALUI KOMPUTER/LAPTOP! *​


Cara membuka Microsoft Excel

1. Melalui tombol Start. Klik tombol start, lalu pilih opsi All program.

2. Melalui shortcut dekstop. Klik dua kali (double klik) shortcut Microsoft Excel yang ada di dekstop.

3. Melalui pencarian Start. Klik tombol start, lalu pada kolom pencarian ketikkan Microsoft Excel, dan klik program Microsoft Excel.

Cara membuka Microsoft Word

1. Lakukan klik padda menut START.

2. Pilihlah All Program.

3. Pilihlah pada bagian Microsoft Office.

4. Pilihlah Microsoft Office Word 2007.

5. Tunggu load aplikasi.

6. Aplikasi Microsoft Word perlahan terbuka.

12. Tuliskan langkah-langkah membuka atau mengaktifkan aplikasi Microsoft Word minimal 2 cara​


Dalam membuka aplikasi perangkat lunak Microsoft Word, terdapat 4 cara yaitu:

Cara ini adalah cara yang paling mudah. Pengguna tinggal men-double click ikon Microsoft Word yang ada pada tampilan utama (desktop) komputer/laptop dengan menggunakan peranti keras bernama mouse . Namun, pastikan bahwa sudah terdapat ikon Microsoft Word dalam bentuk jalan pintas (shortcut).Jika pada desktop pengguna tidak ada shortcut Microsoft Word, maka cara mengaktifkannya adalah dengan menekan tombol start (logo Windows) yang ada di pojok sebelah kiri bawah pada panel utama (taskbar) menggunakan mouse . Kemudian pengguna dapat men-scroll mouse ke bawah untuk mencari program Microsoft Word, setelah itu tekan sekali ikon beserta tulisan Microsoft Word menggunakan tombol mouse sebelah kiri.Hampir sama dengan cara yang kedua, cara ini menggunakan menu start . Alih-alih menggunakan mouse, cara ini dapat menggunakan peranti keras bernama keyboard. Pengguna dapat menekan logo Windows pada keyboard, lalu akan muncul panel yang sama jika menggunakan mouse. Ketik kata kunci yaitu Word maka panel akan menunjukkan program yang dicari, kemudian tekan tombol enter pada keyboard maka program akan berjalan.Dalam penggunaan kedepannya jika ingin akses yang mudah, pengguna juga dapat membuat shortcut pada taskbar. Caranya adalah dengan terlebih dahulu membuka program Microsoft Word menggunakan salah satu dari 3 cara di atas. Setelah program sudah berjalan, pengguna tinggal menekan tombol sebelah kanan mouse pada ikon Microsoft Word yang aktif di taskbar, kemudian akan muncul panel kecil berisikan beberapa opsi dan pilihlah opsi pin to taskbar (Windows 8-later).

Pembahasan

Microsoft Word adalah sebuah perangkat lunak/program yang dikembangkan oleh perusahaan teknologi Microsoft. Program ini berfungsi untuk memproses kata dan membuat dokumen secara digital. Microsoft pertama kali meluncurkan program ini pada tahun 1983 dengan nama Multi-Word Tool. Kemudian, pengembang melakukan berbagai macam pembaruan pada program ini. Sehingga terdapat banyak versi yang ada di pasaran. Namun untuk saat ini, versi terbarunya dirilis tahun 2019 bernama Word yang menjadi satu dengan program yang lain di Microsoft Office 2019. Selain itu, Microsoft juga merilis program ini agar dapat digunakan dibeberapa Sistem Operasi, yaitu Windows, macOS, Android, dan iOs.
