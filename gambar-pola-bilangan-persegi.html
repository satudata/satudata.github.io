2. 1. Gambar kan pola bilangan persegi ke-7 2. Gambar kan pola bilangan segitiga ke-6 3. Gambar kan pola bilangan persegi panjang ke-8 4. Tentu kan suku ke-10 pada pola bilangan persegi panjang 5. Tentu kan suku ke-15 pada pola bilangan ganjil


1. Gambarkan pola bilangan persegi ke 7.

2. Gambarkan pola bilangan segitiga ke 6.

3. Gambarkan Pola Bilangan persegi panjang ke8

4. Tentukan suku ke - 10 pada bilangan persegi panjang

5. Tentukan suku ke 15 pada pola bilangan ganjil.

Jawab :

1. Pola bilangan Persegi :

U1 = 2

U2 = 4

U3 = 9

U4 = 16

U7 = ...

Rumus : Un = n².

maka,

[tex]U7 = {7}^{7} = \boxed{\red{49}}

[/tex]

2. Pola bilangan Segitiga

U1 = 1

U2 = 3

U3 = 6

U6 = ....

Rumus : Un = ½ × n (n + 1)

maka,

[tex]U6 = \frac{1}{2} × 6 \: (6+1) = \boxed{\red{21}}[/tex]

3. Pola Bilangan Persegi Panjang

U1 = 2

U2 = 6

U3 = 12

U8 = ....

Rumus : Un = n (n + 1)

maka,

[tex]U8 = 8 (8+1) = 8 × 9 = \boxed{\purple{72}}[/tex]

4. Pola Bilangan Persegi Panjang

U1 = 2

U2 = 6

U10 = ...

Rumus : Un = n (n +1)

maka,

[tex]U10 = 10 (10+1) = 10 × 11 = \boxed{\pink{110}}[/tex]

5. Pola bilangan Ganjil

U1 = 1

U2 = 3

U3 = 5

U15 = ...

Rumus : Un = 2n - 1.

maka,

[tex]U15 = 2 × 15 - 1 = 30 - 1 = \boxed{\blue{29}}[/tex]

Kesimpulan :Pola bilangan Persegi ke 7 adalah 49Pola bilangan segitiga ke 6 adalah 21Pola bilangan persegi panjang ke 8 adalah 72Pola bilangan persegi panjang ke 10 adalah 110Pola bilangan ganjil ke 15 adalah 29.
