Apa sajakah yang penting diketahui mengenai String?

Di bawah ini adalah penjelasan mengenai String :

1. Cara membuat objek String pada java ada 2 cara.

Untuk membuat objek String dapat dilakukan dengan dua cara, yaitu:

String a = “halo”; //membuat objek String dengan String literal

String a = new String(“halo”); //membuat objek String dengan operator new

2. String bersifat Immutable dan final

String bersifat immutable, artinya sekali sudah dibuat, maka kita tidak dapat memodifikasi isi dari String tersebut. Jika kita memodifikasi String dengan method toLowerCase(), toUpperCase() ataupun method lainnya, maka akan selalu menghasilkan String yang baru. Dikarenakan String adalah final, maka tidak ada jalan siapapun untuk melakukan extend String atau meng-override fungsional apapun dari String.

3. String diatur dalam String pool

Setiap kali kita membuat objek String dengan menggunakan String literal, maka Java Virtual Machine (JVM) pertama kali akan memeriksa konten dari objek yang dibuat. Jika terdapat objek yang sama dalam String pool yang memiliki konten yang sama, maka referensi akan dikembalikan pada objek tersebut, tanpa membuat objek baru. JVM tidak melakukan pemeriksaan String pool jika kita membuat objek dengan menggunakan operator new.

6. Berapa banyak string 10 bit yang banyaknya bit 1 string tersebut sama dengan banyaknya bit 0 ?


Dalam 1 bit string kemungkinan nilai yang sesuai adalah antara 0 atau 1. Maka dalam 10 bit string akan terdapat 10 nilai yang berisi 0 atau 1.

Misal : 1110001110, 0000011100.

Dan untuk memenuhi syarat, banyaknya jumlah bit yang bernilai 1 sama dengan jumlah bit yang bernilai 0, maka string tersebut akan terdiri dari 5 bit bernilai 0 dan 5 bit bernilai 1. Contoh string yang memenuhi syarat tersebut antara lain 1111100000, 1010101010, 0101010101, 0001101110, dst.

Oleh karena itu kita bisa menggunakan rumus permutasi yang mengandung k unsur yang sama yaitu

[tex]P(n,k1,k2,..kt) = \frac{n!}{k1!k2!...kt!}[/tex]


Pembahasan

Dengan persyaratan 5 bit string bernilai 1 dan 5 bit string bernilai 0, maka banyak kemungkinan string yang terjadi adalah

[tex]P(n,k1,k2) = \frac{10!}{5!5!}[/tex]

dimana

k1 adalah syarat 5 bit string harus 1

k2 adalah syarat 5 bit string harus 0

Maka banyaknya kemungkinan 10 bit string dengan syarat 5 bit bernilai 1 dan 5 bit bernilai 0 adalah

[tex]P(n,k1,k2) = \frac{10!}{5!5!}[/tex]

                       = 252

Banyaknya string dengan bobot 5 yang mampu disusun oleh programmer tersebut adalah 126

Penjelasan:

Dalam menghitung bobot dari programmer kita dapat menggunakan permutasian dengan cara berikut yaitu:

(1) 2,2,1,0,0,0 = [tex]\frac{6!}{2!3!} = 60[/tex]

(2) 1,1,1,2,0,0 = [tex]\frac{6!}{3!21} = 60[/tex]

(3) 1,1,1,1,1,0 = [tex]\frac{6!}{5!} = 6[/tex]

Sehingga total adalah 126.

Urutan, di sisi lain, menggabungkan beberapa objek dari  grup dengan memperhatikan urutannya. Dalam urutan, urutan itu penting. Contoh: Saya memiliki sebuah kotak dengan tiga bola  merah, hijau dan biru. Istilah urutan mengacu pada beberapa cara untuk mengatur satu set objek secara berurutan. Kombinasi ini berarti banyak cara untuk memilih item dari sekumpulan besar objek, jadi urutannya tidak masalah. Urutan ini digunakan untuk menentukan berapa banyak elemen berbeda yang dapat ditempatkan dalam lingkaran. Barisan adalah susunan dari berbagai unsur  dalam urutan tertentu. Atau, perhatian diberikan pada penempatan pesanan. Kombinasi adalah susunan yang tidak memperhatikan urutan.

Bahasa Pascal

Dalam bahasa Pascal, tipe data string dituliskan sebagai string. Pada kompilator Pascal lama, string terpanjang yang dapat dibuat terdiri dari 255 karakter. Hal ini terjadi karena Pascal menggunakan 1 byte data untuk menyimpan panjang efektif string.

Contoh kode Pascal yang menggunakan tipe data string:

program str;

var s: string[100];  //jika panjang maksimum tidak dimasukkan, kompilator akan menganggapnya 255

begining

 su:= 'Hello world';

Bahasa C

Dalam bahasa C tidak ada tipe data khusus untuk menyimpan string. String biasanya disimpan sebagai larik dari karakter-karakter. Berbeda dengan bahasa Pascal, string pada bahasa C merupakan sebuah null-terminated string, sehingga kapasitasnya hanya dibatasi oleh kapasitas memori komputer. Seperti pada tipe data larik pada bahasa C, pengguna dapat memberikan batas maksimum karakter yang dapat ditampung.

Contoh kode C yang menggunakan tipe data string:

#include <stdio.h>

int main(){

 char s[100]; /* membuat string dengan panjang maksimum 99 karakter, karena karakter terakhir harus null */

 s = "Hello world";

 printf("%s\n", s);

}

Bahasa C++

Bahasa C++ merupakan turunan dari bahasa C sehingga representasi string sebagai larik karakter masih berlaku. Namun bahasa C++ juga menyediakan tipe data string yang terdapat dalam C++ Standard Template Library (STL).

Contoh kode C++ yang menggunakan tipe data string pada STL:

#include <string>

#include <iostream>

using namespace std;

int main(){

 string s;  //panjang maksimum tidak perlu didefinisikan

 s = "Hello world";

 cout<<s<<endl;

}

23. Seorang programer akan membuat kata sandi berupa string terdiri dari 6 angka. Programer tersebut berkeinginan menggunakan angka 0,1 atau 2. Bobot string didefinisikan sebagai jumlah angka- angka dalaam string tersebut. Banyaknya string dengan bobot 5 yang dapat disusun oleh programer tersebut adalah


Kelas: 11
Mapel: Matematika
Kategori: Peluang
Kata kunci: permutasi unsur yang sama
Kode: 11.2.2 (Kelas 11 Matematika Bab 2-Peluang)

Seorang programer akan membuat kata sandi berupa string terdiri dari 6 angka. Programer tersebut berkeinginan menggunakan angka 0,1 atau 2. Bobot string didefinisikan sebagai jumlah angka- angka dalaam string tersebut. Banyaknya string dengan bobot 5 yang dapat disusun oleh programer tersebut adalah

Pembahasan:

kemungkinan nya adalah :
0-1-1-1-1-1
0-0-1-1-1-2
0-0-0-1-2-2

kemungkinan diatas masih bisa bertukar posisi bilangan nya, sekarang gunakan permutasi unsur yang sama, jika dari n buah bilangan terdapat k, l, dan n unsur yang sama, maka kemungkinan nya adalah:
[tex]P= \frac{n!}{k!\times l!\times m!} [/tex]

untuk bilangan : 0-1-1-1-1-1
seluruh bilangan = 6 buah
bilangan yang sama:
1 = 5 buah
banyak kemungkinan [tex] =\frac{6!}{5!} = \frac{6\times 5!}{5!}=6 [/tex]

untuk bilangan : 0-0-1-1-1-2
seluruh bilangan = 6 buah 
bilangan yang sama:
0 = 2 buah
1 = 3 buah
banyak kemungkinan [tex]= \frac{6!}{2!\times 3!}= \frac{6\times 5\times 4\times 3!}{2\times 1\times 3!}= \frac{120}{2}=60 [/tex]

untuk bilangan : 0-0-0-1-2-2
seluruh bilangan = 6 buah
0 = 3 buah
2 = 2 buah
banyak kemungkinan [tex]= \frac{6!}{2!\times 3!}= \frac{6\times 5\times 4\times 3!}{2\times 1\times 3!}= \frac{120}{2}=60 [/tex]

Banyaknya string dengan bobot 5 yang dapat disusun oleh programer tersebut
= 6 + 60 + 60
= 126
