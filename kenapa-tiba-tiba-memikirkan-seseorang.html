10. Quiz Pelit__________________________Udin sedang bersama mantannya untuk melakukan kencan pertama mereka di sebuah restoran cepat saji. Tiba tiba, steak yang sedang dimakan Udin jatuh ke lantai restauran. Udin langsung panik dan memikirkan soal matematika :[tex] \sf {}^{4} log \: 1.024 + {}^{4}log \: 16.384 = ....[/tex]______Mantap Udin, pacarmu pasti bangga.​


Hasil dari ⁴log1024 + ⁴log16.384 sama dengan 12

-PENDAHULUAN-

Pengertian Logaritma

Logaritma ialah operasi invers yg merupakan kebalikan dari perpangkatan dengan penulisan a log b = c <—> a² = c.

ket:

a => basis atau bisa disebut dgn bilangan pokok.

b => numerus

c => hasil dari logaritma

-PEMBAHASAN-

⁴log1024 + ⁴log16.384

= ⁴log4⁵ + ⁴log4⁷

= 5 + 7

= 12

Kesimpulan:

Jadi, Hasil dari ⁴log1024 + ⁴log16.384 sama dengan 12
