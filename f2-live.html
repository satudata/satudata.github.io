6. Dua vektor gaya F1 dan F2 bertitik tangkap sama saling mengapit sudut a. Jika | F1 + F2 | = √3| F1 - F2 ¦ dan F1 = F2 = F, maka nilai a adalah


VEKTOR

Dua vektor gaya F₁ dan F₂ bertitik tangkap sama saling mengapit sudut α.

BBESAR jumlah kedua vektor
|F₁ + F₂|² = F₁² + F₂² + 2 F₁ F₂ cos α

BESAR selisih kedua vektor
|F₁ - F₂|² = F₁² + F₂² - 2 F₁ F₂ cos α

untuk
• F₁ = F₂ = F
• |F₁ + F₂| = √3 |F₁ - F₂|

diperoleh
|F₁ + F₂| = √3 |F₁ - F₂| ... kuadratkan
|F₁ + F₂|² = 3 |F₁ - F₂|² ... substitusi
(F² + F² + 2F² cos α) = 3 (F² + F² - 2F² cos α)
2F² + 2F² cos α = 3 (2F² - 2F² cos α)
2F² + 2F² cos α = 6F² - 6F² cos α)
8F² cos α = 4F² ; F ≠ 0
2 cos α = 1
cos α = ½
α = 60° ✔️

28. Dua buah vektor gaya F1 dan F2 bertitik tangkap sama saling mengapit sudut a. Jika |F1 + F2| = √3 |F1 - F2| dan |F1| =2 |F2|, hitunglah a


Vektor merupakan besaran yang memiliki arah, untuk menentukan besar perpindahan suatu benda dapat digunakan vektor satuan  

Pembahasan  

Untuk menyelesaikan soal diatas gunakan rumus cosinus penjumlahan vektor

|F1| =2 |F2|

misalkan |F2|=p maka  |F1|=2p

gunakan persamaan yang diketahui

|F1 + F2| = √3 |F1 - F2|  

[tex]\sqrt{(2p)^2+p^2+2.2p.p.cosa} =\sqrt{3} \sqrt{(2p)^2+p^2-2.2p.p.cosa} \\4p^2+p^2+4p^2cosa=3(4p^2+p^2-4p^2cosa)\\5p^2+4p^2cosa==15p^2-12p^2cosa\\16cosa=10\\cosa=\frac{5}{8} \\a=51,3[/tex]

Jadi sudut a adalah 51,3
