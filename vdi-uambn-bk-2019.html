3. manakah yang lebih unggul antar VDI, VHD DAN VMDK


Pengalokasian ukuran virtual drive secara dinamis : VDI, VHD, VMDK dapat mendukung fitur tersebut

Bisa melakukan snapshot : Semua format dapat melakukan snapshot

Kekuatan performa tidak berdampak pada tipe virtual drive, tetapi merefleksikan kepada performa Host itu sendiri dari segala aspek seperti Storage, CPU, GPU, dan resource lainnya

Jika ingin memindahkan data pada nantinya tanpa harus khawatir akan ukurannya yang besar, maka VMDK adalah pilihannya. VMDK mendukung pemisahan drive kurang lebih 2GB per ukuran virtual drive. Dan juga sebaliknya, jika ingin file yang compact pilih VHD atau VDI.
