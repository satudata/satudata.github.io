13. 100MB sama dengan berapa GB


100 MB = 0,098 GB. Soal ini membahas konversi satuan kapasitas penyimpanan file dalam komputer.

Pembahasan

MB itu singkatan dari Megabyte (Indonesia: Megabita). Sedangkan, GB itu singkatan dari Gigabyte (Indonesia: Gigabita). MB maupun GB adalah satuan kapasitas media penyimpanan dalam komputer. Perlu diingat, bedakan MB dengan Mb ataupun GB dengan Gb. Kalau Mb itu Megabit dan Gb itu Gigabit. Mb = 1/8 MB dan Gb = 1/8 GB. Sedangkan 1 GB = 1024 MB. Maka,

100 MB = 100/1024 = 0,098 GB

29. berapa persentase compress file jika datanya 300mb ke 100mb?​


(300 - 100) / 300 x 100% = 66.67%

Jadi, persentase kompresi dari file 300MB menjadi 100MB adalah 66.67%.


30. Kenapa Jika Kita Mendownload Video Di suatu Web File Nya Lebih Besar Atau Lebih KecilMisal Di web A 720p =100mbDan Di Web b 720p =200 mbTolong Dijelaskan kaka​


Penjelasan:

Karena web tersebut memiliki perbedaan dalam sisi kompresi meski resolusi sama.

untuk video, kompresi dibagi menjadi 2. yaitu video itu sendiri dan suara (sound).

faktor kompresi untuk video banyak, salah satunya ukuran. tetapi faktor sample rate , image quality, dan format yang codec yg digunakan juga pengaruh (H264, mp4, avi, etc)

untuk audio, yg menentukan adalah sampe rate (192k,128k,etc.) dan juga codec (aac, mp3, wav, etc).

makanya size bisa beda.
