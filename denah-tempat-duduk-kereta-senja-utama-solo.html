9. \Kereta Api Senja Utama berangkat dari Jakarta pukul 06.20 akan 11.35, maka kecepatan kereta api adalah .......km/jam di stasiun Cirebon selama 15 menit. Jika kereta tiba di Pekalongan pukul menempuh perjalanan 450 km menuju kota Pekalongan. Kereta berhenti


Jawab:

90 km/jam

Penjelasan dengan langkah-langkah:

Maksudmu:

Kereta Api Senja Utama berangkat dari Jakarta pukul 06.20 akan menempuh perjalanan 450 km menuju kota Pekalongan. Kereta berhenti di stasiun Cirebon selama 15 menit. Jika tiba di Pekalongan pukul, 11.35, maka kecepatan kereta api adalah:

Diketahui:

Jarak = 450 kmWaktu keberangkatan = 06:20Layover di Cirebon 15 menitWaktu ketibaan = 11:35

Ditanya: kecepatan rata-rata

Kecepatan rata-rata = Jarak total/waktu total

Kecepatan = 450 km/((11:35-00:15)-06:20)

Kecepatan = 450 km/((11:20-06:20)

Kecepatan = 450 km/5 jam

Kecepatan rata-rata = 90 km/jam

10. Kereta api senja utama biasanya sampai di stasiun gambir pukul 05.00namun hari ini terlambat 2 1/2 jam pukul berapa kereta api itu tiba.


-----------------------------------
> PEMBAHASAN SOAL <
-----------------------------------

• Karena kereta tersebut mengalami keterlambatan waktu, maka waktu tibanya adalah Waktu Biasanya dengan Jumlah Waktu Keterlambatannya

---------------------------------
> KESIMPULAN SOAL <
---------------------------------

2 ½ Jam = 2 Jam + (½ x 60) Menit
2 ½ Jam = 2 Jam + 30 Menit

Waktu Tiba = 05.00 + 2 Jam 30 Menit = 07.30

----------------------------------------
Materi : Satuan Waktu
Kelas : III
Kategori : Operasi Satuan Waktu

Jawaban:

Kecepatan kereta tersebut adalah 86 2/3 km/jam

Penjelasan dengan langkah-langkah:

Jarak = 260 km

Waktu = 10.30 - 07.30 = 3 jam

Jarak = Kecepatan x Waktu

260 km = Kecepatan x 3

Kecepatan = 260 km/3jam

Kecepatan = 86 2/3 km/jam
