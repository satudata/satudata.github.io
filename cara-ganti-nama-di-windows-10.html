14. bagaimana cara mengganti windows 7 ke windows10?


Jawaban:

yang telah menjalankan Windows 7 atau Windows 8 versi legal. Sistem operasi versi Home akan di-upgrade menjadi Windows 10 Home sedangkan versi Pro akan di-upgrade menjadi Windows 10 Pro.

Setelah memahami ketentuannya, berikut cara upgrade Windows 10 secara gratis di perangkat komputer kalian seperti dikutip Forbes:

1. Masuk ke situs download Windows 10.

2. Klik 'Download Tool Now' untuk mengunduh Windows 10 Media Creation Tool.

3. Setelah proses download selesai, buka file yang diunduh dan terima syarat dan ketentuan lisensi.

4. Pilih opsi 'Upgrade this PC now' lalu klik Next.

5. Pilih opsi 'Keep personal files and apps'.

6. Klik 'Install'. Proses ini akan memakan waktu cukup lama dan memerlukan beberapa kali restart.

7. Setelah selesai install buka aplikasi Settings > Windows Update > Activation untuk mengaktivasi lisensi digital Windows 10.

8. Atau masukkan lisensi Windows 7 atau Windows 8/8.1 yang asli dan legal jika kalian belum pernah mengaktivasi sistem operasi versi lama tersebut.

Penjelasan:

Untuk memastikan jika lisensi kalian sudah teraktivasi dengan benar, buka aplikasi Settings > Update & Security > Activation

22. tuliskan langkah langkah mengganti nama file menggunakan windows explorer


- Buka Windows Explorer dengan cara klik Start - My Computer atau tekan tombol Windows+E di keyboard, kemudian pilih My Computer
- Selanjutnya, cari direktori/tempat file yang akan diganti namanya
- Kalau sudah ketemu filenya, klik 1x file tersebut, lalu klik kanan dan pilih Rename
- Kemudian, ganti namanya dan tekan tombol Enter di keyboard
Semoga membantu...1. Buka Windows Explorer
2. Cari file yang ingin di ganti namanya (Rename)
3. Klik file-nya
4. Klik kanan
5. Klik Rename
6. Lalu tulis nama file yang baru
