PENDAHULUAN

Pengertian Teorema Pythagoras :

Teorema Pythagoras adalah Jumlah kuadrat sisi siku-sikunya sama dengan sisi miringnya(hepotenusa).

Penerapan Teorema Pythagotas :

Terdapat berbagai permasalahan yang dapat di selesaikan dengan menggunakan teorema pythagoras.Guna memudahkan dan menyelesaikannya,diperlukan bantyan sketsa.

Contoh soal :

Sebuah kapal berlayar dari pelabuhan O menuju ke pelabuhan A ke arah timur sejauh 60 mill kemudian menuju ke pelabujan B ke arah selatan sjauh 80 mill.Tentukan jarak perpindahan kapal dari pelabuhan O menuju ke pelabuhan B...?

Jawaban :

[tex]OB²=OA²+AB²[/tex]

[tex]OB=\sqrt{OA²+AB²}[/tex]

[tex]=\sqrt{60²+80²}[/tex]

[tex]=\sqrt{3.600+6.400}[/tex]

[tex]=\sqrt{10.000}[/tex]

[tex]=100 mil=[/tex]

Jadi perpindahan kapal dari pelabuhan adalah 100 mill

Pengertian Tripel Pythagoras :

Tripel pythagoras adalah kelompok 3 bilangan bulat positif yang memenuhi kuadrat bilangan tetbesar sama dengan jumlah kuadrat dua bilangan lain.

Cara menentukan dan memeriksa Tripel Phthagoras :

Teorema pyhagoras,dapat digunakan untuk menentukan kumpulan bilangan yang merupakan tripel pythagoras.Caranya dengan menguadratkan bilangan 1 dan bilangan ke 2 kemudian di tambah dan hasilnya merupakan bilangan ke 3.

Contoh Soal :

Perhatikan kelompok 3 bilangan ini :

a.)6,8 dan 10b.)5,12 dan 13

Apakah bilangan tersebut merupakan sisi-sisi segitiga siku siku ?

Jawaban a :

[tex]a.) \sf {a.6,8\: dan\: 10}[/tex]

[tex]\sf {10²=100}[/tex]

[tex]\sf {6²+8²=36+64}[/tex]

[tex]\sf {=100}[/tex]

Karena 10²=6²+8²,maka segitiga tersebut merupakan segitiga siku-siku ✔

Jawaban b :

[tex]b.)\sf{5,12,dan\: 13}[/tex]

[tex]\sf{13²=169}[/tex]

[tex]\sf{5²+12²=25+144}[/tex]

[tex]\sf{=169}[/tex]

Karena 13²=5²+12²,maka segitiga tersebutmerupakan segitiga siku-siku.✔

Berdasarkan hasil penjabaran di atas ,kelompok 3 bilangan ( 6,8,10 dan 5,12,13 ) merupakan segitiga siku-siku ,karena kedua kelompok bilangan tersebut memenuhi teorema Pythagoras maka kedua kelompok tersebut dinamakan tripel pithagoras.

29. Tentukan Keliling Segitiga ABC


#Matematika SMP

Step-1
Perhatikan segitiga siku-siku ABD, dengan ∠BAD = 60° dan ∠ABD = 30° 
Menurut ketentuan dalam sebuah segitiga siku-siku yang memuat sudut 30° dan 60°, maka perbandingan panjang sisi-sisi dalam ΔABD adalah,

                                    AB : AD : BD = 2 : 1 : √3

Sekarang kita hitung panjang sisi AD
AB : AD = 2 : 1
8 : AD = 2 : 1
2 x AD = 8
AD = 4 cm ⇒ digunakan dalam perhitungan pada ΔACD
Selanjutnya mencari panjang sisi BD
AB : BD = 2 : √3
8 : BD = 2 : √3
2 x BD = 8√3
BD = 4√3 cm

Step-2
Perhatikan segitiga siku-siku ACD, dengan ∠ACD = 60° dan ∠CAD = 30° 
Sekali lagi, menurut ketentuan maka perbandingan panjang sisi-sisi dalam ΔACD adalah,

                                    AC : CD : AD = 2 : 1 : √3

Menghitung panjang sisi AC
AC : AD = 2 : √3
AC : 4 = 2 : √3
√3 x AC = 8 cm
AC = 8/√3 ⇒ setelah dikali akar sekawan menjadi AC = [8√3] / 3 cm 

Mencari panjang sisi CD
CD: AD = 1 : √3
CD: 4 = 1 : √3
√3 x CD = 4
CD = 4/√3 ⇒ setelah dikali akar sekawan menjadi CD = [4√3] / 3 cm 

Step-3
BC = BD + CD
BC = 4√3 + [4√3] / 3
BC = [16√3] / 3 cm

Final step
Keliling ΔABC = AB + BC + AC
                       = 8 + [16√3] / 3 + [8√3] / 3
                       = 8 + [16√3 + 8√3] / 3
                       = 8 + [24√3] / 3
                       = (8 + 8√3) cm    
Jadi, keliling segitiga ABC sebesar (8 + 8√3) cm

Materi SMP :
Gunakan perbandingan sudut 60° dan 30° = AB : AC : BC = √3 : 1 : 2

sudut ABC = 30°

AB : BC = √3 : 2
8 / BC = √3 / 2
BC√3 = 2 × 8
BC = 16/√3
BC = 16/√3 × √3/√3
BC = 16/3 √3 cm

sudut ACB = 60°

AB : AC = √3 : 1
8 / AC = √3 / 1
√3 AC = 8 × 1
AC = 8 / √3
AC = 8/√3 × √3/√3
AC = 8/3 √3 cm

keliling ∆ ABC = AB + BC + AC
= (8 + 16/3 √3 + 8/3 √3) cm
= (8 + 24/3 √3) cm
= (8 + 8√3) cm
= 8 (1 + √3) cm
