1. Cara mencari file file yang hilang pada laptop cara nya gimana?


Kemungkinan pertama. Kalau "hilang". Tinggal ketik nama file yang hilang di Search Bar di Windows Explorer (Yang ada tulisan "Search" di pojok kanan agak atas). Sebelum mengetik, pilih folder dimana kemungkinan file hilang. Kalau belum ketemu, ganti folder lain atau ganti ketikan nama yang dikira lebih sesuai.

Kemungkinan kedua. Ternyata bukan hilang, melainkan terhapus (belum permanen). Biasanya file masih terdapat di Recycle Bin di Windows Explorer. Tinggal cari gambar tempat sampah (Recycle Bin) >> Klik 2x >> Cari file yang terhapus >> Klik 2x (Perhatikan Origin, disitulah tempat file nanti kembali) >> Pilih Restore. File akan kembali ke lokasi di Origin tadi.

Kemungkinan ketiga. File terhapus permanen, sudah dicari di Recycle Bin tidak ada berarti file terhapus permanen dari komputer.

Kemungkinan tambahan. File terkena virus dan disembunyikan di suatu tempat. Ini agak susah, file yang disembunyikan virus besar kemungkinan kalau virusnya dihapus file ikut terhapus. Cari di google kalau yang ini.

14. Apakah file yang sudah ada di Onedrive tetap ada jika kita berhenti mengaitkanya pada laptop?


Anda tidak akan kehilangan file atau data dengan menghapus linkOneDrive dari komputer Anda. Anda tetap dapat mengakses file dengan masuk ke OneDrive.com.

1. Pilih ikon awan OneDrive berwarna putih atau biru di bilah tugas atau bilah menu.

Catatan: Anda mungkin perlu mengklik panah Perlihatkan ikon tersembunyi Tombol Perlihatkan ikon tersembunyi. di samping area pemberitahuan untuk melihat ikon OneDrive . Jika ikon tidak muncul di area pemberitahuan, OneDrive mungkin tidak berjalan. Pilih Mulai, ketikkan OneDrive dalam kotak pencarian, lalu pilih OneDrive dalam hasil pencarian

2. Pilih pengaturan> Ikon Bantuan dan Pengaturan OneDrive Bantuan & Pengaturan .

3. Pada tab Akun, klik Lepas tautan PC ini, lalu Lepas tautan akun.

Apa yang seharusnya dilakukan Farhan agar lebih mudah mencari file tersebut?​


Jawaban:

Misalkan laptop Farhan memakai sistem operasi Windows 10:

- saat sudah tampil di File Explorer, klik This PC

- klik di kotak bertuliskan Search This PC

- ketikkan nama file yg ingin dicari dan tekan tombol Enter

Penjelasan tambahan:

Amatilah lampiran (sebagai contoh tampilan File Explorer):

→ This PC ditunjukkan tanda panah biru

→ Search This PC ditunjukkan tanda panah hijau

Adalah tidak apa-apa bila Farhan lupa menyimpan file miliknya, namun setidak-tidaknya Farhan harus ingat nama file miliknya. Contoh nama file:
