3. dua suku berikutnya dari pola: 4, 8, 14, 22, ..., adalah


Dua suku berikutnya dari pola: 4, 8, 14, 22, ..., adalah

Pembahasan :

Kita lihat pola bilangannya dengan cara mencari selisih antar dua sukunya
4 .... 8 .... 14 .... 22 ...... ___ ..... ___
.. +4 .. +6 ... +8 ... +10 ...... +12

Jadi bilangan tersebut selalu bertambah 4, 6, 8, 10, 12, 14 dan seterusnya
22 + 10 = 32
32 + 12 = 44

4, 8, 14, 22, (32), (44)
Jadi dua suku berikutnya adalah 32 dan 44

Jika yang ditanyakan rumus suku ke n :
Bisa menggunakan rumus aritmatika bertingkat
4 .... 8 .... 14 .... 22 ...... 32 ..... 44
.. +4 .. +6 ... +8 ... +10 .... +12
.......+2.....+2.....+2.......+2

angka pertama baris 1 => a + b + c = 4
angka pertama baris 2 => 3a + b = 4
angka pertama baris 3 => 2a = 2

2a = 2
=> a = 1

3a + b = 4
=> 3(1) + b = 4
=> b = 1

a + b + c = 4
1 + 1 + c = 4
c = 2

Jadi suku ke n nya adalah
Un = an² + bn + c
Un = 1n² + 1n + 2
Un = n² + n + 2
