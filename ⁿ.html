Penjelasan dengan langkah-langkah:

Misal a = 3ⁿ dan b = 3⁻ⁿ

Diketahui 3ⁿ - 3⁻ⁿ = 3, maka:

a² + b² = (a - b)² + 2ab

(3ⁿ)² + (3⁻ⁿ)² = (3ⁿ - 3⁻ⁿ)² + 2(3ⁿ)(3⁻ⁿ)

(3ⁿ)² + (3⁻ⁿ)² = 3² + 2(3ⁿ⁻ⁿ)

(3ⁿ)² + (3⁻ⁿ)² = 9 + 2(1)

(3ⁿ)² + (3⁻ⁿ)² = 11

a² + b² = (a + b)² - 2ab

(3ⁿ)² + (3⁻ⁿ)² = (3ⁿ + 3⁻ⁿ)² + 2(3ⁿ)(3⁻ⁿ)

11 = (3ⁿ + 3⁻ⁿ)² + 2

3ⁿ + 3⁻ⁿ = √13

3³ⁿ + 3⁻³ⁿ = (3ⁿ)³ + (3⁻ⁿ)³

a³ + b³ = (a + b)(a² - ab + b²), maka:

(3ⁿ)³ + (3⁻ⁿ)³ = (3ⁿ + 3⁻ⁿ) [(3ⁿ)² + (3⁻ⁿ)² - 3ⁿ (3⁻ⁿ)]

3³ⁿ + 3⁻³ⁿ = √13 (11 - 1)

3³ⁿ + 3⁻³ⁿ = 10√13

log (3³ⁿ + 3⁻³ⁿ) : log [10(3ⁿ + 3⁻ⁿ)]

= log (10√13) : log (10√13)

= 1

Jika 3ⁿ-3⁻ⁿ = 3, maka nilai dari
[log(3³ⁿ+3⁻³ⁿ)] ÷ [log(10(3ⁿ+3⁻ⁿ)] = 1.

[tex]\log\left(3^{3n}+3^{-3n}\right)\div\log\left(10\left(3^{n}+3^{-n}\right)\right)[/tex]

dengan [tex]3^n-3^{-n}=3[/tex] dapat dinyatakan juga oleh

[tex]\begin{aligned}&\frac{\log\left(3^{3n}+3^{-3n}\right)}{\log\left(10\left(3^{n}+3^{-n}\right)\right)}=\frac{\log\left(a^3+b^3\right)}{\log\left[10\left(a+b\right)\right]}\end{aligned}[/tex]

dengan [tex]a = 3^n[/tex], [tex]b=3^{-n}[/tex], dan

[tex]\begin{aligned}a^3+b^3&=(b+3)^3+b^3\\&=b^3+9b^2+27b+27+b^3\\&=2b^3+9b^2+27b+27\\&=(2b+3)b^2+6b^2+27b+27\\&=(2b+3)b^2+(2b+3)3b+18b+27\\&=(2b+3)b^2+(2b+3)3b+(2b+3)9\\&=(2b+3)\left(b^2+3b+9\right)\\&=(2b+3)\left(b(b+3)+9\right)\\&=(2b+3)\left(ab+9\right)\\&=(2b+3)\left(3^n\cdot3^{-n}+9\right)\\&=(2b+3)\left(1+9\right)\\a^3+b^3&=10(2b+3)=10(a+b)\\\end{aligned}[/tex]
