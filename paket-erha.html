Jawaban:

E. x² - 6x + 5

Penjelasan dengan langkah-langkah:

Diketahui f(x) = x - 2 dan g(x) = x² - 2x - 3.

(g o f)(x) = g(f(x))

= g(x - 2)

= (x - 2)² - 2(x - 2) - 3

= x² - 4x + 4 - 2x + 4 - 3

= x² - 6x + 5

Rumus (g o f)(x) adalah x² - 6x + 5.

6. Banyak sumbangan sebanyak 240 paket dengan perbandingan 8÷9÷7 yaitu paket alquran÷paket makanan÷paket pakaian .selisih antara paket makanan ÷ paket pakaian adalah


Jumlah sumbangan 240 paket

Perbandingan paket; al-Quran : makanan : pakaian ➡ 8 : 9 : 7

Selisih paket makanan dan pakaian

= (9-7)/(8+9+7) × 240

= 2/24 × 240

= 20 paket

paket Cargo = y

( I ) y - x = 50

y = -20/-2

y = 10
