9. hitunglah selisih luas kamar tidur anak dan kamar tidur pembantu sebenarnya!


Kamar tidur anak =>

Luas sebenarnya = Luas pada peta : Skala

~ panjang sebenarnya ~

=> 4 cm : 1/150

=> 4 cm × 150/1

=> 600 cm/1

=> 6 m

~ Lebar sebenarnya ~

=> 3 cm : 1/150

=> 3 cm × 150/1

=> 450 cm

=> 4,5 m

Luas sebenarnya kamar tidur anak

=> p x l

=> 6 m x 4,5 m

=> 27 m^2

**************************************************

Kamar tidur pembantu =>

Luas sebenarnya = Luas pada peta : Skala

~ panjang sebenarnya ~

=> 3 cm : 1/150

=> 3 cm × 150/1

=> 450 cm

=> 4,5 m

~ Lebar sebenarnya ~

=> 2,5 cm : 1/150

=> 2,5 × 150/1

=> 375 cm

=> 3,75 m

Luas sebenarnya kamar tidur pembantu

=> p x l

=> 4,5 m x 3,75 m

=> 16,875 m^2

Selisih luas kamar tidur anak dengan kamar tidur pembantu sebenarnya

=> 27 m^2 - 16,875 m^2

=> 10,125 m^2

10. Pada gambar denah sebuah rumah tertulis skala 1 : 125. Jika panjang kamar tidur pada gambar 4 cm , hitunglah berapa panjang kamar tidur sebenarnya


Jawaban:Skala adalah hasil perbandingan antara jarak peta dengan jarak sebenarnya

Rumus" skala:

Skala = jarak peta : jarak sebenarnya

Jarak peta = jarak sebenarnya x skala

Jarak sebenarnya = jarak peta : skala

Pembahasan

Penyelesaian dengan Langkah-langkah:

Diketahui:

Skala = 1 : 50

Panjang sebenarnya = 8 m = 800 cm

Lebar sebenarnya = 6 m = 600 cm

Ditanya:

ukuran kelas pada gambar adalah ...

Jawab:

Gunakan rumus ↓

Jarak peta = jarak sebenarnya x skala

Untuk ukuran Panjang ↓

Panjang peta = panjang sebenarnya x skala

                     = 800 cm x ¹/₅₀

                     = 800 cm : 50

                     = 16 cm

Untuk ukuran Lebar ↓

Lebar peta = lebar sebenarnya x skala

                 = 600 cm x ¹/₅₀

                 = 600 cm : 50

                 = 12 cm

Jadi ukuran kelas pada gambar adalah 16 cm x 12 cm

17. Hitunglah selisih luas kamar tidur anak dan kamar tidur pembantu sebenarnya


1.) kamar tidur anak

Jp = p × l

= 3 × 4

= 12 cm²

Skala = 1 : 150

Js = Jp ÷ Sk

= 12 ÷ 1 : 150

= 12 × 150

= 1.800 cm

= 18 m

2.) kamar tidur pembantu

Jp = p × l

= 2,5 × 3

= 7,5 cm

Sk = 1 : 150

Js = Jp ÷ Sk

= 7,5 ÷ 1 : 150

= 7,5 × 150

= 1.125 cm

= 11,25 m

Selisih : 18 - 11,25 = 6,75 m

30. Delapan orang peserta wisata laki-laki harus menginap dalam 1 kamar dengan dua tempat tidur dan 2 kamar dengan tiga tempat tidur. Banyak cara penempatan peserta dalam kamar adalah …


KAIDAH PENCACAHAN

n = 8
k1 = 3
k2 = 2
k3 = 3
Cara =__?

Ada elemen identik maka gunakan rumus permutasi elemen identik
C = n! / (k1! k2! k3!)
C = 8! / (3! 2! 3!)
C = 8(7)(6)(5)(4)(3!) / (3! 2! 3!)
C = 3.360 / 6
C = 560 cara
