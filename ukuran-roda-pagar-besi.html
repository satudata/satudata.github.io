30. Seorang pemborong mendapat pesanan dua jenis bentuk pagar. pagar jenis 1 seharga rp 300.000.00/m dan pagar jenis 2 seharga 450.000.00/m. tiap m2 pagar jenis 1 memerlukan 4 m besi pipa dan 6 m besi beton, sedangkan adalah pagar jenis 2 memerlukan 8m besi pipa dan 4m besi beton. jika persediaan yang ada 640 besi pipa dan 480 besi beton. maka pendapatan maksimum dari hasil penjualan pagar adalah


Dimisalkan Jenis 1 = x dan Jenis 2 = y
4x + 8y lebih dari samadengan 640
6x + 4y lebih dari samadengan 480

dua persamaan di atas dicari titik koordinatnya
(i) .... 4x + 8y lebih dari samadengan 640
x + 2y lebih dari sama dengan 160
x = 0 》》 (0,80)
y = 0 》》 (160,0)

(ii) .... 6x + 4y lebih dari samadengan 480
3x + 2y lebih dari samadengan 240
x = 0 》》 (0, 120)
y = 0 》》 (80, 0)

(iii) .... dua persamaan tadi dieliminasi untuk menghasilkan titik potong (x,y)
6x + 4y = 480 (x2)
4x + 8y = 640

12x + 8y = 960
4x + 8y = 640
dikurang
8x = 320
x = 40

4x + 8y = 640
4(40) + 8y = 640
160 + 8y = 640
8y = 480
y = 60

titik potongnya (40,60)

dari kesemua titik tadi digambar dan membentuk bidang dengan didapatkan titik titik luar : (0,80) (80,0) (40,60)

(0,80) = 0 + 450.000 x 80 = 36.000.000
(80,0) = 300.000 x 80 + 0 = 24.000.000
(40,60) = 300.000 x 40 + 450.000 x 60 = 12.000.000 + 27.000.000
= 39.000.000

Jadi pendapatan maksimumnya adalah Rp. 39.000.000
