Teknik yang perlu di perhatikan untuk melakukan start pada lari jarak pendek adalah:

1. Posisikan kaki kanan ke depan dan kaki kiri di belakang
2. bungkukkan badan ke depan dan letakkan tangan di belakang garis start.

4. Angkat panggul ke atas hingga posisi pantat lebih tinggi dari pundak.
5. Tolakan lari pada balok sekuat-kuatnya ketika start.

8. jelaskan salah satu macam teknik start lari jarak pendek​


Jawaban:

a. Teknik pertama adalah start pendek atau bunch start untuk jarak 100 m.

Start pendek adalah lutut kaki belakang (kaki kiri) diletakkan didepan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.

b. Teknik kedua adalah start menengah atau medium start untuk jarak 200 m.

Start menengah adalah lutut kaki belakang (kaki kiri) diletakkan sejajar dengan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.

c. Teknik ketiga adalah start panjang atau long start untuk jarak 400 m.

Start panjang adalah lutut kaki belakang (kaki kiri) diletakkan dibelakang tumit kaki depann (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.

11. jelaskan teknik dasar start jongkok teknik berlari dan teknik memasujan garis finish dalam melakukan lari jarak pendek


Teknik lari jarak pendek terbagi menjadi tiga, yaitu start jongkok, gerakan lari, dan teknik memasuki garis finish. 
1). Start jongkok 
Cara melakukan start jongkok adalah sebagai berikut: 
• Posisi berjongkok dengan kedua kaki bertumpu pada sandaran blok start, lutut kaki belakang berada sejajr dengan ujung kaki depan. 
• Kedua lengan lurus sejajr dengan bahu, dan jari-jari tangnan diletakkan dibelakang garis start. 
• Berat badan bertumpu di kedua tangan, sehingga sikap seimbang dapat dipertahankan sampai ada aba-aba. 
Start jongkok yang digunakan oleh pelari jarak pendek, dapat dibedakan menjadi 3, yaitu: 
• Start jongkok pendek (bunch start) : jarak kaki saat jongkok 14-28 meter 
• Start jongkok menengah (medium start) : jarak kaki saat jongkok 35-42 meter 
• Start jongkok panjang (long start) : jarak kaki saat jongkok 50-70 meter

16. jelaskan cara melakukan teknik start jongkok pada lari jarak pendek


Cara melakukan teknik start jongkok pada lari jarak pendek dilakukan dari posisi berjongkok dengan salah satu kaki di depan dan satuny lalgi berada di belakang, kedua tangan menyentuh tanah sebagai tumpuan badan, selanjutnya ketika tembakan atau peluit di bunyikan maka dengan segera lakukan gerakan berlari.

Penjelasan:Cara melakukan teknik start jongkok pada lari jarak pendek dapat dilakukan dengan langkah berikut: Lutut kaki belakang diletakan pada ujung kaki depan menggunakan jarak satu kepal tangan. Kedua lengan lurus sejajar menggunakan bahu, telapak tangan (jari-jari) letakan dibelakang garis start dengan telapak tangan membentuk "V" terbalik. Pandangan lurus ke lintasan Berat badan berada dikedua tangan. Ketika aba-aba "Siap" pindahkan berat badan ke depan,kemudian pada aba-aba "ya" atau suara pistol secara reflek serta cepat melesat berlari kedepan ke depan.

19. teknik start yang dilakukan pada lari jarak pendek


Start yang dilakukan pada lari jarak pendek adalah start jongkok.

Pembahasan

Lari jarak pendek adalah salah satu olahraga lari yang termasuk kedalam cabang atletik.  Lari pendek menempuh jarak 100 m, 200 m, dan 400 m. Dalam lari jarak pendek start yang digunakan adalah start jongkok.

Start jongkok pada lari jarak pendek terdiri dari tiga jenis, yakni start jongkok penden, start jongkok medium, dan start jongkok panjang. Posisi jongkok uyang digunakan pada start jongkok dapat menghasilkan kecepatan lari yang maksimal. Start jongkok merupakan start yang paling sulit dilakukan. Dalam pelaksanaannya, start ini dilakukan dengan menempatkan balok start atau papan tumpuan kaki yang disusun sedemikian rupa untuk menambah tolakan ketika berlari.

Dalam melakukan start jongkok, terdapat beberapa hal yang harus diperhatikan, diantaranya.

Penempatan kaki pada start jongkok dilakukan dengan meletakkan kaki terkuat pada pijakan depan.Posisi lutut disesuaikan dengan jenis start jongkok yang digunakan.Posisi telapak tangan selalau terbuka.Posisi kepala selalu menengadah kedepan.Posisi badan sedikit condong ketika melakukan start dan lari awal setelah melakukan start.

Start Pendek

Start pendek diawali dengan berdiri tegak dengan kedua kaki rapat. Kemudian, kedua tangan berada di samping badan. Adapun cara melakukannya adalah sebagai berikut:

Ambil ancang-ancang dengan posisi kaki kiri (kaki yang terkuat) diletakkan di depan dan ditekuk.

Posisikan lutut kaki kanan diletakkan di sebelah kaki kiri dan beri jarak sekitar satu kepal.

Posisikan kedua tangan diletakkan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat atau seperti membentuk huruf V.

Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

Angkat panggul hingga posisinya lebih tinggi dari pundak.
