7. cara membuat ellipse pada corel draw


Klik Ellipse tool pada toolbox

- Klik tahan dan geser mouse untuk menggambar objek elips.

- Bentuk dan radius dapat diatur pada pada property bar.

- Misalnya elips akan diubah ke bentuk Pie dan ubah besar sudut

- Untuk membuat objek lingkaran / bulat yang memiliki ukuran diameter yang sama maka dapat dilakukan dengan menggeser mouse untuk membuat objek elips, tekan tombol Ctrl

12. apa syarat minimum untuk menginstal program corel draw?


Sistem Operasi Windows 2000, Windows XP atau Windows Vista.
Ø Computer PC berprosesor minimum Pentium III, 600Mhz, atau yang lebih cepat.
Ø Membutuhkan RAM minimum 256 MB, disarankan menggunakan 512 MB atau lebih untukmperformance yang lebih stabil.

Ø Monitor dengan kartuVGA yang mampu menampilkan resolusilayar 1024 x 768.

Ø Mouse atau tablet.

19. Cara Blok Koneksi Corel Draw supaya tidak terdeteksi internet dengan aplikasi


Jawaban:1. Buka control panel

N.B: Cara membuka control panel adalah:

1. Search "control panel" di search bar yang terdapat di windows taskbar (disebelah ikon windows)

2. Tekan windows + r untuk membuka windows run, kemudian ketik control panel dan tekan enter.

3. Buka windows --> buka file "windows system" --> buka control panel

2. Klik "system and security"3. Klik "windows defender firewall"4. Klik "advanced settings" (membutuhkan izin administrator)5. Klik "outbound rules"6. Pilih "new rules"7. Pilih "program"8. Klik "next"9. Klik "this program path"10. Klik browse dan carilah file "CorelDraw.exe"11. Klik file tersebut12. Tekan next13. Pilih block the connection14. Centang semua opsi (tergantung mau memblok koneksi apk tersebut di jaringan internet publik, private, atau domain.15. Terakhir, namakan rule baru tersebut menjadi "block corel X7" dan tambahkan deskripsi (opsional).Selesai.N.B: Ini tidak hanya berlaku untuk coreldraw saja, melainkan untuk semua program juga bisa. Tinggal ganti saja format rulenya dan browse file yang diinginkan (contoh: word.exe atau excel.exe)
