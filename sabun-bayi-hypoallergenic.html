Jawaban:

5A + 3B = Rp. 45.000,00

Penjelasan dengan langkah-langkah:

4A + 2B = 34.000

3A + B = 23.000 × 2

6A + 2B = 46.000

4A + 2B = 34.000

di eleminasi menjadi

2A = 12.000

A = 6.000

3A + B = 23.000

B = 23.000 - 3(6.000)

B = 5.000

maka 5A + 3B = 5(6.000) + 3(5.000) = 45.000
