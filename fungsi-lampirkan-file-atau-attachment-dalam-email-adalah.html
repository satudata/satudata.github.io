9. File yang dilampirkan pada pesan email adalah ... 


Mengirim lampiran dengan pesan Gmail

Melampirkan File

Di ponsel atau tablet Android, buka aplikasi Gmail Gmail.

Tap Tulis Tulis.

Tap Lampirkan Lampirkan.

Ketuk Lampirkan file atau Sisipkan dari Drive.

Pilih file yang akan dilampirkan.

Catatan: Untuk beberapa jenis file, seperti file ZIP, Anda perlu mendownload file manager dari Google Play sebelum dapat melampirkannya.

Menghapus Lampiran

Setelah menambahkan lampiran, Anda dapat menghapusnya. Di sebelah kanan nama lampiran, tap Tutup Tutup.

Batas Ukuran Lampiran

Akun Gmail: 20 MB

Akun non-Gmail: Sampai dengan 25 MB, tergantung pada penyedia

16. maksimum jumlah file yang dapat di lampirkan pada email


jumlahnya gak tau yg penting ukuran file tidak lebih dari 25 mb Gmail – membatasi maksimal 25MB untuk sekali kirim dan terima. Solusi untuk dapat mengirim lampiran lebih dari 25MB adalah dengan menggunakan Google Drive (sampai 15GB untuk versi gratis dan 1TB untuk yang berbayar). Alternatif lain bisa pakai Google Chrome dan DropBox Extension.Yahoo – 25MB dan unlimited jika menggunakan DropBox tetapi tentu tergantung pada ruang penyimpanan yang anda miliki dan untuk akun gratis biasanya hanya mendapatkan space 2GB.Hotmail/Outlook.com – 10MB atau 15GB jika upload terlebih dahulu ke OneDrive.

22. pengertian lampiran file dalam email


berkas atau dokumen yang menyertai email yang dimaksud. biasanya dokumen berisi data pelengkap dari isi email yang dikirim oleh pengirim Attachment Mungkin banyak yang belum dapat menjelaskan tentang penggertian attachment atau definisi attachment secara detail.
Pengertian attachment adalah fasilitas yang disediakan pada suatu website atau email untuk dapat mengirimkan dokumen baik berupa gambar, data, musik, dan dokumen lainnya yang selanjutnya akan diupload ke internet.

Dengan adanya attachment sangat memudahkan kita kalau misalnya mau bertukar informasi atau mengirim dokumen penting, karena yang perlu kita lakukan hanyalah mengupload dokumen tersebut ke server, maka orang lain sudah dapat mendownloadnya dari tempat lain. Fungsi attachment Attachment adalah perintah mengirim file melalui email. Kadang saat
kita mengirimkan surat kita ingin mengikutkan lampiran surat yang lain sehingga surat yang kita buat menjadi lebih lengkap tetapi tidak terlalu panjang.
Dalam lampiran attachment surat ini ada beberapa dokumen yang dapat kita lampirkan juga, misalnya: document word ataupun power point.
Singkatnya, fungsi Attachment File dalam sebuah email yaitu untuk
mengirimkan suatu file (baik berupa video,gambar,animasi dll)
melalui email ke
alamat email tujuannya, karena attachment file itu sendiri adalah
perintah – perintah untuk mengirimkan suatu file melalui email.
Sedangkan cara melakukan Attachment File ialah :
- klik compose pada pilihan yang ada pada email anda
- klik attachment file
- Klik Browse
- Pilih file yang diinginkan
- Klik Attach File
- Pilih Done
- Klik Send
