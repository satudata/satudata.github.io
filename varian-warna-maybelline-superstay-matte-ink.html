10. cara membuat matte semelting​


a. Tahap kominisi

Membebaskan atau meliberasi mineral-mineral tembaga dari ikatan zat-zat pengotornya dengan cara operasi peremukan dan penggerusan. Target dari Proses Pembuatan Tembaga ini ialah ukuran partikel bijih yang bisa memperoleh tingkat recoveri maksimal ketika tahap Konsentrasi Flotasi.

b. Tahab Konsentrasi Flotasi

Setelah ukurannya sesuai, dilanjutkan dengan tahap pemisahan mineral atau konsentrasi.Memisahkan mineral-mineral Cu-Fe-S dan Cu-S pengotoran yang cukup efektif dengan melakukan metode flotasi. Dengan menggunakan metode ini, bisa meningkatkan kadar tembaga di konsentrat menjadi 30 %.

c. Tahap Matte Smelting

Masuk ketahap ini konsentrat tembaga dileburkan berupa lelehan matte. Proses peleburan ini dalam keadaan oksidatif menghasilkan lelehan matte (sulfida yang kaya akan tembaga serta sedikit mengandung besi) dan slag (lelehan dari campuran oksida besi dan oksida logam pengotor serta fulk), gas buang. Tahap ini menghasilkan matte yang kandungan tembaganya setara 45% sampai 75%.

d. Tahap Konversi Matte

Tahap Konversi Matte menjadi blister copper, melalui dioksidasi dan kandungan meningkat menjadi 90 %.Concerter dihembuskan udara melewati sejumlah tuyeres yang terendam dalam lelehan. Dalam proses ini ditambakanpula oksigen murni, silika sebagai fliks, revert dan scrap. Slag yang akan dihasilkan memiliki senyawa besi –silika.

e. Tahap Fire refining,

proses ini biasa disebut pemurnian dilakukan dalam rotary-furnace, reverberatory-furnace, heart-furnace yang bisa diteliti dengan 2 tahap. Tahap pertama, oksidasi selektif kepada sulfur dan elemen pengotor. Tahap kedua, deoksidasi guna penurunan kandungan oksigen dalam tembaga.

Tahap fire refining di atas sanggup memperoleh logam tembaga yang mempunyai kandungan 99 %. Proses selanjutnya, melakukan pelarutan tembaga secara elektrokimia melalui tembaga anode dan mengendapkannya kembali di permukaan katode. Mineral-mineral pengotor yang terdapat pada tembaga anode mengapung ke permukaan dan tidak mengendap.Proses Pembuatan Tembaga yang terakhir ini ialah tahapan Elektrorefining.

16. Cara membuat matte smaleting


Matte adalah istilah yang digunakan dalam bidang pyrometalurgi yang diberikan pada fase logam sulfida cair yang biasanya terbentuk selama peleburan tembaga, nikel, dan logam tidak mulia lainnya. Biasanya, matte adalah fase di mana logam utama yang diekstraksi pulih sebelum proses reduksi akhir (biasanya mengkonversi) untuk menghasilkan logam mentah. Mattes juga dapat digunakan untuk mengumpulkan kotoran dari fase logam, seperti dalam kasus peleburan antimon. Mattes cair tidak larut dalam fase terak dan logam. Ketidaksuburan ini, dikombinasikan dengan perbedaan dalam gravitasi spesifik antara mattes, terak, dan logam, memungkinkan untuk pemisahan fase cair.

ちょっとまって / ちょっと待って

Chotto matte

Tunggu sebentar

Bentuk penggunaannya terkesan digunakan secara kasual (memiliki kesan yang santai), salah satu ungkapan ini sebaiknya digunakan dalam situasi yang tidak resmi/informal.

Penggunaan salah kalimat tersebut sebagai bentuk ungkapan yang menyatakan kepada orang lain agar diminta untuk menunggu sebentar.

Karena kata ちょっと待って (chotto matte) digunakan kepada orang yang sudah kenal akrab dengan diri sendiri atau seumuran seperti halnya kepada teman/sahabat, saudara, rekan. terkesan tidak resmi atau santai.

Jawaban :

Chotto matte artinya "Tunggu bentar".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Simpelnya, kalimat ini terdiri dari 2 kata, yaitu Chotto (Bentar) dan Matte (Tunggu). Bila digabung akan berarti "Tunggu bentar". Kalimat ini merupakan kalimat perintah untuk menunggu sebentar secara kasual. Biasa diucapkan ke teman atau orang yang sederajat. Pada kalimat ini juga bisa ditambahkan kata Kudasai di akhir untuk membuat kalimatnya menjadi lebih sopan.

Untuk pembahasan lebih lanjut, akan saya bahas masing-masing kata nya.

1. ちょっと / Chotto

Berarti "Bentar", "Agak", "Dikit", dsb. Kata ini merupakan versi kasualnya dari kata 少し (Sukoshi) yang artinya serupa. Perbedaan lainnya yaitu kata Chotto juga bisa digunakan untuk menolak sesuatu secara sopan. Contoh penggunaannya misalkan kita ingin menolak sesuatu, dapat mengatakan 'Sumimasen, sore wa chotto...', yang secara harfiah berarti "Maaf, itu agak...".

2. 待って / まって / Matte

Berarti "Tunggu". Kata ini merupakan bentuk Te dari kata kerja 待つ (Matsu), yang berarti "Tunggu". Fungsi dari bentuk Te ini bisa untuk menjadi perintah, bisa juga untuk penyambung, dan lainnya. Pada kalimat ini, kata Matsu diubah ke bentuk Te untuk membuat perintah. Untuk membuat perintah yang lebih halus, bisa dengan ditambahkan lagi kata Kudasai di belakang.

Kedua kata tersebut bila digabung akan menjadi ちょっと待って (Chotto matte), yang berarti "Tunggu bentar".

Bagaimana cara untuk mengubah suatu kata kerja ke bentuk Te?

Sebelumnya, perlu diketahui bahwa kata kerja Bahasa Jepang dibagi menjadi 3 kelompok. Masing-masing kelompok kata kerja memiliki ciri tersendiri, dan berbeda cara untuk dikonjugasikannya. Untuk mengubah KK ke bentuk Te, caranya tiap kelompok adalah sebagai berikut.

KK kelompok 1 : Untuk yang berakhiran 'u', 'tsu', 'ru', diubah menjadi 'tte'. Untuk yang berakhiran 'mu', 'nu', 'bu', diubah menjadi 'nde'. Untuk yang berakhiran 'ku', diubah menjadi 'ite'. Untuk yang berakhiran 'gu', diubah menjadi 'ide'. Untuk yang berakhiran 'su', diubah menjadi 'shιte'.KK kelompok 2 : Dengan mengubah akhiran 'ru' menjadi 'te'.KK kelompok 3 : Dengan mengubah 'suru' menjadi 'shιte', dan mengubah 'kuru' menjadi 'kite'.

Contoh :

Kau → KatteTatsu → TatteSuwaru → SuwatteYomu → YondeShinu → ShindeAsobu → AsondeAruku → AruiteOyogu → OyoideKesu → KeshιteMiru → MiteTaberu → TabeteBenkyousuru → BenkyoushιteMottekuru → Mottekite

Pola '∼te kudasai'

Fungsi dari pola ini adalah untuk membuat ungkapan permohonan atau perintah yang halus. Kata Kudasai ini memiliki arti "Mohon" atau "Tolong". Sedangkan 'te' di sini merupakan kata kerja dalam bentuk Te, yang fungsinya membuat suatu perintah.

Pola kalimat :

== [KK](Bentuk Te) + kudasai

Contoh :

Oshieru → Oshiete kudasaiYameru → Yamete kudasaiMatsu → Matte kudasai
