Dijawab :
⇔ r = U₂/U₁₁
       = 9/3

⇔               Sn = (a*(r^n - 1))/(r - 1)
⇔          3.279 = (3*(3^n - 1))/(3 - 1)
⇔          3.279 = (3*(3^n - 1))/(2)
⇔       3.279*2 = (3(3^n - 1))/2)*2
⇔           6.558 = 3*(3^n - 1)
⇔    3*(3^n - 1) = 6.558 => dibagi 3
⇔ 3*(3^n - 1)/3 = 6.558/3
⇔          3^n - 1 = 2.186
⇔              3^n  = 2.186+1
⇔               3^n = 2.186
⇔               3^n = 3^7

17. jumlah n suku pertama deret geometri dirumuskan dengan Sn=2³ⁿ- 1. tentukan suku ke-n deret tersebut


Jawab:

Penjelasan dengan langkah-langkah:

Jawab:

[tex]\boxed{\bf U_{n} =7.8^{n-1}}[/tex]

Diketahui :

Rumus [tex]S_{n}[/tex] = 2³ⁿ- 1

Ditanya :

Rumus [tex]U_{n}[/tex] ?

Penjelasan dengan langkah-langkah:

Rumus Umum Deret Geometri

[tex]\boxed{\bf S_{n}=\frac{a(r^{n}-1) }{r-1} }[/tex]  untuk r > 1

[tex]\boxed{\bf S_{n}=\frac{a(1-r^{n}) }{1-r} }[/tex]  untuk r < 1

[tex]\boxed{\bf U_{n}=a.r^{n-1} }[/tex]

∴Substitusi n=1 untuk mencari a karena [tex]S_{1}=a[/tex]

[tex]S_{n}=2^{3n} -1[/tex]

[tex]S_{1}=2^{3.1} -1[/tex]

    [tex]=2^{3} -1[/tex]

    [tex]=8 -1[/tex]

  a = 7

∴Substitusi n=2

[tex]S_{n}=2^{3n} -1[/tex]

[tex]S_{2}=2^{3.2} -1[/tex]

    [tex]=2^{6} -1[/tex]

    [tex]=64 -1[/tex]

    = 63

∴Mencari nilai [tex]U_{2}[/tex]

[tex]U_{2}=S_{2}-a[/tex]

    = 63 - 7

    = 56

∴Mencari rasio

[tex]r = \frac{U_{n} }{U_{n-1} }[/tex]

  [tex]=\frac{U_{2} }{U_{1} }[/tex]

[tex]=\frac{56}{7}[/tex]

= 8

∴Mencari rumus [tex]U_{n}[/tex]

[tex]U_{n}=a.r^{n-1}[/tex]

[tex]\boxed{\bf U_{n} =7.8^{n-1}}[/tex]

Semoga membantu ^_^
