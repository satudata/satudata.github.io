20. 5. Perhatikan gambar dua muatan titik berikut! Dimana letak titik P agar kuat medan listrik di titik P tersebut sama dengan nol?


[tex]\\[/tex]Letak titik P agar medan listrik di titik P = 0 adalah 2 cm di kiri muatan Q₁ .

Pendahuluan :

Hukum Coulomb adalah hukum yang mempelajari tentang dua buah muatan listrik. Hukum Coulomb pertama kali dikemukakan oleh seorang Ilmuwan yang berasal dari Perancis bernama Charles Augustin Coulomb.

Listrik dalam bahasa Inggris disebut Electric, yang diambil dari nama elektron yaitu partikel sub atom yang bermuatan listrik negatif.

Muatan listrik terdiri dari 2 jenis yaitu muatan positif dan negatif.

Jenis interaksi antar muatan :

1. Saling tarik menarik, interaksi ini akan terjadi jika muatan yang berbeda jenis didekatkan.

2. Saling tolak menolak, interaksi ini akan terjadi jika muatan yang sejenis didekatkan.

Persamaan hukum Coulomb dapat ditulis sebagai berikut :

[tex]\boxed{\sf\ F = k. \frac{Q_1.Q_2}{r^2}}[/tex]

Keterangan :

k = konstanta

Q1 = Q2 = besar muatan (C)

r = jarak (m)

F = Gaya (N)

Pembahasan :

Diketahui :

Q₁ = 1 µC = 10-⁶ C

Q₂ = 4 µC = 4 × 10-⁶ C

rAB = 2 cm = 2 × 10-² m

rAC = x cm

rBC = (2 + x) cm

k = 9 × 10⁹ Nm²/C²

Ditanya :

letak titik P agar medan listrik tsb = 0?

Jawab :

Misal Q₁ = A, Q₂ = B, titik P = C

A dan B berbeda jenis, maka titik P harus diletakkan di kiri muatan A. Persamaannya menjadi :

EAC - EBC = 0

EAC = EBC

k. qA/rAC² = k. qB/rBC²

qA/rAC² = qB/rBC²

1/x² = 4/(2 + x)²

(2 + x)²/x² = 4

(2 + x)/x = 2

2x = 2 + x

x = 2 cm

Kesimpulan :

Letak titik P agar medan listrik di titik P = 0 adalah 2 cm di kiri muatan Q₁

30. Pada titik-titik sudut B dan D sebuah bujur sangkar ABCD masing-masing diletakkan partikel bermuatan -q. Agar kuat medan listrik di titik A nol, maka di titik C harus diletakkan sebuah partikel bermuatan sebesar...


Pada titik sudut B dan D dari persegi ABCD, setiap partikel bermuatan negatif q. Agar kuat medan listrik di titik A menjadi nol, maka titik C harus diletakan sebuah partikel bermuatan positif dengan +2q√2.

Penjelasan dan langkah-langkah

Diketahui:

Titi B = Titik D = -q

Ea = 0

Ditanyakan:

Muatan di titik C:  qc = ?

Jawab:

Untuk mencari kuat medan listrik digunakan rumus sebagai berikut:

                                        [tex]E = k \frac{q}{r^{2} }[/tex]

Dengan:

E adalah kuat medan listrik (N/C)

k adalah konstanta (9 x 109 Nm2/C2)

q adalah muatan listrik (C)

r adalah jarak (m)

Untuk membuat kuat medan listrik di A menjadi nol, maka Ebd yang dihasilkan harus seimbang dengan kuat medan listrik yang dihasilkan oleh muatan di C dengan arah yang  berlawanan dari Ebd sehingga:

Langkah pertama: Menghitung kuat medan listrik titik BD

[tex]E_{BD} = \sqrt{E_{B} ^{2} + E_{D} ^{2} + 2 . E_{B} .E_{D} cos 90^{0}}\\E_{BD} =\sqrt{E_{B}^{2} + E_{D} ^{2} + 0} \\E_{BD} = \sqrt{(k \frac{q}{r^{2} })^{2} + (k \frac{q}{r^{2} })^{2} } \\E_{BD} =\sqrt{2 (k \frac{q}{r^{2} })^{2} } \\E_{BD} = k \frac{q}{r^{2} } \sqrt{2}[/tex]

Langkah kedua: Menghitung kuat medan listrik dititik AC dengan qₐ = 0 maka:

[tex]E_{AC} =E_{BD} \\k \frac{q_{C}}{r^{2} } = k \frac{q_{}}{r^{2} } \sqrt{2} \\\frac{q_C}{r^{2}} = \frac{q_{}}{r^{2} } \sqrt{2} \\\frac{q_C}{(r\sqrt{2} )^{2}} =\frac{q_{}}{r^{2} } \sqrt{2} \\\\\frac{q_C}{2r^{2}} =\frac{q_{}}{r^{2} } \sqrt{2} \\q_C = 2q\sqrt{2}[/tex]

Agar Kuat medan Eac arahnya berlawanan dengan resultan Ea(bd) maka muatan di C harus positif.
