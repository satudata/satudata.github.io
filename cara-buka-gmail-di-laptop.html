Cara Buat Akun Baru Gmail di Komputer/Laptop:

Masukkan alamat www.gmail.com pada web browser Anda.

Klik create an account, sebuah kotak akan terbuka.

Isi data diri Anda dengan lengkap.

Pada isian setelah nama, Anda bisa memilih nama untuk username.

Jika ternyata terdapat tulisan That username is taken. Try another. Itu tandanya orang lain telah memakai username yang Anda pilih. Anda bisa mengakalinya dengan menambah angka.

Isi semua kolom yang belum terisi dengan data yang benar.

Klik Next Step, akan muncul agreement, atau kotak persyaratan, klik pada I Agree.

Anda telah memiliki akun Gmail sekarang.

Cara termudah untuk membuka situs di laptop adalah dengan menggunakan chrome, atau aplikasi sejenis seperti firefox, Microsoft Edge dan lain sebagainya. Kemudian terdapat kolom URL, dapat diketik atau di copy-paste link situs yang ingin di tuju.

Jawaban:

   

Menggunakan Google Translate

Siapa bilang Google Translate hanya bisa digunakan untuk menerjemahkan bahasa saja? Google Translate memiliki keunggulan lain lho, yakni bisa membuka situs yang diblokir oleh pemerintah. Bagaimana cara buka situs yang diblokir di laptop menggunakan Google Translate? Simak langkahnya berikut ini.

Langkah 1: Buka Google Translate pada browser kamu. Ketikkan alamat situs yang ingin kamu buka. Lalu, klik alamat tersebut

Langkah 2: Setelah diklik, alamat situs tersebut akan langsung terbuka dan diterjemahkan ke bahasa yang kamu gunakan di Google Translate. Namun, kamu dapat mengembalikannya ke bahasa aslinya dengan mengklik tombol Original

Menggunakan Opera Browser

Langkah 1: Unduh Opera Browser pada laptop kamu. Jika sudah, klik Setting

Langkah 2: Setelah klik Setting, klik centang biru pada menu VPN untuk mengaktifkan fitur VPN tersebut

Langkah 3: Setelah centang biru diklik, otomatis VPN kamu sudah menyala. Sekarang kamu bebas mengakses situs apapun yang kamu inginkan

21. 1. langkah membuka gmail2. cara masuk akun gmail3. cara mendaftarkan aku baru4. cara mengirim file lewat Emali


Misalkan membuka gmail memakai komputer/ laptop:
1. yang pertamakali dilakukan adalah membuka program web browser (misalnya Mozilla Firefox) lalu ketikkan www.gmail.com pada address bar.

2. saat tampil jendela situs www.gmail.com klik tombol Masuk, yang mana nantinya kita diminta mengisikan user name dan password email.

3. untuk mendaftarkan akun baru, klik tombol buat akun (berwarna merah), yang mana nantinya kita diminta mengisikan biodata diri secara lengkap.

4. untuk mengirim file lewat email, dapat memakai tombol Lampirkan file saat menulis email.
