3. Cv abadi karya menyimpan arsip-arsip dengan menggunakan pokok masalah sebagai pedoman mengatur penyimpanan arsipnya. berarti cv abadi karya menyimpan arsip berdasarkan…


CV. Abadi Karya menyimpan arsip-arsip dengan menggunakan pokok masalah sebagai pedoman mengatur penyimpanan arsipnya. Berarti CV Abadi Karya menyimpan arsip berdasarkan sistem subyek/pokok masalah.

Dalam pengaturan arsip terdapat beberapa metode dalam melakukan administrasi penyimpanannya. Metode ini akan berbeda dari satu perusahaan dengan perusahaan lainnya tergantung dari model usaha yang dijalankan dan tujuan dari penyimpanan.  

Pembahasan

Arsip adalah kumpulan dokumen penting yang wajib disimpan sebagai informasi yang penting dan dapat diakses kembali apabila diperlukan di kemudian hari.

Berikut tata cara penyimpanan arsip:

1. Melakukan pemeriksaan terlebih dahulu atas lembar disposisi surat apakah surat tersebut sudah dapat diarsip atau belum (meneliti tanda pelepas surat). Tanda pelepas surat pada umumnya memiliki tanda disposisi yang menyatakan surat dapat diarsip.2. Mengindeks surat dengan kode yang disesuaikan dengan system yang digunakan untuk penyimpanan dokumen untuk mempermudah penyimpanan dan penemuan kembali dokumen yang diperlukan.3. Menyortir dan memilah dokumen sesuai dengan bagian, masalah atau tujuan dokumen. 4. Menyimpan dokumen ke dalam folder/map di dalam lemari arsip (filing cabinet) tahan api. 5. Menata arsip menggunakan sistem penyimpanan arsip yang terdiri dari beberapa sistem seperti:

        a. Sistem abjad

        b. sistem tanggal

        c. sistem nomor

        d. sistem wilayah

        e. sistem subyek/pokok masalah.

11. Jika diketahui CV. Abadi mempunyai fungsi Produksi kain Q=5KL dan Q = (X+3)


Jawaban:

Untuk membuat kurva isoquant dari fungsi produksi CV. Abadi, pertama-tama kita perlu menentukan beberapa nilai untuk K dan L. Setelah itu, kita bisa menghitung nilai Q untuk setiap kombinasi K dan L tersebut dengan menggunakan fungsi produksi Q=5KL. Kemudian, kita bisa menggambarkan nilai-nilai Q tersebut dalam bentuk grafik, dengan K sebagai sumbu x dan L sebagai sumbu y.

Misalkan kita menggunakan nilai-nilai K dan L sebagai berikut:

K = 0, 1, 2, 3, 4

L = 0, 1, 2, 3, 4

Maka, kita bisa menghitung nilai-nilai Q dengan menggunakan fungsi produksi Q=5KL sebagai berikut:

Q = 0, 5, 10, 15, 20 (untuk K = 0)

Q = 0, 5, 10, 15, 20 (untuk K = 1)

Q = 0, 10, 20, 30, 40 (untuk K = 2)

Q = 0, 15, 30, 45, 60 (untuk K = 3)

Q = 0, 20, 40, 60, 80 (untuk K = 4)

Setelah itu, kita bisa menggambarkan nilai-nilai Q tersebut dalam bentuk grafik, dengan K sebagai sumbu x dan L sebagai sumbu y. Hasilnya akan menjadi kurva isoquant yang menunjukkan berbagai kombinasi K dan L yang dapat digunakan untuk menghasilkan produksi Q yang sama.

Sebagai contoh, berikut ini adalah kurva isoquant yang mungkin dihasilkan untuk Q = 20:

(0,0) (1,0) (2,0) (3,0) (4,0)

| | | | |

(0,1) (1,1) (2,1) (3,1) (4,1)

| | | | |

(0,2) (1,2) (2,2) (3,2) (4,2)

| | | | |

(0,3) (1,3) (2,3) (3,3) (4,3)

| | | | |

(0,4) (1,4) (2,4) (3,4) (4,4)

Keterangan:

Titik (K,L) menunjukkan nilai K dan L untuk setiap kombinasi.

Garis putus-putus menunjukkan batas-batas kurva isoquant.

Soal tak disertai dengan pertanyaan apapun. Oleh sebab itu maka jawaban hanya akan menjelaskan apa yang dimaksud dengan fungsi produksi. Fungsi produksi bisa dimaknai sebagai hubungan di antara input dan output produksi yang dihasilkan pada suatu periode. Fungsi produksi juga dapat kita artikan sebagai gambaran yang memberikan petunjuk bagaimana perilaku produsen memproduksi produknya (barang atau jasa).

Pembahasan

Fungsi produksi penting untuk dipahami dengan baik sebab memiliki sejumlah peran penting, antara lain sebagai berikut:

Fungsi produksi berperan dalam proses perencanaan.Fungsi produksi berperan dalam proses pengolahan.Fungsi produksi berperan dalam proses pengendalian serta perawatan.Fungsi produksi berperan dalam menentukan jasa penunjang.

Fungsi produksi, sebagaimana yang dinyatakan di awal jawaban, adalah fungsi yang secara khusus menunjukkan hubungan atau keterkaitan antara kuantitas input dan kuantitas output yang selanjutnya akan dipakai dalam proses produksi.
