6. Massa nitrogen (N) dalam CO(NH2)2 adalah Mr CO (NH2)2=60, Ar N=14 adalah...


massa N = ((2 x Ar N) / Mr urea) x massa urea
massa N = ((2 x 14) / 60) x massa urea
massa N = (28 / 60) x massa urea
massa N = 7/15 x massa urea

tinggal massa urea dimasukkan ke persamaan yang di atas, nanti kita dapat massa nitrogennya.
