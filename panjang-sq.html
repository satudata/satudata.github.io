5. tentukan panjang PQ PS dan panjang SQ​


A. [tex]PQ^{2} = PR^{2}+QR^{2}[/tex] ⇒ [tex]PQ =\sqrt{ PR^{2}+QR^{2}}[/tex]

[tex]=\sqrt{7^{2}+24^{2}} = \sqrt{49 + 576} = \sqrt{625}=25[/tex]

B. Pertama, kita gunakan rumus persamaan luas segitiga utk menghitung panjang RS yaitu: 1/2.PQ.RS = 1/2.PR.RQ

PQ.RS = PR. RQ ⇒ 25.RS = 7.24 ⇒ RS = 7.24/25 = 6,72cm

Lalu kita bisa menghitung panjang PS dg rumus phytagoras yaitu:

[tex]PS = \sqrt{PR^{2} -RS^{2} } = \sqrt{7^{2} -6,72^{2} } = \sqrt{49 -45,16}[/tex]

[tex]= \sqrt{3,84} = 1,96cm[/tex]

C. SQ = PQ - PS = 25 - 1,96 = 23,04cm
