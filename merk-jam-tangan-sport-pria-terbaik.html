Jawab:

a. asli semua 50-7=43

peluang = 1806/1225

b. 1 asli, 1 palsu

peluang = 301/1225

c. palsu semua

peluang = 42/1225

Penjelasan dengan langkah-langkah:

banyaknya ruang sample=

= 50!/(50-2)!.2! = 48!x49x50/48!x2 = 1225

= 43/(43-1).1 x42/(42-1).1 = 43 x 42 = 1806

peluang = 1806/1225

b. 1 asli, 1 palsu

= 43/(43-1).1 x7/(7-1).1 = 43 x 7 = 301

peluang = 301/1225

c. palsu semua

= 7/(7-1).1 x 6/(6-1).1 = 7 x 6 = 42

peluang = 42/1225
