4. Fiona membeli sepasang sepatu nike dan 2 pasang adidas seharga rp 4.000.000 angel membeli sepasang nike dan sepasang adidas rp 2.500.000 hitunglah: a.nikita membeli sepasang nike b.irene membeli 3 pasang adidas


Jawaban:

Rp. 1.000.000 dan Rp.4.500.000

Penjelasan dengan langkah-langkah:

anggaplah sepatu nike (x) dan sepatu Adidas (y)

x+2y = 4.000.000

x+y = 2.500.000

______________-

y = 1.500.000

artinya, satu pasang sepatu Adidas harganya Rp. 1.500.000. masukkan harga y ini ke salah satu persamaan, contoh x+y=2.500.000

x+1.500.000=2.500.000

x=2.500.000-1.500.000

x=1.000.000

jadi, harga sepasang sepatu nike adalah Rp. 1.000.000

setelah itu kita jawab pertanyaan di bawah:

a. nikita membeli sepasang sepatu nike. uang yang harus ia keluarkan adalah Rp. 1.000.000

b. irene membeli 3 pasang sepatu adidas. uang yang harus ia keluarkan adalah 3 × Rp. 1.500.000 = Rp. 4.500.000

18. Nia membeli sepasang sepatu Nike dan 2 pasang Adidas seharga Rp 4.000.000 Nita membeli sepasang Nike dan sepasang Adidas Rp 2.500.000 hitunglah:a.Karin membeli sepasang Nike b.Tania membeli 3 pasang adidas


Materi : SPLDV

Pemisalan :

( Nike = x ) dan ( Adidas = y )

\_____________________/

x + 2y = Rp 4.000.000

x + y = Rp 2.500.000

__________________-

Eliminasi ( y )

2y - y = Rp 4.000.000 - Rp 2.500.000

[ y = Rp 1.500.000 ]

()()()()()()()()()()()()()()()()()

x + Rp 1.500.000 = Rp 2.500.000

x = Rp 2.500.000 - Rp 1.500.000

[ x = Rp 1.000.000 ]

Hasil / Result :

( Sepatu Nike ) = x = Rp 1.000.000

( Sepatu Adidas ) = y = Rp 1.500.000

Maka x + 3y = Rp 5.500.000
