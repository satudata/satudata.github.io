6. tentukan negasi/ ingkaran/ penyakalan dari " jika saya sopan maka orang segan"


jika orang sopan dan saya tidak seganp = saya sopan
q = orang segan

p ⇒ q = " Jika saya sopan maka orang segan"

~(p ⇒ q) ≡ ~(~p ∨ q)
               ≡ p ∧ ~q
p ∧ ~q ≡ "Saya sopan dan orang tidak segan"
