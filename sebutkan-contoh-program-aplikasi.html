25. Sebutkan contoh program aplikasi web


Mozila Firefox, Chrome, Opera Mini, Safari, Baidu dan lain-lainXAMPP, Wamp Server, AMPPS, Uwamp, EasyPHP,

XAMPP: Program aplikasi pengembangan websitenya berbasis PHP dan MySQL

Wamp Server: merupakan aplikasi yang dapat menjadikan komputer ataupun laptop menjadi sebuah Server atau bisa dikatakan Server offline

AMPPS: merupakan aplikasi webserver ciptaan softaculous dan bisa digunakan baik sebagai webserver localhost ataupun di onlinekan

UWamp: sebuah paket kumpulan yang terdiri dari Apache, MySQL, phpmyadmin, PHP, perl, freetype2 dll

EasyPHP: Software antarmuka wamp yang dapat menjalankan CMS website berhasil SQL di localhost Windows
