JAWABAN1.). Fungsi :

Font, berfungsi untuk mengganti jenis font pada dokumen. Umunya pada micros yaoft Office, font default yang digunakan adalah Calibri.

Font Size, berfungsi untuk menggantu ukuran font sesuai keperluan pengguna. Ukuran font tersedia dalam rentang skala 8 - 72.

Font Color, berfungsi untuk mengganti warna font sesuai keperluan pengguna. Umumnya akan disediakan standart colour, themed colour dan custom colour

Pembahasan

Font adalah salah satu menu pada Microsoft office yang berada pada ribbon Home yang terletak di nomor dua dari kiri tampilan Microsoft Office. Selain 3 perintah di atas, ada beberapa hal yang dapat dilakukan pada menu font yakni :

Increase or Decrease Font Size, digunakan untuk menaikkan atau menurunkan ukuran font pada satu skala diatas atau dibawahnya.

Change Case, digunakan untuk mengubah model susunan font apakah Uppercase, Lowercase, sentece case, atau toggle case

Clear all formatting, digunakan untuk menghilang semua efek hasil format terhadap font yang di select

Text Effct dan typpgraphy, digunakan untuk membuat efek pada font seperti glow, outline, shadow dan sebagainya untuk memperindah dokumen

Text Highlight colour, digunakan untuk memberi highlight atau jika dalam konteks tertulis adalah stabilo. Warna juga dapat disesuaikan

Subscript dan Superscript diguakan untuk memberi pangkat atau angka dibawah seperti pada kimia

Bold, Italic, Underline, Stikethrough, secara berurutan digunakan untuk memberikan efek penebalan, memiiringkan tulisan, memberi garis bawah, dan mencoret tulisan yang diselect.

2.).• Grow Font : fungsi ikon ini adalah akan memperbesar huruf yang digunakan.

•Shrink Font : berfungsi sebaliknya dari fungsi Grow Font yaitu berfungsi untuk mengecilkan huruf atau font yang digunakan.

•Strikethrough : fungsi ikon ini akan membuat efek pada text atau tulisan yang kita pilih bertanda garis tercoret. Biasany digunakan untuk menandai suatu pilihan yang tidak terpilih cukup dicoret.

3.). •Bullets = Ini mempunyai fungsi untuk memberikan tanda bullet pada beberapa paragraf yang anda pilih, bullet ini modelnya hitam bulat, selain itu ada juga model-model lainnya seperti centang. Numbering = sama seperti bullets, letak perbedaanya cuma bentuknya. Kalau Numbering ini menggunakan nomor atau angka dalam penandaan paragraf yang di pilih.

•Numbering = mempunyai fungsi memberikan nomor atau angka pada setiap paragraf sehingga tulisan akan tanpak urut sesuai dengan penomoran yang diinginkan.

4.). Fungsi bold,italic dan underline

Pembahasan:

Bold, Berfungsi untuk mempertebal tulisan atau teks sehingga teks tersebut lebih terang dibandingkan teks yang lain. Contohnya:

BRAINLY

Italic, Berfungsi untuk memiringkan tulisan. atau teks, sehingga memperjelas teks atau bagian tertentu.

Contohnya:

BRAINLY

Underline, Berfungsi untuk memberi garis bawah pada teks atau tulisan.

20. Perbedaan font size,grow font dan shrink font


Jawaban:

Misalkan pada program pengolah kata seperti Ms. Word 2007:

Font Size: digunakan untuk mengubah ukuran huruf dg mengisikan angka pada kotak isian yg disediakan (semakin besar angka, maka semakin besar ukuran huruf & sebaliknya).

Grow font: digunakan untuk mengubah ukuran huruf menjadi lebih besar secara otomatis.

Shrink font: digunakan untuk mengubah ukuran menjadi lebih kecil secara otomatis.

Penjelasan:

Pada program Ms. Word 2007, font size, grow font dan shrink font terdapat dalam tab ribbon Home di group Font. Perhatikan gambar berikut ini:

Font size: ditunjukkan tanda panah merah

Grow font: ditunjukkan tanda panah biru

Shrink font: ditunjukkan tanda panah hijau

26. ribbon font berfungsi untuk pengaturan ??​


Jawaban:

Ribbon adalah area yang digunakan untuk menampilkan tab dan menu dari tab yang aktif

Fungsi Ribbon Pada Group Clipboard:

-Paste : untuk menempelkan hasil copy atau cut.

-Cut : untuk memotong teks/objek terpilih.

-Copy : untuk menggandakan teks/objek terpilih.

-Format Painter : digunakan untuk meniru format halaman dokumen ke dalam dokumen lainnya.

Fungsi Ribbon pada Group Font:

-Font : memilih jenis huruf.

-Font Size : untuk mengatur ukuran huruf.

-Grow Font : untuk memperbesar ukuran huruf secara instan.

-Shrink Font : untuk memperkecil ukuran huruf secara instan.

-Change Case : untuk mengubah status huruf kapital/huruf kecil.

-Clear Formatting : untuk menghapus pemformatan teks terpilih.

-Bold : untuk menebalkan teks terpilih.

-Italic : untuk memiringkan Teks terpilih.

-Underline : untuk memberikan garis bawah pada teks terpilih.

-Strikethrough : memberikan tanda coret padateks terpilih.

-Subscript : untuk mengetik karakter pemangkatan.

-Superscript : untuk mengetik karakter pemangkatan.

-Text Effect : untuk memberikan efek artistik pada teks terpilih.

-Text Highlight Color : untuk memberikan warna stabilo di belakang teks terpilih.

-Font Color : untuk mengatur warna teks terpilih.

Fungsi Ribbon pada Group Paragraph:

-Bullets : untuk memberikan tanda bullet di tiap paragraf terpilih.

-Numbering : untuk memberikan format penomoran di tiap paragraf terpilih.

-Decrease Indent : untuk menggeser baris kedua paragraf ke kiri.

-Increase Indent : untuk menggeser baris kedua paragraf ke kanan.

-Left-to-Right : untuk menjadikan arah teks berjalan dari kiri ke kanan (format Latin).

-Right-to-Left : untuk menjadikan arah teks berjalan dari kanan ke kiri (format Arab).

-Sort : untuk menyortir data.

-Show Paragraph Marks : untuk menyembunyikan tanda koreksi paragraf.

-Align Text Left : untuk mengatur teks rata kiri.

-Center : untuk mengatur teks rata tengah.

-Align Text Right : untuk mengatur teks rata kanan.

-Justify : untuk mengatur teks rata kanan-kiri.

-Line Spacing : untuk mengatur jarak antar baris teks.

-Shading : untuk mengatur warna latar teks terpilih.

-Border : untuk memberikan garis tepi pada teks terpilih.

Fungsi Ribbon pada Group Styles:

-Heading Styles : berisi pilihan format judul/subjudul dari paragraf terpilih.

-Change Styles : berisi pilihan pengaturan tema paragraf.

Fungsi Ribbon pada Group Editing

-Find : untuk mencari kata tertentu berdasarkan keyword tertentu dalam suatu file

-Replace : untuk mencari dan mengganti kata yang ditemukan untuk diganti dengan kata tertentu dalam suatu file

-Select : digunakan untuk memilih objek atau teks tertentu di dalam suatu file.

Jawaban:

Fungsi dari Ribbon font adalah sebagai tempat dimana terdapat banyak tools atau alat yang digunakan untuk mengedit atau membuat document.

27. Italic adalah fungsi untuk ... a. menebalkan font b. memiringkan font c. membesarkan ukuran font d. menghapus font


Jawaban:

Italic berfungsi untuk B. Memiringkan font

Pembahasan

Perintah untuk Italic adalah Ctrl + I yakni merupakan jenis baris perintah dalam Microsoft office yang berfungsi untuk membuat teks, tulisan maupun naskah menjadi berbentuk miring.

SESUAI OPSI SOAL:

Opsi A. Menebalkan font = Bold.Opsi B. Memiringkan font = Italic.Opsi C. Membesarkan ukuran font = Font size (bukan hanya untuk membesarkan ukuran font tapi juga bisa untuk mengecilkan ukuran font).Opsi D. Menghapus font = ini bisa digunakan untuk mereset font yang sebelum-sebelumnya pernah dipakai.

28. shortcut untuk grow font, shrink font, change case dan format painter


-.-.-.-.-.-.-.-.- oOOOo .-.-.-.-.-.-.-.-.

2. Shortcut Shrink Font (Mengecilkan ukuran font): "CTRL + <"
