17. volume tabung : 7.392t : 12hitunglah lpt

r² = 7.392x7/(22x12) = 196
r = √196
r = 14 cm

Lpt = 2πr ( r + t )
= 2 x 22/7 x 14 x ( 14 + 12 )
= 88 x 26
= 2.288 cm²Volume Tabung
