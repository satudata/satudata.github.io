Jawaban:

Menggunakan Akun Microsoft

buka menu Start Gambar berjudul Windowsstart.png. Anda bisa membukanya dengan mengeklik logo Windows yang berada di bagian bawah kiri taskbar.

Buka program Microsoft Office. Klik program Microsoft Office, seperti Microsoft Word atau Microsoft Excel, untuk membukanya.

Apabila belum memasang Microsoft Office di komputer, Anda bisa mengunduhnya di internet.

Klik tombol Activate. Tombol ini berada di bawah ikon kunci.

Klik tombol Sign In. Tombol ini berada di opsi "Activate".

Masuk ke akun Microsoft. Tiklah alamat surel (surat elektronik atau dikenal juga sebagai email) dan kata sandi (password) akun Microsoft.

Ikuti petunjuk pada layar dan klik tombol Next. Tindakan ini akan menyelesaikan proses pengaktifan Microsoft Office.

Apabila jumlah komputer yang memakai Microsoft Office melebihi batas kuota yang disediakan Office 365, Anda mungkin harus menonaktifkan Microsoft Office di salah satu komputer.

Menggunakan Kode Produk di Program Microsoft Office

Buka program Microsoft Office. Klik program Microsoft Office, seperti Microsoft Word atau Microsoft Excel, untuk membukanya.

Apabila memiliki kode produk, tetapi Microsoft Office belum dipasang di komputer, Anda bisa mengaktifkan Microsoft Office melalui situs webnya.

Klik tombol Activate. Tombol ini berada di bawah ikon kunci.

Klik tombol Enter Product Key. Tombol ini berada di opsi "Activate".

ketiklah kode produk dan klik tombol Continue. Tiklah kode produk yang terdiri dari 25 digit tanpa tanda pisah (-).

Klik tombol Redeem Online. Tombol ini berada di boks yang bernama "Add this key to an account".

Klik tombol Sign In atau tombol Create a new account. Apabila Anda sudah memiliki akun Microsoft, tiklah alamat surel dan kata sandi untuk masuk ke akun tersebut. Apabila Anda tidak memiliki akun Microsoft, klik tombol "Create a new account" dan ikuti petunjuk yang muncul pada layar untuk membuat akun baru.

Klik tombol Finish Activation. Setelah itu, proses pengaktifan akan selesai dan kode produk akan dihubungkan dengan akun Microsoft.
