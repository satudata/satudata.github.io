4. bilangan yang merupakan tripel pythagoras


Bilangan yang merupakan tripel pythagoras adalah 3, 4, 5. Tripel pythagoras merupakan tiga buah bilangan yang merupakan ukuran sisi-sisi segitiga siku-siku.  Misal suatu segitiga memiliki sisi a, b dan c dengan c adalah sisi terpanjang, maka berlaku rumus

a² + b² = c²

Pembahasan

Untuk menentukan tripel pythagoras, bisa kita gunakan ukuran sisi-sisi pada segitiga siku-siku yaitu (a² + b²), (a² – b²) dan 2ab dengan a, b adalah bilangan asli dan a ≠ b

11. menentukan nilai variabel jika diketahui 3 bilangan dengan variabel adalah tripel pythagoras


Contoh soal menentukan variabel jika diketahui tiga bilangan dengan variabel merupakan triple phytagoras. Berikut akan diberikan tiga contoh soal menentukan variabel pada panjang sisi segitiga siku-siku. Simak pembahasan berikut.

Pembahasan

Pada segitiga siku-siku berlaku teorema phytagoras: " kuadrat sisi miring (hipotenusa) sama dengan jumlah kuadrat sisi siku-sikunya".

Teorema phytagoras dapat dirumuskan sebagai berikut:

x² + y² = r²r² - y² = x²r² - x² = y²

dengan:

r = sisi miring

x, y = sisi siku-siku

Contoh soal triple phytagoras dengan variabel.

1. Apabila bilangan a + 4, 4a, dan 4a + 2 membentuk triple phytagoras, tentukan nilai a!

Jawab:

a + 4, 4a, dan 4a + 2 membentuk triple phyatgoras. Karena 4a + 2 merupakan sisi aling panjang, maka diperoleh sisi miring segitiga siku-siku tersebut adalah 4a + 2.

r = 4a + 2, x = a + 4, y = 4a

x² + y² = r²

(a + 4)² + (4a)² = (4a + 2)²

(a + 4)(a + 4) + 16a² = (4a + 2)(4a + 2)

a² + 4a + 4a + 16 + 16a² = 16a² + 8a + 8a + 4

a² + 4a + 4a + 16 + 16a² - 16a² - 8a - 8a - 4 = 0

a² + 8a  - 16a + 16 - 4 = 0

a² - 8a + 12 = 0

(a - 6)(a - 2) = 0

a = 6 atau a = 2

∴ Maka nilai a yang memenuhi adalah a = 6 atau a = 2.

2. Tiga bilangan t, 2t + 2, dan 2t + 3 membentuk triple phytagoras, tentukan nilai t!

Jawab:

t, 2t + 2, dan 2t + 3 membentuk triple hytagoras. Karena 2t + 3 merupakan sisi yang paling panjang, maka sisi miring diantara ketiga bilangan tersebut adalah 2t + 3. Sehingga diperoleh:

r = 2t + 3, x = t, y = 2t + 2

x² + y² = r²

t² + (2t + 2)² = (2t + 3)²

t² + (2t + 2)(2t + 2) = (2t + 3)(2t + 3)

t² + t² + 4t + 4t + 4 = t² + 6t + 6t + 9

2t² + 8t + 4 = t² + 12t + 9

2t² + 8t + 4 - t² - 12t - 9 = 0

2t² - t² + 8t - 12t + 4 - 9 = 0

t²  - 4t  - 5 = 0

(t - 5)(t + 1) = 0

t = 5 atau t = -1

Ingat! nilai x = t, artinya salah satu panjang sisi segitiga siku-siku tersebut adalah t, maka t tidak boleh bernilai negatif. Sehingga nilai t yang memenuhi adalah t = 5.

∴ Maka nilai t adalah 5.

3. Diketahui bilangan - bilangan 2p, 3p + 3, 5p - 3 membentuk triple phytagoras, tentukan nilai p!

Jawab:

5p - 3 merupakan sisi terpanjang, maka sisi miring dari bilangan triple phytagoras tersebut adalah 5p -3. Sehingga diperoleh:

r = 5p - 3, x = 2p, y = 3p + 3

x² + y² = r²

(2p)² + (3p + 3)² = (5p - 3)²

4p² + (3p + 3)(3p + 3) = (5p - 3)(5p - 3)

4p² + 9p² + 9p + 9p + 9 = 25p² - 15p - 15p + 9

13p² + 18p + 9 = 25p² - 30p + 9

13p² + 18p + 9 - 25p² + 30p - 9 = 0

13p² - 25p² + 18p + 30p + 9 - 9 = 0

- 12p² + 48p = 0   : -12

p² - 4p = 0

p(p - 4) = 0

p = 0 atau p = 4

Ingat! karena x = 2p, artinya salah satu panjang sisi segitiga siku-siku tersebut adalah 2p, sehingga p ≠ 0. Maka nilai yang memenuhi tiga bilangan tersebut adalah p = 4.

∴ jadi nilai p = 4.

14. bilangan tripel pythagoras adalah​


Jawab:

Bilangan tripel phytagoras adalah tiga bilangan asli yang memenuhi ketentuan teori phytagoras, contoh:  

3, 4, 5

5, 12, 13

7, 24, 25

8, 15, 17

9, 40, 41

11, 60, 61

12, 35, 37

13, 84, 85

15, 112, 113

16, 63, 65

Penjelasan:

Tripel pythagoras merupakan kelompok tiga bilangan asli yang memenuhi ketentuan yaitu kuadrat bilangan terbesar sama dengan jumlah kuadrat dua bilangan lainnya.

Perhatikan contoh kelompok tiga bilangan berikut.

3, 4, 5

Kuadrat bilangan terbesar :  5² = 25

Jumlah kuadrat dua bilangan lainnya :  3² + 4² = 9 + 16 = 25.

Oleh karena 5² = 3² + 4², kelompok tiga bilangan tersebut merupakan tripel pythagoras.

6, 8, 12

Kuadrat bilangan terbesar :  12² = 144

Jumlah kuadrat dua bilangan lainnya :  6² + 8² = 36 + 64 = 100

Oleh karena 12² ≠ 6² + 8², kelompok tiga bilangan tersebut bukan tripel pythagoras.

17. bilangan terkecil dari tripel pythagoras adalah 33. tentukan tripel pythagoras. jelaskan bagaimana kalian menemukan dua bilangan lainnya


Kelas : VIII

Pelajaran : Matematika

Kategori : Teorema Pythagoras

Kata Kunci : Tripel Pythagoras

 

Pembahasan Materi:

Teorema Pythagoras adalah teori yang menyatakan bahwa kuadrat sisi terpanjang suatu segitiga siku-siku sama dengan jumlah kuadrat sisi-sisinya yang lain

Misal : suatu segitiga siku-siku jika sisi siku-sikunya adalah p dan q, maka panjang sisi terpanjang (sisi miring) misal r dapat dicari dengan rumus p² + q² = r²

Tripel Pythagoras adalah tiga bilangan bulat positif (misal: a, b dan c, a,b<c) yang memenuhi sifat hubungan a² + b² = c²

Cara umum yang dapat digunakan untuk menemukan bilangan-bilangan tripel pythagoras adalah

a = x² - y²

b = 2xy

c = x² + y²

dimana x dan y adalah dua bilangan bulat positif berbeda (x > y)

 

Pembahasan Soal:

Cara 1

Diketahu bilangan terkecil tripel pythagoras adalah 33

Misal a = 33, akan dicari nilai b dan c

Karena bilangan terpendeknya ganjil, maka asumsi bahwa x² - y² = 33

x² - y² = 33

(x – y).(x + y) = 33

Pasangan bilangan bulat dengan hasil perkalian 33 adalah (1,33) dan (3,11)

 

Kemungkinan 1 (x – y).(x + y) = 1.33

Asumsi bahwa (x – y) = 1 dan (x + y) = 33

x = (33+1)/2 = 34/2 = 17

y = (33-1)/2 = 32/2 = 16

maka diperoleh :

a = 17² - 16² = 289 – 256 = 33

b = 2xy = 2.17.16 = 544

c = x² + y² = 289 + 256 = 545

 

Kemungkinan 2 (x – y).(x + y) = 3.11

Asumsi bahwa (x – y) = 3 dan (x + y) = 11

x = (11+3)/2 = 14/2 = 7

y = (11-3)/2 = 8/2 = 4

maka diperoleh :

a = 7² - 4² = 49 – 16 = 33

b = 2xy = 2.7.4 = 56

c = x² + y² = 49 + 16 = 65

 

Cara 2

Diketahu bilangan terkecil tripel pythagoras adalah 33

Karena bilangan terkecil adalah kelipatan 3, maka dapat dibandingkan dengan bilangan tripel pythagoras dengan bilangan terkecil 3.

Telah diketahui bahwa (3,4,5) adalah bilangan tripel pythagoras sehingga untuk setiap x bilangan bulat positif, bilangan-bilangan (3x,4x,5x) juga merupakan tripel pythagoras

Jadi

a = 3.11 = 33

b = 4.11 = 44

c = 5.11 = 55

 

Jadi dua bilangan lainnya yang mungkin adalah

(1) 544 dan 545

(2) 56 dan 65

(3) 44 dan 55

Teorama Pythagoras  

Untuk setiap segitiga siku-siku berlaku : luas persegi pada sisi miring (hipotenusa) sama dengan jumlah luas persegi pada sisi yang lain (sisi siku-sikunya).  

Tripel (tigaan) Pythagoras adalah tiga bilangan asli yang tepat untuk menyatakan panjang sisi-sisi suatu sgitiga siku-siku.

Pembahasan

Bilangan terkecil dari tripel pythagoras adalah 33. Kita cari faktor dari 33 yaitu 3 × 11. Bilangan 3 terdapat pada tripel pythagoras 3, 4, dan 5. Dan bilangan 11 merupakan kelipatannya.

Untuk menemukan dua bilangan lainnya kita bisa mengalikan 4 dan 5 dengan 11.

4 × 11 = 44

5 × 11 = 55

Jadi yang merupakan tripel pythagoras adalah 33, 44, dan 55
