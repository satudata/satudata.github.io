1. warung bu inah menjual setoran nasi goreng jika harga 3 porsi soto dan dua porsi nasi goreng adalah 49.000 sedangkan harga dua porsi 1 dan 4 porsi nasi goreng adalah 70.000 harga 1 porsi soto dan 1 porsi nasi goreng adalah​


Jawaban:

Misalkan

Harga 1 porsi soto = x

Harga 1 porsi nasi goreng = y

Harga 3 porsi soto dan 2 porsi nasi goreng adalah Rp49.000,00

→ 3x + 2y = 49.000

Harga 2 porsi soto dan 4 porsi nasi goreng adalah Rp70.000,00

→ 2x + 4y = 70.000

Eliminasi y :

3x + 2y = 49.000 | ×2|

2x + 4y = 70.000 | ×1|

6x + 4y = 98.000

2x + 4y = 70.000

4x = 28.000

x = 28.000/4

x = 7.000

Substitusi x = 7.000

3x + 2y = 49.000

3(7.000) + 2y = 49.000

21.000 + 2y = 49.000

2y = 49.000 - 21.000

2y = 28.000

y = 28.000/2

y = 14.000

Harga 1 porsi soto = 7.000

Harga 1 porsi nasi goreng = 14.000

Harga 1 porsi soto dan 1 porsi nasi goreng

7.000 + 14.000 = 21.000

7. sebuah restoran menyediakan tiga jenis makanan favorit, yaitu mie goreng, sate rusa, dan nasi goreng. Keluarga firdaus membayar 380 ribu untuk 1 porsi mie goreng, 3 porsi sate rusa, 2 porsi nasi goreng, sedangkan keluarga syahrul membayar 520 ribu untuk 5 porsi sate rusa dan 2 porsi nasi goreng. Harga tiap porsi tetap dan harga 3 porsi mie goreng sama dengan harga 1 porsi nasi goreng. Jika x, y, dan z masing-masing mewakili harga setiap porsi mie goreng, sate rusa, dan nasi goreng a. buatlah model matematika masalah diatas, kemudian nyatakan dalam bentuk persamaan matriks b. berapakah harga tiap porsi ketiga jenis makanan itu


Model matematika pada soal ini adalah

Firdaus : x + 3y + 2z = 380.000

Syahrul : 5y + 2z = 520.000

Matriknya (terlampir pada gambar)

Harga masing-masing porsi ketiga jenis makanan adalah:

a. mie goreng Rp 20.000,-

b. sate rusa Rp 80.000,-

c. nasi goreng Rp 60.000,-

Sistem persamaan linear tiga variabel (SPLTV) adalah bentuk perluasan dari sistem persamaan linear dua variabel (SPLDV).

Ciri-ciri SPLTV adalah menggunakan relasi tanda sama dengan, memiliki 3 variabel, dan masing-masing variabel bersuku satu atau berpangkat satu.

Pembahasan

Diketahui:

Keluarga Firdaus menghabiskan Rp 380.000,- untuk 1 porsi mie goreng, 3 porsi sate rusa, dan 2 porsi nasi goreng.

Kelurga Syahrul menghabiskan Rp 520.000,- untuk 5 porsi sate rusa dan 2 porsi nasi goreng.

harga 3 porsi mie goreng = harga 1 porsi nasi goreng

Ditanya:

a. bagaimana model matematika dan persamaan dalam bentuk matriks?

b. berapa harga per porsi dari mie goreng, sate rusa, dan nasi goreng?

Jawab:

a. Model matematika dari soal tersebut.

Dimisalkan:

mie goreng = x

sate rusa = y

nasi goreng = z

Maka model matematikanya adalah

Keluarga Firdaus

x + 3y + 2z = 380.000 ........................(1)

Keluarga Syahrul

5y + 3z = 520.000 ..............................(2)

3 posi mie goreng = 1 porsi nasi goreng

3x = z  ......................................................(3)

Persamaan matriksnya (terlampir pada gambar)

b. Menentukan harga per porsi masing-masing jenis makanan

Untuk menentukan harga per porsi dari ketiga jenis makanan yang merupakan sistem persamaan 3 variabel, kita bisa menggunakan metode substitusi, metode eliminasi dan metode campuran substitusi - eliminasi.

1. Substitusi persamaan (3) ke persamaan (1) dan (2)

 x + 3y + 2z = 380.000

          x + 3y + 2(3x) = 380.000

          x + 3y + 6x = 380.000

          7x + 3y = 380.000  .................................(4)

 5y + 2z = 520.000

          5y + 2(3x) = 520.000

          5y + 6x = 520.000

          6x + 5y = 520.000 .......................................(5)

2. Eliminasi y dari persamaan (4) dan (5)

   7x + 3y = 380.000 ⇒ dikali 5 ⇒ 35x + 15y = 1.900.000

   6x + 5y = 520.000 ⇒ dikali 3 ⇒ 18x + 15y = 1.560.000

                                                         17x           = 340.000

                                                          x              = 340.000 : 17

                                                          x              = 20.000

Subtitusikan nilai x pada persamaan (3) maka diperoleh:

  3x = z ⇒ 3 x 20.000 = z

  z = 60.000

3. Setelah diperoleh nilai x = 20.000 dan z = 60.000 maka untuk mencari nilai y dapat dilakukan dengan substitusi nilai x dan z ke persamaan (1), sehingga diperoleh:

  x + 3y + 2z = 380.000

  20.000 + 3y + (2 x 60.000) = 380.000

  20.000 + 3y + 120.000 = 380.000

  3y = 380.000 - (20.000 + 120.000)

  3y = 380.000 - 140.000

  3y = 240.000

  y = 240.000 : 3

  y = 80.000

Jadi harga per porsi dari tiga jenis makanan adalah

a. mie goreng = Rp 20.000,-

b. sate rusa = Rp 80.000,-

c. nasi goreng = Rp 60.000,-

nasi goreng = x

kebab = y

maka :

2x + 1y = 25.000

2(10.000) + 1y = 25.000

20.000 + 1y = 25.000

1y = 25.000 - 20.000

y = 5.000,00

Harga 2 Kebab ( y )

= 2 x 5.000,00

= 10.000,00

PENYELESAIAN :

-- Membeli 2 porsi nasi goreng dan 1 porsi kebab

--» 1 porsi nasgor harganya Rp 10.000,00

--» Total harga Rp 25.000,00

= 10.000 × 2

= 20.000, 2 nasgor

= Total - dua nasgor

= 25.000 - 20.000

= 5.000 (kebab)

--» 2 porsi kebab

= 2 × 5.000

= (2 × 5)000

= 10.000,00

Jadi harga 2 porsi kebabnya adalah Rp. 10.000,00
