12. Apa perbedaan window 7 ultimate dengan profesional?


Dari Segi Penggunanya

• Microsoft memang menerbitkan setiap versi dengan target pengguna yang berbeda.

• Windows 7 Starter Edition ditujukan untuk netbooks yang notabene memiliki specs minimum.

• Windows 7 Home Basic ditujukan untuk pengguna yang hanya butuh fitur dasar dari Windows 7.

• Windows 7 Home Premium dan Windows 7 Profesional ditujukan untuk digunakan oleh pengguna personal dengan aktivitas berkomputer yang cukup intens.

• Windows 7 Enterprise lebih ditujukan untuk perusahaan besar.

• Windows 7 Ultimate ditujukan untuk pengguna advanced yang antusias dan ingin fitur optimal dari Windows 7.

Dari Segi Fitur

• Perbedaan versi Windows 7 akan lebih jelas terlihat dengan membandingkan fitur dan kemampuannya. Untuk itu WinPoin sudah memberikan daftar fitur dari setiap versi Windows 7:

Dari Segi harga

Dari segi harga untuk di Indonesia, harga setiap versi Windows 7 juga tentu saja berbeda.

• Windows 7 Starter dijual dengan harga sekitar $62

• Windows 7 Home Basic dijual dengan harga sekitar $100

• Windows 7 Home Premium dijual dengan harga sekitar $125

• Windows 7 Profesional dijual dengan harga sekitar $160

• Windows 7 Ultimate dijual dengan harga sekitar $210


13. berapa lama cooldown ultimate atlas jika tidak membeli fleeting time? ​


Selain stat di atas, item ini juga memiliki pasif unik yang bernama Timestream.

Nah, Fleeting Time ini emang dikenal sebagai salah satu item yang berguna banget untuk mempersingkat cooldown dari skill hero.

Kenapa? Karena selain stat di atas, pasif unik yang bernama Timestream tersebut bakal mengurangi 35% cooldown skill ultimate dari hero.

Tapi, efek dari pasif unik ini bakal muncul setelah kalian berhasil mendapatkan kill atau assist dari hero lawan terlebih dahulu.

Fleeting Time ini biasanya dibutuhkan oleh hero yang memiliki magic dengan cooldown skill ultimate yang cukup lama seperti Heart Guard milik Angela yang berdurasi 1 menit lebih.

Dengan pasif Timestream dari Fleeting Time, maka Angela nggak harus menunggu lama untuk mengeluarkan skill ultimate-nya.

Item ini bisa digunakan untuk hero mage dan juga support yang bisa dipilih sebagai item pertama atau kedua agar lebih terasa manfaatnya.(*)

25. bagaimana cara mengatasi server busy pada windows 7 ultimate ?  


hmmmm. coba agan bongkar pc nya trs bersihkan debu-debu didalamnya1.     Jalankan icon start pada windows 7, ketikkan msconfig.exe

2.     Jika langkah 1 berhasil maka akan masuk ke menu System Configuration. Pilih option tab General, arahkan pilihan dari yang semula Normal Startup dipindah ke Selective Startup agar aktif. Clean atau bersihkan tanda centang di kotak Load Startup Items dengan cara mengklik tanda centang yang masih ada (tombol on/off).

3.     Pindahkan tab option dari General ke Services, dan centang (klik) pada kotak bawah kiri yang bertuliskan “Hide All Microsoft Services” dan selanjutnya tekan tombol “Disable all” untuk menonaktifkan services aplikasi yang tengah berjalan. Eksekusi pilihan dengan tombol “Apply” dan “Ok”.

4.     Langkah selanjutnya akan diminta untuk restart sistem saat itu atau nanti. Pilih saja restart tanpa menunda, dan saat masuk setelah booting sistem operasi windows 7 pesan “server busy” sudah tak ada lagi.  

5.     Untuk menormalkan kembali langkah booting sistem konfigurasi dapat dengan mencari sumber dan penyebab aplikasi apasaja yang selalu minta konek ke server melalui jaringan untuk update lebih dahulu sebelum masuk ke sistem operasi yang terbuka sempurna. Jika sudah ditemukan penyebanya, kembalikan opsinya ke Sistem konfigurasi normal startup dari pilihan opsi selective startup. Jika belum ketemu juga, pakai selective startup juga ndak masalah asal nggak bener-bener ngaruh ke unjuk kerja aplikasi-aplikasi lainnya.
