1. Cara Mengaktifkan Bluetooth di Laptop ASUS X453MA , Karna laptop ini berbeda, tidak ada lambang bluetooth nya di F1-F12


Intinya, coba buka system tray (ikon2 yg biasanya ada di pojok kanan, samping jam), kemudian pilih ikon bluetooth (jika ada), dan jika ada, pilih Setting dan klik Turn On Bluetooth
Kalau tidak ada, coba ke control panel, lalu pilih system, kemudian pilih Device Manager. Lalu, coba cari bluetooth, kemudian klik kanan dan klik Enable
Kalau tidak ada, kemungkinan besar driver bluetoothnya belum dipasang atau laptop tidak ada fasilitas bluetoothnya

15. tombol pada keyboard yang terdiri dari f1 sampai dgn f12 disebut


Tombol pada keyboard yang terdiri dari f1 sampai f12 disebut tombol Function Keys. Selain tombol function keys, terdapat beberapa kelompok tombol lain pada keyboard. Pada keyboard terdapat tombol Escape Keys. Tombol typing keyboard. Tombol cursor keys, dan numeric keypad sertaTombol kontrol (Control keys).

Pebahasan

Keyboard adalah salah satu hardware komputer yang berfung sebagai alat input. Melalui keyboard kita dapat menginput data berupa angka, huruf, dan simbol ke komputer. Keyboard juag menjadi jalan pintas saat bekerja, jika tidak menggunakan mouse.

Seperti yang disebutkan di atas, keyboar terdiri dari beberapa kelompok tombol. Berikut ini kita bahas fungsi dari Function Keys dari f1 sampai f12.

F1 digunakan untuk tombol helpF2 digunakan untuk rename, change volumeF3 digunakan untuk SearchingF4 dikombinasikan dengan Alt berfungsi menutup programF5 digunakan untuk RefreshF6 diguanakn untuk AddressingF7 digunakan untuk memeriksaan kesalahan ketik dan ejaan teksF8  sebagai Awal perintah penyorotan atau pemilihan teks atau objekF9 untuk Mengupdate Field (Mail Merge)F10 dkombiasi dengan tombol Shift berfungsi sebagai alternatif klik kanan.  F11 digunakan untuk memasukkan field berikutnya (Mail Merge)F12 digunakan untuk save as, sebagai tombol Alternatif dan untuk mengaktifkan atau memulai penggunaan menu bar.

23. jelaskan apa yang dimaksud f1-f12


1. F1
Biasanya tombol ini digunakan untuk memanggil file help dari suatu software atau aplikasi.Tombol Win + F1 akan membuka Microsoft Windows help and support center.

2. F2
Untuk di windows dapat digunakan sebagai jalan pintas mengubah nama suatu file maupun folder.Tombol kombinasi Alt + Ctrl + F2 digunakan untuk membuka dokumen baru di Microsoft Word.Ctrl + F2 menampilkan jendela print preview dalam Microsoft Word.

3. F3
Sering kali diguanakan untuk membuka fitur pencarian untuk banyak program termasuk Microsoft Windows.Jika kalian sedang bekerja dalam Ms word atau yang lain dapat menggunakan CTRL+F atau mencari file di windows dapat menggunakan Win+FShift + F3 akan mengubah teks dalam Microsoft Word menjadi uppercase(huruf besar) atau lowercase(huruf kecil) atau huruf kapital pada awal setiap kata.

4. F4
Membuka jendela.Mengulangi aksi terakhir yang dilakukan di Word 2000 +Alt + F4 akan menutup program yang sedang aktif dalam Microsoft Windows.Ctrl + F4 akan menutup jendela yang terbuka dalam jendela aktif saat ini di Microsoft Windows.

5. F5
Mungkin tombol ini yang paling populer untuk me reload halaman di browser maupun untuk refresh tampilan windows.Buka window pencarian, penggantian kata di Microsoft Word.Mulai tayangan slide show dalam PowerPoint.
6. F6
Memindahkan kursor ke address bar di Internet Explorer dan Mozilla Firefox.

7. F7
Biasanya digunakan untuk memeriksa ejaan dan tata bahasa (spelling) dalam program-program Microsoft seperti Microsoft Word, Outlook, dllShift + F7 melakukan pemeriksaan Thesaurus pada kata yang disorot.Menyalakan tsobat sisipan browsing di Mozilla Firefox.

8. F8
Fungsi tombol yang digunakan untuk masuk ke menu Windows startup, biasanya digunakan untuk masuk ke Windows Safe Mode.

9. F9
Membuka toolbar Pengukuran dalam Quark 5.0.
10. F10
Pada Microsoft Windows digunakan untuk mengaktifkan menu bar dari aplikasi yang terbuka.Shift + F10 adalah sama dengan klik kanan pada icon yang disorot, file, atau link internet.

11. F11
Modus layar penuh di pada web browser.

12. F12
Membuka menu Save As pada Microsoft Word.Ctrl + Shift + F12 mencetak/print dokumen di Microsoft Word.
