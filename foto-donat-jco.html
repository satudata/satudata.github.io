Yang dibeli Sandy = 1 kotak donat + 1 Ice tea

Ditanyakan : Model Matematika SPLDV permasalahan di atas

Jawab :

Belanjaan Annisa

2 kotak donat + 1 ice tea = Rp200.000,00 - Rp20.000,00

= Rp180.000,00.

Belanjaan Sandy

1 kotak donat + 1 ice tea = Rp100.000,00.

Pemisalan

kotak donat = xice tea = y

Model Matematika SPLDV

Annisa ==>> 2x + y = 180.000

Sandy ==>> x + y = 100.000

Jadi penulisan model matematika dari pernyataan di atas adalah :

2x + y = 180.000

Jawaban:

27 cara

A = himpunan jumlah donat = (10,12,15)

B = himpunan rasa = (keju, coklat, kacang)

[tex]banyak \: cara \: = {b}^{a} \: = \: {3}^{3} = 3 \times 3 \times 3 = 27[/tex]

jadi, banyak cara untuk mengikat donat-donat tersebut ada 27 cara
