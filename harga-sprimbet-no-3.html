9. 3 kao dan 4 topi dijual eharga 960000,dua kao dan 5 topi di jual 990000 tentuian 2harga topi dan 3 harga kao.


Jawaban singkat:

Harga 2 topi dan 3 kao = 660.000

Penjelasan dg langkah²:

Misalkan kao = K dan topi = T, maka:

3K + 4T = 960.000 (persamaan 1)

2K + 5T = 990.000 (persamaan 2)

- Persamaan 1 dikalikan 5 dan persamaan 2 dikalikan 4

- Hasil perkaliannya dikurangkan (amati lampiran)

- Hasil akhirnya ditemukan nilai K = 120.000

Masukkan nilai K dalam perhitungan persamaan 2:

2K + 5T = 990.000

240.000 + 5T = 990.000

5T = 990.000 - 240.000

5T = 750.000

T = 150.000

Harga 2 topi dan 3 kao = (2 x 150.000) + (3 x 120.000)

Harga 2 topi dan 3 kao = 300.000 + 360.000

Harga 2 topi dan 3 kao = 660.000

29. Pada suatu toko harga gula 4/3 harga beras harga beras 2/5harga telur selisih harga gula dan beras RP2.400,00 berapakah harga telur


Misal :
gula = a
beras = b
telur = c

a = 4/3 b
a : b = 4 : 3

b = 2/5 c
b : c = 2 : 5

a : b : c = 8 : 6 : 15

Selisih a dan b = 2.400

Harga telur
= 15/(8 - 6) x Rp. 2.400
= 15/2 x Rp. 2.400
= Rp. 18.000

➡️Bab: Perbandingan
➡️Kata Kunci: (-)

Pembahasan
⬇️⬇️⬇️⬇️⬇️

gula = 4/3 beras

gula : beras = 4 : 3


beras = 2/5 telur

beras : telur = 2 : 5


gula : beras : telur = 8 : 6 : 15

telur

= 15/(8 - 6) × 2.400

= 15/2 × 2.400

= 15 × 1.200

= 18.000[tex] [/tex]
