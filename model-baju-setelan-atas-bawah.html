14. Harga awal satu setel baju adalah rp. 78.000,00. harga satu setel baju itu setalah dipotong diskon 25% adalah rp. ...


~ Diskon, Untung, dan Rugi

_______________________________

Pembahasan :

Diketahui :Harga awal = Rp 78.000,00%Diskon = 25%

Ditanya :Harga baju setelah di diskon = ... ?

Penyelesaian :

Harga baju setelah di diskon

= ( 100% - %Diskon ) × Harga awal

[tex] = (100\% - 25\%) \times 78.000[/tex]

[tex] = 75\% \times 78.000[/tex]

[tex] = \frac{75}{100} \times 78.000[/tex]

[tex] = 75 \times 780[/tex]

[tex] = 58.500[/tex]

Kesimpulan :

Jadi, harga satu setel baju setelah di diskon adalah Rp 58.500,00

dengan rumus keuntungan maksimum hasilnya adalah Rp 2.000.000. caranya : 3x + 2y = 60    Dengan Fungsi 100.000x + 60.000 y
1x + 2y = 40  
Petidakasamaanya, 3x + 2 y = 60 ..... x = 0; y = 30  -----> ( 0,30)
                                                            y =  0;  x = 20 -----> ( 20, 0 )
                                 1x + 2y = 40 .......x = 0;  y= 20 ---------> ( 0,20 )
                                                             y = 0;  x= 40 ---------> ( 40, 0 )
sehingga jika digamabarkan akan membentuk diagram dengan fungsi
 A. 100.000( 0 ) + 60.000 ( 30 ) = Rp 1800.000
B . 100.000 ( -20 ) + 60.000 ( 60 ) = Rp. 1600.000
c. 100.000 ( 20 ) + 60.000 ( 0 ) = Rp.2000.000. jadi keuntungan maksimum adalah Rp. 2000.000.
