Penjelasan dengan langkah-langkah:

Harga asli => 200.000,00Harga Bayar => 175.000,00Diskon ??

Diskon 15%

200.000,00 - 15%

= ( 200.000,00 - ( 200.000,00 × 15% ))

= ( 200.000,00 - 30.000 )

= 170.000 ( Maka Salah )

-------------------------------------------------------

Diskon 12,5% => Benar

200.000,00 - 12,5%

= ( 200.000,00 - ( 200.000,00 × 12,5% )

= ( 200.000,00 - 25.000 )

= 175.000,00 ( Maka benar )

Jadi diskon yg di dapatkan adalah 12,5%
