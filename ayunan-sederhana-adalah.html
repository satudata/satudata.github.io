5. suatu ayunan sederhana melakukan getaran sebanyak 480 kali dalam 2 menit. periode ayunan sederhana tersebut adalah


Mapel : Fisika

Kelas : 8 Smp

Bab : Getaran


Penyelesaian

Dik : n = 480 Kali
t = 2 menit
= 2 x 60 detik
= 120 detik

Dit : periode ayunan

Jawab :

T = t / n
= 120 detik / 480 kali
= 0,25 Sekon


Jadi periode ayunan sederhana tersebut adalah 0,25 Sekon


Keterangan

T = Periode ayunan ( sekon )
t = waktu bergetar ( detik / sekon )
n = jumlah getaran ( kali )
