10. Sebutkan cara menghitung range di excel


Jawaban:

Dalam Ms. Excel, range adalah sekumpulan cell. Range dapat dihitung melalui tampilan Name Box.

Di tampilan Name Box akan tampak jumlah baris dan jumlah kolom yg terlibat pada range.

Penjelasan:

Misalkan pada gambar di bawah, terdapat range dari cell C3 sampai F12. Melalui tampilan Name Box dapat dihitung bahwa range itu terdiri dari 10R dan 4C.

Artinya range itu terdiri dari 10 ROW (10 baris) dan 4 Column (4 kolom). Amati gambar berikut ini dg seksama (perhatikan di bagian yg ditunjukkan tanda panah merah).

*range dapat berupa sekumpulan cell yg kosong & dapat pula sekumpulan cell yg berisi data*

17. Cara menghitung nama yang sama pada excel


Jawaban singkat:

Caranya dg memakai fungsi COUNTIF

Penjelasan panjang:

Misalkan di Ms. Excel 2013, ingin mencari nama Joko dengan pencarian mulai dari sel B7 sampai sel B13, maka penulisan rumusnya:

=COUNTIF(B7:B13;"Joko")

Untuk lebih jelasnya, amati di lampiran:

- hasil rumusnya ditunjukkan tanda panah merah

- penulisan rumusnya ditunjukkan tanda panah biru

29. Cara mencetak lembar kerja pada microsoft excel


Jawaban:

Mencetak satu atau beberapa lembar kerja

Pilih lembar kerja yang ingin Anda cetak.

Klik File > Cetak,atau tekan CTRL+P.

Klik tombol Cetak atau sesuaikan Pengaturan sebelum Anda mengklik tombol Cetak.

Mencetak satu atau beberapa buku kerja

Semua file buku kerja yang ingin Anda cetak harus dalam folder yang sama.

Klik File > Buka.

Tahan CTRL klik nama setiap buku kerja untuk dicetak, lalu klik Cetak.

Mencetak semua atau sebagian lembar kerja

Klik lembar kerja, lalu pilih rentang data yang ingin Anda cetak.

Klik File, lalu klik Cetak.

Di Pengaturanbawah , klik panah di samping Cetak Lembar Aktif dan pilih opsi yang sesuai.

Di bawah Pengaturan, klik Cetak Seluruh Lembar Kerja

Klik Cetak.

Catatan: Jika lembar kerja memilih area cetak yang ditentukan, Excel hanya akan mencetak area cetak tersebut. Jika Anda tidak hanya ingin mencetak area cetak yang ditetapkan tersebut, pilih kotak centang Abaikan area cetak. Pelajari selengkapnya tentang mengatur atau mengosongkan area cetak.

Mencetak tabel Excel

Klik sel dalam tabel untuk mengaktifkan tabel.

Klik File, lalu klik Cetak.

Di bawah Pengaturan, klik panah di samping Cetak Lembar Aktif dan pilih Cetak Tabel Terpilih.

Mencetak Tabel yang Dipilih

Klik Cetak.

Mencetak buku kerja ke satu file

Klik File lalu klik Cetak, atau tekan CTRL+P.

Di bawah Printer, pilih Cetak ke File.

Opsi Cetak ke File

Klik Cetak.

Dalam kotak dialog Simpan Output Cetak Sebagai, masukkan nama file lalu klik OK. File akan disimpan dalam folder Dokumen Anda

Penting: Jika Anda mencetak file yang disimpan di printer berbeda, ganti halaman dan penspasian font mungkin berubah.
