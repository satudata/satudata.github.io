Misal: jaket = j

kaos = k

Eka : 3j + 4k = 360.000

Eko : j + 5k = 285.000

Eki : 2j + 2k = ?

eliminasi:

3j + 4k = 360.000 |×1|

j + 5k = 285.000 |×3|

---------------------------- (-)

=

3j + 4k = 360.000

3j + 15k = 855.000

---------------------------- (-)

- 11k = - 495.000

k = - 495.000 ÷ - 11

k = 45.000

subtitusi:

j + 5k = 285.000

j + 5(45.000) = 285.000

j + 225.000 = 285.000

j = 285.000 - 225.000

j = 60.000

hasil:

2j + 2k =

2(60.000) + 2(45.000) =

120.000 + 90.000 = 210.000

Jadi, uang yang harus di bayar Eki pada kasir adalah 210.000

13. Dokter memberikang masker persegi panjang kepada Zara. Pangjang 33 cm. Dan lebarnya 65 cm. Berapakah keliling masker Zara?


Keliling masker adalah 196 cm

Pendahuluan

Persegi panjang adalah bangun datar yang memiliki dua buah sisi sejajar. Sifat-sifat persegi panjang antara lain yaitu :

Memiliki empat buah sisiMemiliki dua diagonalMemiliki dua simetri lipatMemiliki dua simetri putar

PenyelesaianDiketahuipanjang (p) = 33 cmlebar (ℓ) = 65 cmDitanyakan

Keliling

Jawab

Keliling = 2 × (p + ℓ)

= 2 × (33 + 65)

= 2 × 98

= 196 cm

Total 25% keuntungan dari 100 jaket = Rp. 5000.000 x 25% = Rp. 1.250.000

Jadi, harga jual setiap jaket adalah Rp.50.000 + (Rp. 1250.000 : 100) = Rp. 50.000 + Rp. 25.000 = Rp. 75.000
