22. N + 30 =100 + 71N Nilai N ====================== Dapat Fountain Of Widsom :)


Jawaban:

Nilai N = -1

================================

Penjelasan dengan langkah-langkah:

N + 30 = 100 + 71N

→ N - 71N = 100 - 30

→ -70N = 70

→ N = -70 ÷ 70

→ N = -1

Jadi, nilai N sama dengan -1

=================================

Jawaban:

1. Susun ulang suku

N + 30 = 100 + 71 N

N + 30 = 71 N + 100

2. Kurangi Kedua ruas persamaan dengan 30

N + 30 = 71 N + 100

N + 30 - 30 = 71 N + 100 - 30

3. Sederhanakan

Kurangkan bilangannya

N = 71 N + 70

4. Kurangi kedua ruas persamaan dengan 71 N

N = 71 N + 70

N - 71 N = 71 N + 70 - 71 N

5. Sederhanakan

Gabungkan suku sejenis

-70 N = 70

6. Bagi kedua ruas persamaan dengan suku yang sama

-70 N = 70

[tex] \frac{ - 70n}{ - 70} = \frac{70}{ - 70} [/tex]

7. Sederhanakan

Hapus suku yang ada di baik pembilang maupun penyebut.

Bagi bilangannya.

N = -1
