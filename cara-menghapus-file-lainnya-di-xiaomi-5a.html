13. cara menghapus benar-benar file yang telah dihapus sebelumya pada komputer ataupun gadged...


dengan menghapusnya dari recycle bin
- Cara pertama: Klik/pilih file, lalu klik kanan lalu klik Delete atau tekan tombol Del di keyboard, kemudian klik Ok (tapi, ntar masuk ke Recycle Bin). Kemudian, masuk ke Recycle Bin (penyimpanan sementara sebelum dihapus secara permanen), klik file yang tadi dihapus lalu tekan tombol Del dan klik Ok atau klik kanan di Recycle Bin, kemudian pilih Empty Recycle Bin
- Cara ke2 (langsung kehapus permanen): klik/pilih file, kemudian tekan Ctrl+Del dan pilih Ok (note: hati-hati dalam menggunakan cara ini karena file hilang secara permanen/tidak bisa direstore/dikembalikan lagi)

27. Bagaimanakah cara menghapus nama file​


Penjelasan:

Metode pertama, Anda pergi ke folder yang ingin Anda hapus dan klik kanan di atasnya. Daftar pilihan akan muncul. Dari antara orang-orang pilihan Anda harus klik pada tab ditandai sebagai "Hapus" untuk menghapus folder dari lokasi saat ini.

Hal yang penting untuk dicatat adalah bahwa menggunakan metode ini folder hanya dapat dihapus dari posisinya sekarang, untuk menghapus folder untuk baik folder harus juga akan dihapus dari recycle bin. Setelah itu dihapus dari recycle bin kemudian itu hilang dari perangkat Anda untuk kebaikan.

Jika folder yang hadir dalam USB drive, kemudian setelah mengikuti metode yang disebutkan di atas, folder akan dihapus dari sistem untuk selama-lamanya dan Anda tidak perlu khawatir tentang menghapusnya dari recycle bin juga.

29. cara mengembalikan file yang terhapus di colornote


1. klik ikon menu => setting

2. carilah opsi back up kemudian klik

3. Sekarang anda dapat melakukan restore data catatan ColorNote lama anda ke Hp android baru anda atau lainnya- Buka Colornote, buka menu atau tanda strip 3 di pojok kiri atas
- Setelah terbuka menunya, pilih Tong Sampah (atau kalau pake b.inggris: Trash). Nah disitulah tempat file & note yg telah terhapus disimpan sebelum dihapus permanen (mirip Recycle Bin di komputer)
- Setelah dibuka Trash/tong sampahnya, pilih file yang telah terhapus, kemudian tekan & tahan file yg akan dikembalikan (contoh di lampiran ingin mengembalikan file "Test123"), lalu pilih Pemulihan (atau kalau b.inggrisnya Restore), lali balik lagi ke menu Catatan/Note, dan nanti kamu akan menemukan file/note yg sebelumnya terhapus...
