Diketahui :
Harga mesin fotokopi = 5x harga komputer
Harga 5 komputer + 2 mesin foto kopi = Rp 60.000.000,-

Ditanyakan :
Harga mesin foto kopi?

Jawab :
Mesin fotokopi = a
Komputer = b

Harga mesin fotokopi = 5x harga komputer, berarti :
a = 5b

5 komputer + 2 mesin foto kopi = Rp 60.000.000,- Operasi aljabarnya adalah :
5b + 2(5b) = Rp 60.000.000,-
5b + 10b = Rp 60.000.000,-
15b = Rp 60.000.000,-
b = 60.000.000 : 15
b = 4.000.000,-
Jadi, harga komputer = b = Rp 4.000.000,-

Harga mesin fotokopi =
a = 5b
a = 5 ( Rp 4.000.000,-)
a = Rp 20.000.000,-

Jadi, harga mesin fotokopi = Rp 20.000.000,-
