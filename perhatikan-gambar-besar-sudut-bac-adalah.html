5. perhatikan gambar! Besar sudut BAC adalah​


Jawab:

Besar sudut BAC adalah 50° (jawaban : C).

Penjelasan dengan langkah-langkah:

Jumlah sudut yang dibentuk oleh sebuah segitiga serta sudut yang dibentuk dari satu sisi sebuah garis lurus adalah sama-sama bernilai 180°. Dengan demikian berlaku

∠ BAC + ∠ ABC + ∠ ACB = ∠ ABC + ∠ DBC

∠ BAC + ∠ ACB = ∠ DBC, masukkan nilai-nilai yang diketahui,

(y + 10°) + (2y + 10°) = 140°

3y + 20° = 140°

3y = 140° - 20°

3y = 120°

y = 120° / 3

y = 40°

Dengan demikian sudut BAC adalah

∠ BAC = y + 10°

∠ BAC = 40° + 10°

∠ BAC = 50°

14. Perhatikan gambar ! besar sudut BAC adalah .... *​


⟩ Garis dan Sudut

Diketahui :

Sudut BAC = (2x + 4)°Sudut ABC = (8x + 1)°Sudut ACB = (4x + 7)°

Ditanya :

Besar sudut BAC ...?

Dijawab :

Mencari nilai x

(2x + 4)° + (8x + 1)° + (4x + 7)° = 180°

14x + 4 + 1 + 7 = 180°

14x + 12 = 180°

14x = 180° - 12

14x = 168°

x = 168°/14

x = 12

Mencari besar sudut BAC

=> (2x + 4)°

=> (2(12) + 4)°

=> (24 + 4)°

=> 28° (B.)

- KesimpulanJadi, jawaban yang tepat adalah 28 derajat
