• Dealer A membayar 92.000.000 untuk 4 sepeda motor jenis 1 dan 2 sepeda motor jenis 2

• Dealer B membayar 108.000.000 untuk 4 sepeda motor jenis 1 dan 3 sepeda motor jenis 2

• Berapa dealer C harus membayar jika akan membeli 2 sepeda motor jenis 1 dan 5 sepeda motor jenis 2

Jawab :

Sepeda motor jenis 1 = x

Sepeda motor jenis 2 = y

( eliminasi )

4x + 2y = 92.000.000

4x + 3y = 108.000.000

___________________-

-y = -16.000.000

y = 16.000.000

( substitusi ke persamaan 1 )

4x + 2y = 92.000.000

4x + 2 (16.000.000) = 92.000.000

4x + 32.000.000 = 92.000.000

4x. = 92.000.000 - 32.000.000

4x. = 60.000.000

x. = 60.000.000 ÷ 4

x. = 15.000.000

Dealer C membeli 2 sepeda motor jenis 1 dan 5 sepeda motor jenis 2 :

2x + 5y

2 ( 15.000.000 ) + 5 ( 16.000.000)

30.000.000 + 80.000.00

110.000.000

jadi, dealer C harus membayar sebesar :

%U = 15%

Untung ;

[tex] \frac{15}{100} \times 18.000.000 = 2.700.000[/tex]

Hj ;
= Rp18.000.000 + Rp2.700.000

U = Hj - Hb
