2x+3y+2z=450.000

x+2y+2z=300.000 -

-------------------------------

x + y = 150.000

50.000 + y = 150.000 , maka y = 100.000

Pers.(3) :

x+2y+2z=300.000

50.000 + 2*(100.000) + 2z = 300.000

250.000 + 2z = 300.000

2z = 50.000

z = 25.000

Diskon Baju merk B = 15%

Harga 3 celana merk A = 150.000

Maka harga 1 celana nya = 150.000 : 3

= 50.000

Harga 2 baju merk B = 200.000

Maka harga 1 baju merk B = 200.000 : 2

= 100.000

Diskon Celana merk A = 10/100 x 50.000

= 10 x 500

= 5.000

maka harga celana merk A setelah terkena diskon menjadi 50.000-5.000

= Rp.45.000,00

Diskon Baju merk B = 15/100 x 100.000

= 15 x 1.000

= 15.000

Maka harga baju merk B setelah terkena diskon = 100.000-15.000

= Rp.85.000,00

Penjelasan dengan langkah-langkah:

A=10/100*(150.000/3)

=10/100*50.000

=5000

=50.000-5000= 45.000

maka Celana A =3*45.000=Rp. 135.000

B=15/100*(200.000/2)

=15/100*100.000

=15.000

=100.000-15.000= 85.000

maka Baju B = 2*85.000=Rp. 170.000

135.000+170.000 = Rp 305.000
