1. Sebuah rumah kecil akan dibangun di lahan terbuka sehingga udara luar terasa panas di siang hari dan dingin di malam hari. Bagian depan rumah akan dibuat dengan dinding kaca tebalSeorang arsitek memiliki hasil eksperimen nilai konduktivitas termal tiga jenis kaca dengan ketebalan bervariasi seperti berikut. Jenis kaca yang tidak akan dapat digunakan untuk menyekat kedua ruang adalah


P =Q/t = k A ΔT /d

P = laju perpindahan kalor

k = konduktivitas termal zat

A = luas penampang

ΔT perubahan suhu

P d = k A ΔT

d/k = A ΔT/P

d/k sebaiknya jangan bernilai besar. krn d/k sebanding dengan ΔT atau perubahan suhu.

kalau d/k besar, perubahan suhu besar, akibatnya suhu ruangan makin panas.

Harus dicari d/k tiap kaca

1. Bening d = 0,5 k = 1,286

d/k = 0,5/ 1,286 = 0,389

2. Bening d = 0,8 k = 1,967

d/k = 0,8/1,967 = 0,407

3. Rayban d = 0,5 k = 1,327

d/k = 0,5/1,327 = 0,377

4. Rayban d = 0,8 k = 2,014

d/k = 0,8/2,014 = 0,397

5. Buram d = 0,3 k = 0,851

d/k = 0,3/0,851 = 0,353

6. Buram d = 0,5 k = 1,296

d/k = 0,5/1,296 = 0,386

Nilai d/k terbesar ada di no 2

Jadi kaca yang tidak dipakai kaca bening dengan ketebalan 0,8 cm
