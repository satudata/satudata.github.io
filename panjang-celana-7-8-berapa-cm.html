y = baju

eliminasi x

4x + 3y = 500.000 (×1) mjd 4x + 3y = 500.000

4x + 3y = 500.000
------------------------------ (-)
11y = 660.000
y = 660.000/11 = 60.000 (harga baju)

2x + 7y = 580.000
2x = 580.000 - 7y
2x = 580.000 - 7 (60.000)
2x = 580.000 - 420.000
2x = 160.000
x = 80.000 (harga celana)

harga 2lusin celana panjang dan 4 lusin baju
= (2×12)x + (4×12)y
= 24x + 48y
= 24 (80.000) + 48 (60.000)

= 4.800.000
