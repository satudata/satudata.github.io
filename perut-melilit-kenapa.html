5. sebuah transformator mempunyai lilitan skunder 1000 lilitan dan primer 5000 lilitan. jika arus pada lilitan primer 0,2A berapa arus pada lilitan skunder


Diketahui :

Ns= 1.000 lilitan

Np= 5.000 lilitan

Ip = 0,2 A

Ditanya :

Is ?

Rumus :

Np/ Ns x Is/ Ip

Is = Np.Ip / Ns

= 5.000 . 0,2 / 1.000

= 1.000 / 1.000

= 1 A

Keterangan :

1.Untuk huruf awal huruf besar { N, I, dan V }

2.Dan untuk sisanya menggunakan huruf kecil { p dan s}.

3. keterangan / sama dengan per

Transformator

------------------------

Transformator step up berfungsi untuk menaikkan tegangan

Transformator step down berfungsi untuk menurunkan tegangan

Is = kuat arus sekunder

Ip = Kuat arus primer

Vp = tegangan primer

Vs =tegangan sekunder

Np = lilitan primer

Ns = lilitan sekunder

Jika sebuah transformator bertipe step up maka

-.Vs lebih besar dari Vp

-. Ns lebih besar dari Np

Jika sebuah transformator bertipe step down maka

-. Vp lebih besar dari Vs

-. Np lebih besar dari Ns

Tambahan rumus :

Perhitungan efesiensi trafo (π)

π = Pout / Pin x 100 %

π = Vs/ Vp. Is/ Ip x 100%

Keterangan rumus :

Pout = daya listrik pada kumparan sekunder

Pin = daya listrik pada kumparan primer

26. Lilitan primer: 220 volt 450 lilitan. Lilitan sekunder: 1100 volt berapakah jumlah lilitan sekunder?


Diketahui :

Vp = 220 v

Np = 450 lilitan

Vs = 1100 v

Ditanya : Ns ?

Dijawab :

Vp / Vs = Np / Ns

220 / 1100 = 450 / Ns

0,2 = 450 / Ns

Ns = 450 / 0,2

Ns = 2.250 Lilitan

np = 450 lilitan

Vp = 220 volt

Vs = 1100 volt

ns = ...?

Vp : Vs = np : ns

220 : 1100 = 450 : ns

0,2 = 450 : ns

ns = 450 : 0,2

ns = 2250 lilitan

Jadi jumlah lilitan sekunder adalah 2250 lilitan

Keterangan :

Vp = tegangan primer

Vs = tegangan sekunder

np = jumlah lilitan primer

ns = jumlah lilitan sekunder

-. Vs lebih besar dari Vp

-. Ns lebih besar dari Np

Jika sebuah transformator bertipe step down maka

-. Vp lebih besar dari Vs

-. Np lebih besar dari Ns

Perhitungan efesiensi trafo (π)

Keterangan rumus :

Pout = daya listrik pada kumparan sekunder
