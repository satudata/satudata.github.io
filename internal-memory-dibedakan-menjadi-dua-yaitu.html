1. Memory internal dan memory external, kemukakan perbedaan antara keduanya dan berikan contoh alatnya


Dewasa ini perangkat komputer untuk penyimpanan data mengalami peningkatan dengan adanya opsi pilihan penyimpan internal dan eksternal.  Tentunya kedua opsi tersebut yaitu memory internal dan memory external mempunyai perbedaan. Berikut ini perbedaannya:

Sederhananya, memori internal adalah media penyimpanan utama telepon seluler, sedangkan memori eksternal memperluas atau melengkapinya ketika diperlukan kapasitas penyimpanan yang lebih besar. Beberapa orang sering menyebutn memori internal adalah ROM, tetapi memori internal tertanam dan tidak langsung dapat diganti  oleh pengguna umum. Ini secara teknis berbeda dari memori eksternal, yang dapat dilepas sesuka hati. Memori internal dibangun langsung ke dalam ponsel, kemampuan  memori internal untuk mentransfer data lebih cepat dan lebih stabil daripada memori eksternal, tetapi dalam hal ini juga dipengaruhi oleh standar memori yang digunakan. Penyimpanan eksternal, di sisi lain, dapat mendukung penyimpanan data pribadi yang lebih luas. Belum lagi beberapa produsen telah menghapus dukungan untuk itu, dan itu tergantung pada kapasitas apa yang kita miliki.

Pembahasan

Ponsel memiliki dua opsi media penyimpanan untuk mengelola berbagai jenis file dan membuatnya mudah diakses. Yaitu di dalam dan di luar.  Kedua jenis media penyimpanan ponsel ini termasuk dalam keluarga memori non-volatile, yang dapat menyimpan  data dalam jumlah besar tanpa menggunakan listrik.

Keduanya dapat digunakan untuk menyimpan file yang cenderung berubah hanya sebagai akibat dari operasi komputasi tertentu, seperti menghapus data, mencopot pemasangan, memformat, atau mengatur ulang ke pengaturan pabrik.

14. Perbedaan secondary memory dengan main memory


Secondary dan main memory adalah dua jenis perangkat memori yang terdapat dalam sebuah sistem komputer dan berfungsi menyimpan data.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan dari kedua memori tersebut. Berikut kakak akan mencoba menjawabnya.

MAIN MEMORI atau memori utama adalah perangkat penyimpanan data pada komputer dan umumnya berbentuk RAM.

Memori utama berfungsi sebagai media penyimpanan data sementara yang kemudian akan diolah oleh unit processor untuk kemudian disampaikan kepada pengunakan melalui media output.

Sementara itu, SECONDARY MEMORY atau memori tambahan adalah perangkat penyimmpanan data jangka panjang dan biasanya berbentuk ROM.

Dalam praktiknya, memori tambahan juga kerap difungsikan untuk menyediakan ruang penyimpanan data tambahan ketika tidak tersedia ruang cukup di memori utama.

28. 33. Jenis memory terbagi menjadi dua yaitu... A. SSD dan HDD B. Harddisk dan Flashdisk C. External dan Internal Memory D. Read Only Memory (ROM dan Read Acces Memory (RAM) E. 32-Bit dan 64-bit


Jenis memori dibagi menjadi dua yaitu C. External dan Internal Memory.

Memori komputer adalah setiap perangkat fisik yang digunakan untuk menyimpan data, informasi, atau instruksi sementara atau permanen.

Pembahasan

Jenis memory terbagi menjadi dua

1. Memori Internal

Memori internal dapat diakses secara langsung oleh prosesor yang memiliki fungsi sebagai pengingat. Memori utama dibagi lagi menjadi dua bagian: RAM (Random Access Memory) dan ROM (Read Only Memory). Selain itu, terdapat pula memori yang disebut CACHE MEMORI, CMOS, DRAM, SDRAM, DIMM.

2. Memori Eksternal

Memori Eksternal merupakan memori tambahan yang berfungsi untuk menyimpan data atau program. Contoh: Hardisk, Flashdisk. Konsep dasar memori eksternal adalah menyimpan data bersifat tetap (non volatile), baik pada saat komputer aktif atau tidak.

29. keuntungan menggunakan memory internal dibandingkan memory eksternal


Jawaban:

Berikut adalah beberapa keuntungan menggunakan memori internal dibandingkan dengan memori eksternal:

Kecepatan Akses Data yang Lebih Cepat: Memori internal umumnya memiliki kecepatan akses data yang lebih cepat daripada memori eksternal seperti kartu memori atau flash drive. Ini dapat mempercepat proses membaca dan menulis data pada perangkat Anda.

Ukuran yang Lebih Kecil: Memori internal biasanya lebih kecil dalam ukuran daripada memori eksternal. Ini membuatnya lebih mudah untuk dimasukkan ke dalam perangkat elektronik seperti ponsel cerdas, laptop, atau tablet. Ukuran yang lebih kecil juga membuat memori internal lebih mudah untuk dipindahkan dari satu perangkat ke perangkat lain.

Dapat Dipercaya: Memori internal lebih dapat diandalkan daripada memori eksternal karena kebanyakan perangkat elektronik dirancang untuk bekerja dengan memori internal. Karena itu, kemungkinan terjadinya masalah dengan memori internal cenderung lebih rendah daripada memori eksternal.

Keamanan yang Lebih Tinggi: Memori internal umumnya lebih aman daripada memori eksternal. Karena memori internal tidak dapat dengan mudah dilepas dari perangkat, ada risiko lebih rendah untuk kehilangan atau dicuri. Selain itu, memori internal juga dapat dienkripsi untuk memberikan tingkat keamanan yang lebih tinggi.

Kompatibilitas yang Lebih Baik: Memori internal umumnya lebih mudah dipasangkan dengan perangkat elektronik daripada memori eksternal. Karena memori internal biasanya dirancang khusus untuk perangkat tertentu, maka kompatibilitas lebih mudah diatur dan diatur oleh produsen perangkat elektronik.

Namun, ada juga beberapa kekurangan dalam menggunakan memori internal. Memori internal cenderung lebih mahal dan kapasitasnya seringkali lebih kecil dibandingkan dengan memori eksternal. Jadi, memori internal mungkin tidak cukup jika Anda memiliki banyak data atau file besar yang perlu disimpan.
