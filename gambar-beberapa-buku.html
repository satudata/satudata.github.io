=> 2y = 58
=> y = 58/2 = 29

6. Harga Sebuah Buku Tulis dan buku gambar RP 8.000 sedangkan harga dua buku tulis dan buku gambar RP 11.000 tentukan A.Model Matematika B.Harga 1 buku tulis dan 1 buku gambar C.Harga 5 Buku tulis dan 4 buku gambar ​


Jawab:

a.) Jadi, model matematika untuk persamaan tersebut adalah

2a + b = 11.000

a + b = 8.000

b.) Jadi, harga sebuah buku tulis dan sebuah buku gambar, adalah Rp8.000,-.

c.) Jadi, harga 5 buku tulis dan 4 buku gambar, adalah Rp35.000,-.

Penjelasan dengan langkah-langkah:

a.)

2a + b = 11.000

a + b = 8.000

b.)

Kedua persamaan pada poin (a), dikurangkan.

a = 3.000

Untuk mencari b, disubstitusikan

2(3.000) + b = 11.000

6.000 + b = 11.000

b = 11.000 - 6.000

b = 5.000

Substitusikan ke dalam model soal

a + b = c

3.000 + 5.000 = 8.000

Harga 1 buku tulis dan 1 buku gambar adalah Rp8.000,-.

c.)

Harga barang pada poin (a), disubstitusikan ke dalam model soal

5a + 4b = c

5(3.000) + 4(5.000) = c

15.000 + 20.000 = c

c = 35.000

Harga 5 buku tulis dan 4 buku gambar adalah Rp35.000,-.

==========

Kesimpulan

Jawaban untuk poin (a),

2a + b = 11.000

a + b = 8.000

poin (b),

Rp8.000,-.

poin (c),

Rp35.000,-.
