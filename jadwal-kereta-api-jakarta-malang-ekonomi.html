9. menurut jadwal pemberangkatan kereta api paralyangan dari jakarta ke bandung berangkat pukul 06.00 dari tiba pukul 08.40 akan tetapi karena terjadi suatu kerusakan maka kereta api tersebut mengalami keterlambatan pemberangkatan selama 47 menit a. pukul berapa kereta api itu berangkat dari jakarta? b. pukul berapa kereta api itu tiba di bandung? +bagaimana menghitungnya


Mapel : Matematika
Kategori : SD
Bab : Waktu


Cara: 
a). Karena pemberangkatannya terlambat 47 menit
06.00 + 47 menit = 06.47

b). Hitunglah lama perjalanan bila kereta tersebut berangkat tepat waktu!
08.40 - 06.00 = 02.40 (2 jam 40 menit)
Karena pemberangkatannya pada pukul 06.47, maka kereta tersebut akan tiba pada 06.47 + 02.40 = 07.87. Karena menit itu hanya sampai 60 menit, maka 07.87 = 07.87 - 00.60 + 01.00 = 08.27

Jawaban :
a). Jadi, kereta tersebut berangkat pada pukul 06.47
b). Jadi, kereta tersebut tiba pada pukul 08.27
