- klik menu FORMAT

- klik drop cao

misal, obat I = x dan obat II = y

8x + 2y ≥ 16
x + y ≥ 5
2x + 7y ≥ 20

f(x, y) = 15000x + 30000y

buat grafik terlebih dahulu untuk menentukan titik penyelesaian (lihat lampiran)

Didapatkan 4 titik ujung, yaitu :

(0, 8)
(1, 4)
(3, 2)
(10, 0)

f(0, 8) = 15000(0) + 30000(8) = 240000
f(1, 4) = 15000(1) + 30000(4) = 135000
f(3, 2) = 15000(3) + 30000(2) = 105000
f(10, 0) = 15000(10) + 30000(0) = 150000

Jadi, banyaknya masing-masing obat yang harus dibeli untuk mendapatkan harga minimum adalah 3 obat I dan 2 obat II
