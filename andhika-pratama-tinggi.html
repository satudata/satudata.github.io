Jarak rumah Andhika ke sekolahnya adalah 1,5 km + 2 hm.

maka,

1,5 km + 2 hm

= 1,5 km + 0,2 km

= 1,7 km

1,7 km = 1700 m

Jadi, jarak rumah Andhika ke sekolahnya adalah 1700 m.

dapat dinyatakan dengan

Jarak rumah Andhika ke Perpusda - jarak rumah Andhika ke sekolah

= 3860 m - 1700 m

= 2160 m

atau 2,16 km.

Jadi, jarak sekolah ke Perpusda adalah 2,16 km.

26. Perbandingan uang Andhika dan Raisa adalah 3:7. Selisih uang keduanyaadalah Rp50.000,00. Jumlah uang Andhika dan Raisa adalah....​


PENYELESAIAN SOAL.

Rumus :

[tex] \frac{jumlah \: perbandingan}{selisih \: perbandingan} \times selisih [/tex]

Maka,

[tex] \frac{10}{4} \times 50.000 \\ \\ 10 \times 12.500 \\ \\ = 125.000[/tex]

Kesimpulan.Jadi, uang Andika dan Raisa adalah Rp 125.000

DETAIL JAWABAN.Kelas : 6Bab : 9Materi : PerbandinganMapel : MatematikaKode Kategorisasi : 6.2.9

Jawaban:

Perbandingan uang Andhika dan Raisa adalah 3:7. Selisih uang keduanya

adalah Rp50.000,00. Jumlah uang Andhika dan Raisa adalah....

Dik:

Andhika : Raisa = 3 : 7

Selisih uang = Rp50.000,00

Dit:

Jumlah uang keduanya?

Dijawab:

Uang Andhika

[tex] = > \frac{3}{7 - 3} \times 50.000[/tex]

[tex] = > \frac{3}{4} \times 50.000[/tex]

[tex] = > \frac{3}{4} \times \frac{50.000}{1} [/tex]

[tex] = > \frac{150.000}{4} [/tex]

[tex] = > 37.500[/tex]

Uang Raisa

[tex] = > \frac{7}{7 - 3 } \times 50.000[/tex]

[tex] = > \frac{7}{4} \times 50.000[/tex]

[tex] = > \frac{7}{4} \times \frac{50.000}{1} [/tex]

[tex] = > \frac{350.000}{4} [/tex]

[tex] = > 87.500[/tex]

Jumlah Uang keduanya

=> Rp37.500 + Rp87.500

=> Rp125.000

Adhedhasar konsepsi secara tradisional

laire Aksara Jawa digathukake karo

legenda Aji Saka.

.

Aksara Jawa ana 20 jinise :

Ha = ꦲNa = ꦤCa = ꦕRa = ꦫKa = ꦏDa = ꦢTa = ꦠSa = ꦱWa = ꦮ La = ꦭPa = ꦥDha = ꦝJa = ꦗYa = ꦪNya = ꦚMa = ꦩGa = ꦒBa = ꦧTha = ꦛNga = ꦔ

Ukara Baim Putra Andhika :

Baim : aksara ba, ha + suku, ma

+ pangkon

Putra : aksara pa + suku, ta + cakra

Andhika : aksara ha, na + pasangan

dha + wulu, ka

Nulis aksara Jawa

jawaban terlampi

Pembahasan

Aksara Jawa ing dhuwur ana sing nggunakake aksara murda lan ora nggawe aksara murda.

________________________

Aksara Jawa sing dhuwur ora nggunakake aksara murda.

---------------------------------------------

➡️ Baim Putra Andhika = ꦧꦲꦩ꧀ꦥꦸꦠꦾꦲꦤ꧀ꦝꦶꦏ

Baim = aksara Ba, aksara Ha + wulu, aksara MaPutra = pasangan aksara Pa + suku, aksara Ta + cakraAndhika = aksara Ha, aksara Na, pasangan aksara Dha + wulu, aksara Ka

⚫ Baim Putra Andhika, nggunakake sandhangan swara wulu ( i ), suku ( u ) lan sandhangan panyigeg wanda Cakra ( sesulih ra ).

_____________________________

Aksara jawa sing ngisor nggunakake aksara murda lan aksara swara.

------------------------------------------------------

➡️ Aksara murda yaiku aksara sesirah kang sinebut huruf kapital. Aksara murda digawe kanggo pakurmatan utawa menghormati lan lumrahe kanggo nulis jenenge wong utawa pangkat lan drajat sarta papan Panggonan.

➡️ Aksara murda cacahe ana 8 :

Na, Ka, Ta, SaPa, Nya, Ga, Ba

➡️ Aksara swara yaiku aksara jawa kang digunakake kanggo nulis huruf vokal.

➡️ Aksara swara cacahe ana 5 :

A, I, U, E, O

➡️ Baim Putra Andhika = ꦨꦲꦶꦩ꧀ꦦꦸꦠꦿꦄꦤ꧀ꦝꦶꦑ

Baim = aksara murda Ba, aksara Ha + wulu, aksara MaPutra = pasangan aksara murda + suku, aksara Ta + cakraAndhika = aksara swara, aksara Na, pasangan aksara Dha + wulu, aksara murda Ka.

★ catetan :

aksara sing diwenehi garis ngisor iku nggunakake aksara murda lan aksara swara.
