Penjelasan dengan langkah-langkah:

x² - 2x - 4 = 0

a = 1 ; b = -2 ; c = -4

akar²nya adalah y₁ dan y₂

y₁ + y₂ = -b/a = -(-2)/1

y₁ + y₂ = 2

y₁ . y₂ = c/a = -4/1

y₁ . y₂ = -4

misal akar barunya p dan q

p = y₁/y₂

q = y₂/y₁

p + q = y₁/y₂ + y₂/y₁

p + q = y₁²/y₁.y₂ + y₂²/y₂y₁

p + q = (y₁² + y₂²)/(y₁.y₂)

p + q = [(y₁ + y₂)² - 2 (y₁.y₂)]/(y₁.y₂)

p + q = [(2)² - 2(-4)]/(-4)

p + q = (4 + 8)/4

p + q = 3

p . q = y₁/y₂ . y₂/y₁

p . q = y₁.y₂/y₁.y₂

p . q = 1

persamaan kuadrat baru

x² - (p + q)x + p.q = 0

x² - 3x + 1 = 0
