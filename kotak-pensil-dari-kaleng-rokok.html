5. 1. Toko alat tulis Sahabat Siswa menyediakan pensilwarna dalam kemasan kotak kecil, kemasan kotakbesar, dan kemasan kaleng. Jumlah kemasan pensilwarna di toko tersebut 144 buah. Jikaperbandingan antara banyak kemasan kotak kecil,kemasan kotak besar, dan kemasan kaleng adalah3:3:2, banyak kemasan kaleng adalah ...a. 32c. 48b. 36d. 54​


Toko alat tulis Sahabat Siswa menyediakan pensil warna dalam kemasan kotak kecil, kemasan kotak besar, dan kemasan kaleng. Jumlah kemasan pensil warna di toko tersebut 144 buah. Jika perbandingan antara banyak kemasan kotak kecil, kemasan kotak besar, dan kemasan kaleng adalah 3 : 3 : 2, banyak kemasan kaleng adalah 36.  

Pembahasan  

Diketahui

Jumlah kemasan pensil warna di toko tersebut 144 buah Perbandingan banyak kemasan kotak kecil, kemasan kotak besar, dan kemasan kaleng = 3 : 3 : 2

Ditanyakan

Banyak kemasan kaleng = … ?

Jawab

Kotak kecil : kotak besar : kaleng = 3 : 3 : 2

Misal

Kemasan kotak kecil = 3x Kemasan kotak besar = 3x Kemasan kaleng = 2x

Jumlah kemasan pensil warna di toko tersebut 144 buah

Kotak kecil + kotak besar + kaleng = 144

3x + 3x + 2x = 144

8x = 144

x = [tex]\frac{144}{8}[/tex]

x = 18

Jadi banyak kemasan kaleng adalah  

= 2x

= 2(18)

= 36
