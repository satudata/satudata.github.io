14. Tentukan nilai titik balik maksimum dan minimum fungsi f (x) = x³ - 3x


Titik Balik

f(x) = x³ - 3x

Titik balik → f'(x) = 0

f'(x) = 0
3x² - 3 = 0
x² = 1
x = 1 atau x = -1

f(1) = 1³ - 3(1) = -2
f(-1) = (-1)³ - 3(-1) = 2 → maksimum

Titik balik maksimum = (-1,2) → jawaban
