1. Tentukan persamaan kuadrat yg akar-akarnya kuadrat dari akar-akar persamaan kuadrat 17 dan -5​


~Persamaan Kuadrat

_____________________

Diketahuiakar-akar pertama ( x₁ ) = 17akar-akar kedua ( x₂ ) = - 5

Ditanya

Bentuk persamaan kuadrat

» Pembahasan

Untuk kasus ini merupakan materi menyusun persamaan kuadrat, dimana untuk menyusun persamaan kuadrat jika diketahui akar-akarnya dapat menggunakan rumus:

x² - (x₁ + x₂)x + (x₁ × x₂) = 0

Maka:

x² - (x₁ + x₂)x + (x₁ × x₂) = 0

x² - (17 + ( - 5 ))x + (17 × ( - 5 )) = 0

x² - 12x + ( - 85 ) = 0

x² - 12x - 85 = 0

Kesimpulan

Jadi, Persamaan kuadrat yg akar-akarnya kuadrat dari akar-akar persamaan kuadrat 17 dan - 5 adalah x² - 12x - 85 = 0

Maka, persamaan kuadrat yang akar-akarnya dari akar-akar persamaan kuadrat 17 dan -5 adalah

x²-12x-85=0

PEMBAHASAN:

Diketahui:

x₁ = 17

x₂ = -5

Ditanya:

Tentukan persamaan kuadrat dari akar-akar persamaan kuadrat 17 dan -5

Penyelesaian:

13. Tentukan persamaan kuadrat yang akar akarnya berlawanan dari akar akar persamaan kuadrat x² + 3x + 5 = 0


Dimisalkan : akar-akar PK : x² + 3x + 5 = 0 adalah α dan β

Maka bsrlaku :

[tex]\alpha\:+\:\beta\:=\:-\frac{b}{a}\:=\:-\frac{3}{1}\:=\:-3[/tex]

[tex]\alpha.\beta\:=\:\frac{c}{a}\:=\:\frac{5}{1}\:=\:5[/tex]

Diketahui PK baru mempunyai akar-akar yang berlawanan dengan akar-akar PK tersebut.

Maka, akar-akar PK baru adalah : –α dan –β

Sehingga, untuk PK baru berlaku :

[tex]-\alpha\:+\:(-\beta)\:=\:-(\alpha\:+\:\beta)\:=\:-(-3)\:=\:3[/tex]

[tex]-\alpha\:+\:(-\beta)\:=\:-\frac{b}{a}[/tex]

[tex]-\frac{b}{a}\:=\:3\:=\:-\frac{-3}{1}[/tex]

Didapatkan untuk PK baru :

=> b = –3 dan a = 1

Untuk PK baru juga berlaku :

[tex](-\alpha).(-\beta)\:=\:\alpha.\beta\:=\:5[/tex]

[tex](-\alpha).(-\beta)\:=\:\frac{c}{a}[/tex]

[tex]\frac{c}{a}\:=\:5\:=\:\frac{5}{1}[/tex]

Didapatkan untuk PK baru :

=> x = 5 dan a = 1

Sehingga, untuk PK baru didapatkan :

a = 1 ; b = –3 ; dan c = 5

Jadi, PK baru :

[tex]\boxed{\boxed{x^2\:-\:3x\:+\:5\:=\:0}}[/tex]
