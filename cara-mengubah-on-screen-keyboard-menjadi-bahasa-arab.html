10. beberapa tombol keyboard seperti enter,shift,capslock,numlock,spacebar,tab dan print screen merupakan bagian dari


beberapa tombol keyboard yang sebutkan merupakan bagian dari tombol-tombol utama pada keyboard komputer. Berikut ini adalah penjelasan tentang masing-masing tombol tersebut:

Enter: Tombol Enter digunakan untuk memberi isyarat ke sistem bahwa Anda telah selesai mengetik sebuah perintah atau teks, sehingga sistem akan memproses perintah atau teks tersebut.Shift: Tombol Shift digunakan untuk mengaktifkan huruf kapital (uppercase) atau karakter khusus pada tombol-tombol lain yang terdapat di sebelah kiri tombol Shift.Caps Lock: Tombol Caps Lock digunakan untuk mengaktifkan huruf kapital (uppercase) secara permanen. Sehingga semua huruf yang diketik setelah tombol Caps Lock diaktifkan akan ditulis dengan huruf kapital.Num Lock: Tombol Num Lock digunakan untuk mengaktifkan atau menonaktifkan penggunaan tombol-tombol angka di bagian kanan keyboard. Jika tombol Num Lock diaktifkan, tombol-tombol angka tersebut akan berfungsi sebagai tombol angka, sedangkan jika tombol Num Lock dinonaktifkan, tombol-tombol tersebut akan berfungsi sebagai tombol kursor.Spacebar: Tombol Spacebar digunakan untuk memberikan jarak antar kata atau untuk memindahkan kursor ke kanan sejauh satu spasi.Tab: Tombol Tab digunakan untuk memindahkan kursor ke kanan sejauh satu tab atau untuk mengatur tingkat paragraf dalam dokumen teks.Print Screen: Tombol Print Screen digunakan untuk mengambil tangkapan layar (screenshot) dari layar komputer Anda. Hasil tangkapan layar tersebut dapat disimpan sebagai file gambar atau dicetak langsung.

Cara mengganti layout keyboard menjadi huruf Arab: (Windows 10)

1. Navigasikan ke Start Menu dengan cara menekan tombol Windows

2. Pilih menu Settings

3. Pilih menu Time & Language

4. Navigasikan ke tab Region & language atau Language untuk versi 1809 Windows 10

5. Pilih Add a language

6. Pilih opsi Arabic

7. Pilih next dan check Install language pack and set as my Windows display language

8. Install language pack

9. Tekan bahasa yang sedang diinstall

10. Pilih Add a keyboard

11. Pilih bahasa Arab

12. Language telah terpasang
