24. Himpunan penyelesaian dari 78 + 8n ≥ 2n adalah....a. HP= {-12, -11, -10, ...}b. HP= {..., -16, -15, -14}c. HP= {..., -15, -14, -13}d. HP= {-13, -12, -11, ...}​


[tex]\huge\tt{ \pink P\purple e \green n \red d \blue a \orange h \pink u \purple l \green u \red a \blue n }[/tex]

Pertidaksamaan Linear Satu Variabel adalah Pertidaksamaan yang memuat satu variabel, berpangkat tertinggi satu,dan memuat tanda ketidaksamaan (<,>,≤,≥ atau ≠ )

Contoh soal PtLSV

Himpunan penyelesaian dari Pertidaksamaan

2x + 11 ≤ 23

=> 2x + 11 - 11 ≤ 23 - 11

=> 2x ≤ 12

=> x ≤ 12 ÷ 2

=> x ≤ 6

Maka HP = {0,1,2,3,4,5,6}

[tex]\huge\tt{ \pink P \purple e \green m \red b \blue a \orange h \pink a \purple s \green a \red n \blue}[/tex]

78 + 8n ≥ 2n

=> 78 + 8n - 2n ≥ 0

=> 78 + 6n ≥ 0

=> 6n ≥ -78

=> n ≥ -78 ÷ 6

=> n ≥ -13

Maka, HP = {-13,-12,-11,...} atau opsi D.
