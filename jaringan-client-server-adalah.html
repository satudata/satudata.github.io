4. jaringan peer to peer adalah jaringan di mana setiap komputer dapat berperan sebagai server sekaligus client. Kapan sebuah komputer akan menjadi server dan kapan menjadi client?


Jaringan peer to peer adalah jaringan di mana setiap komputer dapat berperan sebagai server sekaligus client. Kapan sebuah komputer akan menjadi server dan kapan menjadi client? Jaringan peer to peer merupakan jaringan yang lebih dikenal dengan jaringan torrent. Karena pada file-file yang berbagi menggunakan sistem torrent biasanya menggunakan jaringan peer to peer untuk proses pembagian filenya.

Pada jaringan peer to peer semua komputer akan terhubung dan semua komputer dapat menjadi server dan menjadi client. Ketika salah satu komputer meminta suatu file, maka komputer lain akan bertindak sebagai servernya, dan komputer yang meminta file tersebut akan menjadi klien.

Demikian pembahasan mengenai jaringan peer to peer dimana jaringan peer to peer merupakan jaringan yang terdiri dari beberapa komputer, namun seiring perkembangan jaman, jaringan peer to peer dikenal sebagai jaringan torrent. Pada jaringan peer to peer, semua komputer bisa menjadi server dan bisa menjadi client.
