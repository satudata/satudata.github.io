4. apakah hardware dan software itu? bagaimana cara kerja pada umumnya di hardware? dan apa contoh hardware?


Hardware ( Perangkat Keras ) adalah komponen pada komputer yang dapat dilihat dan dapat disentuh fisik.
Software ( Perangkat Lunak ) adalah data-data yang terdapat pada sebuah komputer yang di format kemudian disimpan secara digital.Perangkat ini hanya dapat dilihat, tetapi tidak dapat disentuh oleh fisik.

Cara kerja hardware : 1. digunakan untuk memasukan ( input ) intruksi dari pngguna komputer (user), ex: keyboard, mouse, dsb. 2. untuk memproses atau memasukkan/input dri pengguna, ex:prossesor pada sebuah komputer. 3. untuk mnghasilkan suatu proses ( output ) dari pengguna komputer (user), ex: monitor, printer, speaker.
