5. Tentukan titik berat


Pada soal di atas, kita diminta untuk menentukan letak titik berat dari gambar di samping. Letak titik berat benda di atas adalah 5[tex]\frac{2}{23}[/tex] cm.

Penjelasan dengan Langkah-Langkah

Kita pisahkan lebih dahulu bidang datar pada gambar menjadi dua bangun, yaitu persegi panjang ABCE dan segitiga CDE.

Langkah 1:

Bidang ABCE

[tex]A_{1}[/tex] = 12 × 16 = 192

[tex]Y_{1}[/tex] = [tex]\frac{1}{2}[/tex] × 16 = 8

Bidang CDE

[tex]A_{2}[/tex] = [tex]\frac{1}{2}[/tex] × 16 × (16-7) = 54

[tex]Y_{2}[/tex] = ([tex]\frac{2}{3}[/tex] × (16 - 67) + 7 = 13

Langkah 2:

Titik berat = [tex]\frac{A_{1}Y_{1}-A_{2}Y_{2} }{A_{1}-A_{2} }[/tex]

= [tex]\frac{1928-54.13}{192-54}[/tex]

= [tex]\frac{1536-702}{138}[/tex]

= [tex]5\frac{12}{138} = 5\frac{2}{23}[/tex]cm

Jadi, letak tiitk berat benda di atas adalah [tex]5\frac{2}{23}[/tex] cm.

Hallo!
1. cari x1, y1, dan A1 dari bangun pertama (dataran dari T)
x1 = 2,0cm
y1 = 4+1 = 5cm
A = p.l = 4.2 = 8cm^2

2. cari x2,y2, dan A2 dari bangun kedua (lurusan T)
x2= 1+1=2cm
y2= 2cm
A2= p.l = 2.4 = 8cm^2

jadi x0 = x1A1 + x2A2 / A1+A2 = 2.8 + 2.8 / 8+8 = 16+16 / 16 = 2 cm
jadi y0 = y1A1 + y2A2 / A1+A2 = 5.8 + 2.8 / 8+8 = 40 + 16 / 16 = 3,5 cm
Maka Titik Beratnya (x0,y0) = (2 ; 3,5)

23. Tentukan titik berat benda!


Ditanya : titik berat benda

Jawab :

Benda 1( silinder pejal)

Y1= ½ t = 10 cm

X1 =  0 cm

V1 = π r²t = π . 10² (20) = 2000π

Benda 2 (kerucut)

Y2 = ¼ t + 20 = ¼ (15) + 20 = 23,75

X2= 0 cm

V2= 1/3 π r² t = 1/3 pi 102 .15 = 500 pi

Y0  = (V1.Y1+V2.Y2)/(V1+V2)

     = 2000 π (10) + 500π (23,75) / (2000π+ 500 π)

     =12,75

Jadi titik berat benda berada pada (0;12,75) cm

26. Tentukan koordinat titik berat


Jawaban:

Langkah pertama yang sanggup kita lakukan dalam memilih titik berat yaitu mengamati bentuk bendanya dan menguraikan benda tersebut menjadi beberapa bentuk (jika bisa). Anggaplah benda terdiri dari beberapa bentuk yang sudah umum dan sanggup dengan simpel ditentukan letak titik beratnya contohnya persegi, segitiga, setengah bulat dan sebagainya.

Selanjutnya, kita tentukan titik berat untuk masing-masing bentuk yang sudah kita uraikan sebelumnya. Sesudah kita peroleh koordinat untuk masing-masing bentuk, maka selanjutnya kita hitung resultan dari keseluruhan bentuk untuk mendapat koordinat titik berat bendanya.

Karena suatu benda sanggup dibagi menjadi beberapa partikel contohnya partikel 1, 2, 3, dan seterusnya sampai sebanyak n, maka setiap partikel akan mempunyai berat dan letak titik berat masing-masing. Titik berat masing-masing partikel ditetapkan dengan (xn,yn), nilai n sesuai dengan penomoran.

Titik berat benda totalnya ialah resultan dari keseluruhan partikel dan ditetapkan dengan (xo,yo). Nilai xo dan yosanggup kita hitung memakai rumus diberikut :

xo =W1.x1 + W2.x2 + W3.x3 + ... + Wn.xn / W1 + W2 + W3 + ... +Wn

yo =W1.y1 + W2.y2 + W3.y3 + ... + Wn.yn / W1 + W2 + W3 + ... + Wn

Keterangan :

W1 = berat partikel pertama

W2 = berat partikel kedua

W3 = berta partikel ketiga

n = banyak partikel
