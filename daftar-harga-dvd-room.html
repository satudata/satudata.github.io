3. Harga 2 DVD dan 6 radio adalah 1.400.000, sama dengan harga 3 DVD dan 2 radio . hitunglah harga 4 DVD dan 3 radio.???!!!


Sistem Persamaan Linear Dua Variabel
SMP Kelas VIII

Misal
harga DVD = x
harga radio = y

2x + 6y = 1.400.000
x + 3y = 700.000
x = 700.000 - 3y ...(1)

3x + 2y = 1.400.000 ...(2)

Substitusi
3x + 2y = 1.400.000
3(700.000 - 3y) + 2y = 1.400.000
2.100.000 - 9y + 2y = 1.400.000
-9y + 2y = 1.400.000 - 2.100.000
-7y = -700.000
y = 100.000

x = 700.000 - 3y
x = 700.000 - 3(100.000)
x = 700.000 - 300.000
x = 400.000

maka
4x + 3y = 4(400.000) + 3(100.000)
             = 1.600.000 + 300.000
             = 1.900.000

7. pak bandi menjual DVD player dengan harga rp 2.725.000.00 dengan keuntungannya 9%.Tentukan harga beli DVD player tersebut​


Harga pembelian DVD player tersebut adalah Rp 2.500.000. Hal yang dapat dilakukan untuk mendapat harga pembelian DVD player tersebut adalah

Menghitung jumlah keuntungan dari penjualan DVD player.Menghitung harga pembelian DVD player.Penjelasan dengan langkah-langkah:

Diketahui:

Harga jual: Rp 2.725.000Untung = 9%

Ditanya:

Berapakah harga beli DVD player?

Jawab

Langkah 1 : Mencari jumlah keuntungan dari penjualan DVD player.

Untung = % untung × harga beli ( diumpakan n)

            = 9% × n

            = 0,09 × n

            = 0,09 n

Langkah 2: Menghitung harga beli DVD player.

Harga beli + untung = harga jual

n + 0,09n = Rp 2.725.000

1,09 n = Rp 2.725.000

n = [tex]\frac{Rp 2.725.000}{1,09}[/tex]

 = Rp 2.500.00

8. Seorang pedagang elektronik menjual sebuah DVD player seharga Rp840.000,00 dan memperoleh keuntungan 5%.Harga beli DVD player tersebut adalah......


Jawaban:

800.000

Penyelesaian :

Harga beli = 100/(100 + %U) × Harga jual

Harga beli = 100/(100 + 5) × 840.000

Harga beli = 100/105 × 840.000

Harga beli = 800.000

==============================

Detail Jawaban

Mapel : Matematika

Kelas : 7

Materi : Aritmatika Sosial

Kode Soal : 2

Kode Kategorisasi : 7.2.7

Penyelesaian :

Harga beli DVD player :

[tex] \tt{ = \frac{100\%}{100\% + \%untung} } \times harga \: jual \\ \tt{ = \frac{100 \%}{100\% + 5\%} } \times 840.000 \\ \tt{ = \frac{100\%}{105\%} } \times 840.000 \\ \tt{ = \frac{100}{105} \times 840.000} \\ \tt{ = 100 \times 8.000} \\ \tt{ = Rp800.000}[/tex]

Jadi harga beli DVD player tersebut adalah Rp800.000

17. Bahasa natural sebuah persewaan DVD mempunyai aturan dalam menyewa DVD yaitu jumlah yang harus dibayar oleh penyewa adalah harga perkeping dikalikan dengan jumlah DVD yg di sewa. Harga 1 keping DVD 3000 rupiah

1. a) Bahasa Natural

       

        Algoritma Menyewa DVD

                     {algoritma yang digunakan untuk menentukan harga menyewa DVD}

     

        Deklarasi

                      Variabel harga merupakan integer

                      Variabel jumlah merupakan integer

   

        Algoritma

                      1. Masukan harga DVD " 3.000 "

                      2. Kalikan dengan jumlah DVD yang disewa

                      3. Tulis hasil

                      4. Selesai

     

    b) Pseudecode

         Algoritma menyewa DVD

         {algoritma yang digunakan untuk menentukan harga menyewa DVD}

     

         Deklarasi

                       Harga  :  integer

                       Jumlah :  integer

         Algoritma

                       read (harga)

                       read (jumlah)

                       harga sewa ← (harga * jumlah)

                       write (harga sewa)

                       end

     c) Flowchart

                         ↓

             (harga*jumlah DVD)

                         ↓

                      cetak

                  harga sewa

                         ↓

                       end

2. a) Bahasa Natural

       Algoritma Gaji Karyawan

                     Variabel hari masuk merupakan integer
