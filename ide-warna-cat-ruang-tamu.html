Jawaban

VARIABEL:

b = 1 kaleng cat biru

p = 1 kaleng cat putih

t = ruang tamu

r = ruang tidur

persediaan cat biru Pb: 200 x b = 200b

persediaan cat putih Pp: 360 x p = 360p

biaya cat ruang tamu: Bt = 500.000

biaya cat ruang tidur: Br = 400.000

kebutuhan ruang tamu: Kt = 2b + 6p

kebutuhan ruang tidur: Kr = 4b + 2p

Jika diasumsikan bahwa ingin diketahui berapa harga cat biru dan cat putih, maka:

- harga 1 kaleng cat biru adalah Rp.70.000

- harga 1 kaleng cat putih adalah Rp.60.000

Pembahasan:

dapat dicari dengan langkah sbb:

Kebutuhan ruang tamu = Biaya Cat ruang tamu

2b + 6p = 500.000

2b = 500.000 - 6p

b = (500.000 - 6p)/2

b = 250.000 - 3p ... persamaan 1

Kebutuhan ruang tidur = Biaya Cat ruang tidur

4b + 2p = 400.000

2p = 400.000 - 4b

p = (400.000 - 4b)/2

p = 200.000 - 2b ... persamaan 2

gabungkan persamaan 1 dan 2

b = 250.000 - 3p

b = 250.000 - 3(200.000 - 2b)

  = 250.000 - 600.000 + 6b

  = -350.000 - 6b

b - 6b = -350.000

   -5b = -350.000

     b = -350.000/-5

     b = 70.000

Jadi harga 1 kaleng cat biru adalah Rp.70.000

p = 200.000 - 2b

p = 200.000 - (2 x 70.000)

p = 200.000 - 140.000

p = 60.000

Jadi harga 1 kaleng cat putih adalah Rp.60.000

29. Seorang pemborong mempunyai persediaan cat warna coklat 100 kaleng dan warna abu-abu 240 kaleng. pemborong tersebut mendapat tawaran untuk mencat ruang tamu dan ruang tidur disuatu gedung. setelah dikalkulasi ternyata satu ruang tamu menghabiskan 1 kaleng cat warna cokelat dan 3kaleng warna abu-abu . sedangkan 1 ruang tidur menghabiskan 2 kaleng cat warna coklat dan dua kaleng cat warna abu abu jika biaya yang ditawarkan pada pemborong adalah untuk setiap ruang tamu rp.30.000 dan tiap ruang tidur rp.25.000 .berapakah pendapatan maksimum yang dapat diterima pemborong tersebut?


Misalkan:

Banyaknya ruang tamu yang dicat = x
Banyaknya ruang tidur yang dicat = y 
x ≥ 0, y ≥ 0

Cat cokelat yang diperlukan:
x + 2y ≤ 100      (1)
Cat abu-abu yang diperlukan:
3x + 2y ≤ 240      (2)

Gambarkan kedua pertidaksamaan pada bidang Cartesius.
Akan diperoleh titik-titik solusi dari daerah solusi, sebagai berikut:
A(0,50) dari perpotongan x + 2y = 100 dan sumbu y
B(70,15) dari perpotongan x + 2y = 100 dan 3x + 2y = 240
C(80, 0) dari perpotongan 3x + 2y = 240 dan sumbu x

Fungsi objektif:  f(x,y) = 30.000x + 25.000y
A(0,50)  → f(0,50) = 30.000(0) + 25.000(50) = 1.250.000
B(70,15)  → f(70,15) = 30.000(70) + 25.000(15) = 3.750.000
C(80, 0)  → f(80,0) = 30.000(80) + 25.000(0) = 2.400.000

Pemborong akan menerima pendapatan maksimum sebesar 3.750.000, dengan mengecat 70 ruang tamu dan 15 ruang tidur.
