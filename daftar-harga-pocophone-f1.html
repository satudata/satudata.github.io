1. bila sudut antara f1 dan f2 adalah 120 derajat dan R=20N maka harga f1 dan f2 adalah


R = 20 N = √F1² + F2² + 2F1F2 Cos 120°
F1² + F2² + 2F1F2 Cos 120°= 20² = 400

Cos 120° = - 0,5
F1² + F2² + 2F1F2 (-0,5) = 400
F1² + F2² = 400 + F1F2
F1² + F2² - F1F2 = 400 
F1² + F2² - (√F1F2)² = 20² 
F1 + F2 - √F1F2 = 20
F1 dan F2 sama yaitu : 20N

Pembuktian
F1 + F2 - √F1F2 = 20
20 + 20 - √20(20) = 20
40 - √400 = 20
40 - 20 = 20
20 = 20

21. Dua vektor gaya F1 dan F2 bertitik tangkap sama saling mengapit sudut a. Jika | F1 + F2 | = √3| F1 - F2 ¦ dan F1 = F2 = F, maka nilai a adalah


VEKTOR

Dua vektor gaya F₁ dan F₂ bertitik tangkap sama saling mengapit sudut α.

BBESAR jumlah kedua vektor
|F₁ + F₂|² = F₁² + F₂² + 2 F₁ F₂ cos α

BESAR selisih kedua vektor
|F₁ - F₂|² = F₁² + F₂² - 2 F₁ F₂ cos α

untuk
• F₁ = F₂ = F
• |F₁ + F₂| = √3 |F₁ - F₂|

diperoleh
|F₁ + F₂| = √3 |F₁ - F₂| ... kuadratkan
|F₁ + F₂|² = 3 |F₁ - F₂|² ... substitusi
(F² + F² + 2F² cos α) = 3 (F² + F² - 2F² cos α)
2F² + 2F² cos α = 3 (2F² - 2F² cos α)
2F² + 2F² cos α = 6F² - 6F² cos α)
8F² cos α = 4F² ; F ≠ 0
2 cos α = 1
cos α = ½
α = 60° ✔️
