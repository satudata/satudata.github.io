7. Jelaskan perbedaan suction filter dan pressure line filter.


Suction filter dan pressure line filter adalah jenis filter yang digunakan dalam sistem hidrolik untuk menghilangkan kontaminan dari cairan.

Suction filter terpasang pada garis hisap (suction line) pump dan digunakan untuk menyaring cairan sebelum ditarik ke dalam pump. Filter ini biasanya terletak dekat dengan tangki atau reservoir dan digunakan untuk melindungi pump dari kontaminan yang mungkin ada di cairan.

Sedangkan pressure line filter terpasang pada sisi tekanan (pressure side) pump dan digunakan untuk menyaring cairan setelah ditekan oleh pump. Filter ini biasanya terletak dekat dengan beban atau aktor dan digunakan untuk melindungi komponen sistem setelah filter dari kontaminan yang mungkin ada di cairan.

Secara umum, suction filter digunakan untuk melindungi pump dari kontaminan, sedangkan pressure line filter digunakan untuk melindungi komponen sistem setelah filter. Kedua jenis filter ini penting untuk mempertahankan keandalan dan kinerja sistem hidrolik.

Berapa harga satu bungkus pupuk dasar, satu bungkus batu hias, dan satu saringan filter masing-masing!


Jawaban:

pupuk dasar = 100.000

batu hias = 50.000

saringan = 80.000

Penjelasan dengan langkah-langkah:

pupuk dasar = X

batu hias = Y

saringan = Z

persamaan 1 : 2x + 2y + z = 380.000

persamaan 2 : x + z = 180.000

persamaan 3 : x + 3y + z = 330.000

metode eliminasi per 1&3

2x + 2y + z = 380.000 |×3| 6x + 6y + 3z = 1.140.000

x + 3y + z = 330.000 |×2| 2x + 6y + 2z = 660.000

____________________-

4x + z = 480.000 (per 4)

metode eliminasi per 2&4

4x + z = 480.000

x + z = 180.000

_______________-

3x = 300.000

x = 300.000 : 3

x= 100.000 (pupuk dasar)

metode substitusi per 2

x + z = 180.000

(100.000) + z = 180.000

z = 180.000 - 100.000

z = 80.000 ( saringan)

metode substitusi per 1

2x + 2y + z = 380.000

2(100.000) + 2y + (80.000) = 380.000

200.000 + 2y + 80.000 = 380.000

2y = 380.000 - 200.000 - 80.000

2y = 100.000

y = 100.000 : 2

y = 50.000 (batu hias)

20. jelaskan perbedaan suction filter dan pressure line filter


Perbedaan antara Suction filter dan Pressure Line filter adalah terletak pada kegunaan dan tempat pemasangannya. Dimana keduanya sama merupakan filter, filter adalah alat yang  berfungsi untuk menyaring kotoran - kotoran atau kontaminan yang berasal dari komponen sistem hidrolik seperti bagian-bagian kecil yang mengelupas, kontaminasi akibat oksidasi dan sebagainya yang dapat mengganggu kinerja dari sebuah sistem liquid atau lainnya.


Pembahasan

Nah suction filter dan pressure line filter merupakan dua dari tiga jenis macam - macam filter yang dibedakan berdasarkan penempatannya. Perbedaan diantara keduanya adalah

Suction filter adalah filter yang dipasang pada saluran hisap dan kemungkinannya dipasang di dalam tangki. Kebanyakan sistem hidrolik selalu memasang suction filter.

Sedangkan, Pressure line filter, adalah filter yang dipasang pada saluran tekan dan berfungsi untuk mengamankan komponen-komponen yang dianggap penting dari sebuah sistem mesin.


Filter selain kedua tersebut adalah Return line filter, yaitu filter yang dipasang pada saluran balik (return) untuk menyaring agar kotoran tidak sampai masuk ke dalam tangki.

26. Perbedaan low pass filter dan high pass filter


Jawaban & Penjelasan:

Perbedaan utama antara high pass filter dan low pas filter adalah bahwa rangkaian filter lolos tinggi melewatkan sinyal dengan frekuensi lebih tinggi dari frekuensi cut off sedangkan filter lolos rendah melewatkan sinyal dengan frekuensi lebih rendah daripada frekuensi cut off.

high pass filter dan low pas filter juga bervariasi dalam perancangan sirkuit; filter lolos tinggi terdiri dari kapasitor diikuti oleh resistansi secara paralel. Sedangkan rangkaian low pass filter terdiri dari resistor diikuti kapasitor.

low pas filter digunakan sebagai filter anti-aliasing sedangkan high pass filter digunakan dalam penguat audio untuk menyambungkan atau menghilangkan distorsi akibat sinyal frekuensi rendah seperti derau.
