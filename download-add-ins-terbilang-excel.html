18. Tipe dalam penyimpanan yang digunakan untuk menyimpan file yang terdapat macro didalamnya dan macro yang sudah dibuat akan tersimpan di dalam file, yaitu tipe…. Pilih salah satu: a. Excel Binary Workbook b. Excel Macro-Enable Workbook c. Excel Workbook d. Excel Add-In​


Jawaban:

b. Excel Macro-Enable Workbook

Penjelasan:

Berikut ini sedikit penjelasan dalam penggunaannya.

1. Excel Workbook, Menyimpan dengan tipe ini seperti menyimpan file excel atau workbook pada umumnya. Pada file ini tidak terdapat Macro.

2. Excel Macro-Enable Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya. Macro yang sudah dibuat akan tersimpan di dalam file.

3. Excel Binary Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya akan tetapi macro akan tersimpan pada system Microsoft Office Excel.

4. Excel Add-In, Menyimpan dengan tipe ini digunakan untuk menyimpan file macro yang dapat anda gunakan sebagai Add-in atau program tambahan dalam Program Excel.

Langkah-langka Cara menyimpan file Macro :

1. Lakukan Penyimpanan dengan tombol SAVE

2. Akan tampil peringatan “the following features cannot be saved in Macro-free workboks” peringatan ini menginformasikan bahwa file ini terdapat VB Project (macro). Jika anda pilih Yes maka Macro yang sudah dibuat akan dihapus, dikarenakan workbook biasa tidak diperkenenkan untuk menyimpan macro.

3. Pilih No Untuk menyimpan macro.

4. Lakukan penyimpanan didalam folder tujuan. Ubahlah Save as Type Menjadi Excel macro-enable Workbook (xlsm).

5. Klik Save

21. Apa makna yang dikandung dalam penggunaaan bahasa Belanda dalam kepanjangan INS (INS kayutanam)


Jawaban:

makna yang dikandung dalam penggunaaan bahasa Belanda dalam kepanjangan INS (INS kayutanam) adalah:

Pembahasan

Jika diterjemahkan ke dalam bahasa Belanda, INS atau Indonesisch Nederlansche School adalah Sekolah Indonesia yang didirikan pada masa pendudukan Belanda. Lebih lengkapnya, Sekolah INS Kayutanam adalah sekolah yang didirikan sudah ada sejak zaman Belanda, namun sistem pendidikan yang tidak diambil dari sistem pendidikan Belanda, melainkan eksklusif atau berdiri sendiri, yang terletak di Kayu Tanam, Padang Pariaman, Sumatera Barat.  

Sekolah pada masa pendudukan Belanda tidak lain ditujukan untuk mencetak para pegawai murah yang ditempatkan di instansi kolonial Belanda. Di samping itu,  sekolah pada masa itu menggunakan bahasa Belanda sebagai salah satu bahasa pengantar dan mengajarkan pelajaran contohnya belajar sejarah yang tidak berdasarkan sejarah dari perspektif orang Indonesia atau Indonesia Sentris.  

INS Kayutanam sendiri didirikan untuk mendirikan sekolah yang melawan sistem pendidikan barat (Belanda) yang mencetak muridnya sebagai pegawai rendahan yang akan ditempatkan di instansi pemerintah Belanda. Reaksi-reaksi INS Kayutanam ini terlihat pada beberapa hal seperti:  

1. Meniadakan bahasa Belanda sebagai mata pelajaran sekolah, namun mempelajari bahasa asing lain seperti bahasa inggris, bahasa Jerman dan bahasa Perancis

2. Mengajarkan sejarah yang Indonesiasentris atau membuang kurikulum pembelajaran sejaah yang nederlando-sentris atau menurut perspektif Belanda,  

3. Mengajakan keterampilan yang kurang dibutuhkan sebagai pegawai kolonial seperti menggambar, kerajinan tangan atau seni kriya dan seni musik.

Jadi, Makna INS dalam INS Kayutanam adalah sekolah kejuruan dengan kurikulum Inovatif yang berdiri sejak penajajahan Belanda, yang bertujuan untuk mendidik para murid agar dapat menyesuaikan pendidikannya dengan kebutuhan rakyat Indonesia dan bukan untuk kebutuhan pekerja kolonial Belanda.

25. 3.Pemberian keterangan axis table dapatmenggunakan fitur ....A. Add InsB. TablesC. Chart ElementD. Chart LayoutE. FormulaFxcel untukMicrocoft​


Materi: MS. Office Excel - Charts

Jawaban:

Pemberian keterangan axis table dapat menggunakan fitur C. Chart Element.

Penjelasan:

Axis table/label/title adalah keterangan informasi mengenai grafik (chart) yang dibuat dan berada di bawah grafik tersebut. Axis table bisa ditambahkan menggunakan fitur Add Chart Element yang merupakan Chart Element (berada di samping kiri grup alat Chart Styles). Add Chart Element berada di tab menu/ribbon Design setelah menyeleksi grafik.

Jadi, opsi yang benar adalah opsi C. Chart Element.
