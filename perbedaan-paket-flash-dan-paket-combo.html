10. Jalur yang digunakan untuk membedakan paket data satu VLAN dengan paket data VLAN yang lain dan didalamnya terdapat paket data yang memuat informasi VLAN ID disebut​


Jawaban:

Tagged VLAN (Trunking)

Penjelasan:

Jika ada lebih dari satu VLAN, maka diperlukan cara untuk membedakan paket data satu VLAN dengan paket data VLAN yang lain. Cara tsb adalah dengan menambahkan sebuah Tag (atau VLAN ID) ke dalam paket data. Oleh karena itu dinamakan “Tagged VLAN” karena didalamnya terdapat paket data yang memuat informasi VLAN ID.

Sebaliknya dari “Tagged VLAN” (jalur Akses), dinamakan “Untagged VLAN” karena paket data didalamnya tidak memuat informasi VLAN ID.

Ditanya :b = ?

2a + 3b + c = Rp. 61.000

2a + b + c  = Rp. 41.000

___________________-

2b = Rp. 20.000

Kesimpulan :

26. Harga Paket Combo Ayam dari suatu brand makanan terkenal adalah Rp 185.000,00 dan semua produknya dikenakan PPN sebesar 10%. Jika seseorang akan membeli Paket Combo tersebut, jumlah uang yang harus dikeluarkan adalah .... A. Rp 200.000,00 B. Rp 203.500,00 C. Rp 205.000,00 D. Rp 208.500,00​


Jawab:

Harga paket = Rp185.000,00

PPN 10%

PPN = 10%/100% x Rp185.000,00

= Rp18.500,00

Uang yang harus dikeluarkan = Rp185.000,00 + PPN = Rp185.000,00 + Rp18.500,00 = Rp203.500,00

Jawab:

B. Rp 203.500,00

Penjelasan dengan langkah-langkah:

Diketahui : Harga Awal = Rp 185.000,00

                  PPN 10%

Ditanya    : Harga Akhir

Dijawab   :

[tex]Harga Akhir = Harga Awal + (Harga Awal \times 10 \%)\\Harga Akhir = 185.000 + (185.000\times 10 \%)\\Harga Akhir = 185.000 + 18.500\\Harga Akhir = 203.500[/tex]

∴ Jadi, jumlah uang yang harus dikeluarkan adalah Rp 203.500,00

29. pak Deni membagikan 5 paket dengan berat berbeda untuk korban bencana alam. paket pertama beratnya lebih ringan 2kg daripada paket kedua. paket kedua beratnya lebih ringan 2kg daripada paket ketiga. begitu seterusnya hingga paket kelima. jika total berat kelima paket 90kg, maka total berat paket yang paling ringan dengan paket yang paling berat adalah ..kg​


Jawab:

Jumlah berat paket yang paling berat dan paling ringan adalah 36 kg

Penjelasan dengan langkah-langkah:

Untuk menyelesaikan soal diatas dapat digunakan pengetahuan deret aretmatika, berat masing masing paket akan memenuhi deret berikut

x, x-2, x-4, x-6, x-8

sehingga totalnya 90 maka

x + x-2 + x-4 + x-6 + x-8=90

5x-20=90

5x=110

x=22

sehingga jumlah paling berat dan paling ringan

x+x-8=2x-8=2*22-8=36 kg
