30. Jika 1 liter susu dijual dengan harga Rp.200,00 maka yang laku 800 liter sehari.Setiap harga diturunkan Rp.10,00 maka permintaan bertambah 100 liter.Berapakah harga 1 liter susu itu sebaiknya supaya penjual mendapat uang sebanyak banyaknya ......


misal : fungsi permintaan dinyatakan dengan : f (x) = ax + b
kasus 1 : 800 = 200a + b
kasus 2 : 900 = 190a + b
               ---------------------- - 
              - 100 = 10.a
                    a = - 100/10
                    a = - 10
 800 = 200.a + b
 800 = 200. - 10 + b
 800 = - 2000 + b
     b = 800 + 2000
     b = 2800
jadi , rumus fungsi permintaan :
    f (x) = - 10x + 2800
uang yang diterma : g(x) = (- 10x + 2800) .x
                                g (x) = - 10x² + 2800x
uang yang diterima mencapai maksimum jika g' (x) = 0
g' (x) = - 20x + 2800
      0 = - 20x + 2800
  20x = 2800
      x = 2800/20
      x = 140
jadi, agar pendapatan maksimum susu harus dijual seharga Rp. 140,- / liter
