6. Cara print 4 gambar dalam 1 kertas di word adalah .....


Masukan gambar foto dengan menu Insert – Picture – This Device.
Buka Drive dan Folder menyimpan gambar tersebut.
Blok untuk memilih banyak foto.
Klik Insert.
Ubah format pengeditan, dengan klik kanan salah satu foto, pilih Wrap Text – Square.
Lakukan tahap di atas pada foto yang lain.
Seleksi semuanya dengan klik kiri, kemudian tahan ctrl, dan klik gambar yang lain.
Pada tab ribbon, pilih Format, ubah nilai Height, misalkan 6.
Geser foto dengan klik salah satu lalu tekan tanda arah panah.
Ulangi langkah ke.9 pada foto yang lain hingga rapi.
Tekan ctrl & P bersamaan.
Jika sudah yakin rapi, klik kiri ikon Print atau tekan Enter.

11. JelaskAn fUngsi print previev pada microsoft word dan excel


Untuk melihat printed version dokumen atau halaman sebelum benar-benar dicetak dalam bentuk kertas.

Print Preview Edit Mode adalah Menu Ribbon lengkap yang menampilkan Print Preview.
Fungsi Print Preview Edit Mode adalah untuk menampilkan print out di layar monitor dengan menu yang lebih lengkap. Sebagian dari Menu Ribbon Print Preview Edit Mode juga dapat disetting dengan Menu Page Layout atau  melalui menu Print ( File --> Print ). Tetapi tidak ada salahnya jika ingin lebih mempercepat proses print out dengan menampilkan Print Preview Edit Mode pada Customize Quick Accsess Toolbar.

Bagaimana caranya?
Cara untuk menampilkan Print Preview Edit Mode pada Customize Quick Accsess Toolbar sama dengan menampilkan icon Print Preview and Print atau menampilkan icon Camera pada Microsoft Excel atau juga sama dengan menampilkan icon Save All.
