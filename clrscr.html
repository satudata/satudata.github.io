15. pengertian uses,crt,dan clrscr dan berikan fungsinya


Uses CRT dan CLRSCR adalah dua jenis instruksi yang dapat dimasukkan pada aplikasi Command Prompt atau BIOS pada sistem operasi Windows.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan dari kedua fungsi tersebut. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

JIKA DIKETIKKAN KE DALAM LAYAR COMMAND PROMPT DAN DITEKAN ENTER, MAKA FUNGSI USES CRT AKAN MENGATUR JUMLAH KARAKTER MAKSIMAL YANG DAPAT DIKETIK DALAM SUATU NASKAH PERINTAH YAITU HANYA 80 KARAKTER YANG TERSEBAR KE DALAM 25 BARIS.

SEMENTARA ITU, JIKA DIKETIKKAN KE DALMA LAYAR APLIKASI YANG SAMA DAN DITEKAN ENTER, MAKA FUNGSI CLRSCR AKAN MEMBERSIHKAN SELURUH REKAMAN MASUKAN DAN TULISAN YANG PERNAH DIKETIKKAN DI MONITOR PADA SETIAP LANGKAH AWAL EKSEKUSI PROGRAM.

19. Jelaskan dan sebutkan definisi dan fungsi dari : 1. Uses Crt 2. Clrscr 3. Begin 4. Gotoxy 5. Read 6. Readln 7. Write 8. Writeln 9. Var 10. Const


1. Uses Crt : menggunakan library/unit Crt, yang di dalamnya terdapat beberapa fungsi yang dapat digunakan untuk penanganan screen dan keyboard. contoh-contoh fungsinya clrscr, gotoxy, textcolor, textmode, dll.

2. Clrscr : membersihkan layar

3. Begin : memulai program

4. Gotoxy : memindahkan kursor ke posisi (x,y)

5. Read : membaca atau menerima masukan sampai ditemukan spasi, sehingga kursor ke samping

6. Readln : membaca atau menerima masukan sampai ditemukan karakter newline, sehingga kursor ke bawah

7. Write : mencetak ke layar tanpa diikuti karakter newline, sehingga kursor berada di samping

8. Writeln : mencetak ke layar dengan diakhiri karakter newline '\n', sehingga kursor berada di bawah

9. Var : mendeklarasikan variabel (nilainya bisa berubah-ubah)

10. Const: mendeklarasikan konstanta (nilainya tidak bisa diubah)
