Rumus dari arah Gaya Lorentz adalah F = q(v × B). Keterangan dari rumus itu antara lain: F Merupakan gaya (dalam satuan/unit newton) B Merupakan medan magnet (dalam unit tesla) q ialah muatan listrik (dalam satuan coulomb) v ialah arah kecepatan muatan (dalam unit meter per detik) Berikutnya ini untuk gaya Lorentz yang ada karena arus listrik, I, dalam sebuah medan magnet (B), rumusnya yang akan terlihat terdiri dari berikut ini: Rumus = F = IL ×B F F yakni gaya yang di ukur dalam unit satuan newton I ialah arus listrik dalam ampere B yakni medan magnet dalam satuan tesla ialah perkalian silang vektor, dan L yakni panjang kawat listrik yang dialiri listrik dalam satuan meter. 270000

F = q(v × B)= 18.000(3x5)=18.000x15= 270000

Sebuah Tembaga memiliki panjang awal 0,0027 Cm pada suhu 90°C. Apabila Tembaga dipanaskan hingga suhu 230°F serta apabila diketahui Kuat Arusnya sebesar 6000 mA,berapakah besar Gaya Lorentznya? (koef muai : 0,000018,Kuat Medan Magnet : 50 T)

JAWABAN : 8, 102916 x [tex]10^{-3}[/tex] Newton

PEMBAHASAN

Rumus mencari panjang akhir ( L )

[tex]\boxed{L = Lo ( 1 + \alpha \times \triangle t )} [/tex]

Ket :

L = Panjang akhir ( satuannya meter )

Lo = Panjang mula - mula ( satuannya meter )

a = Koefisien muai ( satuannya /°C )

Δt = Perubahan suhu ( satuannya °C )

.

Rumus mencari gaya Lorentz ( F )

[tex]\boxed{F = B \times I \times L}[/tex]

Ket :

F = Gaya Lorentz ( satuannya Newton )

B = Kuat medan magnet ( satuannya tesla )

I = Kuat arus listrik ( satuanya ampere )

L = Panjang ( satuannya meter )

Diketahui

Lo = 27 x [tex]10^{-4}[/tex] cm = 27 x [tex]10^{-4}[/tex] : 100 = 27 x [tex]10^{-6}[/tex] Meter

α = 18 x [tex]10^{-6}[/tex]/C°

Δt = 230°F - 90°C = ( ( 230 °F − 32 ) × [tex]\frac{5}{9}[/tex] ) - 90°C = 110°C  - 90°C = 20°C

I = 6000mA = 6000 : 1000 = 6 Ampere

B = 50 Tesla

Ditanya

F?

Dijawab

Mencari panjang akhir

L = Lo ( 1 + α x Δt )

L = 27 x [tex]10^{-6}[/tex] ( 1 + 18 x [tex]10^{-6}[/tex] x 20 )

L = 27 x [tex]10^{-6}[/tex] ( 1 + 36 x [tex]10^{-5}[/tex] )

L = 27 x [tex]10^{-6}[/tex] ( 1 + 0,00036 )

L = 27 x [tex]10^{-6}[/tex] x 1, 00036

L = 0, 00002700972

L = 2, 700972 x [tex]10^{-5}[/tex] Meter

Mencari gaya lorentz

F = B x I x L

F = 50 x 6 x  2,700972 x [tex]10^{-5}[/tex]

F = 300 x 2,700972 x [tex]10^{-5}[/tex]

F = 8,102916 x [tex]10^{-3}[/tex] Newton

Kesimpulan

Jadi besar gaya Lorentznya adalah 8,102916 x [tex]10^{-3}[/tex] Newton
