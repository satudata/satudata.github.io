4. Data nilai energi ikatan rata-rata diketahui sebagai berikut. ikatan energi ikatan (kj mol–1 ) c–h 410 c–f 485 f–f 159 h–f 565 jika entalpi reaksi : c2h5f(g) + 2f2(g) → c2h3f3 (g) + 2hf (g) nilai entalpi reaksi di atas adalah


Jawaban:

Nilai entalpi reaksi (ΔH reaksi) pada reaksi di atas adalah -962 kJ/mol. Rumus yang dapat digunakan adalah ΔH reaksi = ∑ΔH EI kiri - ∑ΔH EI kanan. Adapun langkah yang harus dilakukan terlebih dahulu adalah menentukan struktur kimia dari masing-masing senyawa. Hal ini dilakukan agar dapat diketahui ikatan yang terjadi.

Penjelasan:

Rumus struktur dari reaksi di atas adalah:

   H  F                         H  F

    |   |                           |   |

H-C-C-H  + 2F-F  → H-C-C-F   + 2H-F

    |   |                           |   |

   H  H                        H  F

∑ EI Kiri = [(5× EI C-H) + (1× EI C-C) + (1× EI C-F) + (2× EI F-F)]

∑ EI Kiri = 2.853 kJ/mol

∑ EI Kanan = [(3× EI C-H) + (3× EI C-F) + (2× EI H-F) + (1× EI C-C)]

∑ EI Kanan = 3.815 kJ/mol

 

Dengan demikian, ΔH reaksi adalah:

ΔH reaksi = 2.853 kJ/mol - 3.815 kJ/mol

ΔH reaksi = -962 kJ/mol

Penjelasan:

Diketahui reaksi:

[tex]C_{2}H_{4}[/tex] + HCl → [tex]C_{2}H_{5}Cl[/tex]     ΔH = +44 kJ/mol

H   H                         H    H

Ι     Ι                           Ι     Ι

C = C  + H - Cl → H - C - C - Cl

Ι     Ι                           Ι     Ι

H   H                          H   H

ΔH = ∑Energi Ikatan pemutusan - ∑Energi Ikatan pembentukan

atau :

ΔH = ∑Energi Ikatan reaktan - ∑Energi ikatan produk

∑Energi Ikatan reaktan = (EI C=C + (4 x EI C-H) + EI H-Cl)

                                       = 614 + 4(C-H) + 431

                                       = 1045 + 4(C-H)

∑Energi ikatan produk  = (EI C-C + (5 x EI C-H) + EI C-Cl

                                       = 348 + 5(C-H) + 328

                                       = 676 + 5(C-H)

ΔH = ∑Energi Ikatan reaktan - ∑Energi ikatan produk

44 = (1045 + 4(C-H)) - (676 + 5(C-H))

44 = 1045 + 4(C-H) - 676 + 5(C-H)

44 = 369 - (C-H)

44 - 369 = -(C-H)

-325 = -(C-H)

(C-H) = 325 kJ/mol

Jadi energi ikatan C - H adalah 325 kJ/mol

Menghitung perubahan entalpi dapat dilakukan dengan banyak cara:

1. Dengan kalorimeter

Biasanya kalorimeter sederhana dapat terbuat dari gelas atau wadah yang bersifat isolator (seperti styrofoam). Ada juga kalorimeter lain dengan tingkat ketelitian yang lebih tinggi yaitu kalorimeter bom. Biasanya digunakan untuk menentukan perubahan entalpi dari reaksi - reaksi pembakaran yang melibatkan gas.

Rumus mencari perubahan entalpi dengan kalorimeter:

q total = q sistem + q kalorimeter

q sistem = m . c . ΔT

q kalorimeter = C . ΔT

Keterangan:

q = kalor (J)

m = massa (gr)

c = kalor jenis zat (J/g.K)

C = kapasitas kalorimeter (J/K)

ΔT = perubahan suhu (K)

2. Dengan hukum hess

Hukum Hess menyatakan bahwa "perubahan entalpi suatu reaksi hanya tergantung pada keadaan awal (zat-zat pereaksi) dan keadaan akhir (zat-zat hasil reaksi) dari suatu reaksi dan tidak tergantung bagaimana jalannya reaksi." Biasanya dinyatakan dalam Siklus Hess atau diagram entalpi (tingkat energi)

3. Dengan data entalpi pembentukan (ΔH°f)

Jika diketahui data entalpi pembentukannya, maka kita bisa gunakan persamaan ini untuk menemukan perubahan entalpinya:

ΔH = ∑H°f produk - ∑ΔH°f reaktan

produk = yang berada di ruas kanan

reaktan = yang berada di ruas kiri

4. Dengan data energi ikat

Energi ikat rata-rata adalah energi rata - rata yang diperlukan untuk memutuskan sebuah ikatan dari seluruh ikatan suatu molekul gas menjadi atom-atom gas. Untuk menghitungnya perubahan entalpinya, kita bisa gunakan persamaan berikut:

ΔH = ∑Energi ikat reaktan - ∑Energi ikat produk

12. 1. Gunakan data energi ikatan untuk menghitung AH Reaksi dari reaksi berikut. ​


Jawaban:

Gunakan data energi ikatan untuk menghitung ΔH reaksi dari reaksi berikut :

A. C₃H₈ (g) + Cl₂ (g) → C₃H₇Cl (g) + HCl (g)

B. C₃H₈ (g) + 5O₂ (g) → 3CO₂ (g) + 4H₂O (g)

C. C₄H₈ (g) + HBr (g) → C₄H₉Br (g)

Jawaban Pendahuluan

Termokimia merupakan cabang ilmu kimia yang mempelajari kalor atau energi yang menyertai suatu reaksi kimia, baik yang diserap atau dilepaskan. Keseluruhan energi dalam bentuk kalor yang terdapat dalam suatu sistem disebut entalpi. Entalpi dalam sistem bersifat tetap selama tidak terjadi pertukaran energi antara sistem dengan lingkungan. Entalpi dinyatakan dengan huruf H dengan satuan Joule (J). Entalpi suatu rekasi dapat ditentukan dengan beberapa langkah, salah satunya dengan energi ikatan.

Pembahasan

Energi ikatan rata-rata merupakan energi ikatan atomik dibagi dengan jumlah ikatan yang ada dalam satu molekul. Energi ikatan adalah energi untuk melepaskan satu ikatan antara dua atom dalam satu molekul.

Penentuan entalpi reaksi dapat diketahui dengan energi ikatan rata-rata atom yang terlibat dan reaksi kimia, yang dirumuskan ∆H reaksi = jumlah energi pemutusan ikatan reaktan – jumlah energi pemutusan ikatan produk.

Untuk mengetahui ikatan yang terbentuk maka harus menggambarkan ikatan antar atom pada setiap molekul yang terlibat dalam reaksi.

Beberapa data energi ikatan yang dapat digunakan sebagai berikut :

C=O = 799 kJ

C=C = 614 kJ

C–C = 348 kJ

C–H = 414 kJ

C–Cl = 328 kJ

C–Br = 276 kJ

H–Cl = 200 kJ

H–Br = 243 kJ

O–H = 463 kJ

O=O = 495 kJ

Cl–Cl = 242 kJ

A. C₃H₈ (g) + Cl₂ (g) → C₃H₇Cl (g) + HCl (g)

Untuk reaksi C₃H₈ (g) + Cl₂ (g) → C₃H₇Cl (g) + HCl (g) digambarkan ikatannya sebagai berikut :

CH₃ – CH₂ – CH₃ + Cl–Cl → CH₃ – CH₂ – CH₂ – Cl  + H–Cl (untuk setiap ikatan C – H )

∆H = Ʃ Dreaktan – Ʃ Dproduk

∆H = {8 C–H + 2C–C + Cl–Cl} – {7 C–H + 2C–C + C–Cl + H–Cl}

Pada ruas kiri dan kanan masing-masing terdapat C–H dan C–C sehingga dapat disederhanakan sebagai berikut :

∆H = {C–H + Cl–Cl} – {C–Cl + H–Cl}

∆H = {414 + 242} – {328 + 200} kJ

∆H = {656} – {528} kJ

∆H = +128 kJ

B. C₃H₈ (g) + 5O₂ (g) → 3CO₂ (g) + 4H₂O (g)

Untuk reaksi C₃H₈ (g) + 5O₂ (g) → 3CO₂ (g) + 4H₂O (g) digambarkan ikatannya sebagai berikut :

CH₃ – CH₂ – CH₃ + 5 O=O → 3 O=C=O  + 4 H–O–H (untuk setiap ikatan C – H )

∆H = Ʃ Dreaktan – Ʃ Dproduk

∆H = {8 C–H + 2C–C + 5O=O} – {6C=O + 8 O–H}

∆H = {8 · 414 + 2 · 348 + 5 · 495} – {6 · 799 + 8 · 463}

∆H = {3.312 + 696 + 2.475} – {4.794 + 3.704} kJ

∆H = {6.483} – {8.498} kJ

∆H = - 2.015 kJ

C. C₄H₈ (g) + HBr (g) → C₄H₉Br (g)

Untuk reaksi C₄H₈ (g) + HBr (g) → C₄H₉Br (g) digambarkan ikatannya sebagai berikut :

H₃C – CH₂ – CH = CH₂ + H–Br → H₃C – CH₂ – CH₂ – CH₂ – Br  (untuk setiap ikatan C – H )

∆H = Ʃ Dreaktan – Ʃ Dproduk

∆H = {8C–H + 2C–C + C=C + H–Br} – {9C–H + 3C–C + C–Br}

Pada ruas kiri dan kanan masing-masing terdapat C–H dan C–C sehingga dapat disederhanakan sebagai berikut :

∆H = {C=C + H–Br} – {C–H + C–C + C–Br}

∆H = {614 + 243} – {414 + 348 + 276} kJ

∆H = {857} – {1.038} kJ

∆H = - 181 kJ

Kesimpulan

A. C₃H₈ (g) + Cl₂ (g) → C₃H₇Cl (g) + HCl (g), ∆H = + 128 kJ

B. C₃H₈ (g) + 5O₂ (g) → 3CO₂ (g) + 4H₂O (g), ∆H = - 2.015 kJ

C. C₄H₈ (g) + HBr (g) → C₄H₉Br (g), ∆H = - 181 kJ

Perubahan Entalpi Berdasarkan Data Energi Ikatan

Energi ikatan adalah energi rata-rata yang diperlukan untuk memutuskan satu ikatan dalam 1 mol molekul menjadi atom penusunnya.

ΔH = Ʃpemutusan ikatan (kiri) – Ʃpembentukan ikatan (kanan)

Menentukan perubahan entalpi

Persamaan termokimia yang terjadi adalah

C₂H₄ + HCI ---> C₂H₅CI

16. Tolong bantu jawab ya. Perubahan entalpi suatu reaksi dapat ditentukan dengan menggunakan... (1) kalorimeter (2) hukum Hess (3) data energi ikatan (4) data energi potensial


Jawab :

1. Berdasarkan eksperimen / percobaan

Salah satu cara yang digunakan untuk mengukur perubahan entalpi reaksi adalah dengan kalorimetri, merupakan suatu proses pengukuran jumlah panas dari sistem reaksi dengan menggunakan kalorimeter. Berdasarkan fungsinya, kalorimeter dibedakan menjadi :

Kalorimeter tipe reaksi (sederhana), adalah kalorimeter untuk menentukan kalor reaksi dari semua reaksi, kecuali reaksi pembakaran. Kalorimeter tipe ini memiliki bejana yang terbuat dari Styrofoam, namun ada pula yang terbuat dari aluminium. Kalorimeter tipe reaksi dapat juga digunakan untuk menentukan kalor jenis logam. Kalorimeter tipe Bom, berfungsi untuk menentukan jumlah kalori dalam bahan makanan berdasarkan reaksi pembakaran (biasanya dioksidasi dengan oksigen).

Rumus yang digunakan untuk menghitung jumlah kalor bila kalor dari kalorimeter diabaikan adalah sebagai berikut.

q = m × c × ΔT

Namun, bila kalor dari kalorimeter diperhitungkan, rumusnya menjadi :

q = (m × c × ΔT) + (C × ΔT)

Keterangan :

q = kalor reaksi (J)

m = massa zat (g)

c = kalor jenis zat (J/g°C atau J/gK)

ΔT = perubahan suhu (°C atau K)

C = kapasitas kalor zat (J/°C atau J/K)

Perlu diketahui juga, yang dimaksud dengan kalor jenis (c) adalah jumlah kalor yang dibutuhkan untuk menaikkan suhu 1 gram zat sebesar 1 ◦C sedangkan kapasitas kalor adalah jumlah kalor yang dibutuhkan untuk menaikkan suhu suatu zat sebesar 1°C atau 1.

2. Berdasarkan Hukum Hess

Tidak semua reaksi kimia berlangsung dalam satu tahap, contohnya reaksi pembuatan belerang (baik melalui proses kontak maupun kamar timbal) dan reaksi pembuatan besi dari biji besi. Namun, menurut Hess (1840) berapa pun tahap reaksinya, jika bahan awal dan hasil akhirnya sama, akan memberikan perubahan entalpi yang sama.

3. Berdasarkan Entalpi Pembentukan Standar

Data dari entalpi pembentukan standar dapat juga digunakan untuk menghitung H reaksi (ΔH). Zat-zat pereaksi mengurai membentuk unsur-unsurnya, kemudian unsur-unsur hasil uraian tersebut membentuk zat baru. Rumus yang digunakan adalah :

ΔH = Σ ΔHf hasil reaksi – Σ ΔHf pereaksi

4. Berdasarkan Energi Ikatan  

Energi ikatan adalah energi yang digunakan untuk memutuskan ikatan kimia dari 1 mol senyawa yang berbentuk gas menjadi atom-atom gas pada keadaan standar. Misalnya untuk memutuskan ikatan 1 mol oksigen diperlukan energi sebesar 498,3 kJ/mol. Artinya, energi ikatan O = O dalam molekul O2 = 498,3 kJ.

Energi ikatan juga disebut sebagai energi disosiasi, yang dilambangkan dengan D. Semakin banyak jumlah ikatan antar atom atau jumlah pasangan terikat dari suatu atom, maka nilai energi ikatan semakin besar dan ikatan antar atom juga semakin kuat.

Secara umum, perhitungan entalpinya dirumuskan dengan:

ΔH reaksi = Σ energi ikatan pereaksi yang putus - Σ energi ikatan zat hasil reaksi yang terbentuk.

Dari rumus ini dapat ditentukan:

a. H dari reaksi yang bersangkutan

b. energi ikatan rata-rata dari suatu molekul

c. energi disosiasi ikatan

17. Gunakan data energi ikatan untuk menghitung ^H reaksi dari reaksi berikut C3H(g)+5O2(g)→3CO2(g)+4H2O(g) ?


Gunakan data energi ikatan untuk menghitung ΔH reaksi dari reaksi berikut C₃H₈(g)  +  5 O₂(g)  ==>  CO₂(g)  +  4 H₂O(g) ?

Diketahui:

Energi Ikatan:

C-H = 413 kJ

C-C = 348 kJ

O=O = 495 kJ

C=O = 799 kJ

O-H = 463 kJ

Ditanya: ΔH reaksi.....?

Jawab:

Nilai ΔH reaksi berdasarkan pada energi ikatan dapat dihitung dengan persamaan:

ΔH reaksi = ∑ Energi Ikatan pada Pereaksi - ∑ Energi Ikatan pada Produk

untuk mengetahui total energi ikatan pada pereaksi maka langkah pertama yang harus dilakukan yaitu menggambarkan strukturnya terlebih dahulu.

C₃H₈(g)  +  5 O₂(g)  ==>  CO₂(g)  +  4 H₂O(g)

CH₃-CH₂-CH₃ + 5 O=O ==> O=C=O + 4 H-O-H

sehingga nilai energi ikatan pada pereaksi yaitu:

2 x  energi Ikatan C-C = 2 x 348 kJ = 696 kJ

8 x energi ikatan C-H = 8 x 413 kJ = 3304 kJ

5 x energi ikatan O=O = 5 x 495 kJ = 2475 kJ

maka jumlah total energi ikatan pada pereaksi yaitu:

∑ Energi Ikatan pada Pereaksi = 696 kJ + 3304 kJ + 2475 kJ

∑ Energi Ikatan pada Pereaksi = 6475 kJ

Selanjutnya kita cari nilai energi ikatan pada produk, yaitu:

2 x energi ikatan C=O = 2 x 799 kJ = 1598 kJ

8 x energi ikatan O-H = 8 x 463 kJ = 3704 kJ

maka jumlah total energi ikatan pada produk yaitu:

∑ Energi Ikatan pada produk = 1598 kJ + 3704 kJ

∑ Energi Ikatan pada Pereaksi = 5302 kJ

maka nilai ΔH reaksi bisa kita hitung, yaitu:

ΔH reaksi = ∑ Energi Ikatan pada Pereaksi - ∑ Energi Ikatan pada Produk

ΔH reaksi = 6475 kJ - 5302 kJ

ΔH reaksi = 1.173 kJ

Jadi nilai ΔH reaksi pembakaran dari 1 mol Propana adalah sebesar 1.173 kJ.

24. berdasarkan data energi ikatan tersebut,maka besarnya entalpi penguraian 10,95 gram HCl adalah


Data energi ikatan rata-rata :

Ikatan   Energi Ikatan

H – Cl       432 kj/mol

H – H        436 kj/mol

Cl – Cl      432 kj/mol

Berdasarkan data energi tersebut, maka besarnya entalpi penguraian 10,95 gram HCl adalah…..

A. 1,2 kj

B. 12 kj

C. -1,2 kj

D. -1 kj

E. -12 kj

Jawaban : C. -1,2 kj

Pembahasan

Reaksi kimia selalui disertai dengan perubahan energi yang berwujud kalor baik kalor yang dilepas atau diterima oleh sistem. Sistem merupakan segala sesuatu yang menjadi pusat perhatian, sedangkan lingkungan merupakan segala sesuatu diluar sistem

Keseluruhan enegi yang dimiliki oleh sistem dalam keadaan tertentu disebut energi dalam. Energi hanya bergantung pada keadaan sistem (seperti suhu, volum, tekanan dan jumlah mol) dan tidak bergantung pada jalan yang dilalui sistem. Energi tidak dapat diukur tetapi hanya perubahan energi saja yang dapat diukur.

Jika perubahan dilakukan pada tekanan tetap, maka perubahan energi dalam disebut perubahan entalpi (∆H). Perubahan entalpi dari reaksi kimia dapat ditentukan dari besarnya energi ikatan.

Energi ikatan merupakan energi yg diperlukan utk memutuskan ikatan suatu molekul gas menjadi atom-atom dalam keadaan gas. Pada reaksi kimia, terjadi proses pemutusan ikatan antar atom-atom dari senyawa yang bereaksi (memerlukan energi) dan proses penggabungan ikatan kembali dari atom-atom.  

Penentuan perubahan entalpi reaksi dari energi ikatan menggunakan rumus yaitu :

∆H reaksi = ∑ energi ikatan reaktan - ∑ energi ikatan produk

Penyelesaian Soal

Diketahui :

Ikatan   Energi ikatan

H – Cl      432 kj/mol

H – H       436 kj/mol

Cl – Cl     432 kj/mol

Massa HCl = 10,95 gram

Ar H = 1    Ar Cl = 35,5

Ditanya  : besarnya entalpi penguraian 10,95 gram HCl…?

Jawab :

Perubahan entalpi reaksi penguraian (∆Hd) menyatakan perubahan entalpi penguraian 1 mol senyawa menjadi unsur-unsurnya.

Persamaan reaksi penguraian senyawa HCl yaitu :

2HCl(g) → H₂(g) + Cl₂(g)

2 H - Cl → H - H + Cl - Cl

∆H reaksi = ∑ energi ikatan reaktan - ∑ energi ikatan produk

∆H reaksi = (2 x H – Cl) – (H – H + Cl – Cl)

∆H reaksi = (2 x 432 kj/mol) – (436 kj/mol + 432 kj/mol)

∆H reaksi = 864 kj/mol – 868 kjmol

∆H reaksi = - 4 kj/mol

Perubahan entalpi penguraian HCl untuk 1 mol HCl adalah sebesar - 4kj/mol

Mr HCl = Ar H + Ar Cl

Mr HCl = 1 + 35,5 = 36,5  

Mol =  massa / Mr

Mol HCl =  10,95 gram ÷ (36,5 gram/mol)

Mol HCl = 0,3 mol

Besarnya perubahan entalpi penguraian 10,95 gram HCl yaitu:

∆H reaksi = - 4 kj/mol x 0,3 mol

∆H reaksi = -1,2 kj

Jadi, besarnya perubahan entalpi penguraian 10,95 gram HCl adalah sebesar -1,2 kj

∆H = Ʃ Dreaktan – Ʃ Dproduk

∆H = {8 C–H + 2C–C + Cl–Cl} – {7 C–H + 2C–C + C–Cl + H–Cl}

∆H = {C–H + Cl–Cl} – {C–Cl + H–Cl}
