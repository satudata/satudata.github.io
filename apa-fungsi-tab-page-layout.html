6. sebutkan 3 tombol di tab page layout dan sebutkan fungsinya


Jawaban:

Misalkan 3 tombol di tab page layout dalam program pengolah angka:

1. Margins

tombol untuk mengatur batas kertas dokumen aktif, batas yang dapat disetting adalah:

- batas atas (top margin)

- batas bawah (bottom margin)

- batas kanan (right margin)

- batas kiri (left margin)

2. Orientation

tombol untuk mengatur orientasi kertas dokumen aktif, pilihannya yaitu:

- Portrait (mengatur orientasi kertas secara vertikal/ tegak lurus)

- Landscape (mengatur orientasi kertas secara horizontal/ mendatar)

3. Size

tombol untuk mengatur ukuran kertas (paper size) dokumen aktif.

8. jelaskan fungsi fungsi yang terdalat dalam tab home,tab insert Dan page layout pada Microsoft excel


1. Home
Home merupakan menu utama yang akan tampil pada saat pertama kali anda membuka word 2007. Menu home terdiri dari toolbar standar yaitu font, tool bar formatting dan paragraph, style dan editing serta clipboard.(klik untuk memperbesar gambar)
2. Insert
Tab ini digunakan untuk memasukkan instruksi-instruksi ke dalam dokumen. Menu insert terdiri dari : Pages, Tables, Illustrations, Links, Header & Footer, Text, Symbol.(klik untuk memperbesar gambar)
3. Page Layout
Tab page Layout digunakan untuk mengatur tata letak data, yang meliputi Page Setup, memberiwarna background pada lembar kerja, mengatur spasi antar baris, membuat kolom pada teks dan mengatur indentasi. Menu page layout terdiri dari : Themes, Page Setup, Page Background, Paragraph, Arrange.(klik untuk memperbesar gambar)
4. References
Tab ini dugunakan untuk memasukkan berbagai perintah seperti menuliskan footnote (catatan kaki), menyisipkan gambar, dan membuat tanda citasi. Perhatikan tampilan tab references
Tab References. References terdiri dari : Table of Contents, Footnotes, Citations & Bibliography, Captions, Index, Table of Authorities.(klik untuk memperbesar gambar)
5. Mailings
Tab ini berguna untuk membuat mail merger (surat massal), mengetik teks pada amplop, dan membuat label. Mailings terdiri dari : create, start mail merge, write & insert Fields, preview results, finish.(klik untuk memperbesar gambar)
6. Review
Tab Review berguna untuk mengecek kesalahan tata bahasa serta memproteksi dokumen. Review terdiri dari : proofing Comments, Tracking, changes, Compare, Protect.(klik untuk memperbesar gambar)
7. View
Tab View digunakan untuk menampilkan atau mencetak dokumen pada layar, mengatur ukuran cetakannya, dan menampilkan garis bantu pada lembar kerja. Menu View berfungsi untuk mengatur tampilan layar dari dokumen yang kita kerjakan. Berikut ini merupakan berbagai fungsi dari tool-tool yang terdapat dalam menu View.

30. Jelaskanlah fungsi-fungsi yang terdapat dalam tab home tab insert dan page layout pada Microsoft Excel 2007 !


Pada Microsoft Exel 2007 tab Home berfungsi untuk menampung perintah-perintah yang paling sering digunakan oleh penguna untuk mengedit dokumen, seperti group Clipboard yang berisi paste, cut, copy, dan format paitnter, serta masih banyak sub menu lainnya. Tab insert berfungsi menyisipkan opjek atau ilustrasi seperti gambar, clip Art, Shpes, SmartArt, Cart, table, persamaan matematika, dan simbol. Tab page layout berfungsi untuk mengatur tampilan kertas dari dokumen.

Pembahasan

Tab Home, terdiri atas:

Group Clipboard, terdiri atas empat, yaitu:Paste, untuk menempatkan hasil copy atau cut.Cut, untuk memotong atau memindahkan bagian cell atau range yang ditandai.Copy, untuk menyalin satu bagian cell atau rangeFormat Painter, untuk menentukan format painter atau menyalin format.Group Font terdiri atas 10 tombol, antara lain:Font, untuk menentukan jenis font yang akan digunakan.Font size, untuk mengubah ukuran font.Bold, untuk menentukan efek mencetak tebal.Underline, untuk menentukan efek garis bawah.Italic, untuk menentukan efek mencetak miringGroup Alignment, terdiri atas 11 tombol atau ikon, antara lain:Top align, untuk membuat isi cell rata atas.Bottom align, untuk membuat isi cell rata bawah.Middle align, untuk membuat isi cell berada di tengah.Align text right, untuk membuat isi cell rata kanan.Align text left, untuk membuat isi cell rata kiri.Group Number, terdiri atas 5 simbol atau ikon, antara lain:Number format, untuk menampilkan pilihan format angka.Accounting number format, untuk menampilkan mata uang.Percent style, untuk membuat angka dalam bentuk persen.Group Styles, terdiri atas 3 ikon yaitu:Format as tabel, untuk mengatur tabel.Conditional formatting, untuk pengaturan bersyarat.Cell style, untuk menampilkan cell.Group Cells, terdiri atas 3 ikon, yaitu:Insert, untuk menyisipkan cell, baris, atau kolom pada worksheet.Delete, untuk menghapus isi cell.Format, untuk pengaturan ukuran cell.Group Editing, terdiri atas 5 ikon, antara lain:Sum, fungsi yang digunakan untuk menampilkan hasil penjumlahan.Fill, untuk mengisi nilai secara otomatis.Clear, untuk menghapus format, isi atau semuanya.

Tab Insert, terdiri atas:

Group Tabels, terdiri atas 2 ikon yaitu:Pivot tabel, untuk membuat pivot data dan tabel.Table, untuk membuat tabel, serta mengatur dan menganalisis data.Group Illustrations, terdiri atas 4 ikon, antara lain:Picture, untuk menyisipkan gambar dari file.Clip Art, untuk menyisipkan gambar Clip Art.Shapes, untuk memilih model-model shapes dalam pembuatan flowchart.Group Charts, terdiri atas 7 ikon antara lain:Column, untuk menyisipkan grafik batang vertikal.Line, untuk menyisipkan grafik garis.Pie, untuk grafik lingkaran.Bar, untuk menyisipkan grafik batang horizontal.Group Text, terdiri atas 6 ikon antara lain:Text Box, untuk menyisipkan teks di dalam kotak.Header and footer, untuk mengedit heder dan footer dalam dokumen.WordArt, untuk membuat WordArt.Signature Line, untuk menyisipkan tanda tangan dalam workbook.Group Links, berfungsi untuk menghubungkan lembar kerja dengan link ke tempat yang lain.

Tab Menu Page Layout, terdiri atas:

Group Themes, terdiri atas 4 ikon, yaitu:Themes, untuk pengaturan desain workbook.Colors, untuk pengaturan warna pada tema.Fonts, untuk pengaturan huruf pada tema.Effets, untuk pengaturan efek pada temaGroup Page Setup, terdiri atas 7 ikon antara lain:Margins, untuk mengatur worksheet.Orientation, untuk mengatur orientasi kertas untuk mencetak. Size, untuk memilih ukuran kertas untuk mencetak. Print Area, untuk memilih area tertentu untuk dicetak.Group Scale to fit, terdiri atas 3 ikon, yaitu:width, untuk mengatur lebar halaman sebelum dicetak.Height, untuk mengatur tinggi halaman sebelum dicetak.Scale, untuk mengatur persentase tampilan halaman.Group Sheet Options, berfungsi untuk mengatur properti yang ada pada sheet seperti grid dan headings.Group Arrange, terdiri atas 6 ikon, diantaranya:Bring to font, untuk meletakkan objek gambar di atas atau depan objek lain.Send to back, berfungsiuntuk meletakkan objek gambar di belakang atau bawah objek lain.Align, untuk mengatur perataan beberapa gambar dari beberapa objek.
