Penjelasan dengan langkah-langkah:

Diketahui:

Hp = a

Laptop = b

(a/b) = (3/7)

    a = (3/7) x b ..... (persamaan 1)

b-a = 2.000.000

   a = b-2.000.000..... (persamaan 2)

Pertanyaan:

2a+b = ....

Jawab:

(I)

Subtitusi pers-1 ke pers-2:

a = b-2.000.000

(3b/7) = b-2.000.000  .... kalikan 7

3b  = 7b-14.000.000

4b = 14.000.000

b = 3.500.000

(II)

subtitusi nilai b ke pers-2:

a= 3.500.000-2.000.000

a= 1.500.000

(III)

masukkan nilai a untuk menjawab pertanyaan:

2a+b = (2 x 1.500.000) + 3.500.000

        = 3.000.000 + 3.500.000

        = 6.500.000
