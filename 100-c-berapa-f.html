°R = 4/5 x °C
     = 4/5 x 100 = 80 °C

100°C = ... °F
°F = (9/5 x °C) + 32
     = (9/5 x 100) + 32
     = 180 + 32
     = 212 °F

100°C = ... K
K = °C + 273
   = 100 + 273
   = 373 K
100⁰c = 100+273 = 373⁰k
100°c= 4/5X100⁰ = 80R
100⁰C = 9/5 X100+32 =212⁰F
