barang yg diproduksi a dan b yadiketahui : jika mesin a,b dan c bekerja bersama-sama menghasilkan 222 unit 
                   jika mesin a dan c bekerja bersama-sama menghasilkan 138 unit
                   jika masin b dan c bekerja bersama-sama menghasilkan 147 unit

ditanya :.Berapakah hasil yang diproduksi jika hanya mesin a dan b yang beroperasi?

jawab :
bentuk kalimat yang diketahui diubah kedalam bentuk matematika dulu,

a + b + c = 222        (1)
a + c = 138               (2)
b + c = 147               (3)

a + b = ....                 (4)

eliminasi a dan c dari pers (1) dan (2),
a + b + c = 222        (1)
a + c = 138               (2)
____________-
b = 84

eliminasi b dan c dari pers (1) dan (2),
a + b + c = 222        (1)
b + c = 147               (3)
____________-
a = 75

subtitusikan nilai a dan b ke pers (4)
a + b  =
75 + 84 = 159

jadi, hasil yang diproduksi jika hanya mesin a dan b yang beroperasi adalah 159 unit
