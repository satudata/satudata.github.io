1. Apakah kepanjangan dari RAM..?? Dan Apakah itu RAM?


RAM (Random Access Memory)

Merupakan jenis memori yang isinya dapat diganti-ganti selama komputer dihidupkan sebagai suatu penyimpanan data yang dapat dibaca atau ditulis dan dapat dilakukan secara berulang-ulang dengan data yang berbeda-beda. Random artinya data yang disimpan pada RAM dapat diakses secara acak. Modul memori RAM yang umum diperdagangkan berkapasitas 128 MB, 256 MB, 512 MB, 1 GB, 2 GB, dan 4 GB

RAM dibagi lagi menjadi dua jenis, yaitu jenis statik dan dinamik.

RAM statik menyimpan satu bit informasi dalam sebuah flip-flop. RAM statik biasanya digunakan untuk aplikasi-aplikasi yang tidak memerlukan kapasitas memori RAM yang besar.RAM dinamik menyimpan satu bit informasi data sebagai muatan. RAM dinamik menggunakan kapasitas gerbang substrat sebuah transistor MOS sebagai sel memori elementer.

19. 4. Apakah kepanjangan dari CPU, RAM, ROM, dan ALU? sing-masing​


Kepanjangan dari masing-masing singkatan tersebut adalah:

RAM = Random Access MemoryCPU = Central Processing UnitROM = Read Only MemoryALU = Arithmetic Logic Unit.

PEMBAHASAN:

RAM

Komputer atau PC jika memiliki RAM yang besar, maka pemprosesan datanya akan semakin cepat. Sehingga bisa dikatakan bahwa RAM dapat mempercepat kinerja komputer karena RAM adalah tempat penyimpanan data sementara saat komputer dijalankan dan dapat diakses secara acak (random). Adapun fungsi RAM yaitu mempercepat pemprosesan data pada komputer atau PC

CPU

Otak komputer adalah CPU dimana CPU akan bertugas untuk menerima dan melaksanakan perintah dan data dari perangkat lunak. Sebegitu pentingnya fungsi dari CPU maka jika tanpa ada CPU komputer tidak akan berfungsi sebagaimana mestinya. Program yang sudah disimpan dalam memori utama, akan diambil lalu diuji dengan perintah oleh  CPU.

ROM

ROM berbeda dengan RAM dimana ROM memiliki sifat permanen. Artinya program atau data yang disimpan didalam ROM tidak mudah hilang ataupun berubah-ubah meskipun aliran listrik sudah dimatikan. Menyimpan data di dalam ROM juga tidak mudah, namun untuk membaca data merupakan hal yang mudah. Biasanya program atau datanya diisi oleh pabrik yang memproduksinya. Hal ini membuat ROM digunakan untuk menyimpan firmware.

ALU

ALU merupakan bahasa mesin. ALU bertugas untuk melakukan semua perhitungan aritmatika sesuai perintah program. Adder merupakan dasar penjumlahan sehingga sirkuit elektronik untuk melakukan semua operasi aritmatika. Ada juga tugas lain dari ALU yaitu membuat keputusan dari operasi logika sesuai dengan perintah program. Operasi logika diantaranya perbandingan dua operand dengan menggunakan operator logika tertentu, yaitu sama dengan (=), tidak sama dengan (¹), kurang dari (), dan lebih besar atau sama dengan (³).
