1. sebuah pipa tembaga mempunyai panjang 1 m pada suhu 55°. berapakah panjang pipa tembaga pada suhu 175°c


Diketahui :
Lo =1  m
To = 55 C
Tt = 175 C
Ditanyakan : Lt ?

Dijawab :
α = 0,000017/C

Jadi Panjang akhir :
Lt = Lo (1 + α x (Tt-To)
    = 1 (1 + 0,000017 x (175-55))
    = 1 (1 + 0,000017 x 120)
    = 1 (1 + 0,00204)
    = 1,00204 m

2. Panjang pipa tembaga 1 Meter dengan suhu 25°C Koefisien tembaga 0,000017/°C berapakah panjang pipa tembaga pada suhu 50°C ?


ΔL = Lo x α x ΔT

     = 1 x 0.000017 x 25

     = 4,25 x 10^-4 m

L akhir = Lo + ΔL

           = 1 + 4,25 x 10^-4

           = 1,000425 m

7. Sebuah pipa tembaga mempunyai panjang 1 m pada suhu 25°C. panjang pipa tembaga pada suhu 175°C adalah .... (alfa tembaga = 1,7 × 10-5/°C)


ΔL = L0 x α x ΔT

ΔL = Perubahan panjang
L0 = Panjang awal = 1 m
α = koefisien muai panjang (1,7 x 10-5/°C)
ΔT = Perubahan suhu (175 - 25 = 150°C)

ΔL = 1 x 1,7 x 10-5 x 150
ΔL = 255 x 10-5
ΔL = 0,00255 m

Panjang pipa akhir = 1 + 0,00255 = 1,00255 m 
Semoga membantu, maaf kalau salah

21. Sebuah pipa tembaga mempunyai 1 m pada suhu 25°C.Panjang pipa tembaga pada suhu 175°C adalah TOLONG DIJAWAB SECEPATNYA YA


Lt = Lo (1 + α x Δt)
Lt = panjang akhir (m, cm)
Lo= panjang awal (m, cm)
α = koefisien muai panjang (/°C)
Δt = perbedaan suhu (°C)

Diket :
Lo = 1 m
α tembaga = 0,000017/°C
Δt = 175 - 25 = 150°C

Lt = 1 (1 + 0,000017 x 150)
Lt = 1 (1 + 0,00255)
Lt = 1 . 1,00255
Lt = 1,00255 m

jadi panjang pipa tembaga pada suhu 175°C adalah 1,00255 meter.
