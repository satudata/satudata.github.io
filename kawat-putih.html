[tex]\boxed{\boxed{Penjelasan\:Nomor\:1}}[/tex]

[tex]\boxed{Diketahui}[/tex]

Hambatan meningkat 10%

[tex]\boxed{Ditanya}[/tex]

Perubahan suhu.

[tex]\boxed{Penyelesaian}[/tex]

Untuk mencari perubahan suhu tersebut bisa menggunakan persamaan :

Rt = R₀ ( 1 + α ΔT )

dimana :

Rt = 10% R₀ + R₀

atau

Rt = 1,1 R₀

sehingga :

Rt = R₀ ( 1 + α ΔT )

1,1 R₀ = R₀ ( 1 + α ΔT )

1,1 = 1 + α ΔT

dimana :

α = 6,18 x 10⁻³ /°C   (koefisien suhu hambatan jenis besi)

sehingga :

1,1 = 1 + 6,18 x 10⁻³ /°C ΔT

1,1 - 1 = 6,18 x 10⁻³ /°C ΔT

0,1 = 6,18 x 10⁻³ /°C ΔT

ΔT = 0,1 / 6,18 x 10⁻³ /°C

ΔT  = 0,162 x 10² °C

ΔT  = 16,2°C

[tex]\boxed{Kesimpulan}[/tex]

Besarnya perubahan suhu yang harus diberikan adalah 16,2°C.

[tex]\boxed{\boxed{Penjelasan\:Nomor\:2}}[/tex]

[tex]\boxed{Diketahui}[/tex]

Panjang A dan B : La = Lb

Bahan A dan B    : ρa = ρb

Diameter kawat pejal (da) = 1 mm

Diameter luar kawat tabung (db) = 2 mm

Diamater dalam kawat tabung (db') = 1 mm

[tex]\boxed{Ditanya}[/tex]

Perbandingan antara hambatan kawat A dan kawat B.

[tex]\boxed{Penyelesaian}[/tex]

Untuk mencari Perbandingan antara hambatan kawat A dan kawat B bisa menggunakan cara sebagai berikut :

Luas penampang kawat A

Luas penampang kawat A adalah :

Aa = 1/4 π (da)²

Luas penampang kawat B

Luas penampang kawat B adalah :

Ab = 1/4 π (db - db')²

Persamaan hambatan

Persamaan hambatan pada kawat adalah :

R = ρ L / A

Perbandingan antara hambatan kawat A dan kawat B.

Perbandingan tersebut adalah :

Ra : Rb = ρa La / Aa   : ρb Lb / Ab

karena La = Lb dan ρa = ρb, maka :

Ra : Rb = Ab : Aa

Ra : Rb = 1/4 π (db - db')² : 1/4 π (da)²

Ra : Rb = (db - db')² :  (da)²

Ra : Rb = (2 mm - 1 mm)²  : ( 1 mm)²

Ra : Rb = 1 mm² : 1 mm²

Ra : Rb = 1 : 1

[tex]\boxed{Kesimpulan}[/tex]

Perbandingan antara hambatan kawat A dan kawat B adalah 1 : 1.

sehingga :
