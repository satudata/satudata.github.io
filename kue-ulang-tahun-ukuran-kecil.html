2. Setelah di ukur,diameter paling atas kue ulang tahun adalah 14cm.setiap diameter di bawahnya lebih besar 7cm.tentukan volume kue ulang tahun bertingkat tiga tersebut


Jawaban:

Tabung adalah bangun ruang dengan sisi lengkung

Rumus-rumus yang ada pada bangun tabung

1) Luas alas tabung = π x r²

2) Luas selimut tabung = Keliling alas x tinggi

= 2 x π x r x t atau = π x d x t

Keliling alas = panjang selimut

3) Luas permukaan tabung tanpa tutup = luas alas + luas selimut

 = π x r² + 2 x π x r x t

 = π x r x (r + 2t)

4) Luas permukaan tabung dengan tutup =  luas alas + luas tutup + luas selimut

 = π x r² + π x r² + 2 x π x r x t

 = 2 x π x r² + 2 x π x r x t

 = 2 x π x r x (r + t)

5) Volume tabung = π x r² x t atau luas alas x t

r = jari-jari

d = diameter

t = tinggi

Penyelesaian Soal

Soal ini kurang lengkap, karena tidak disertai foto kuenya, atau informasi tentang kuenya.

Diketahui:

Kue tart susun 3,

Diameter paling atas 14 cm

Diameter dibawahnya selalu selisih 7 cm.

Tinggi kue tar 45 cm

Ditanya:

Volume kue tar dalam satuan cm³

Jawab:

Kue tar susun 3,

Tinggi kue 45 cm,

Berarti tinggi tiap susunnya 45 cm : 3 = 15 cm

Diameter paling atas = 14 cm

Jari-jari kue paling atas = 7 cm

Diameter kue tar bagian tengah = 21 cm

Jari-jari kue tar bagian tengah = 10,5 cm

Diameter paling bawah = 28 cm

Jari-jari kue paling atas = 14 cm

Volume kue tar paling atas

= Volume tabung

= π x r² x t

= \frac{22}{7}722 x 7 x 7 x 15

= 2.310 cm³

Volume kue tar bagian tengah

= Volume tabung

= π x r² x t

= \frac{22}{7}722 x 10,5 x 10,5 x 15

= 5.197,5 cm³

Volume kue tar paling bawah

= Volume tabung

= π x r² x t

= \frac{22}{7}722 x 14 x 14 x 15

= 9.240 cm³

Volume kue tar mainan

=  2.310 cm³ + 5.197,5 cm³ + 9.240 cm³

= 16.747,5 cm³
