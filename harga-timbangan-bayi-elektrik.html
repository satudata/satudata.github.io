25. Di posyandu desa X terdapat 5 bayi yang sedang ditimbang. Rata-rata berat kelima bayi tersebut sama dengan mediannya. Ketika datang satu bayi lagi ditimbang, ternyata rata-rata berat bayinya menjadi naik 1 kg tetapi mediannya tidak berubah. Jika berat dari kelima bayi awal diurutkan mulai dari yang paling ringan, tentukan selisih berat dari bayi ke-4 dan bayi yang baru datang!


Selisih berat dari bayi ke-4 dan bayi yang baru datang adalah sebesar 6 kilogram.

Pembahasan

Materi  pada soal merupakan soal statistika sederhana. Pada umumnya dalam mengolah data sederhana, dapat ditentukan nilai rata-rata(mean), nilai tengah(median), dan nilai yang paling sering muncul(modus). Cara menentukan mean, median, dan modus adalah sebagai berikut :

1. Cara menentukan mean

Pada umumnya, rumus menentukan rata-rata adalah

[tex]\boxed{\bold{mean = \frac{jumlah \hspace{1 mm} data}{banyak \hspace{1 mm} data} }}[/tex]

2. Cara menentukan median

Untuk menentukan nilai tengah/median, data harus diurutkan terlebih dahulu lalu dilihat nilai yang berada di barisan paling tengah. Jika banyak data adalah bilangan ganjil, maka mediannya adalah data ke [tex]\frac{n + 1}{2}[/tex] dengan n adalah banyak data. Sedangkan untuk banyak data berupa bilangan genap, maka median dari data tersebut adalah jumlah data ke [tex]\frac{n}{2}[/tex] dan data ke [tex]\frac{n + 2}{2}[/tex] lalu dibagi dengan 2 dengan n adalah banyak data.

3. Cara menentukan modus

Untuk menentukan nilai modus, maka tinggal dilihat nilai yang paling sering muncul pada kumpulan data tersebut. Modus dari kumpulan data tersebut bisa saja ada 2 atau lebih, tidak harus 1 nilai.

Gunakan cara-cara diatas dalam menyelesaikan soal.

Penyelesaian

Diketahui :

Banyak bayi mula-mula adalah 5 bayi.

Rata-rata kelima bayi mula-mula sama dengan mediannya.

Banyak bayi ditambah 1, sehingga rata-ratanya naik 1 kg namun mediannya tetap.

Ditanyakan :

Selisih berat bayi ke-4 dengan bayi yang baru datang.

Jawaban :

Dimisalkan berat kelima bayi mula-mula tersebut dilambangkan a, b, c, d, e dengan a ≤ b ≤ c ≤ d ≤ e. Karena rata-rata kelima bayi tersebut sama dengan mediannya, maka dengan rumus rata-rata didapat

[tex]\bold{mean = median}[/tex]

[tex]\bold{\frac{a + b + c + d + e}{5} = c}[/tex]

[tex]\bold{a + b + c + d + e = 5c}[/tex] ... (persamaan 1)

Misal berat bayi yang baru datang adalah f, karena rata-ratanya bertambah 1 kilogram, maka didapat

[tex]\bold{mean = c + 1}[/tex]

[tex]\bold{\frac{a + b + c + d + e + f}{6} = c + 1 }[/tex]

[tex]\bold{a + b + c + d + e + f = 6c + 6}[/tex]

substitusikan nilai a + b + c + d + e = 5c pada persamaan 1, maka didapat

[tex]\bold{a + b + c + d + e + f = 6c + 6}[/tex]

[tex]\bold{5c + f = 6c + 6}[/tex]

[tex]\bold{f = c + 6}[/tex]

Karena f = c + 6, maka dapat dipastikan bahwa f > c sehingga f merupakan data pada urutan ke 4, 5, atau 6. Dari sini didapat

1. Jika f merupakan data pada urutan ke-4

Maka urutan datanya adalah a, b, c, f, d, e sehingga didapat

[tex]\bold{median = \frac{c + f}{2} }[/tex]

[tex]\bold{median = \frac{c + c + 6}{2} }[/tex]

[tex]\bold{median = c + 3 }[/tex]

Karena seharusnya mediannya tetap, maka f bukan data ke-4.

2. Jika f merupakan data pada urutan ke-5 atau ke-6

Maka urutan datanya adalah a, b, c, d, f, e atau a, b, c, d, e, f sehingga didapat

[tex]\bold{median = \frac{c + d}{2} }[/tex]

Agar mediannya tetap bernllai c, maka haruslah c = d sehingga f merupakan data ke-5 atau data ke-6.

Karena d merupakan data ke-4 dengan c = d dan f = c + 6, maka selisih berat bayi ke-4 dengan bayi yang baru datang adalah f - d = (c + 6) - c = 6 kilogram.
