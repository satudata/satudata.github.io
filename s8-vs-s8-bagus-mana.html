6. Selesaikan soal deret geometri berikut!U3 = 8 dan U5 = 32 S8 =U2 = 3 dan U4 = 27 S8=​


Rumus umum suku ke-n deret geometri :

[tex]U_n = U_1 \times {r}^{(n - 1)} [/tex]

Rumus umum jumlah deret geometri sampai suku ke-n :

[tex]S_n = U_1 \times \frac{{r}^{n} - 1}{r - 1}[/tex]

1.

[tex]U_3 = 8[/tex]

[tex]U_1 \times {r}^{(3 - 1)} = 8[/tex]

[tex]U_1 \times {r}^{2} = 8[/tex]

[tex]U_1 = \frac{8}{ {r}^{2}} \: \: \: ... \: \: \: (i)[/tex]

[tex]U_5 = 32[/tex]

[tex]U_1 \times {r}^{(5 - 1)} = 32[/tex]

[tex]U_1 \times {r}^{4} = 32[/tex]

Subtitusikan nilai U₁ pada ( i ) :

[tex]\frac{8}{{r}^{2}} \times {r}^{4} = 32[/tex]

[tex]8 \times {r}^{2} = 32[/tex]

[tex]{r}^{2} = 32 \div 8 = 4 \: \: \: > \: \: \: r= 2[/tex]

Subtitusikan nilai r ke ( i ) :

[tex]U_1 = \frac{8}{{2}^{2}} = \frac{8}{4} = 2[/tex]

Sehingga :

[tex]S_8 = 2 \times \frac{{2}^{8} - 1}{2 - 1}[/tex]

[tex]S_8 = 2 \times \frac{256 - 1}{1}[/tex]

[tex]S_8 = 2 \times 255[/tex]

[tex] \boxed{ \boxed{S_8 = 510}}[/tex]

2.

[tex]U_2 = 3[/tex]

[tex]U_1 \times {r}^{(2 - 1)} = 3[/tex]

[tex]U_1 \times r = 3[/tex]

[tex]U_1 = \frac{3}{r} \: \: \: ... \: \: \: (i)[/tex]

[tex]U_4 = 27[/tex]

[tex]U_1 \times {r}^{(4 - 1)} = 27[/tex]

[tex]U_1 \times {r}^{3} = 27[/tex]

Subtitusikan nilai U₁ pada ( i ) :

[tex] \frac{3}{r} \times {r}^{3} = 27[/tex]

[tex]3 \times {r}^{2} = 27[/tex]

[tex]{r}^{2} = 27 \div 3 = 9 \: \: \: > \: \: \: r= 3[/tex]

Subtitusikan nilai r ke ( i ) :

[tex]U_1 = \frac{3}{3} \: \: \: > \: \: \: U_1 = 1[/tex]

Sehingga :

[tex]S_8 = 1 \times \frac{{3}^{8} - 1}{3 - 1}[/tex]

[tex]S_8 = \frac{6.561 - 1}{3 - 1}[/tex]

[tex]S_8 = \frac{6.560}{2}[/tex]

[tex] \boxed{ \boxed{S_8 = 3.280}}[/tex]
