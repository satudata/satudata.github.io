= 125.600 ribu

= 200.000 ribu

Total biaya seluruhnya = 125.600 + 200.000
= 325.600 ribu

Cara:
Faktor:
56:2^3×7
64:2^3×3^2
FPB=2^3=8

56merah:8=7merah
64putih :8 =8putih
7merah+8putih=15 rantai gelang

Jadi,pada setiap rantai ada 7merah dan 8putih sehingga ada 15 rantai gelang.

56 = 8 x 7

56 = 2³ x 7


64 = 2 x 2 x 2 x 2 x 2 x 2

64 = 2⁸


FPB = 2³

FPB = 8
