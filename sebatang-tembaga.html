1. Harga sebatang tembaga adalah 2 kali harga sebatang besi jika harga 2 batang tembaga dan 3 batang besi rp420.000.00 maka harga sebatang tembaga adalah


Tembaga = x
Besi = y

x = 2y
2x + 3y = 420.000

2x + 3y = 420.000
4y + 3y = 420.000
7y = 420.000
y = 60.000

x = 2y
x = 2 × 60.000
x = 120.000

Jadi, harga sebatang tembaga adalah Rp.120.000

Maaf Kalau Salahtembaga = a
besi = b

a = 2b
2a + 3b = 420.000

2(2b) + 3b = 420.000
7b = 420.000
b = 60.000

a = 2b
a = 2(60.000)
a = 120.000

maka harga sebatang tembaga adalah Rp.120.000

8. Sebatang tembaga pada suhu 28°C panjangnya 80m. Berapakah pertambahan panjangnya, jika tembaga dipanaskan sampai suhunya 78°C?


Diketahui = Lo = 80 m
                 T1 = 28 C
                 T2 = 78 C
                   α = 0,000017/derajat celcius = 17 x 10^-6 C
Ditanya = pertambahan panjang..?
Jawab = Lt  = Lo ( 1+ α x Δt)
                 = 80 ( 1 + 0,000017 x (78-28))
                 = 80 + 0,00136 x 50
                 = 80 + 0,068
                 = 80,068 m
Maka pertambahan panjang = Lt - Lo
                                         = 80,068 m - 80 = 0,068 m
Atau dengan cara 
Δl = α . lo . ΔT
    = 0,000017 . 80 .(78 - 28)
    = 0,068 m
