15. Sebutkan seb menu yang terdapat di menu page layout


-Icon Perintah di Group Themes

1. Themes: digunakan untuk memilih tema halaman yang meliputi warna halaman berikut warna hurufnya.

2. Colors: digunakan untuk mengatur warna tema halaman.

3. Fonts: digunakan untuk mengatur tema huruf yang akan diterapkan ke halaman dokumen aktif.

4. Effect: digunakan untuk mengatur tema efek terhadap objek Shape yang terdapat di halaman dokumen.

-Icon Perintah di Group Page Setup

1. Margins: digunakan untuk mengatur batas teks di halaman dokumen.

2. Orientation: digunakan untuk mengatur posisi kertas.

3. Size: digunakan untuk mengatur ukuran halaman.

4. Columns: digunakan untuk mengatur jumlah kolom teks.

5. Breaks: digunakan untuk mengatur kontinuitas halaman maupun kolom teks.

6. Line Numbers: digunakan untuk mengatur kontinuitas nomor baris teks.

7. Hypenation: digunakan untuk memisahkan suku kata dengan tanda hubung secara otomatis.

-Icon Perintah di Group Page Background

1. Watermark : digunakan untuk memberikan efek tanda air di belakang teks.

2. Page Color : digunakan untuk mengatur warna latar halaman.

3. Page Borders : digunakan untuk mengatur garis tepi halaman.

-Icon Perintah di Group Paragraph

1. Left: menggeser tulisan yang dibog di lembar kerja Microsoft Word dari kiri ke kanan.

2. Right: menggeser tulisan pada lembar kerja dari kanan ke kiri halaman.

2. Before: membuat jarak antara dari baris pertama dan baris ke dua dan seterusnya.

3. After sama dengan fungsi dari Before.

-Icon Perintah di Group Arrange

1. Position: meletakan gambar pada lembar kerja sesuai dengan kebutuhan.

2. Warp Text: menyisipkan gambar kedalam naskah yang ada pada pada lembar kerja Microsoft Word.

3. Bring Forward: mentukan picture ke arah depan.

4. Send Backward: mentukan picture ke arah belakang pada lembar kerja.

5. Selection Pane: memilih jenis gambar yang ingin dimasukan pada lembar kerja.

6. Align: membuat tulisan pada lembar kerja pada Microsoft Word untuk rata kanan dan rata kiri.

7. Group: membuat sekelompok group menjadi 1 kelompok, sesui dengan kebutuhan.

8. Rotate: mengatur gambar dengan cara memutar gambar desngan susuai kebutuhan

25. sebutkan submenu dalam menu layout






1. New.

New adalah submenu dalam menu file yang di dalam submenu tersebut terdapat beberapa jenis pemilihan file mana yang akan digunakan. Misalnya saja, blank document, blog post, recent templates, sample templates, my templates dan new from existing. Dari semua hal tersebut, terdapat jenis atau bentuk yang berbeda-beda untuk menulis dokumen. Jadi, tinggal pilih jenis mana yang akan digunakan. Fungsi dari submenu new adalah untuk membuat dokumen baru.

2. Open.

Open adalah salah satu submenu dalam menu file yang pengguna mengklik submenu tersebut maka akan muncul beberapa dokumen atau beberapa file yang sudah disimpan sebelumnya. Selanjutnya, tinggal memilih saja file atau dokumen mana yang akan dibuka. Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.

3. Save.

Save adalah salah satu submenu dalam menu file yang di jika pengguna mengklik submenu tersebut maka pengguna berarti akan menyimpan file atau dokumen mereka. Kemudian, pengguna harus memilih di mana file atau dokumen tersebut akan diletakkan. Pengguna juga dapat memilih dalam bentuk atau tipe apa file atau dokumen mereka akan disimpan. Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.

4. Save as.

Save as adalah submenu dalam menu file yang fungsinya hampir sama dengan submenu save. Pengguna harus memilih mau diletakkan dimana dokumen atau file mereka. Pengguna juga akan bisa memilih dalam bentuk atau tipe apa dokumen atau file mereka yang akan disimpan. Fungsi dari submenu save as dalam menu file ini adalah untuk menyimpan dokumen atau file dengan format file yang berbeda dari penyimpanan yang pertama.

5. Print.

Print adalah submenu dalam menu file yang jika pengguna mengklik submenu tersebut berarti pengguna akan mencetak dokumen mereka. Dalam submenu print, terdapat beberapa hal untuk mengecek kembali apakah dokumen yang akan dicetak sudah benar atau belum. Fungsinya adalah untuk mencetak dokumen dengan printer.
