4. untuk melakukan lari jarak jauh menggunakan start? ​


Jawaban:

Lari jarak jauh menggunakan tipe start berdiri (standing start)

Penjelasan:

Setiap nomor olahraga lari pasti melakukan awalan yang dinamakan start, dalam melakukan start memiliki macam-macam start yang berbeda, yaitu start jongkok, start melayang dan start berdiri. Penggunaan start tersebut disesuaikan dengan nomor lari yang sedang di perlombakan.

Untuk lari jarak jauh menggunakan tipe start berdiri (standing start), start ini juga digunakan pada nomor lari jarak menengah, adapun cara melakukan start yang baik dan benar dijelaskan pada pembahasan di bawah ini :

Bukalah kedua kaki selebar bahu.

Buka salah satu kaki dengan cara menggesernya ke belakang, kira-kira dengan jarak tiga telapak kaki, usahakan jaraknya sesuai.

Kaki belakang di jinjitkan dan letakkan titik tumpu pada jari kaki dan tumit.

Lutut bagian depan ditekuk secukupnya, hingga posisi badan menjadi lebih rendah.

Sedikit condongkan badan ke depan mengikuti tekukan kaki yang ada di depan.

Tetap usahakan dada terbuka lebar agar mempermudah pernafasan sehingga tubuh tidak menjadi tegang.

Usahakan konsentrasi tetap terjaga dan selalu memandang ke depan.

Posisi tangan mengepal, namun usahakan tetap rileks.

Tetap tenang dan fokus mendengarkan aba-aba selanjutnya untuk berlari.

14. Start yang digunakan pada lari jarak jauh


          Start yang digunakan pada lari jarak jauh adalah Start Berdiri, Start yang digunakan pada lari jarak pendek adalah Start Jongkok, dan Start yang digunakan pada lari estafet adalah Start Jongkok dan Start Melayang.

Pembahasan

           Penjelasan macam-macam start adalah sebagai berikut :  

START JONGKOK

           Start Jongkok (crouching start) adalah start yang digunakan pada cabang olahraga atletik lari jarak pendek. Start Jongkok digunakan pada panjang lintasan 100m, 200m, dan 400m.

________________

START BERDIRI

           Start Berdiri (Standing Start) adalah start yang digunakan pada cabang olahraga atletik lari jarak menengah dan lari jarak jauh. Start berdiri digunakan pada panjang lintasan 800m, 1500m, 5000m, dan 10.000m.

________________

START MELAYANG

           Start Melayang (Flying Start) adalah start yang digunakan dalam olahraga cabang atletik lari estafet. Start melayang digunakan oleh pemain ke-2, ke-3, dan ke-4 dalam olahraga estafet

17. lari jarak jauh menggunakan start


Jawaban:

Lari jarak jauh menggunakan start berdiri (standing start).

Penjelasan:

Lari jarak jauh merupakan salah satu cabang olahraga atletik. Lari jarak jauh merupakan olahraga lari yang menempuh jarak 5.000 m dan 10.000 m. Lari jarak jauh menggunakan start berdiri (standing start). Teknik dasar yang digunakan dalam lari jarak jauh adalah dengan memposisikan badan untuk condong ke depan dengan membentuk sudut ±10°. Kedua lengan diayunkan secara santai dengan jarak beberapa sentimeter di atas pinggang. Telapak kaki di sisi luar kaki bagian tengah digunakan untuk pendaratan.

Selain penggunaan start yang berbeda teknik dasar yang digunakan dalam lari jarak jauh juga berbeda dengan lari jarak pendek. Lari jarak jauh menggunakan start berdiri sedangkan lari jarak pendek menggunakan start jongkok. Teknik berlari dalam lari jarak jauh juga sangat berbeda dengan lari jarak pendek. Pelari harus mengeluarkan tenaganya semaksimal mungkin dari mulai start hingga finish dalam lari jarak pendek. Sedangkan, pelari diharuskan untuk mengatur tenaga dan kecepatannya dari start hingga finish agar tidak kelelahan dalam lari jarak jauh.

Pada saat berlari dalam lari jarak jauh, kecepatan harus diatur secara konstan (tetap) dan mengatur tenaga dan pernafasan agar tidak mudah lelah. Apabila mendekati finish kurang lebih 1 km hingga 500 meter, maka pelari harus meningkatkan kecepatannya untuk dapat mencapai garis finish sesegera mungkin.

Jadi, dari keterangan di atas dapat disimpulkan bahwa lari jarak jauh menggunakan start berdiri (standing start).
