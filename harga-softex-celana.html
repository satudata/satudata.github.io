PENDAHULUAN

Sistem Persamaan Linear Dua Variabel adalah persamaan yg terdiri atas dua persamaan linier yang mempunyai dua variabel (x dan y) atau bisa juga dengan variabel lain, misal (a dan b) . Menentukan himpunan penyelesaian dari SPLDV ada beberapa metode, diantaranya :

) Metode Grafik) Metode Substitusi) Metode Eliminasi) Metode Gabungan (Substitusi dan Eliminasi)

PENYELESAIANDiketahuiHarga sebuah celana dua,kali harga sebuah bajuDua buah celana dan sebuah baju harga nya Rp. 625.000.00DitanyaHarga Sebuah baju________________Misal

c = Celana

b = Baju

Buat Persamaan

c = 2b ... Persamaan Pertama

2c + b = Rp625.000 ... Persamaan Kedua

Substitusikan

2c + b = 625.000

2(2b) + b = 625.000

4b + b = 625.000

5b = 625.000

b = 625.000 : 5

b = Rp125.000

Substitusikan nilai b ke persamaan pertama

c = 2b

c = 2(125.000)

c = Rp250.000

PEMBUKTIAN

= 2c + b = 625.000

= 2(250.000) + 125.000 = 625.000

= 500.000 + 125.000 = 625.000

= 625.000 = 625.000 [ BENAR ]

Kesimpulan

Harga sebuah celana adalah Rp250.000
