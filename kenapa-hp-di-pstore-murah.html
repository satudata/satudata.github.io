Penjelasan dengan langkah-langkah:

= 10² + 20² x 30² - 40²

= (10x10) + (20x20) x (30x30) - (40x40)

= 100 + 400 x 900 - 1.600

= 100 + 360.000 - 1.600

= 360.100 - 1.600

= 358.500
