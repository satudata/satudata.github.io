5. 5. Harga 3 kaleng cat dan 2 kuas adalah Rp. 32.000,00, sedangkan harga 2 kaleng cat dan 2 kuas Rp. 21.000,00. Harga 1 kaleng cat dan 1 kuas adalah ....


Jawab:

Rp11.500

Penjelasan dengan langkah-langkah:

diketahui :

perumpamaan =

kaleng cat = x

kuas = y

3x + 2y = Rp32.000

2x + 2y = Rp21.000

ditanya ; harga 1 kaleng cat dan 1 kuas? (x + y)

jawab :

3x + 2y = Rp32.000 => pers. 1

2x + 2y = Rp21.000 => pers. 2

eliminasi pers. 1 dan pers. 2

3x + 2y = Rp32.000

2x + 2y = Rp21.000

________________(-)

x            = Rp11.000

substitusi x = Rp11.000 ke pers. 1

3x + 2y = Rp32.000

3(Rp11.000) + 2y = Rp32.000

Rp33.000 + 2y = Rp32.000

2y = Rp33.000 - Rp32.000

2y = Rp1.000

y = Rp1.000 ÷ 2

y = Rp500

harga 2 kaleng cat dan 1 kuas

= x + y

= Rp11.000 + Rp500

= Rp11.500

jadi, harga 1 kaleng cat dan 1 kuas adalah Rp11.500

18. Seorang pekerjaan bangunan membeli 2 kaleng cat dan 3 kuas seharga 101500.esoknya 1 kaleng cat dan 2 kuas 53500 .harga 1 kaleng cat dan 1 kuas


Seorang pekerja bangunan membeli 2 kaleng cat dan 3 kuas dengan harga Rp. 101.500. Kemudian besoknya dia membeli 1 kaleng cat dan 2 kuas dengan harga Rp. 53.500. Maka harga 1 kaleng cat dan 1 kuas adalah Rp. 48.000.

Penyelesaian Soal :

Misalkan Harga kaleng cat = x

               Harga kuas = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Seorang pekerja bangunan membeli 2 kaleng cat dan 3 kuas dengan harga Rp. 101.500" yaitu :

2x + 3y = 101.500     ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Kemudian besoknya dia membeli 1 kaleng cat dan 2 kuas dengan harga Rp. 53.500" yaitu :

x + 2y = 53.500      ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai x dengan menggunakan cara sebagai berikut :

2x + 3y = 101.500    ║×2║    4x + 6y = 203.000

x + 2y = 53.500       ║×3║    3x + 6y = 160.500

______________________________________ -

                                                        x = 42.500

LANGKAH KEEMPAT (IV)

Subtitusikan nilai x pada persamaan 2 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

x + 2y = 53.500

42.500 + 2y = 53.500

2y = 53.500 - 42.500

2y = 11.000

y = [tex]\frac{11.000}{2}[/tex]

y = 5.500

LANGKAH KELIMA (V)

Hitung harga 1 kaleng cat dan 1 kuas dengan menggunakan cara sebagai berikut :

x + y = Rp. 42.500 + Rp. 5.500

        = Rp. 48.000

∴ Kesimpulan harga 1 kaleng cat dan 1 kuas adalah Rp. 48.000.

26. Sandi membeli 2 kuas cat berukuran kecil dan 3 kuas cat berukuran besar dengan harga 53.000,00. Anton membeli 4 kuas cat berukuran kecil dan 2 kuas cat berukuran besar dengan harga Rp 58.000. Gibran membeli 2 kuas cat berukuran kecil dan 2 kuas cat berukuran besar pada toko yang sama, Gibran membayar dengan uang Rp 100.000,00 Uang kembalian yang diterima Gibran adalah​


Jawaban:

63.000

Penjelasan dengan langkah-langkah:

Diketahui:

Misal

kuas cat kecil = x

kuas cat besar = y

Maka

2x + 3y = 53.000

4x + 2y = 58.000

2x + 2y = ?

Ditanya:

Jika 2x + 2y dibayar 100.000 berapa kembaliannya

Jawab:

"menentukan Y"

2x + 3y = 53.000 | x2

4x + 3y = 58.000 | x1

=

4x + 6y = 106.000

4x + 3y = 58.000

Eliminasi X dengan dikurangkan

3y = 48.000

y = 48.000:3

y = 16.000

"menentukan X dengan substitusi Y"

2x + 3y = 53.000

2x + 3(16.000) = 53.000

2x + 48.000 = 53.000

2x = 53.000 - 48.000

2x = 5.000

x = 2.500

"menentukan 2x + 2y"

2x + 2y

=2(2.500) + 2(16.000)

=5000 + 32.000

=37.000

"menentukan kembaliannya"

100.000 - 37.000

=63.000

Berarti uang kembalian yang diterima Gibran adalah Rp. 63.000,00
