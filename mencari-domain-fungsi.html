15. carilah domain fungsi dari akar x kuadrat - 3x + 2


[tex]f(x)=\sqrt{x^2-3x+2}[/tex]
Domainnya adalah ketika hasil dari pemetaan oleh x tidak negatif karena sifat akar pangkat 2 akan menghasilkan bukan bilangan real jika negatif. Maka:
[tex]\sqrt{x^2-3x+2}\geq 0 \\ x^2-3x+2\geq 0 \\ (x-1)(x-2)\geq 0 \\ x\leq 1$ atau $x\geq 2[/tex]

Maka, domain dari fungsi tersebut adalah:
[tex]D=\{x|x\leq-1$ atau $x\geq 2,x\in R\}[/tex]

27. Carilah domain natural fungsi f(x) = 1/3x-1


Diketahui:

f(x) = 1/(3x - 1)

Ditanyakan:

Df

Jawab:

Karena f(x) = 1/(3x -1) adalah fungsi rasional, maka agar f(x) ada:

3x - 1 ≠ 0

3x ≠ 1

x ≠ (1/3)

Df = {x| x ≠ (1/3)}

29. Kalkulus Fungsi: Cari domain dan range dari f (t) = 4 / 3-t


Jawab:

Domain : t ≠ 3

Range : f(t) ≠ 0

Penjelasan dengan langkah-langkah:

f (t) = 4/ 3 - t

Domain : 3 - t ≠ 0 ⇒ t ≠ 3

f(t)  = 4/3 - t

f(t) [ 3 - t ] = 4

3 f(t) - f(t) t = 4

f(t) t = 3f(t) - 4

t = 3 f(t) - 4/ f(t)

Range : f(t) ≠ 0
