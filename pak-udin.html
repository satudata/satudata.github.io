24. Pak Udin membeli kambing dengan harga rp3.000.200 karena ingin membeli yang lebih besar Pak Udin menjualnya dengan harga rp3.900.000 Tentukan berapakah persentase keuntungan Pak Udin​


• Pertanyaan •

Pak Udin membeli kambing dengan harga rp3.000.200 karena ingin membeli yang lebih besar Pak Udin menjualnya dengan harga rp3.900.000 Tentukan berapakah persentase keuntungan Pak Udin?

• Jawaban •29.9%• Penjelasan •

Keterangan :

L = Laba (Untung)

HB = Harga Beli

HJ = Harga Jual

=> Diketahui :

HB = Rp. 3.000.200

HJ = Rp. 3.900.000

=> Ditanya :

L = ?

=> Dijawab :

L = HJ - HB

= 3.900.000 - 3.000.200

= Rp. 899.800

%L = L/HB × 100%

= 899.800/3.000.200 × 100%

= 29.9%

• Kesimpulan •

Jadi, persentase keuntungan Pak Udin adalah 29.9%
