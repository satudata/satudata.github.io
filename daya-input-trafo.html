14. efisiensi trafo 80%, jika daya output trafo 16 watt maka besar input trafo adalah​


Jawaban:

Besar input trafo adalah 20 watt.

Penjelasan:

Kita dapat menggunakan rumus dasar efisiensi trafo:

Efisiensi = (Daya output / Daya input) x 100%

Kita diminta untuk mencari besar input trafo, yang dapat kita sebut sebagai "P_in". Kita sudah diketahui efisiensi trafo sebesar 80%, yang dapat kita tulis sebagai 0.8, dan daya output trafo sebesar 16 watt.

Maka:

0.8 = (16 / P_in) x 100%

0.8 x P_in = 16

P_in = 16 / 0.8

P_in = 20 watt

Jadi, besar input trafo adalah 20 watt.

PEMBAAHASAN LEBIH LANJUT :

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -





Berikut penjelasan dari jawaban diatas :





Diketahui :

Pp = 500 watt

η = 80 %



Ditanyakan : Ps...........?





Dijawab :

Trafo merupakan alat yang digunakan untuk mengubah tegangan listrik bolak balik menjadi lebih besar atau lebih kecil.

Trafo sendiri dibagi menjadi 2 yaitu trafo step up (menaikan tegangan) dan trafo step down (menurunkan tegangan). Adapun ciri ciri trafo step up yaitu nilai Vs > Vp (tegangan sekunder lebih besar dibandingkan tegangan primer) , Ns > Np (jumlah lilitan sekunder lebih besar dibandingkan jumlah lilitan primer), Is < Ip (kuat arus sekunder lebih kecil dibandingkan dengan kuat arus primer). Untuk ciri ciri trafo step down yaitu nilai Vs < Vp (tegangan sekunder lebih kecil dibandingkan tegangan primer) , Ns < Np (jumlah lilitan sekunder lebih kecil dibandingkan jumlah lilitan primer), Is > Ip (kuat arus sekunder lebih besar dibandingkan dengan kuat arus primer).





Untuk mencari rumus efisiensi transformator adalah sebagai berikut :

η = (Is x Vs / Ip x Vp) x 100% atau η = (Ps/Pp) x 100%

Ket:

η = efisiensi trafo

Pp = daya masukan primer

Ps = daya keluaran sekunder

Vp = Tegangan pada kumparan primer (masukkan)

Vs = Tegangan pada kumparan sekunder (keluaran)

Ip = Arus pada kumparan primer

Is = Arus pada kumparan sekunder



Sekarang tinggal mencari daya outputnya sebagai berikut :

η = (Ps/Pp) x 100%

80% = (Ps/500 watt) x 100%

80 x 500 = Ps x 100%

Ps = 80 x 500 / 100

Ps = 40.000/100

Ps =400 watt

Jadi besarnya nilai daya outputnya adalah 400 watt

26. Sebuah trafo mempunyai daya input 25 watt sedangkan daya outputnya 20 watt. Berapakah efisiensi trafo tersebut?​


Jawaban:

[tex]\bf Efisiensi\ trafo\ 80\% [/tex]

Penjelasan:

[tex]\sf\large Diketahui: [/tex]

[tex]\sf\large P_{Primer}=25\ Watt [/tex]

[tex]\sf\large P_{Sekunder}=20\ Watt [/tex]

Ditanyakan besarnya efisiensi transformator.

Jawab:

Gunakan rumus:

[tex]\sf\large η=\Large{\frac{P_{Sekunder}}{P_{Primer}}}\large×100\% [/tex]

[tex]\sf\large η=\Large{\frac{20}{25}}\large×100\% [/tex]

[tex]\sf\large η=0,8×100\% [/tex]

[tex]\sf\large η=80\% [/tex]

Jadi efisiensi transformator tersebut adalah [tex]\sf\large80\% [/tex]
