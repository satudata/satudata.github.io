2. Sebutkan cara mengubah urutan nomor halaman di excel


Jawaban:

How to Provide Page Numbers in Microsoft Excel

As with other documents, documents created using Microsoft Excel should also be given a page number so that no errors occur in the order in which they are read. Numbering in Microsoft Excel is relatively more complicated than numbering in Microsoft Word but that does not mean that in Microsoft Excel numbering cannot be done. To create page numbers in Microsoft Excel you can use the following method.

Click the Page Layout ribbon menu, then click the page setup button, or

Press the keyboard keys ALT + P, S, P

Figure: How to display the Page Setup dialog in Microsoft Excel

Then in the Page Setup dialog select the Header / Footer tab

Decide where to place the page number whether in the header (the top of the document), or in the Footer (the bottom of the document). For example, select and click on the footer then click the Custom Footer button .

Next in the Footer dialog , select where to place the page number, whether on the left ( left section ), in the middle ( Center Section ), or on the right ( Right Section ). Suppose we want to put it in the middle of the page, then click an empty area in the Center Section .

Then click the Insert Page Number button.

In the center section area there will be a text & [Page] . (you can also type it directly)

Click the OK button to close the Footer dialog

Click the OK button again to close the page setup dialog .

The page number will now be used when printing a document.

besides giving the number, you can also add the excel page number directly to the footer / excel header .

Jawaban:

Cara membuat nomor urut otomatis di Ms Excel

Ketikkan angka 1 pada sel A1 dan angka 2 pada sel A2.

Blok sel A1 dan A2.

Arahkan mouse pada pojok kanan bawah pada sel A2 sehingga pointer berubah menjadi tanda plus (+).

Klik kiri dan tahan menggunakan mouse kemudian drag atau seret kebawah sampai pad baris yang diinginkan.

Maka nomor urut akan tampil.

14. Cara mengurutkan abjad pada microsoft excel


1. Silahkan sobat buka daftar nama yang akan sobat urutkan namanya sesuai dengan abjad. Jika belum punya datanya, silahkan di buat dulu dengan menggunakan Microsoft excel.
2. Kemudian block seluruh data yang akan kita urutkan namanya sesuai abjad.
3 . Setelah terblock semua, sekarang klik menu home > editing > short & filter > costume short
4 . Kemudian pada kolom column short by pilih nama (kolom yang akan dijadikan patokan untuk menyusun sesuai abjad) dan pada kolom order pilih A-Z jika ingin mengurutkan berdasarkan huruf A ke Z. dan Z-A jika ingin mengurutkan berdasarkan huruf Z-A. kemudian klik ok dan lihat hasilnya.

19. cara mengurutkan nomor file handle pada microsoft excel 2007 secara cepat dan otomatis


Dengan AutoFill
1. Masukkan angka yang diinginkan pada lembar Excel, misalnya angka 1 di sel B2.
2. Lalu klik titik tebal yang ada disebelah kanan bawah sel yang terseleksi (Fill Handle) dan drag ke bawah sesuai keinginan.
3. Selanjutnya klik kotak kecil yang muncul disamping kanannya (Auto Fill Options).
4. Selanjutnya pada kotak Auto Fill Options pilih perintah Fill Series.

21. cara membuat tabel dalam microsoft excel?


1.Buka dokumen Microsoft Excel Anda, anggap saja datanya seperti ini. Blok atau tandai semua baris dan kolom yang terdapat data di dalamnya.
2.Kemudian klik tanda ikon menu Border kemudian pilih All Borders atau Outside Borders. All Borders untuk memberi garis border di semua tabel sementara Outside Borders hanya garis luar data yang Anda tandai.
3.Hasil akhirnya seperti ini. Anda bisa juga memilih jenis garis menggunakan langkah di atas, pilih opsi Line Style atau merubah warna garis dengan opsi Line Color.
