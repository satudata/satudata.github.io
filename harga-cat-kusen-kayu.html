7. tentukan ukuran dari kusen pintu segiempat siku siku yang dibuat dari kayu kusen yang panjangnya 8 meter sehingga luas pintu maksimal


panjang seluruh kayu = keliling kusen
                                    8  = 2 (p + l)   .... kedua ruas bagi dengan 2
                                    4  = p + l
                                     l  = 4 - p
f (L) = p x l
        = p (4 - p)
        = 4p - p²
luas mencapai maksimum jika f' (L) = 0
f (L) = 4p - p²
f'(L) = 4 - 2p
    0 = 4 - 2p
  2p = 4
    p = 2 m
l = 4 - p
  = 4 - 2
  = 2 m
jadi, ukuran kusen agar mencapai luas maksimum adalah : 
panjang = 2 m
     lebar = 2 m

Cat kayu = x

Cat dinding / tembok = y

2x + 5y = 185.000 ... (1)

x + 10y = 280.000 ...(2)

2x + 20y = 560.000

-15y = -375.000

y = 25.000

2x + 5y = 185.000

2x + 5(25.000) = 185.000

2x + 125.000 = 185.000

2x = 185.000 - 125.000

2x = 60.000

x = 30.000

Maka, harga 3kg cat kayu dan 4kg cat tembok adalah :

3x + 4y

= 3(30.000) + 4(25.000)

= 90.000 + 100.000

= 190.000
