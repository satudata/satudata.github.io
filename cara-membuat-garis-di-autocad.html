1. langkah langkah membuat ukuran garis pada AutoCAD


1. Ketik L enter pada keyboard komputer atau laptop Anda.

2. Kemudian klik pada sembarang display autocad untuk menentukan point pertama.

3. Tekan F8 untuk pembuatan garis lurus searah vertical (Y axis) atau horizontal (X axis)

4. Arahkan mouse ke arah sesuai keinginan, misalnya pada contoh ini saya menggerakan mouse ke arah kanan sehingga sejajar dengan sumbu X.

5. Lalu ketik dimensi atau ukuran panjangnya garis sesuai harapan, seperti terlihat di contoh gambar saya memasukkan dimensi 50.

6. Tekan enter untuk mengakhiri perintah Line di autocad dan lihat hasilnya.

20. Daftar layer beserta warna, jenis garis, dan ketebalan garis yang di gunakan dalam AutoCad untuk menggambar detail pondasi?


Perintah layer adalah perintah untuk mengatur penggunaan layer. Layer adalah suatu fasilitas yang memungkinkan penggunaan autocad mengorganisasi objek-objek gambar autocad.Tujuh pengelompokkan objek gambar ke dalam layer adalah untuk memudahkan proses menggambar apabila gambar yang sedang ditangani berupa gambar yang kompleks. Dengan menggunakan layer, anda dapat memilah-milah gambar yang ada ke dalam beberapa lapisan (layer=lapisan).Setiap layer diidentifikasi dengan nama layer, warna layer, tipe garis, dan lain-lain. Jika satu layer hendak dihapus atau diedit, kita dapat mengatur sedemikian rupa sehingga layer lain yang terdapat pada gambar yang sama tidak ikut terhapus atau teredit (dengan cara mengunci layer tidak ingin diedit).

22. Bagaimana cara meng-import gambar dari autocad ke sketchup?


Copy gambar yang ingin anda import ke file baru agar gambar yang tidak perlu tidak memberatkan sketchup. seperti pada gambar saya hanya mengambil gambbar denah.Hapus layer selain layer tembok dan kolom.Explode semua gambar hal ini sangat penting agar memudahkan langkah selanjutnya di sketchup. Selain itu jika anda biasanya membuat tembok dengan multi line, jika tidak anda explode maka tidak akan terimport di sketchup.Bukua sketchup anda kemudian klik pada menu File>ImportSetelah muncul window seperti diatas pilih jenis filenya yaituAutoCAD files (*.dwg, *.dfx).Sebelum menekan open klik pada Options terlebih dahulu. Pada bagian Scale pilih unit yang anda gunakan dalam file DWG tersebut. Saat pertama kali unit defaultnya adalah feet, jika tidak anda atur maka ukuran pada sketchup tidak akan sesuai dengan ukuran di AutoCAD.Kemudian tekan Open maka gambar akan terimoprt ke sketchup.Gambar yang anda import akan berada dalam satu grup. Langkah berikutnya adalah membuat face atau bidang pada denah tersebut. Pertama download dulu plugin make_faces.rb kemudian instal. cara menginstalnya bisa anda lihat disini. setelah itu masuk ke group dan pilih semua lalu klik kanan dan pilih make face, jika tidak ada pilihan make face anda bisa membuat shortcutnya caranya bisa anda lihat disini.setelah selesai hasilnya akan seperti berikut.

1.Cara membuat segi banyak di autocad dengan Polygon teknik circumscribed.

Jika panjang sisi 3 mm

History..

1. Ketik “POL” di keyboard

2. Ketik Spasi

3. Masukkan nilai “3” (Jumlah sisi polygon dalam contoh)

4. Ketik “cen” untuk menentukan titik tengah lingkaran

5. Ketik spasi kemudian klik objek circle

6. Klik “Circumsribed” untuk memilih opsi sisi Luar lingkaran

7. Masukkan nilai radius (Contoh 20.1)

8. Ketik spasi untuk mengakhiri perintah

Catatan :

pada langkah ke-4 anda tidak mesti menggambar circle dulu untuk membuat polygon, objek circle di tutorial ini hanya menrupakan contoh yang membuktikan bagaimana perintah polygon dengan teknik “circumscribed” berjalan dengan baik.

2. Cara membuat segi banyak di autocad dengan Polygon teknik inscribed

Polygon dengan teknik ini yaitu dengan cara menentukan penggambaran polygon berada di dalam lingkaran yang anda tentukan jari-jarinya.

History..

1. Ketik “POL” di keyboard

2. Ketik Spasi

3. Masukkan nilai “3” (Jumlah sisi polygon dalam contoh)

4. Ketik “cen” untuk menentukan titik tengah lingkaran

5. Ketik spasi kemudian klik objek circle

6. Klik “Inscribed” untuk memilih opsi sisi Luar lingkaran

7. Masukkan nilai radius (Contoh 20.1)

8. Ketik spasi untuk mengakhiri perintah

Catatan :

pada langkah ke-4 anda tida mesti menggambar circle dulu untuk membuat polygon, objek circle di tutorial ini hanya menrupakan contoh yang membuktikan bagaimana perintah polygon dengan teknik “Inscribed” berjalan dengan baik.

3. Cara membuat segi banyak di autocad dengan Polygon teknik edge

Teknik ini dilakukan dengan cara menggambar polygon dengan menentukan panjang sisi polygon, maka keseluruhan sisi akan memiliki panjang yang sama berdasarkan nilai yang anda kehendaki.

History..

1. Ketik “POL” di keyboard

2. Ketik spasi

3. masukkan nilai 3 (jumlah sisi pada contoh tutorial ini

4. Ketik spasi

5. Ketik “e” untuk opsi Edge

6. Masukkan nilai 3mm (Panjang sisi pada contoh tutorial ini)

Catatan :

Jika ingin polygon tersebut memiliki putaran sudut pada bagian bawahnya anda tinggal memasukkan nilai sudut setelah angka panjang sisi polygon yang anda kehendaki contoh panjang sisi 50 sudut 45 derajat maka ketikkan “50<45” pada langkah ke 6, jika anda ingin lebih jauh bagaimana system putaran sudut ini bekerja pada softwere tinjaulah konten System koordinat dan putaran sudut.

29. apa saja icon yang ada di aplikasi AutoCAD?


Ada berbagai icon yang terdapat dalam aplikasi AutoCAD. Berikut ini adalah beberapa icon yang sering digunakan dan fungsinya.

Pull-down menus: menu-menu AutoCAD yang disusun secara menurun.Properties toolbar: mengakses toolbar untuk properti ruang kerja.Standard toolbar: mengakses toolbar dasar.Layer toolbar: mengakses toolbar untuk later.Edit toolbar: mengakses fitur edit gambar.Draw toolbar: mengakses fitur menggambar.Save: menyimpan file yang sudah pernah disimpan sebelumnya.Save as: menyimpan file baru.Open: membuka dokumen AutoCAD.Close: menutup lembar kerja AutoCAD.Export: mengubah jenis ekstensi file hasil kerja.Page setup: mengatur ukuran kertas, orientasi, dan hal-hal lain yang berhubungan dengan kertas lembar kerja.Zoom: memperbesar maupun memperkecil ukuran tampilan lembar kerja.Text style: mengubah jenis huruf yang digunakan.Line: membuat garis lurus sambung.Circle: membuat lingkaran.Erase: menghapus objek yang tidak diinginkan dari lembar kerja..
