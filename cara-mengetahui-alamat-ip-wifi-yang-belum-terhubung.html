3. cara mengetahu alamat ip private kelas c​


Jawaban:

IP address kelas C biasanya digunakan untuk jaringan komputer berukuran kecil seperti halnya LAN (Local Area Network), dimana perangkat keras jaringan komputer yang digunakan tidak terlalu banyak dan hanya digunakan pada satu area saja. Tiga bit pertama pada IP address kelas C selalu diatur “111” dalam bilangan biner. Network ID-nya terdiri dari 24 bit dan host ID-nya terdiri dari 8 bit. Sehingga, kelas ini dapat membentuk sekitar 2 juta network dengan masing-masing network mampu menampung 256 host.

5. tuliskan hubungan antara domain name system (DNS) dengan alamat ip ( IP address)​


Jawaban:

Pengertian IP address

Alamat IP (Internet Protocol Address atau sering disingkat IP) adalah deretan angka biner antar 32-bit sampai 128-bit yang dipakai sebagai alamat identifikasi untuk tiap komputer host dalam jaringan Internet. Panjang dari angka ini adalah 32-bit (untuk IPv4 atau IP versi 4), dan 128-bit (untuk IPv6 atau IP versi 6) yang menunjukkan alamat dari komputer tersebut pada jaringan Internet berbasis TCP/IP.

Pengertian Domain Name System (DNS)

Domain Name System (DNS) adalah distribute database system yang digunakan untuk pencarian nama komputer (name resolution) di jaringan yang mengunakan TCP/IP (Transmission Control Protocol/Internet Protocol). DNS biasa digunakan pada aplikasi yang terhubung ke Internet seperti web browser atau e-mail, dimana DNS membantu memetakan host name sebuah komputer ke IP address.

Penjelasan:

Hubungnnya adalah DNS dan IP adress adalah sama sama sebagai penamaan sistem, bedana jika DNS terdapat nama contohnya (namadomain.com) sedangkan ip dari namadomain.com (192.178.0.1)

10. Apakah komputer yang terhubung ke jaringan boleh menggunakan alamat (ip address) yang sama?


Komputer mampu terhubung pada jaringan dan alamat yang sama adalah salah karena akan mengakibatkan konflik.

Pembasahan:

Protokol internet atau alamat IP adalah alamat pribadi Anda saat mengakses internet.  IP ini memungkinkan penyedia layanan Internet atau ISP Anda untuk menerima dan mengirim ulang aktivitas Internet yang dilakukan oleh seseorang. Selama Anda tidak berada di  jaringan yang sama, tidak ada masalah. Jika Anda berada di jaringan yang sama, konflik IP akan terjadi dan IP tersebut akan dianggap sebagai alamat yang buruk hingga salah satu atau keduanya mengubah IP. Tidak seperti alamat IP publik, alamat IP pribadi hanya berfungsi dalam jaringan area lokal (LAN). Menjadi lokal,  perangkat  dan tetangga  dapat menggunakan alamat IP pribadi yang sama. Artinya, alamat IP  adalah sekumpulan angka yang terkait dengan komputer, ponsel, atau perangkat "pintar" lainnya yang terhubung ke Internet. Nomor-nomor ini bervariasi dari perangkat ke perangkat dan digunakan untuk saling menghubungi.

16. bagaimana cara memperoleh alamat ip


Instruksi
Pada Windows

1. Bukalah aplikasi Command Prompt. Klik tombol Start>Run, sebuah jendela akan muncul. Ketikkan cmd pada kotak teks yang ada, dan klik OK.

2. Lepaskan alamat IP komputer saat ini. Untuk melakukannya, ketikkan ipconfig/release pada jendela command prompt dan tekan Enter.

3. Dapatkan alamat IP baru. Lakukan hal ini dengan mengetikkan ipconfig/renew pada jendela command prompt dan tekan Enter. Perintah ini seharusnya membuat komputer mendapatkan alamat IP baru, yang berbeda dengan alamat IP sebelumnya.

4. Lakukan restart pada modem Anda. Jika langkah diatas tidak bekerja, Anda bisa memperbarui alamat IP Anda dengan melakukan restart pada modem Anda. Matikan komputer, modem, dan router (jika Anda memilikinya). Tunggulah paling tidak sekitar 5 menit, dan nyalakan semuanya kembali. Jika komputer Anda diatur untuk menerima alamat IP secara dinamis, Anda seharusnya sudah diberikan alamat IP baru secara otomatis.

5. Periksalah untuk memastikana alamat IP Anda telah diganti. Untuk melakukannya, Anda bisa melakukan logon pada sejumlah situs web yang memberitahukan alamat IP Anda.

 
Pada Macintosh

1. Bukalah menu System Preferences. Lakukan ini dengan cara meng-klik ikon System Preferences pada dock, atau dengan cara meng-klik logo Apple pada taskbar dan memilih System Preferences.

2. Bukalah menu Network. Klik pada ikon Network yang ada didalam menu System Preferences, dan lalu pilihlah tipe sambungan Anda pada panel kiri.

3. Bukalah menu Advaced. Setelah memilih tipe sambungan Anda, klik pada tombol Advanced. Pada menu yang muncul, klik pada tab TCP/IP.

4. Perbarui alamat IP Anda. Lakukan hal tersebut dengan meng-klik tombol Renew DHCP Lease. Anda seharusnya melihat alamat IP Anda (yang terletak pada bagian kiri tombol) telah berubah. Setelah alamat IP Anda diubah, klik tombol OK dan tutuplah menu System Preferences.

 
Tips

Jika Anda membutuhkan alamat IP baru untuk melakukan bypass pada sebuah firewall, Anda bisa mencoba mengakses web melalui sebuah server proxy daripada menggunakan alamat IP baru.

Jika Anda mengakses Internet melalui router nirkabel, Anda bisa memperbarui alamat IP Anda melalui layar login router.

29. Routing dengan cara mengirimkan data dari satu alamat ip ke alamat ip lainnya tanta melalui sebuah host disebut


Jawaban:

Routing Langsung

Penjelasan:

Menurut cara pengiriman datanya, routing dibagi menjadi dua yaitu routing langsung dan routing tidak langsung.

Routing langsung adalah routing dengan cara mengirimkan data dari satu alamat IP ke alamat IP lainnya tanpa melalui sebuah host.

Misalnya sebuah komputer dengan alamat IP A ingin mengirimkan data ke alamat IP B maka datanya akan langsung dikirim.

Sedangkan untuk routing tidak langsung, data yang dikirimkan harus melewati host lain.

Seperti misalnya data yang dikirimkan oleh alamat IP A ke alamat IP B harus melewati IP C terlebih dahulu sebelum disampaikan ke alamat IP B.
