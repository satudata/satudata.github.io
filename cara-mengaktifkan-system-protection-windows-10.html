12. mohon bantuannya gan, cara mengaktifkan windows 10...


Windows 10 dapat diaktifkan dengan banyak cara. Salah satunya dengan memasukkan product key untuk edisi Windows yang tepat ke sistem dengan perintah berikut dari cmd:

slmgr /ipk (kode produk)
Contoh:
slmgr /ipk ABCDE-FGIJK-LMNOP-12345-67890

Setelah itu koneksikan komputer Anda ke internet dan jalankan perintah berikut:

slmgr /ato

Kalau muncul dialog bertuliskan "Product activated successfully", maka Windows Anda sudah teraktivasi. Dapat dicek dengan slmgr /dlv ataupun dengan "control system" dari dialog Run.

17. Bagaimana cara mengaktifkan bluetooth di windows 7



1. Periksa untuk melihat bahwa laptop memiliki adaptor Bluetooth yang terpasang. Klik pada menu “Start” di pojok kiri bawah desktop. Klik pada “Control Panel.” Klik pada ” Performance and Maintenance ” dan kemudian pada “System.” Klik “Hardware” tab pada jendela System Properties. Klik pada tombol “Device Manager”. Mencari ikon biru kecil yang mengatakan ” Bluetooth Devices.” Klik pada “+” di samping itu untuk memperluas menu. Harus ada minimal satu perangkat Bluetooth yang terdaftar. Hal ini menunjukkan bahwa laptop Anda memiliki bluetooth adapter. Jika tidak ada artinya laptop tidak mempunyai perangkat bluetooth.
2. Aktifkan perangkat Bluetooth dengan menempatkan switch fisik atau perangkat lunak yang mengaktifkan adaptor. Periksa tepi luar laptop untuk tombol atau switch yang berlabel “Bluetooth,” menunjukkan gambar ikon Bluetooth atau diberi label “Wireless.” Mengubah ini ke posisi “On”. Jika tidak ada switch fisik ada, periksa tombol fungsi di atas keyboard. Carilah yang menunjukkan simbol nirkabel. Ini mungkin terlihat seperti antena kecil dengan gelombang memancar keluar dari atas. Tahan tombol “Fn” atau “Fungsi” tombol dan tekan tombol fungsi yang sesuai sampai komputer menunjukkan bahwa Bluetooth diaktifkan.
3. Cari ikon Bluetooth pada system tray di sudut kanan bawah desktop. Ini akan cocok dengan penampilan ikon Bluetooth Anda terletak di Device Manager. Ini mungkin memiliki simbol merah di tengah yang menunjukkan bahwa Bluetooth dimatikan. Klik kanan pada ikon ini dan pilih “Start Device Bluetooth” dari menu.
4. Uji Bluetooth dengan menghubungkan perangkat Bluetooth. Klik kanan pada ikon Bluetooth pada system tray. Gunakan “Quick Connect” pilihan atau ikuti wizard Bluetooth Setup diakses melalui pilihan “Add a Bluetooth Device”.

30. sebutkan beberapa cara mengaktifkan windows 7​


Jawaban:

melaui internet

1.Klik Start. Klik kanan Computer lalu klik Properties. Jendela System Properties akan terbuka.

2.Klik tautan "Activate Windows Now" di bagian bawah jendela. Windows akan mencoba mendeteksi koneksi internet yang aktif. Apabila ada koneksi yang terdeteksi, opsi "Activate Windows online now" akan muncul dan bisa Anda pilih. Jika tetap tidak muncul, pastikan bahwa Anda benar-benar telah terhubung ke Internet.

3.Masukkan kunci produk (product key) Windows 7 saat diminta. Masukkan kode berisi 25 karakter yang valid agar salinan Windows Anda bisa diaktifkan. Kunci produk Windows bisa berada di bagian bawah laptop, di bagian belakang casing komputer, di wadah cakram DVD Windows 7, atau di buku manual Windows 7 Anda.

Apabila Anda mendapatkan Windows dari internet, Anda akan menerima kunci produk melalui surel (email) konfirmasi.

Apabila kunci produk Anda tetap tidak ada, Anda harus membeli kunci produk yang baru

4.Klik .Next untuk mengaktifkan salinan Windows Anda. Proses aktivasi bisa memerlukan waktu beberapa menit. Setelah proses selesai, akan muncul pesan konfirmasi di jendela aktivasi. Untuk memastikan bahwa Windows Anda telah aktif, buka kembali System Properties (dengan mengeklik kanan Computer lalu memilih Properties). Di bagian bawah jendela akan muncul pesan yang bertuliskan "Windows is activated".
