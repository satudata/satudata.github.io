Mungkin scriptnya seperti ini

// Deklarasi variabel

float x, y;    // Koordinat posisi Baby Doge

float speedX;  // Kecepatan perpindahan horizontal

void setup() {

 size(400, 400);  // Ukuran jendela

 x = width / 2;   // Posisi awal Baby Doge di tengah horizontal

 y = height / 2;  // Posisi awal Baby Doge di tengah vertikal

 speedX = 2;      // Kecepatan perpindahan horizontal

}

void draw() {

 background(220);  // Warna latar belakang

 // Menggambar Baby Doge

 fill(255, 0, 0);       // Warna isi Baby Doge (merah)

 stroke(0);             // Warna garis pinggir Baby Doge (hitam)

 ellipse(x, y, 50, 50); // Menggambar lingkaran (tubuh Baby Doge)

 // Menggerakkan Baby Doge

 x += speedX;  // Mengubah posisi Baby Doge berdasarkan kecepatan horizontal

 // Membatasi pergerakan Baby Doge ketika mencapai batas layar

 if (x > width || x < 0) {

   speedX *= -1;  // Membalik arah pergerakan Baby Doge

 }

}

11. Baby ingin belajar di kamar apa dampak jika menggunakan hak tersebut tanpa ada tanggung jawab


Baby ingin belajar di kamar apa dampak jika menggunakan hak tersebut tanpa ada tanggung jawab? Kamar tersebut akan berantakan jika tidak dibereskan

PEMBAHASAN

Tanggung jawab merupakan kewajiban/wajib menanggung dari perbuatan atau akibat yang telah kamu perbuat.

Misalnya :

Santi ingin bermain masak-masakan dirumah Fitri. Tiba² Santi lupa membawa mainnya. Alhasil Fitri meminjamkan mainannya kepada Santi. Setelah Fitri meminjamkan kepada Santi, Santi malah memberantakan semua mainan Fitri. Maka TANGGUNG JAWAB Santi adalah membereskan mainan Fitri tersebut.

KESIMPULAN

Jadi, dampak jika menggunakan hak tersebut tanpa ada tanggung jawab adalah kamar tersebut akan berantakan jika tidak dibereskan
