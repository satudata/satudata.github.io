1. Jelaskan langkah langkah mendownload sebuah file dari halaman web


1)       Kliklah teks dimana file tersebut di link.

Kotak dialog file download akan ditampilkan.

2)    Anda dapat membuka langsung file yang akan di download tersebut dengan mengklik tombol open. Jika anda ingin mendownload file tersebut, klik tombol save.
3)     Tentukan folder penyimpanan file yang akan di download tersebut. Secara otomatis, nama file yang diberikan akan sama dengan nama file asli dari file yang di download tersebut. Namun, jika anda ingin mengganti, anada dapat memberi nama

4)     Klik tombol save untuk memulai proses download file.

Kotak dialog proses pen-download akan ditampilkan.

5)    Tunggu sampai proses download file selesai.

6)    Klik tombol open untuk membuka file yang sudah di download.

7)     Klik tombol open folder untuk membuka folder dimana file tersebut ditempatkan atau klik close untuk menutup kotak dialog.




Pada bagian atas jendela Firefox, klik pada menu File, lalu pilih Save Page As.... Jendela Save As akan muncul. Gunakan jendela Save As untuk memilih lokasi penyimpanan halaman Web sesuai dengan yang Anda kehendaki. Pada panel dropdown Save As Type, pilih format penyimpanan halaman yang diinginkan

23. Apa yang dimaksud dengan download,gprs,web


Download adalah sebuah proses dalam pengambilan file-file tertentu yang terdapat di internet baik melalui web server, FTP server, mail server, server ataupun sistem lain yang identik.

GPRS (General Packet Radio Service)

suatu teknologi yang digunakan untuk pengiriman dan penerimaan paket data. GPRS sering disebut dengan teknologi 2G. GPRS memungkinkan kita untuk mengakses e-mail, internet, mms, dll. Secara teori GPRS memberikan kecepatan akses antara 56kbps sampai 115kbps. GPRS disingkat dengan huruf G.

Website adalah sering juga disebut Web, dapat diartikan suatu kumpulan-kumpulan halaman yang menampilkan berbagai macam informasi teks, data, gambar diam ataupun bergerak, data animasi, suara, video

25. Download dapat dilakukan pada seluruh file dari internet .kelemahan dari download adalah


Kecepatan/Kelambatan Download dipengaruhi oleh suatu kondisi jaringan juga terkadang ada iklan yang muncul dan beberapa situs lain yang menganggu di internet

Jawaban:

1. Tidak Gratis (USD $ 29,95)

2. Menggunakan Memori sekitar 9 MB waktu idle dan 20 MB saat proses download.

3. Browsing jadi lebih sering time out karena seluruh bandwidth digunakan oleh Internet Download Manager.

4. Resume capability indicator terkadang tidak akurat. Jika download dari situs yang tidak mendukung resume (seperti download dari rapidshare atau megaupload dengan free account) disitu akan ditampilkan “Unknown” atau “Yes” padahal tidak bisa diresume.

5.Lokasi (temporary) file download tidak jelas dan jika komputer mati mendadak saat proses download proses akan dimulai dari awal bahkan kadang daftar link hilang dari jendela utama Internet Download Manager.
