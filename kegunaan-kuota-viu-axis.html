9. Jabarkan perbedaan antara vertical axis dengan horizontal axis


Jawaban:

Jawaban

Pendahuluan

Sebelum membahas mengenai perbedaan antara vertikal axis dengan horizontal axis, mari kita bahas terlebih dahulu apa itu vertikal axis dan horizontal axis. Kedua istilah tersebut dapat ditemui pada saat kita membuat chart atau membuat grafik baik menggunakan aplikasi microsoft office seperti pada microsoft word, microsoft excel atau aplikasi lainnya.

Pembahasan

Perbedaan diantara keduanya secara umum dapat dilihat pada letaknya, untuk vertical axis merupakan gari tegak ke atas, sedangkan horizontal axis merupakan garis mendatar pada grafik. Vertical axis biasanya digunakan untuk menampilkan suatu jumlah sedangkan horisontal axis biasanya digunakan untuk menampilkan range waktu.

Kesimpulan

Demikian pembahasan mengenai perbedaan antara vertikal axis dan horizontal axis, keduanya merupakan salah satu komponen pendukung yang ada pada suatu grafik. Keduanya memiliki fungsi agar data yang ditampilkan pada grafik mudah untuk dibaca. 21. Pemberian keterangan axis table dapat menggunakan fitur

Jawaban:

Penjelasan:

Axis table/title/label adalah keterangan axis di bawah suatu chart. Ini dapat diberikan dengan menggunakan fitur Add Chart Element (Chart Element) di tab menu/ribbon Design. Yaitu di grup alat di samping Chart Styles.

Dengan menggunakan fitur di Chart Element, kita dapat memberikan keterangan axis table/title/label.
