17. Tiara membeli 3 gelas jus jambu dan 2 gelas jus jeruk di kantin sekolah dengan harga Rp.46.000.00 di tempat yang sama.putri membeli 2gelas jus jambu dan 5 gelas jus jeruk dengan harga Rp.60.000.00.berpa harga satu gelas jus jambu dan satu gelas jus jeruk??? Kerjakan dalam bentuk persamaan matrik


Persamaan linier dua peubah/variabel dapat dituliskan dalam bentuk berikut: ax + by = c. Untuk mencari nilai x dan y dalam persamaan linier dapat digunakan sistem matriks dengan dua metode yaitu:

metode invers matriks metode determinan matriks.Pembahasan

Pada soal dimisalkan bahwa x adalah harga segelas jus jambu dan y adalah harga segelas jus jeruk. Maka soal di tersebut dapat dituliskan menjadi persamaan linier sebagai berikut:

3x + 2y = 46.000

2x + 5y = 60.000

Persamaan di atas jika dituliskan dalam sistem matriks menjadi:

[tex]\left[\begin{array}{ccc}3&2\\\\2&5\end{array}\right] \left[\begin{array}{ccc}x\\\\y\end{array}\right] =\left[\begin{array}{ccc}46.000\\\\60.000\end{array}\right][/tex]

maka untuk mencari harga x dan y dapat menggunakan dengan dua metode sebagai berikut:

Metode Invers Matriks

Untuk persamaan linier ax + by = c dan px + qy = r maka bentuk invers matriksnya adalah

[tex]\left[\begin{array}{ccc}a&b\\\\p&q\end{array}\right] \left[\begin{array}{ccc}x\\\\y\end{array}\right] = \left[\begin{array}{ccc}c\\\\r\end{array}\right][/tex]

[tex]\left[\begin{array}{ccc}x\\\\y\end{array}\right] =\left[\begin{array}{ccc}a&b\\\\p&q\end{array}\right] ^{-1} \left[\begin{array}{ccc}c\\\\r\end{array}\right][/tex]

[tex]\left[\begin{array}{ccc}x\\\\y\end{array}\right] =\frac{1}{aq-bp} \left[\begin{array}{ccc}q&-b\\\\-p&a\end{array}\right] \left[\begin{array}{ccc}c\\\\r\end{array}\right][/tex]

Maka untuk soal tersebut dapat dituliskan menjadi:

[tex]\left[\begin{array}{ccc}x\\\\y\end{array}\right] = \frac{1}{(3x5)-(2x2)} \left[\begin{array}{ccc}5&-2\\\\-2&3\end{array}\right] \left[\begin{array}{ccc}46.000\\\\60.000\end{array}\right][/tex]

[tex]\left[\begin{array}{ccc}x\\\\y\end{array}\right] = \frac{1}{11} \left[\begin{array}{ccc}(5x46.000)&(-2x60.000)\\\\(-2x46.000)&(3x60.000)\end{array}\right][/tex]

[tex]\left[\begin{array}{ccc}x\\\\y\end{array}\right] = \frac{1}{11} \left[\begin{array}{ccc}230.000&(-120.000)\\\\(-92.000)&180.000\end{array}\right][/tex]

[tex]\left[\begin{array}{ccc}x\\\\y\end{array}\right] =\frac{1}{11} \left[\begin{array}{ccc}110.000\\\\88.000\end{array}\right][/tex]

maka x = [tex]\frac{110.000}{11} = 10.000[/tex]

dan y = [tex]\frac{88.000}{11} = 8.000[/tex]

maka x = Rp 10.000,- dan y = Rp 8.000,-

Metode Determinan Matriks

untuk persamaan ax + by = c dan px + qy = r

bentuk determinan x dan y nya adalah:

[tex]D = \left[\begin{array}{ccc}a&b\\\\p&q\end{array}\right] = aq-bq[/tex]

[tex]Dx = \left[\begin{array}{ccc}c&b\\\\r&q\end{array}\right] = cq-rb[/tex]

[tex]Dy = \left[\begin{array}{ccc}a&c\\\\p&r\end{array}\right] = ar-pc[/tex]

maka dari soal tersebut determinannya adalah

[tex]D = \left[\begin{array}{ccc}3&2\\\\2&5\end{array}\right] = (3x5) - (2x2) = 15 - 4 = 11[/tex]

[tex]Dx = \left[\begin{array}{ccc}46.000&2\\\\60.000&5\end{array}\right] = (46.000x5) - (60.000x2) = 230.000 - 120.000 = 110.000[/tex]

[tex]Dy = \left[\begin{array}{ccc}3&46.000\\\\2&60.000\end{array}\right] = (3x60.000) - (2x46.000) = 180.000 - 92.000 = 88.000[/tex]

maka x = [tex]\frac{Dx}{D} = \frac{110.000}{11} = 10.000[/tex]

dan y =[tex]\frac{Dy}{D} =\frac{88.000}{11} = 8.000[/tex]

jadi x = Rp 10.000,- dan y = 8.000,-
